# Add-AzMlWebServiceRegionalProperty

## Sammanfattning
Skapar egenskaper för webb tjänsten.

## FRÅGESYNTAXEN

```
Add-AzMlWebServiceRegionalProperty -ResourceGroupName <String> -Name <String> -Region <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar nationella egenskaper för Azure Machine Learning för en befintlig webb tjänst.

## BESKRIVS

### Exempel 1: lägga till nya nationella egenskaper för västra centrala USA

```
Add-AzMlWebServiceRegionalProperty -ResourceGroupName "myresourcegroup" -Name "mywebservicename" -Region westcentralus
```

I det här exemplet kommando skapas en regional egenskap för en webb tjänst i regionen "Västra Central USA".
