# Remove-AzOperationalInsightsSavedSearch

## Sammanfattning
Tar bort en sparad sökning från arbets ytan.

## FRÅGESYNTAXEN

```
Remove-AzOperationalInsightsSavedSearch [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-SavedSearchId] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzOperationalInsightsSavedSearch** tar bort en sparad sökning från arbets ytan.

## BESKRIVS

### Exempel 1: ta bort en sparad sökning
```
PS C:\>Remove-AzOperationalInsightsSavedSearch -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace" -SavedSearchId "ContosoSavedSearchId" -Force
```

Det här kommandot tar bort en sparad sökning från arbets ytan.
