# Get-AzBlockchainConsortium

## Sammanfattning
Visar en lista över tillgängliga konsortier för ett abonnemang.

## FRÅGESYNTAXEN

```
Get-AzBlockchainConsortium -Location <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Visar en lista över tillgängliga konsortier för ett abonnemang.

## BESKRIVS

### Exempel 1: skaffa blockchain Consortium.
```powershell
PS C:\> Get-AzBlockchainConsortium -Location eastus

```

Det här kommandot visar konsortiet under ett abonnemang för en viss plats.
