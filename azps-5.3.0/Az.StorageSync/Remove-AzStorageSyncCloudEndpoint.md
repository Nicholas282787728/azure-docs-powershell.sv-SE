# Remove-AzStorageSyncCloudEndpoint

## Sammanfattning
Det här kommandot tar bort den angivna moln slut punkten från en synkroniseringsresurs. Utan åtminstone en moln slut punkt kan ingen annan server slut punkter i den här synkroniseringsresursen synkronisera.

## FRÅGESYNTAXEN

### StringParameterSet (standard)
```
Remove-AzStorageSyncCloudEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-InputObject] <PSCloudEndpoint> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Det här kommandot tar bort den angivna moln slut punkten från en synkroniseringsresurs. Azure-filen dela moln slut punkts referenserna kvarstår inte under den här processen. Det här kommandot är endast avsett för avställning. Borttagning av moln slut punkter är en destruktiv operation. Server slut punkter kan inte synkroniseras utan minst en moln slut punkt. Denna åtgärd ska inte utföras för att lösa synkroniseringsproblem. Om den här Azure-fildelningen läggs till igen i samma synkroniseringsresurs som en moln slut punkt kan den leda till konfliktskapande filer och andra oavsiktliga konsekvenser.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzStorageSyncCloudEndpoint -Force -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName" -Name "myCloudEndpointName"
```

Det här kommandot tar bort moln slut punkten.
