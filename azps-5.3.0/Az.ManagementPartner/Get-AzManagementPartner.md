# Get-AzManagementPartner

## Sammanfattning
Hämtar MPN-ID för den aktuella autentiserade användaren eller tjänstens huvud konto.

## FRÅGESYNTAXEN

```
Get-AzManagementPartner [[-PartnerId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämtar MPN-ID för den aktuella autentiserade användaren eller tjänstens huvud konto.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzManagementPartner
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Skaffa aktuellt Management partner-ID

### Exempel 2
```powershell
PS C:\> Get-AzManagementPartner -PartnerId 4977985
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Skaffa det aktuella Management partner-ID: t med ett partner-ID, om de inte stämmer överens
