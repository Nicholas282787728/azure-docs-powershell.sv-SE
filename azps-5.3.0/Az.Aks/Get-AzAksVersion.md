# Get-AzAksVersion

## Sammanfattning
Lista tillgänglig version för att skapa hanterat Kubernetes-kluster.

## FRÅGESYNTAXEN

```
Get-AzAksVersion -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Lista tillgänglig version för att skapa hanterat Kubernetes-kluster.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzAksVersion -Location westus
```
