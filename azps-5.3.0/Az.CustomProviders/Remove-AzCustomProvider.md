# Remove-AzCustomProvider

## Sammanfattning
Tar bort den anpassade resurs leverantören.

## FRÅGESYNTAXEN

### Ta bort (standard)
```
Remove-AzCustomProvider -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzCustomProvider -InputObject <ICustomProvidersIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort den anpassade resurs leverantören.

## BESKRIVS

### Exempel 1: ta bort en anpassad leverantör.
```powershell
PS C:\> PS C:\> Remove-AzCustomProvider -ResourceGroupName myRg -Name Namespace.Type
```

Ta bort en anpassad leverantör

### Exempel 2: ta bort en anpassad leverantör med PassThru
```powershell
PS C:\> PS C:\> Remove-AzCustomProvider -ResourceGroupName myRg -Name Namespace.Type -PassThru

True
```

Ta bort en anpassad Provider med funktionen PassThru för att indikera framgång eller misslyckande.
