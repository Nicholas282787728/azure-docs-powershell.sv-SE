# Modulen AZ. WindowsIotServices
## Problembeskrivning
Microsoft Azure PowerShell: WindowsIotServices-cmdletar

## AZ. WindowsIotServices-cmdletar
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
Hämta icke-säkerhetsrelaterade metadata för en Windows IoT-enhet.

### [New-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Skapa eller uppdatera metadata för en Windows IoT-enhet.
Det vanliga mönstret för att ändra en egenskap är att hämta metadata och säkerhets-metadata för Windows IoT Device service och sedan kombinera dem med de ändrade värdena i en ny brödtext och uppdatera enhets tjänsten för Windows IoT.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Ta bort en Windows IoT-enhet.

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Uppdaterar metadata för en Windows IoT-enhet.
Det vanliga mönstret för att ändra en egenskap är att hämta metadata och säkerhets-metadata för Windows IoT Device service och sedan kombinera dem med de ändrade värdena i en ny brödtext och uppdatera enhets tjänsten för Windows IoT.
