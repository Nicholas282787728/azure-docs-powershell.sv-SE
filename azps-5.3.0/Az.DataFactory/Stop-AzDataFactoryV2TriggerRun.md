# Stop-AzDataFactoryV2TriggerRun

## Sammanfattning
Stoppar en Utlös ande körning i en data fabrik.

## FRÅGESYNTAXEN

### ByFactoryName (standard)
```
Stop-AzDataFactoryV2TriggerRun [-TriggerName] <String> [-TriggerRunId] <String> [-PassThru]
 [-ResourceGroupName] <String> [-DataFactoryName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Stop-AzDataFactoryV2TriggerRun [-InputObject] <PSTriggerRun> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Stop-AzDataFactoryV2TriggerRun [-TriggerName] <String> [-TriggerRunId] <String> [-PassThru]
 [-DataFactory] <PSDataFactory> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Stop-AzDataFactoryV2TriggerRun** stoppar en Utlös ande körning i en data fabrik som anges med Utlösarens namn och utlösarens kör-ID.
För närvarande endast för TumblingWindowTriggers i WaitingOnDependency-tillstånd.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Stop-AzDataFactoryV2TriggerRun -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -TriggerName "testTumblingWindowTrigger" -TriggerRunId 08586002468005888497807248799CU16
```

Det här kommandot stoppar trigger-körningen med ID ' 08586002468005888497807248799CU16 ' i fabriken WikiADF.
