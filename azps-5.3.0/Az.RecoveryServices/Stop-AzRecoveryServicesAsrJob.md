# Stop-AzRecoveryServicesAsrJob

## Sammanfattning
Stoppar ett Azure Site Recovery-jobb.

## FRÅGESYNTAXEN

### ByObject (standard)
```
Stop-AzRecoveryServicesAsrJob -InputObject <ASRJob> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByName
```
Stop-AzRecoveryServicesAsrJob -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Stop-AzRecoveryServicesAsrJob** stoppar det angivna Azure Site Recovery-jobbet.

## BESKRIVS

### Exempel 1
```
PS C:\> $currentJob = Stop-AzRecoveryServicesAsrJob -Job $Job
```

Försöker stoppa det angivna jobbet och returnerar ett uppdaterat objekt i ASR-jobbet.
