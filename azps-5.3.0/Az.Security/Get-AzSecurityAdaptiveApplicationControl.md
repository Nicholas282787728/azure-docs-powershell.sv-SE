# Get-AzSecurityAdaptiveApplicationControl

## Sammanfattning
Hämtar en lista med program kontroll för den virtuella dator/server gruppen för abonnemanget.

## FRÅGESYNTAXEN

### SubscriptionScope (standard)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Anpassade program kontroller beräknas automatiskt i Azure Security Center, Använd denna cmdlet för att få en lista över anpassade program styr resurser i ett abonnemangs omfång.
