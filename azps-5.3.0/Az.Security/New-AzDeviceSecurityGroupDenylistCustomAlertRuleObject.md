# New-AzDeviceSecurityGroupDenylistCustomAlertRuleObject

## Sammanfattning
Skapa en ny regel för att neka den anpassade varnings listan för säkerhets gruppen enhet (IoT Security)

## FRÅGESYNTAXEN

```
New-AzDeviceSecurityGroupDenylistCustomAlertRuleObject -Enabled <Boolean> -Type <String>
 -DenylistValue <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
New-AzDeviceSecurityGroupDenylistCustomAlertRuleObject cmdlet skapar en ny lista över nekade anpassade notifieringsregler för säkerhets gruppen enhet i IoT-säkerhetslösningen.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> New-AzDeviceSecurityGroupDenylistCustomAlertRuleObject -Enabled $false -Type "SomeRuleType" -DenylistValue @()

RuleType: "SomeRuleType"
DisplayName: "Display name for some rule type"
Description: "Description for some rule type"
IsEnabled: false
ValueType: "String"
DenylistValues: []
```

Skapa en ny regel för att neka en anpassad avisering med rull typen "SomeRuleType" och status inställd på desable
