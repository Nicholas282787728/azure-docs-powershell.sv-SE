# Remove-AzSentinelBookmark

## Sammanfattning
Ta bort ett bok märke.

## FRÅGESYNTAXEN

### BookmarkId. Vis
```
Remove-AzSentinelBookmark -ResourceGroupName <String> -WorkspaceName <String> -BookmarkId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelBookmark -InputObject <PSSentinelBookmark> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzSentinelBookmark** tar bort ett bok märke permanent från en angiven arbets yta.
Du kan skicka ett **Bookmark** -objekt med hjälp av pipeline-operatorn eller så kan du ange de parametrar som krävs.
Du kan använda Confirm-parametern och $ConfirmPreference Windows PowerShell-variabel för att kontrol lera om cmdleten uppmanar dig att bekräfta.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -BookmarkId "MyBookmarkId"
```

Det här kommandot tar bort bok märket från arbets ytan.
