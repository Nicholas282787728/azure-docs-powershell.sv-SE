# Get-AzSentinelAlertRuleAction

## Sammanfattning
Få ett automatiskt svar (varnings regel åtgärd).

## FRÅGESYNTAXEN

### AlertRuleId (standard)
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ActionId
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 -ActionId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSentinelAlertRuleAction** får ett automatiskt svar (varnings regel åtgärd) från den angivna arbets ytan.
Om du anger parametrarna *ActionId* och *AlertRuleId* returneras ett enskilt **AlertRuleAction** -objekt.
Om du inte anger parametern *ActionId* returneras en matris som innehåller alla åtgärder för den specifika varnings regeln på den angivna arbets ytan.
Du kan **använda händelseobjektet för att uppdatera åtgärden,** till exempel om du vill ändra **åtgärden** för en notifieringsregel.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $AlertRuleActions = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId"
```

Det här exemplet får alla **åtgärder** för den angivna notifieringsregeln på den angivna arbets ytan och lagrar den sedan i $AlertRuleActions variabel.

### Exempel 2
```powershell
PS C:\> $AlertRuleAction = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId" -ActionId "MyActionId"
```

Det här exemplet får en **AlertRuleAction** för den angivna notifieringsregeln på den angivna arbets ytan och lagrar den sedan i $AlertRuleAction variabel.
