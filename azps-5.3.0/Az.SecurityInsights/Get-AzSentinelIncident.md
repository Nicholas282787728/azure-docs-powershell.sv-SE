# Get-AzSentinelIncident

## Sammanfattning
Få ett problem.

## FRÅGESYNTAXEN

### WorkspaceScope (standard)
```
Get-AzSentinelIncident -ResourceGroupName <String> -WorkspaceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IncidentId
```
Get-AzSentinelIncident -ResourceGroupName <String> -WorkspaceName <String> [-IncidentId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ID
```
Get-AzSentinelIncident -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSentinelIncident** hämtar en incident från den angivna arbets ytan.
Om du anger parametern *IncidentId* returneras ett enskilt **incident** objekt.
Om du inte anger parametern *IncidentId* returneras en matris med alla händelser på den angivna arbets ytan.
Du kan använda **händelsen incident** för att uppdatera händelsen, till **exempel att lägga till anteckningar om felet**.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $Incidents = Get-AzSentinelIncident -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName"
```

Det här exemplet får alla **händelser** på den angivna arbets ytan och lagrar dem sedan i $incidents variabel.

### Exempel 2
```powershell
PS C:\> $Incident = Get-AzSentinelIncident -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId"
```

Det här exemplet får en **incident** på den angivna arbets ytan och lagrar den sedan i $incident variabel.
