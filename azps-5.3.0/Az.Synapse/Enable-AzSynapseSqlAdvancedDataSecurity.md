# Enable-AzSynapseSqlAdvancedDataSecurity

## Sammanfattning
Aktiverar avancerad data säkerhet på en arbets yta.

## FRÅGESYNTAXEN

### EnableByNameParameterSet (standard)
```
Enable-AzSynapseSqlAdvancedDataSecurity [-ResourceGroupName <String>] -WorkspaceName <String>
 [-DoNotConfigureVulnerabilityAssessment] [-DeploymentName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### EnableByInputObjectParameterSet
```
Enable-AzSynapseSqlAdvancedDataSecurity -InputObject <PSSynapseWorkspace>
 [-DoNotConfigureVulnerabilityAssessment] [-DeploymentName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### EnableByResourceIdParameterSet
```
Enable-AzSynapseSqlAdvancedDataSecurity -ResourceId <String> [-DoNotConfigureVulnerabilityAssessment]
 [-DeploymentName <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Enable-AzSynapseSqlAdvancedDataSecurity** aktiverar avancerad data säkerhet på en arbets yta. Avancerad data säkerhet är ett enhetligt säkerhets paket som innehåller data klassificering, sårbarhets utvärdering och Avancerat skydd för arbets ytan. (Ett nytt lagrings konto skapas automatiskt för att spara säkerhets utvärderingar. Om du har skapat ett lagrings konto för detta ändamål används det i stället.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Enable-AzSynapseSqlAdvancedDataSecurity -WorkspaceName ContosoWorkspace
```

Det här kommandot aktiverar avancerad data säkerhet för arbets ytan.

### Exempel 2
```powershell
PS C:\> Get-AzSynapseWorkspace -Name ContosoWorkspace | Enable-AzSynapseSqlAdvancedDataSecurity
```

Det här kommandot möjliggör avancerad data säkerhet för arbets ytan.

### Exempel 3
```powershell
PS C:\> Enable-AzSynapseSqlAdvancedDataSecurity -WorkspaceName ContosoWorkspace -DoNotConfigureVulnerabilityAssessment
```

Det här kommandot aktiverar avancerad data säkerhet via arbets yta och aktiverar inte automatisk säkerhets problem.
