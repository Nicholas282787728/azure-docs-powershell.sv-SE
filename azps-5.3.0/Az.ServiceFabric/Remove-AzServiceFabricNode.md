# Remove-AzServiceFabricNode

## Sammanfattning
Ta bort noder från en viss nodtyp från ett kluster.

## FRÅGESYNTAXEN

```
Remove-AzServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Använd **Remove-AzServiceFabricNode** för att ta bort noder från en viss nodtyp från ett kluster. Borttagningen fortsätter bara om den uppfyller kluster hälso måtten.

## BESKRIVS

### Exempel 1
```powershell
PS c:> Remove-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeType 'nt1' -NumberOfNodesToRemove 2
```

Det här kommandot tar bort två noder från NodeType ' NT1 '.
