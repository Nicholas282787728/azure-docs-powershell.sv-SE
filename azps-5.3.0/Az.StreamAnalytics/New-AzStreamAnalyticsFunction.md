# New-AzStreamAnalyticsFunction

## Sammanfattning
Skapar eller ersätter en funktion i ett ström analys jobb.

## FRÅGESYNTAXEN

```
New-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzStreamAnalyticsFunction** skapar en funktion i ett Azure Stream Analytics-jobb eller ersätter en befintlig funktion.
Definiera funktionen i en JSON-fil (JavaScript Object Notation).
Du kan ange namnet på funktionen med hjälp av parametern *namn* eller i JSON-filen.
Om du anger namnet på båda sätten måste namnen vara identiska.
Om du vill ersätta en befintlig funktion anger du namnet på den befintliga funktionen.

## BESKRIVS

### Exempel 1: skapa en Stream Analytics-funktion
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob07" -File "C:\Function07.json"
```

Det här kommandot skapar en funktion från filen Function07.jspå.
Namnet på funktionen lagras i. JSON-filen.

### Exempel 2: skapa en Stream Analytics-funktion med namnet ScoreTweet
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet"
```

Det här kommandot skapar en funktion på jobbet med namnet ScoreTweet.

### Exempel 3: ersätta en Stream Analytics-funktion
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet" -Force
```

Det här kommandot ersätter definitionen av den befintliga funktionen med namnet ScoreTweet med definitionen i Function22.jspå.
