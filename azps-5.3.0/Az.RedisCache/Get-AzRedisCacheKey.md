# Get-AzRedisCacheKey

## Sammanfattning
Hämtar snabb tangenterna för en Redis cache.

## FRÅGESYNTAXEN

```
Get-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzRedisCacheKey** hämtar snabb tangenterna för Azure Redis-cachen.

## BESKRIVS

### Exempel 1: Hämta snabb tangenterna för en Redis cache
```
PS C:\>Get-AzRedisCacheKey -ResourceGroupName "MyResourceGroup" -Name "MyCacheKey"
PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Det här kommandot får åtkomst nycklar som heter MyCacheKey.
