# Get-AzRedisCache

## Sammanfattning
Hämtar en Redis cache.

## FRÅGESYNTAXEN

```
Get-AzRedisCache [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzRedisCache** hämtar det angivna Azure Redis-cacheminnet.
Om du inte anger några parametrar får den här åtgärden alla Redis-cachen för det aktuella abonnemanget.

## BESKRIVS

### Exempel 1: Hämta en Redis-cache efter namn
```
PS C:\>Get-AzRedisCache -Name "myexists"

        Name               : myexists
        Type               : Microsoft.Cache/Redis
        HostName           : myexists.redis.cache.windows.net
        Port               : 6379
        ProvisioningState  : succeeded
        SslPort            : 6380
        RedisConfiguration : {}
        EnableNonSslPort   : False
        RedisVersion       : 2.8
        Size               : 1GB
        Sku                : Basic
        Tag                : {}
        Zone               : []
```

Det här kommandot får Redis cacheminnet som heter exist.
