# Set-AzApplicationGatewayClientAuthConfiguration

## Sammanfattning
Ändrar konfigurationen för klientautentisering för ett SSL-profil objekt.

## FRÅGESYNTAXEN

```
Set-AzApplicationGatewayClientAuthConfiguration -SslProfile <PSApplicationGatewaySslProfile>
 [-VerifyClientCertIssuerDN] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzApplicationGatewayClientAuthConfiguration** ändrar konfigurationen för klientautentisering för ett SSL-profil objekt.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "SslProfile01" -ApplicationGateway $AppGw
PS C:\> Set-AzApplicationGatewayClientAuthConfiguration -SslProfile $profile -VerifyClientCertIssuerDN
```

Det första kommandot får den Programgateway som heter ApplicationGateway01 i resurs gruppen som heter ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot får den SSL-profil som heter SslProfile01 för $AppGw och lagrar inställningarna i $profile-variabeln. Det senaste kommandot ändrar konfigurationen för klientautentisering för det SSL-profil objekt som lagras i $profile.
