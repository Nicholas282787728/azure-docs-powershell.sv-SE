# Remove-AzExpressRoutePortIdentity

## Sammanfattning
Tar bort en identitet från en ExpressRoutePort.

## FRÅGESYNTAXEN

```
Remove-AzExpressRoutePortIdentity -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzExpressRoutePortIdentity** tar bort identiteten från ett lokalt Azure ExpressRoutePort-objekt. Använd **Remove-AzExpressRoutePort** för att ta bort den från ExpressRoutePort.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $expressroutePort = Remove-AzExpressRoutePortIdentity -ExpressRoutePort $expressroutePort
```
