# Get-AzNetworkInterfaceTapConfig

## Sammanfattning
Hämtar en tryck konfigurations resurs.

## FRÅGESYNTAXEN

### GetByNameParameterSet (standard)
```
Get-AzNetworkInterfaceTapConfig -ResourceGroupName <String> -NetworkInterfaceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzNetworkInterfaceTapConfig -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzNetworkInterfaceTapConfig** får en Azure knackning-konfiguration för en viss resurs grupp, nätverks gränssnitt och tryck på konfigurations namn eller en lista över konfigurationer i en resurs grupp och ett nätverks gränssnitt.

## BESKRIVS

### Exempel 1: Hämta alla tryck konfigurationer för ett visst nätverks gränssnitt
```
PS C:\> Get-AzNetworkInterfaceTapConfig -ResourceGroupName "ResourceGroup1" -NetworkInterface "sourceNicName"
```

Det här kommandot får trycka på konfigurationer som lagts till för det angivna nätverks gränssnittet.

### Exempel 2: få ett bestämt tryck på konfiguration
```
PS C:\> Get-AzNetworkInterface -ResourceGroupName "ResourceGroup1" -NetworkInterface "sourceNicName" -Name "tapconfigName"
```

Det här kommandot ger specifik tryck konfiguration som läggs till för det angivna nätverks gränssnittet.

### Exempel 3: få en specifik trycknings konfiguration
```
PS C:\> Get-AzNetworkInterface -ResourceGroupName "ResourceGroup1" -NetworkInterface "sourceNicName" -Name "tapconfig*"
```

Det här kommandot får trycka på konfigurationer som lagts till för det angivna nätverks gränssnittet med namn som börjar med "tapconfig".
