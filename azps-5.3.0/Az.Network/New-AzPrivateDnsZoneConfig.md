# New-AzPrivateDnsZoneConfig

## Sammanfattning
Skapar DNS-zonöverföringar för gruppen privata DNS-zoner.

## FRÅGESYNTAXEN

```
New-AzPrivateDnsZoneConfig -Name <String> [-PrivateDnsZoneId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Med **New-AzPrivateDnsZoneConfig** cmdlet kan du skapa ett nytt konfigurations objekt för DNS-zon som kommer att anges i DNS-zonfilen.

## BESKRIVS

### Skapar konfiguration för DNS-zoner
```powershell
PS C:\> $dnsZone = New-AzPrivateDnsZone -ResourceGroupName "rg" -Name "test.vault.azure.com"
PS C:\> $config = New-AzPrivateDnsZoneConfig -Name "test-vault-azure-com" -PrivateDnsZoneId $dnsZone.ResourceId
```

Exemplet ovan skapar DNS Zone och skapar sedan DNS Zone-konfigurering. `New-AzPrivateDnsZone` cmdleten bevaras av modul AZ. PrivateDns.
