# Remove-AzApplicationGatewayClientAuthConfiguration

## Sammanfattning
Tar bort konfigurationen för klientautentisering för ett SSL-profil objekt.

## FRÅGESYNTAXEN

```
Remove-AzApplicationGatewayClientAuthConfiguration -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApplicationGatewayClientAuthConfiguration** tar bort konfigurationen för klientautentisering för ett SSL-profil objekt.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "Profile01" -ApplicationGateway $AppGw
PS C:\> Remove-AzApplicationGatewayClientAuthConfiguration -SslProfile $profile
```

Det första kommandot får den Programgateway som heter ApplicationGateway01 i resurs gruppen som heter ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot får den SSL-profil som heter Profile01 för $AppGw och lagrar den i $profile-variabeln. Med det senaste kommandot tar du bort klientautentisering för den SSL-profil som lagras i $profile.
