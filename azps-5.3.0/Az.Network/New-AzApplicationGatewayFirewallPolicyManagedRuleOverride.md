# New-AzApplicationGatewayFirewallPolicyManagedRuleOverride

## Sammanfattning
Skapar en managedRuleOverride-post för RuleGroupOverrideGroup-posten.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId <String> [-State <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Den **nya-AzApplicationGatewayFirewallPolicyManagedRuleOverride** skapar en ruleOverride-post.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> ruleOverrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId $ruleId -State Disabled
```

Skapar en ruleOverride-post med RuleId som $ruleId och State som inaktive rad.
