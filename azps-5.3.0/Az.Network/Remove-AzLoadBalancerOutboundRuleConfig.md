# Remove-AzLoadBalancerOutboundRuleConfig

## Sammanfattning
Tar bort en konfiguration för utgående regel från en belastningsutjämnare.

## FRÅGESYNTAXEN

```
Remove-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzLoadBalancerOutboundRuleConfig** tar bort en konfiguration för utgående regel från en Azure-belastningsutjämnare.

## BESKRIVS

### Exempel 1: ta bort en regel för utgående trafik från en Azure-belastningsutjämnare
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Remove-AzLoadBalancerOutboundRuleConfig -Name "RuleName" -LoadBalancer $slb
PS C:\>Set-AzLoadBalancer -LoadBalancer $slb
```

Det första kommandot får belastningsutjämnaren som är kopplad till den utgående regel konfiguration som du vill ta bort och sedan spara den i $slb variabel.
Det andra kommandot tar bort den associerade konfigurationen för utgående trafik från belastningsutjämnaren.
Det tredje kommandot uppdaterar belastningsutjämnaren.
