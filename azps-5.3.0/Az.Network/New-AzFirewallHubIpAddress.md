# New-AzFirewallHubIpAddress

## Sammanfattning
IP-adresser assoicated till brand väggen på virtuellt nav

## FRÅGESYNTAXEN

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
IP-adresser assoicated till brand väggen på virtuellt nav. Dessa kan vara offentliga och privata adresser

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

I det här exemplet skapas ett nav-IP-adressprefix med antalet offentliga (public) IP-adresser. HubIPAddress-objektet är ssociated till brand väggen på det virtuella navet.
