# New-AzApplicationGatewayClientAuthConfiguration

## Sammanfattning
Skapar en ny konfiguration för klientautentisering för SSL-profil.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayClientAuthConfiguration [-VerifyClientCertIssuerDN]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzApplicationGatewayClientAuthConfiguration** skapar en ny konfiguration för KLIENTAUTENTISERING för SSL-profil.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $clientAuthConfig = New-AzApplicationGatewayClientAuthConfiguration -VerifyClientCertIssuerDN
```

Kommandot skapar en ny konfiguration för klientautentisering och lagrar den i $clientAuthConfig variabel som ska användas i en SSL-profil.
