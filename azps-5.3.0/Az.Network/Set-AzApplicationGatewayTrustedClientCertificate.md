# Set-AzApplicationGatewayTrustedClientCertificate

## Sammanfattning
Ändrar den betrodda klient certifikat utfärdarens certifikat kedja för en Programgateway.

## FRÅGESYNTAXEN

```
Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzApplicationGatewayTrustedClientCertificate** ändrar den betrodda klient certifikat utfärdarens certifikat kedja för en Programgateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\clientCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Ovan i scenario visar vi hur du uppdaterar ett befintligt kedje objekt för en betrodd klient certifikat utfärdare. Det första kommandot får en Programgateway och lagrar den i $gw variabeln. Det andra kommandot ändrar det befintliga betrodda klient certifikat utfärdarens certifikat för certifikatutfärdarcertifikatet med en ny kedje fil för CA-certifikat. Det tredje kommandot uppdaterar programgatewayen på Azure.
