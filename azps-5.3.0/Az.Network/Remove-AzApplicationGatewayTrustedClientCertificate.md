# Remove-AzApplicationGatewayTrustedClientCertificate

## Sammanfattning
Tar bort kedje objekt för Trusted client-certifikatutfärdare från en Programgateway.

## FRÅGESYNTAXEN

```
Remove-AzApplicationGatewayTrustedClientCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApplicationGatewayTrustedClientCertificate** tar bort kedje objekt för Trusted client-certifikatutfärdare från en Programgateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name "TrustedClientCertificate01"
```

Det första kommandot får en Programgateway och lagrar den i $gw variabeln. Det andra kommandot tar bort den betrodda klient certifikat utfärdarens certifikat kedja med namnet "TrustedClientCertificate01" från Application Gateway som lagras i $gw.
