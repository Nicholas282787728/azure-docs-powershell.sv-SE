# Remove-AzVMAccessExtension

## Sammanfattning
Tar bort VMAccess-tillägget från en virtuell dator.

## FRÅGESYNTAXEN

```
Remove-AzVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzVMAccessExtension** tar bort tillägget för virtuell dator åtkomst (VMAccess) från en virtuell dator.

## BESKRIVS

### Exempel 1

Tar bort VMAccess-tillägget från en virtuell dator. (automatiskt genererat)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzVMAccessExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```
