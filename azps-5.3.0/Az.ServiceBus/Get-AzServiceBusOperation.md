# Get-AzServiceBusOperation

## Sammanfattning
Lista över ServiceBus-åtgärder som stöds

## FRÅGESYNTAXEN

```
Get-AzServiceBusOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzServiceBusOperation** innehåller en lista med de ServiceBus-funktioner som stöds.

## BESKRIVS

### Exempel 1
```
PS C:\> Get-AzServiceBusOperation
```

Här listas ServiceBus-funktioner som stöds
