# New-AzApiManagementNamedValue

## Sammanfattning
Skapar ett nytt namngivet värde.

## FRÅGESYNTAXEN

```
New-AzApiManagementNamedValue -Context <PsApiManagementContext> [-NamedValueId <String>] -Name <String>
 -Value <String> [-Secret] [-Tag <String[]>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzApiManagementNamedValue** skapar ett **värde** för Azure API Management.

## BESKRIVS

### Exempel 1: skapa ett namngivet värde som innehåller Taggar
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>$Tags = 'sdk', 'powershell'
PS C:\> New-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -Name "Property Name" -Value "Property Value" -Tags $Tags
```

Det första kommandot tilldelar två värden till $Tags variabel.
Det andra kommandot skapar ett namngivet värde och tilldelar strängarna i $Tags som taggar på egenskapen.

### Exempel 2: skapa ett namngivet värde med ett hemligt värde
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property12" -Name "Secret Property" -Value "Secret Property Value" -Secret
```

Det här kommandot skapar ett **namngivet värde** som har ett krypterat värde.
