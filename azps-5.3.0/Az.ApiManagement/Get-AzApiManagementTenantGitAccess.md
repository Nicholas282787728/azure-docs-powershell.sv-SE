# Get-AzApiManagementTenantGitAccess

## Sammanfattning
Hämtar git-åtkomst-konfigurationen för en innehavare.

## FRÅGESYNTAXEN

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzApiManagementTenantGitAccess** hämtar konfigurationen för git-åtkomst för en innehavare.
Nycklarna tas inte med i resultatet. Använd **Get-AzApiManagementTenantGitAccessSecret** för att få klient hemlighet.

## BESKRIVS

### Exempel 1: Hämta konfiguration för klient åtkomst
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```
