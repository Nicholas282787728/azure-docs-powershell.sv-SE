# Modulen AZ. Maps
## Problembeskrivning
I det här avsnittet visas hjälp för Azure Maps-cmdletar.

## AZ. kartor cmdlets
### [Get-AzMapsAccount](Get-AzMapsAccount.md)
Får kontot.

### [Get-AzMapsAccountKey](Get-AzMapsAccountKey.md)
Hämtar API-nycklarna för ett konto.
De här nycklarna är den autentiseringsmekanism som används vid kommande samtal till Azure Maps.

### [New-AzMapsAccount](New-AzMapsAccount.md)
Skapar ett Azure Maps-konto.

### [New-AzMapsAccountKey](New-AzMapsAccountKey.md)
Återskapar en konto-nyckeln.

### [Remove-AzMapsAccount](Remove-AzMapsAccount.md)
Tar bort ett Azure Maps-konto.
