# Clear-AzSqlServerAdvancedThreatProtectionSetting

## Sammanfattning
Tar bort de avancerade skydds inställningarna från en server.

## FRÅGESYNTAXEN

```
Clear-AzSqlServerAdvancedThreatProtectionSetting [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Clear-AzSqlServerAdvancedThreatProtectionSetting cmdlet tar bort avancerade inställningar för skydd mot en Azure SQL Server.
Använd den här cmdleten genom att ange parametrarna ResourceGroupName och ServerName för att identifiera den server som cmdleten tar bort.

## BESKRIVS

### Exempel 1: ta bort avancerade skydds inställningar för en databas
```
PS C:\> Clear-AzSqlServerAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Det här kommandot tar bort de avancerade skydds inställningarna från en server som heter Server01.
