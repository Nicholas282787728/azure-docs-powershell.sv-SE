# Get-AzSqlSyncAgentLinkedDatabase

## Sammanfattning
Returnerar information om SQL Server-databaser som är länkade av en synkroniseringsklient.

## FRÅGESYNTAXEN

```
Get-AzSqlSyncAgentLinkedDatabase [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlSyncAgentLinkedDatabase** returnerar information om SQL Server-databaser som är länkade av en synkroniseringsresurs.

## BESKRIVS

### Exempel 1: Hämta de länkade SQL Server-databaserna för en Azure SQL Sync-agent.

Följande exempel returnerar de länkade SQL Server-databaserna som är länkade av en Azure SQL Sync-agent.

```powershell
<!-- Aladdin Generated Example --> 
Get-AzSqlSyncAgentLinkedDatabase -ResourceGroupName MyResourceGroup -ServerName s1 -SyncAgentName 'SyncAgent01'
```
