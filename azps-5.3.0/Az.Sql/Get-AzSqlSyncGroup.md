# Get-AzSqlSyncGroup

## Sammanfattning
Returnerar information om synkroniseringsresurser för Azure SQL-databas.

## FRÅGESYNTAXEN

```
Get-AzSqlSyncGroup [[-Name] <String>] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlSyncGroup** returnerar information om en eller flera synkroniseringsresurser för Azure SQL-databas.
Ange namnet på en synkroniseringsresurs för att visa information för enbart den synkroniseringsresursen.
