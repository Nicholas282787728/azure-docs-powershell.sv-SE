# Get-AzSqlDatabaseBackupLongTermRetentionPolicy

## Sammanfattning
Hämtar en bevarande princip för en tids krävande databas.

## FRÅGESYNTAXEN

```
Get-AzSqlDatabaseBackupLongTermRetentionPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlDatabaseBackupLongTermRetentionPolicy** hämtar den långsiktiga bevarande principen för databasen.
Principen är en Azure Backup-resurs som används för att definiera säkerhets kopierings lagrings princip.

## BESKRIVS

### Exempel 1: hämta den senaste versionen av princip för bevarande
```powershell
PS C:\> Get-AzSqlDatabaseBackupLongTermRetentionPolicy -ResourceGroupName resourcegroup01 -ServerName server01 -DatabaseName database01


ResourceGroupName                      : resourcegroup01
ServerName                             : server01
DatabaseName                           : database01
WeeklyRetention                        : P2W
MonthlyRetention                       : PT0S
YearlyRetention                        : PT0S
WeekOfYear                             : 0
State                                  :
RecoveryServicesBackupPolicyResourceId :
Location                               :
```

Med det här kommandot får du den senaste versionen av bevarande principen för lång sikt för database01
