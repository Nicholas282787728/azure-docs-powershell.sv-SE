# Get-AzSqlSyncAgent

## Sammanfattning
Returnerar information om Azure SQL Sync-agenter.

## FRÅGESYNTAXEN

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlSyncAgent** returnerar information om en eller flera Sync-agenter för Azure SQL.
Ange namnet på en synkroniseringsresurs för att visa information för enbart den synkroniseringsklienten.
