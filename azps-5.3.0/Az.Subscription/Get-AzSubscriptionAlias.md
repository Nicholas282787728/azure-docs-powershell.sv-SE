# Get-AzSubscriptionAlias

## Sammanfattning
Får information om prenumerations Ali Aset

## FRÅGESYNTAXEN

```
Get-AzSubscriptionAlias [-AliasName <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSubscriptionAlias** får information om prenumerations Ali Aset.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzSubscriptionAlias -AliasName "ExistingAliasName"
```

Får information om prenumerations Ali Aset
