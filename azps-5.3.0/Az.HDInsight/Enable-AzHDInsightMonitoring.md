# Enable-AzHDInsightMonitoring

## Sammanfattning
Aktiverar övervakning i ett HDInsight-kluster och relevanta loggar skickas till övervaknings arbets ytan som anges vid aktivering.

## FRÅGESYNTAXEN

```
Enable-AzHDInsightMonitoring [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Enable-AzHDInsightMonitoring** aktiverar övervakning i ett Azure HDInsight-kluster.

## BESKRIVS

### Exempel 1
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

Övervakning aktive ras i HDInsight-klustret och relevanta loggar skickas till övervaknings arbets ytan med ID 1d364e89-bb71-4503-aa3d-a23535aea7bd

### Exempel 2
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

Övervakning aktive ras i HDInsight-klustret och relevanta loggar skickas till övervaknings arbets ytan med ID 1d364e89-bb71-4503-aa3d-a23535aea7bd
