# Use-AzHDInsightCluster

## Sammanfattning
Väljer ett kluster som ska användas med Invoke-RmAzureHDInsightHiveJob cmdlet.

## FRÅGESYNTAXEN

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **use-AzHDInsightCluster** väljer Azure HDInsight-klustret för den Invoke-AzHDInsightHiveJob cmdlet som ska användas för att skicka struktur jobb.

## BESKRIVS

### Exempel 1: Välj ett kluster för överföring av registrerings data fil
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Det här kommandot väljer ett kluster för överföring av registrerings data fil.
