# Set-AzDtlAutoStartPolicy

## Sammanfattning
Ställer in en policy för automatisk start för ett labb i DevTest Labs.

## FRÅGESYNTAXEN

### Aktivera (standard)
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Aktiverar
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzDtlAutoStartPolicy** anger den automatiska start principen för ett labb som tillåter att virtuella laboratorie datorer schemaläggs för automatisk start.
Cmdleten använder den angivna resurs gruppen och namnet på laboratoriet för att ange principen.

## BESKRIVS

### Exempel 1

Ställer in en policy för automatisk start för ett labb i DevTest Labs. (automatiskt genererat)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoStartPolicy -Days Sunday -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
