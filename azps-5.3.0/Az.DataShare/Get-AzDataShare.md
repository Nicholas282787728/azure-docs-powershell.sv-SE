# Get-AzDataShare

## Sammanfattning
Få information om data resurser.

## FRÅGESYNTAXEN

### ByFieldsParameterSet (standard)
```
Get-AzDataShare -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShare -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzDataShare** hämtar information om data resurser i en Azure Data Share-accoount.
Om du anger namnet på en data resurs får denna cmdlet information om den data resursen.
Om du inte anger ett namn hämtas den här cmdleten information om alla data resurser i ett Azure Data Share-konto.

## BESKRIVS

### Exempel 1: Hämta en specifik data resurs
```
PS C:\>Get-AzDataShare -ResourceGroupName "ADS" -AccountName "WikiAdsAccount" -Name "AdsShare"
Name                : AdsShare
Id                  : /subscriptions/f3ead1ff-d0ab-4cf4-9a5a-86f1661d4685/resourceGroups/ADS/providers/Microsoft.DataShare/accounts/WikiAdsAccount/shares/AdsShare
Type                : Microsoft.DataShare/shares
CreatedAt           : 6/11/2019 12:00:00 AM
CreatedBy           : Contoso ADS
ShareKind           : CopyBased
Description         : Example of description  
ProvisioningState   : Succeeded
Terms               : This should not be shared
```

Det här kommandot visar information om data AdsShare i Azure Data Share-WikiAdsAccount och resurs grupp annonser.
