# Save-AzDeploymentTemplate

## Sammanfattning
Sparar en distributionsmall till en fil.

## FRÅGESYNTAXEN

### SaveByDeploymentName (standard)
```
Save-AzDeploymentTemplate -DeploymentName <String> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SaveByDeploymentObject
```
Save-AzDeploymentTemplate -DeploymentObject <PSDeployment> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Save-AzDeploymentTemplate**  sparar en distributionsmall till en JSON-fil.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Save-AzDeploymentTemplate -DeploymentName "TestDeployment"
```

Det här kommandot får distributions mal len från TestDeployment och sparar den som en JSON-fil i den aktuella katalogen.

### Exempel 2: Hämta en distribution och spara dess mall
```
PS C:\>Get-AzDeployment -Name "RolesDeployment" | Save-AzDeploymentTemplate
```

Det här kommandot får distributionen "RolesDeployment" vid den aktuella prenumerations omfattningen och sparar mallen.
