# Get-AzKustoClusterLanguageExtension

## Sammanfattning
Returnerar en lista med språk tillägg som kan köras i Keyword-frågor.

## FRÅGESYNTAXEN

```
Get-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Returnerar en lista med språk tillägg som kan köras i Keyword-frågor.

## BESKRIVS

### Exempel 1: lista alla språk tillägg som är inställda för ett kluster
```powershell
PS C:\> Get-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster

Name
----
R
PYTHON
```

Kommandot ovan returnerar en lista över språk tillägg som kan köras i Keyword-frågor.
