# Get-AzLegacyPeering

## Sammanfattning
Används för att konvertera gamla peering-resurser till resurser för Azure Resource Management (ARM). 

## FRÅGESYNTAXEN

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Kommandot används för att Visa äldre peering-resurser som alla du använder för att konvertera dem till resurser för Azure Resource Management (ARM).
