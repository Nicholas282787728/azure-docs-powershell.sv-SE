# New-AzAttestation

## Sammanfattning
Skapar en attestering

## FRÅGESYNTAXEN

```
New-AzAttestation -Name <String> -ResourceGroupName <String> -Location <String> [-Tag <Hashtable>]
 [-PolicySignersCertificateFile <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
New-AzAttestation cmdlet skapar en attestering i den angivna resurs gruppen.

## BESKRIVS

### Exempel 1
```powershell

Status            : Ready

TagsTable         :

                    CreationYear  2020
```
