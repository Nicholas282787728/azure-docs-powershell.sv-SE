# Test-AzContainerRegistryNameAvailability

## Sammanfattning
Kontrollerar tillgänglighet för ett namn på en behållare.

## FRÅGESYNTAXEN

```
Test-AzContainerRegistryNameAvailability [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Test-AzContainerRegistryNameAvailability cmdlet kontrollerar om ett namn på en behållare är giltig och tillgänglig för användning.

## BESKRIVS

### Exempel 1: kontrollerar tillgänglighet för ett namn på en behållares register
```powershell
PS C:\>Test-AzContainerRegistryNameAvailability -Name 'SomeRegistryName'

NameAvailable Reason Message
------------- ------ -------
         True
```

Det här kommandot kontrollerar tillgängligheten för behållar register namnet \` SomeRegistryName \` .
