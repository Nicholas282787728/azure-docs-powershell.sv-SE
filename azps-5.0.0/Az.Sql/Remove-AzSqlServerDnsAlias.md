# Remove-AzSqlServerDnsAlias

## Sammanfattning
Tar bort DNS-alias för Azure SQL Server.

## FRÅGESYNTAXEN

### Ta bort ett DNS-alias för servrar från cmdlet-indataparametrar
```
Remove-AzSqlServerDnsAlias -Name <String> -ServerName <String> [-ResourceGroupName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Ta bort ett DNS-alias för servrar från instans definitionen för AzureSqlServerDnsAliasModel
```
Remove-AzSqlServerDnsAlias -InputObject <AzureSqlServerDnsAliasModel> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Ta bort ett DNS-alias för servrar från ett Azure Resource-ID
```
Remove-AzSqlServerDnsAlias -ResourceId <String> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
De här kommandona tar bort DNS-aliaset för Azure SQL Server från servern som lämnar servern intakt.

## BESKRIVS

### Exempel 1
```
PS C:\> Remove-AzSqlServerDnsAlias -DnsAliasName aliasName -ServerName serverName -ResourceGroupName rg
```

Tar bort DNS-aliaset för Azure SQL Server med namnet aliasName från servern med namnet serverName
