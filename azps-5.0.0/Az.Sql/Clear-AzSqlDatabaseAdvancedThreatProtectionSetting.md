# Clear-AzSqlDatabaseAdvancedThreatProtectionSetting

## Sammanfattning
Tar bort de avancerade skydds inställningarna från en databas.

## FRÅGESYNTAXEN

```
Clear-AzSqlDatabaseAdvancedThreatProtectionSetting [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Clear-AzSqlDatabaseAdvancedThreatProtectionSetting** tar bort avancerade skydds inställningar från en AzureAzure SQL-databas.
Använd den här cmdleten genom att ange parametrarna *ResourceGroupName* och *servername* för att identifiera den databas som den här cmdleten tar bort.

## BESKRIVS

### Exempel 1: ta bort avancerade skydds inställningar för en databas
```
PS C:\>Clear-AzSqlDatabaseAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot tar bort de avancerade skydds inställningarna från en databas som heter Database01 på den server som heter Server01.
