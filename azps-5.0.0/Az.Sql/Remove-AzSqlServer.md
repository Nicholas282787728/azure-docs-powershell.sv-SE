# Remove-AzSqlServer

## Sammanfattning
Tar bort en Azure SQL Database-Server.

## FRÅGESYNTAXEN

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzSqlServer** tar bort en Azure SQL Database-Server.
Borttagnings åtgärden är asynkron och kan ta lite tid, så kontrol lera att åtgärden är klar innan du utför ytterligare åtgärder som beror på att servern tas bort helt.
Till exempel måste du skapa en ny server med samma namn.

## BESKRIVS

### Exempel 1: ta bort en server
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Det här kommandot tar bort Azure SQL-databasfilen med namnet Server01.
