# Get-AzSqlDatabaseActivity

## Sammanfattning
Hämtar databas operationernas status.

## FRÅGESYNTAXEN

```
Get-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlDatabaseActivity** får statusen för databas åtgärder i Azure SQL-databasen.

## BESKRIVS

### Exempel 1: Hämta status för alla SQL Database-instanser
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Det här kommandot returnerar status för alla SQL databas-instanser i en elastisk pool med namnet ElasticPool01.

### Exempel 2: Hämta status för alla SQL-databas operationer
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot returnerar statusen för alla SQL-databas operationer i en databas.
