# Get-AzSqlDatabaseAdvisor

## Sammanfattning
Får en eller flera rådgivare till en Azure SQL-databas.

## FRÅGESYNTAXEN

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlDatabaseAdvisor** hämtar en eller flera Azure SQL-databas rådgivare för en Azure SQL-databas.
