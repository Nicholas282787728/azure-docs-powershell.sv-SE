# Get-AzServiceBusQueue

## Sammanfattning
Returnerar en beskrivning för den angivna tjänst buss kön.

## FRÅGESYNTAXEN

```
Get-AzServiceBusQueue [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Returnerar en beskrivning för den angivna tjänst buss kön.

Returnerar beskrivningen av kön.

### Exempel 2
```
PS C:\> Get-AzServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Returnerar en lista med köer för det angivna namn området, som standard kommer 100-köer att returneras, om fler än 100 köer ska returneras, Använd-MaxCount-parametern.

### Exempel 3
```
PS C:\> Get-AzServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Returnerar en lista med de första 150-köerna för det angivna namn området
