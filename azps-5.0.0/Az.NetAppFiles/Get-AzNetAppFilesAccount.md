# Get-AzNetAppFilesAccount

## Sammanfattning
Hämtar information om ett Azure NetApp-ANF-konto.

## FRÅGESYNTAXEN

### ByFieldsParameterSet (standard)
```
Get-AzNetAppFilesAccount -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzNetAppFilesAccount -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzNetAppFilesAccount** hämtar information om ett ANF-konto.

## BESKRIVS

### Exempel 1: skaffa ett ANF-konto
```
PS C:\>Get-AzNetAppFilesAccount -ResourceGroupName "MyRG" -Name "MyAnfAccount"

Output:

Location          : westus2
Id                : /subscriptions/mySubs/resourceGroups/MyRG/providers/Microsoft.NetApp/netAppAccounts/MyAnfAccount
Name              : MyAnfAccount
Type              : Microsoft.NetApp/netAppAccounts
Tags              :
ProvisioningState : Succeeded
```

Det här kommandot får kontot MyAnfAccount.
