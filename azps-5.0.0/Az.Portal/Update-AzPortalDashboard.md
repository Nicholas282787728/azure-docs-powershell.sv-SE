# Update-AzPortalDashboard

## Sammanfattning
Uppdaterar en befintlig instrument panel.

## FRÅGESYNTAXEN

### UpdateExpanded (standard)
```
Update-AzPortalDashboard -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-Lens <Hashtable>] [-Metadata <Hashtable>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzPortalDashboard -InputObject <IPortalIdentity> [-Lens <Hashtable>] [-Metadata <Hashtable>]
 [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Uppdaterar en befintlig instrument panel.

## BESKRIVS

### Exempel 1: uppdatera taggarna för en instrument panel
```powershell
PS C:\> Update-AzPortalDashboard -ResourceGroupName my-rg -Name dashbase03 -Tag @{'hidden-title'="My Dashboard Title"; NewTag="NewValue"}

Location Name       Type
-------- ----       ----
eastasia dashbase03 Microsoft.Portal/dashboards
```

Uppdatera taggarna i en instrument panel.
Taggar representeras som en infogad hash.

### Exempel 2: uppdatera instrument panels Taggar med pipelinen
```powershell
PS C:\> Get-AzPortalDashboard -ResourceGroupName my-rg -Name dashbase03 | Update-AzPortalDashboard -Tag @{'hidden-title'="My Dashboard Title"; NewTag="NewValue"}

Location Name       Type
-------- ----       ----
eastasia dashbase03 Microsoft.Portal/dashboards
```

Uppdatera taggarna i en instrument panel återkallas med Get-AzPortalDashboard.
Det här kan användas för att uppdatera märkningarna över en enskild instrument panel eller flera dashboardfs.
