# New-AzCosmosDBGremlinIncludedPath

## Sammanfattning
Skapar ett nytt objekt av typen PSIncludedPath. Det kan skickas som ett parameter värde för set-AzCosmosDBGremlinGraph.

## FRÅGESYNTAXEN

```
New-AzCosmosDBGremlinIncludedPath [-Path <String>] [-Index <PSIndexes[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Objekt som motsvarar Gremlin API-IncludedPath.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $index1 = New-AzCosmosDBGremlinIncludedPathIndex -DataType String -Precision -1 -Kind Hash
New-AzCosmosDBGremlinIncludedPath -Path "/*" -Index $index1
Path Indexes
---- -------
/*   {Microsoft.Azure.Commands.CosmosDB.Models.PSIndexes}
```
