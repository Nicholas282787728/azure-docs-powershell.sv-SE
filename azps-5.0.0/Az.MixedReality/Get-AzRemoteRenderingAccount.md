# Get-AzRemoteRenderingAccount

## Sammanfattning
Skaffa fjärrstyrt konto

## FRÅGESYNTAXEN

### ListParameterSet (standard)
```
Get-AzRemoteRenderingAccount [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetParameterSet (standard)
```
Get-AzRemoteRenderingAccount -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzRemoteRenderingAccount -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämta eller Visa fjärråter givnings konton i vissa abonnemang och resurs grupper.
