# Remove-AzDataLakeStoreTrustedIdProvider

## Sammanfattning
Tar bort den angivna betrodda identitets leverantören i den angivna data Lake Store.

## FRÅGESYNTAXEN

```
Remove-AzDataLakeStoreTrustedIdProvider [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzDataLakeStoreTrustedIdProvider** tar bort den angivna betrodda identitets leverantören i den angivna data Lake Store.

## BESKRIVS

### Exempel 1: ta bort en betrodd identitets leverantör.
```
PS C:\> Remove-AzDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider
```

Tar bort leverantören "" "leverantör" från kontot "ContosoADL"
