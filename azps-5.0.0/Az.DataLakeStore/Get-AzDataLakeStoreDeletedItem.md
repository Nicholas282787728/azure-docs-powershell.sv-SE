# Get-AzDataLakeStoreDeletedItem

## Sammanfattning
Söker efter borttagna poster i pappers korgen som matchar filtret.

## FRÅGESYNTAXEN

```
Get-AzDataLakeStoreDeletedItem [-Account] <String> [-Filter] <String> [-Count <Int32>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzDataLakeStoreDeletedItem** söker igenom borttagna filer och mappar i data Lake Store som matchar angivet filter.
Här visas följande attribut för borttagna objekt – OriginalPath, TrashDirPath, Type, CreationTime.
Detta kan vara tids krävande att köra eftersom det kanske måste söka igenom miljon tals filer i pappers korgen och kan köras som ett jobb.
Varning! det är en bra åtgärd att ta bort filer. Det finns inga garantier för att en fil kan återställas när den har tagits bort. Användningen av detta API aktive ras via väggar. Om ditt ADL-konto inte är whitelisted, följer detta API inte implementerat undantag. Om du vill ha mer information kontaktar du Microsoft support.
