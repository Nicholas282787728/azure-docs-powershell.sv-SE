# Add-AzDataLakeStoreItemContent

## Sammanfattning
Lägger till innehåll i ett objekt i en data Lake Store.

## FRÅGESYNTAXEN

```
Add-AzDataLakeStoreItemContent [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Value] <Object>
 [[-Encoding] <Encoding>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Add-AzDataLakeStoreItemContent** lägger till innehåll i ett objekt i en Azure Data Lake Store.

## BESKRIVS

### Exempel 1: lägga till innehåll i en fil
```
PS C:\>Add-AzDataLakeStoreItemContent -AccountName "ContosoADLS" -Path /abc/myFile.txt -Value "My content here"
```

Det här kommandot lägger till innehåll i filen myFile.txt.
