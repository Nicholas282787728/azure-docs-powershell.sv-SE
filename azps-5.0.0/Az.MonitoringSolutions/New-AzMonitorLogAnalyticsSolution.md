# New-AzMonitorLogAnalyticsSolution

## Sammanfattning
Skapar en logganalys-lösning.

## FRÅGESYNTAXEN

```
New-AzMonitorLogAnalyticsSolution -ResourceGroupName <String> -Location <String> -Type <String>
 -WorkspaceResourceId <String> [-SubscriptionId <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar en logganalys-lösning.

## BESKRIVS

### Exempel 1: skapa en övervaka logganalys-lösning för logganalys-arbetsytan
```powershell
PS C:\> $workspace = Get-AzOperationalInsightsWorkspace -ResourceGroupName azureps-manual-test -Name monitoringworkspace-2vob7n
PS C:\> New-AzMonitorLogAnalyticsSolution -Type Containers -ResourceGroupName azureps-manual-test -Location $workspace.Location -WorkspaceResourceId $workspace.ResourceId

Name                                   Type                                     Location
----                                   ----                                     --------
Containers(monitoringworkspace-2vob7n) Microsoft.OperationsManagement/solutions East US
```

Med det här kommandot skapas en monitor-lösning för logganalys-arbetsytan.

Vanliga typer är:

| Format | Problembeskrivning |
| :-----| :----- |
| SecurityCenterFree |  Azure Security Center – gratis version |
| Uppdatering | Azure Security Center |
| Updates | Uppdaterings hantering |
| ContainerInsights | Azure Monitor för behållare |
| ServiceMap | Tjänst karta |
| AzureActivity | Aktivitets logg analys |
| ChangeTracking | Spåra ändringar och lager |
| VMInsights | Azure Monitor för virtuella datorer |
| SecurityInsights | Azure Sentinel |
| NetworkMonitoring | Övervakaren för nätverks prestanda |
| SQLVulnerabilityAssessment | Utvärdering av SQL-sårbarhet |
| SQLAdvancedThreatProtection | Avancerat skydd för SQL |
| Antimalware | Utvärdering mot skadlig program vara |
| AzureAutomation | Automatiserings hybrid arbetare |
| LogicAppsManagement | Hantering av logiska appar |
| SQLDataClassification | & klassificering av SQL-dataidentifiering |
