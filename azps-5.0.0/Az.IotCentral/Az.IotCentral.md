# Modulen AZ. IotCentral
## Problembeskrivning
Tillhandahåller CRUD hantering för IoT Central program.

## AZ. IotCentral-cmdletar
### [Get-AzIotCentralApp](Get-AzIotCentralApp.md)
Hämtar egenskaper för antingen en eller flera IoT Central program.

### [New-AzIotCentralApp](New-AzIotCentralApp.md)
Skapar ett nytt IoT Central program.

### [Remove-AzIotCentralApp](Remove-AzIotCentralApp.md)
Tar bort ett IoT Central program.

### [Set-AzIotCentralApp](Set-AzIotCentralApp.md)
Uppdaterar metadata för ett IoT Central program.
