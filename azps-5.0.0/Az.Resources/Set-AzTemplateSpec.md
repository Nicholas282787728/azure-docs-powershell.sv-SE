## PROBLEMBESKRIVNING
Ändrar en Templace-spec. Om det angivna namnet och/eller den specifika versionen av mallen inte redan finns skapas den.

När du ändrar innehållet i en mal Linies spec-mall kan innehållet antingen komma från en RAW JSON-sträng (med **UpdateVersionByNameFromJsonParameterSet** parameter uppsättning) eller från en angiven JSON-fil (med **UpdateVersionByNameFromJsonFileParameterSet** parameter uppsättning).

## BESKRIVS

### Exempel 1:
```powershell
PS C:\> $templateJson = @"
{
    "$schema": "http://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
    "contentVersion": "1.0.0.0",
    "parameters": {},
    "resources": []
}
"@
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v1.0' -Location 'West US' -TemplateJson $templateJson
```

Ändrar version "v 1.0" för en mall med namnet "myTemplateSpec". Den angivna versionen är $templateJson som versions mal len för ARM-mallen. Om specifikation och/eller version av rot mal len inte redan finns kommer de att skapas.


**Anmärkningar** 

* Mallen ARM i exemplet är inte en-op eftersom den inte innehåller några faktiska resurser.
* Plats är endast nödvändig när mallen för mallar inte redan finns

### Exempel 2:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v2.0' -Location 'West US' -TemplateFile 'myTemplateContent.json'
```

Ändrar version "v 2.0" för en mall med namnet "myTemplateSpec". Den angivna versionen kommer att få innehållet från den lokala filen "myTemplateContent.jspå" som versionens ARM-mallens innehåll. Om specifikation och/eller version av rot mal len inte redan finns kommer de att skapas.

**Obs!** Plats är endast nödvändig när mallen för mallar inte redan finns

### Exempel 3:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec'  -Location 'West US' -Description 'My updated Template Spec'
```

Ändrar beskrivningen av mallen "myTemplateSpec" i resurs gruppen "myRG". Om mallens spec inte redan finns kommer den att skapas.

**Obs!** Plats är endast nödvändig när mallen för mallar inte redan finns
