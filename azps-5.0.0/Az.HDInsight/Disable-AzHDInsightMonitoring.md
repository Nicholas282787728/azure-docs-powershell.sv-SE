# Disable-AzHDInsightMonitoring

## Sammanfattning
Inaktiverar övervakning i ett HDInsight-kluster och relevanta loggar slutar att flöda till arbets ytan övervakning som anges under aktivera.

## FRÅGESYNTAXEN

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **disable-AzHDInsightMonitoring** inaktiverar övervakning i ett Azure HDInsight-kluster.

## BESKRIVS

### Exempel 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

Övervakning kommer att inaktive ras i HDInsight-klustret och relevanta loggar slutar flöda till övervaknings arbets ytan.

### Exempel 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

Övervakning kommer att inaktive ras i HDInsight-klustret och relevanta loggar slutar flöda till övervaknings arbets ytan.
