# Stop-AzHDInsightJob

## Sammanfattning
Stoppar ett angivet pågående jobb i ett kluster.

## FRÅGESYNTAXEN

```
Stop-AzHDInsightJob [-ClusterName] <String> [-JobId] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Stop-AzHDInsightJob** stoppar ett angivet jobb som körs på ett Azure HDInsight-kluster.

## BESKRIVS

### Exempel 1: stoppa ett jobb i angivet kluster
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential
PS C:\> Stop-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
            -JobId $jobId
```

Det här kommandot stoppar ett jobb i klustret som heter ditt-Hadoop-001.
