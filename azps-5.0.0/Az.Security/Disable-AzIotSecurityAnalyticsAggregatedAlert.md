# Disable-AzIotSecurityAnalyticsAggregatedAlert

## Sammanfattning
Stäng IoT-summerad varning

## FRÅGESYNTAXEN

### SolutionLevelResource (standard)
```
Disable-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName <String> -SolutionName <String> -Name <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Disable-AzIotSecurityAnalyticsAggregatedAlert -InputObject <PSIoTSecurityAggregatedAlert> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ID
```
Disable-AzIotSecurityAnalyticsAggregatedAlert -ResourceId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Disable-AzIotSecurityAnalyticsAggregatedAlert cmdlet avaktiverar en viss aggragated-varning på enheter i IoT Hub. Namnet på de aggregerade varningarna är en kombination av varnings typen och aviseringens aggragted-datum, avgränsade med "/".

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Disable-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName "MyResourceGroup" -SolutionName "MySolutionName" -Name "IoT_SucessfulLocalLogin/2020-03-15"
```

Ignorera den sammanslagna varningen "IoT_SucessfulLocalLogin/2020-03-15" (namn kombinerat med aviserings typen och dess aggregerade datum) från IoT Security-lösningen "MySolutionName" i resurs gruppen "MyResourceGroup"

### Exempel 2
```powershell
PS C:\> Disable-AzIotSecurityAnalyticsAggregatedAlert -ResourceId "/subscriptions/XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX/resourceGroups/MyResourceGroup/providers/Microsoft.Security/iotSecuritySolutions/MySolutionName/analyticsModels/default/aggregatedAlerts/IoT_SucessfulLocalLogin/2020-03-15"
```

Ignorera summerad avisering med resurs-ID "/subscriptions/XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX/resourceGroups/MyResourceGroup/providers/Microsoft.Security/iotSecuritySolutions/MySolutionName/analyticsModels/default/aggregatedAlerts/IoT_SucessfulLocalLogin/2020-03-15"
