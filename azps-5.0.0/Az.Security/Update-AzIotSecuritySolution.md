# Update-AzIotSecuritySolution

## Sammanfattning
Uppdatera en eller flera av följande egenskaper i IoT-säkerhetslösningen: taggar, rekommendations konfiguration, användardefinierade resurser

## FRÅGESYNTAXEN

### ResourceGroupLevelResource (standard)
```
Update-AzIotSecuritySolution -Name <String> -ResourceGroupName <String> [-Tag <Hashtable>]
 [-UserDefinedResource <PSUserDefinedResources>]
 [-RecommendationsConfiguration <PSRecommendationConfiguration[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ID
```
Update-AzIotSecuritySolution -ResourceId <String> [-Tag <Hashtable>]
 [-UserDefinedResource <PSUserDefinedResources>]
 [-RecommendationsConfiguration <PSRecommendationConfiguration[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Update-AzIotSecuritySolution -InputObject <PSIotSecuritySolution> [-Tag <Hashtable>]
 [-UserDefinedResource <PSUserDefinedResources>]
 [-RecommendationsConfiguration <PSRecommendationConfiguration[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Update-AzIotSecuritySolution cmdlet updayes en eller flera av följande egenskaper i en specifik IoT-säkerhetslösning: taggar, rekommendations konfiguration, användardefinierade resurser.
Endast de angivna egenskaperna uppdateras i IoT-säkerhetslösningen.
IoT-säkerhetslösningen samlar in säkerhets data från IoT-enheter och IoT Hub för att förhindra och upptäcka hot.
