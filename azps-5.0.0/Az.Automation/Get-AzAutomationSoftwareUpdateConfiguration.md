# Get-AzAutomationSoftwareUpdateConfiguration

## Sammanfattning
Hämtar en lista över konfigurationer för Azure Automation-programuppdatering.

## FRÅGESYNTAXEN

### ByAll (standard)
```
Get-AzAutomationSoftwareUpdateConfiguration [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationSoftwareUpdateConfiguration -Name <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByVMId
```
Get-AzAutomationSoftwareUpdateConfiguration -AzureVMResourceId <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Get-AzAutomationSoftwareUpdateConfiguration returnerar en lista med konfigurationer för program uppdatering. Om du vill ha en specifik konfiguration för program uppdateringen anger du parametern name. Du kan också lista de konfigurationer för program uppdateringar som riktar sig mot specifik Azure Virtual Machine genom att ange Azure Resource ID för den här virtuella datorn.

## BESKRIVS

### Exempel 1
Skaffa en konfiguration för Azure Automation-programuppdatering med namn.

```powershell
PS C:\> Get-AzAutomationSoftwareUpdateConfiguration -ResourceGroupName "mygroup" -AutomationAccountName "myaccount" -Name "MyWeeklySchedule"
```
