# Remove-AzAutomationSourceControl

## Sammanfattning
Tar bort en Azure Automation-källkod.

## FRÅGESYNTAXEN

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Remove-AzAutomationSourceControl cmdlet tar bort en käll kontroll från Azure Automation.

## BESKRIVS

### Exempel 1
Det här kommandot tar bort Automation Source Control med namnet VSTSNative i kontot som heter devAccount.
Det här kommandot anger parametern *Force* . Därför behöver du inte bekräfta.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
