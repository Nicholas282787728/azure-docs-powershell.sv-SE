# Remove-AzAutomationAccount

## Sammanfattning
Tar bort ett Automation-konto.

## FRÅGESYNTAXEN

```
Remove-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzAutomationAccount** tar bort ett Azure Automation-konto från en resurs grupp.
Mer information om automatiserings konton finns i New-AzAutomationAccount cmdlet.

## BESKRIVS

### Exempel 1: ta bort ett Automation-konto
```
PS C:\>Remove-AzAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

Det här kommandot tar bort ett Automation-konto med namnet ContosoAutomationAccount utan att fråga efter användar verifiering.
