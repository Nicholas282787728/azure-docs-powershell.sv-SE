# Remove-AzStreamAnalyticsOutput

## Sammanfattning
Tar bort ett resultat från ett ström analys jobb.

## FRÅGESYNTAXEN

```
Remove-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzStreamAnalyticsOutput** asynkront tar bort utdata från ett ström analys jobb i Azure.

## BESKRIVS

### EXEMPEL 1: ta bort ett resultat från ett jobb
```
PS C:\>Remove-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Det här kommandot tar bort utdata från jobbet StreamingJob.
