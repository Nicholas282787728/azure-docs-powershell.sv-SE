# Remove-AzDatabricksWorkspace

## Sammanfattning
Tar bort arbets ytan.

## FRÅGESYNTAXEN

### Ta bort (standard)
```
Remove-AzDatabricksWorkspace -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzDatabricksWorkspace -InputObject <IDatabricksIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort arbets ytan.

## BESKRIVS

### Exempel 1: ta bort en Databricks-arbetsyta
```powershell
PS C:\> Remove-AzDatabricksWorkspace -ResourceGroupName testgroup -Name databricks-test
```

Det här kommandot tar bort en Databricks-arbetsyta från en resurs grupp.

### Exempel 2: ta bort en Databricks-arbetsyta efter objekt
```powershell
PS C:\> $dbr = Get-AzDatabricksWorkspace -ResourceGroupName testgroup -Name databricks-test02
PS C:\> Remove-AzDatabricksWorkspace -InputObject $dbr
```

Det här kommandot tar bort en Databricks-arbetsyta från en resurs grupp.
