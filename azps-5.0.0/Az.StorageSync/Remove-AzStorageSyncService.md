# Remove-AzStorageSyncService

## Sammanfattning
Det här kommandot tar bort den angivna tjänsten för lagrings synkronisering.

## FRÅGESYNTAXEN

### StringParameterSet (standard)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Det här kommandot tar bort den angivna tjänsten för lagrings synkronisering. En lagrings tjänst kan bara tas bort när alla synkroniserade grupper och registrerade servrar tas bort först.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Det här kommandot tar bort tjänsten Storage Sync.
