# New-AzFrontDoorRulesEngineRuleObject

## Sammanfattning
Skapa ett PSRulesEngineRule-objekt för att skapa regel motorn.

## FRÅGESYNTAXEN

```
New-AzFrontDoorRulesEngineRuleObject -Name <String> -Priority <Int32> -Action <PSRulesEngineAction>
 [-MatchProcessingBehavior <PSMatchProcessingBehavior>] [-MatchCondition <PSRulesEngineMatchCondition[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapa ett PSRulesEngineRule-objekt för att skapa regel motorn.

Använd cmdlet "New-AzFrontDoorRulesEngineActionObject" för att skapa PSRulesEngineAction-objekt som ska överföras till parametern "-Action".
Använd cmdlet "New-AzFrontDoorRulesEngineMatchConditionObject" för att skapa PSRulesEngineMatchCondition-objekt som ska överföras till parametern "-MatchCondition".
