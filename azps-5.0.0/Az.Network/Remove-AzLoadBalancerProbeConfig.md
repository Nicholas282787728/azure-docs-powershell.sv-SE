# Remove-AzLoadBalancerProbeConfig

## Sammanfattning
Tar bort en PROBE-konfiguration från en belastningsutjämnare.

## FRÅGESYNTAXEN

```
Remove-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzLoadBalancerProbeConfig** tar bort en PROBE-konfiguration från en belastningsutjämnare.

## BESKRIVS

### Exempel 1: ta bort en PROBE-konfiguration från en belastningsutjämnare
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $loadbalancer
```

Det första kommandot får belastningsutjämnaren med namnet MyLoadBalancer och lagrar det sedan i $loadbalancer variabel.
Det andra kommandot tar bort konfigurationen med namnet ' Avsök från belastningsutjämnaren i $loadbalancer.
