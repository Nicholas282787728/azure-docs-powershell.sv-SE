# Remove-AzApplicationGatewayAutoscaleConfiguration

## Sammanfattning
Tar bort autoskalans konfiguration från en Programgateway.

## FRÅGESYNTAXEN

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApplicationGatewayAutoscaleConfiguration** tar bort autoskalans konfiguration från en befintlig Programgateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Med det första kommandot hämtas programgatewayen och lagras i $gw variabel.
Det andra kommandot tar bort autoskalans konfiguration från Application Gateway.
Det tredje kommandot uppdaterar programgatewayen på Azure.
