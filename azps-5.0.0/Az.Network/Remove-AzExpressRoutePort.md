# Remove-AzExpressRoutePort

## Sammanfattning
Tar bort en ExpressRoutePort.

## FRÅGESYNTAXEN

### ResourceNameParameterSet (standard)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzExpressRoutePort** tar bort en ExpressRoutePort.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Tar bort $PortName ExpressRoutePort-resursen i $rg resurs grupp i prenumerationen.

### Exempel 2
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

Tar bort ExpressRoutePort-resursen i InputObject.

### Exempel 3
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

Tar bort ExpressRoutePort-resursen med ResourceId $id.
