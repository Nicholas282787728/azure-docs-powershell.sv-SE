# New-AzApplicationGatewayFirewallPolicyExclusion

## Sammanfattning
Skapar ett undantag för brand Väggs principen

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayFirewallPolicyExclusion -MatchVariable <String> -SelectorMatchOperator <String>
 -Selector <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**New-AzApplicationGatewayFirewallPolicyExclusion-** cmdlet en ny lista över undantags regler för brand Väggs princip.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $exclusionEntry = New-AzApplicationGatewayFirewallPolicyExclusion -MatchVariable "RequestHeaderNames" -SelectorMatchOperator "StartsWith" -Selector "xyz"
```

Det här kommandot skapar en ny uteslutning-post för variabeln med namnet RequestHeaderNames och operatorn StartsWith och Selector som heter xyz. Undantags posten sparas i $exclusionEntry.
