# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## Sammanfattning
Skapar RuleGroupOverride-post i ManagedRuleSets för brand Väggs principen.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** skapar en ruleGroupOverride-post i en managedRuleSet för en brand Väggs princip.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Skapar en RuleGroupOverride-post med grupp namnet som $ruleName och regler som $rule 1, $rule 2. Tilldelar samma $overrideEntry
