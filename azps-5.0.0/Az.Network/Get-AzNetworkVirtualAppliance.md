# Get-AzNetworkVirtualAppliance

## Sammanfattning
Hämta eller Visa en lista över virtuella nätverk enheter.

## FRÅGESYNTAXEN

### ResourceNameParameterSet (standard)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Get-AzNetworkVirtualAppliance kommandon hämtar eller visar resurser för virtuella nätverk enheter.
