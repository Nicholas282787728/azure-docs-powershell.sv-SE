# Get-AzApplicationGatewayAuthenticationCertificate

## Sammanfattning
Hämtar ett autentiseringscertifikat för en Programgateway.

## FRÅGESYNTAXEN

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzApplicationGatewayAuthenticationCertificate** hämtar ett autentiseringscertifikat för en Azure Application Gateway.

## BESKRIVS

### Exempel 1: Hämta ett angivet autentiseringscertifikat
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $pool = Get-AzApplicationGatewayBackendAddressPool -Name "pool01" -ApplicationGateway $appgw
```

Det första kommandot får programgatewayen som heter appGwName och lagrar den i $appgw variabeln.
Det andra kommandot får autentiseringscertifikatet med namnet pool01 och lagrar det i $pool variabeln.
