# Set-AzApplicationGatewayAuthenticationCertificate

## Sammanfattning
Uppdaterar ett autentiseringscertifikat för en Programgateway.

## FRÅGESYNTAXEN

```
Set-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzApplicationGatewayAuthenticationCertificate** uppdaterar ett autentiseringscertifikat för en Azure Application Gateway.

## BESKRIVS

### Exempel 1: uppdatera ett autentiseringscertifikat
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Set-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01" -CertificateFile "C:\cert2.cer"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Med det första kommandot får programgatewayen namnet appGwName och lagras resultatet i $appgw variabel.
Det andra kommandot uppdaterar autentiseringscertifikatet med namnet cert01 i programgatewayen.
Det tredje kommandot uppdaterar programgatewayen.
