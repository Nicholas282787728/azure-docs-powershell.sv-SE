# New-AzApplicationGatewaySslCertificate

## Sammanfattning
Skapar ett SSL-certifikat för en Azure Application Gateway.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewaySslCertificate -Name <String> [-CertificateFile <String>] [-Password <SecureString>]
 [-KeyVaultSecretId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzApplicationGatewaySslCertificate** skapar ett SSL-certifikat för en Azure Application Gateway.

## BESKRIVS

### Exempel 1: skapa ett SSL-certifikat för en Azure Application Gateway.
```
PS C:\> $password = ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force
PS C:\> $cert = New-AzApplicationGatewaySslCertificate -Name "Cert01" -CertificateFile "D:\cert01.pfx" -Password $password
```

Det här kommandot skapar ett SSL-certifikat med namnet Cert01 för standard program-gateway och lagrar resultatet i variabeln som heter $Cert.

### Exempel 2: skapa ett SSL-certifikat med hjälp av nyckel valv hemlighet (version-mindre secretId) och lägga till i en Programgateway.
```
PS C:\> $secret = Get-AzKeyVaultSecret -VaultName "keyvault01" -Name "sslCert01"
PS C:\> $secretId = $secret.Id.Replace($secret.Version, "") # https://<keyvaultname>.vault.azure.net/secrets/
PS C:\> $cert = New-AzApplicationGatewaySslCertificate -Name "Cert01" -KeyVaultSecretId $secretId
```

Få hemligheten och skapa ett SSL-certifikat med `New-AzApplicationGatewaySslCertificate` .
Obs! as-version-mindre secretId tillhandahålls här, programgatewayen synkroniserar certifikatet med jämna mellanrum med ett valv.

### Exempel 3: skapa ett SSL-certifikat med hjälp av nyckel valv hemlighet och Lägg till det i en Programgateway.
```
PS C:\> $secret = Get-AzKeyVaultSecret -VaultName "keyvault01" -Name "sslCert01"
PS C:\> $secretId = $secret.Id # https://<keyvaultname>.vault.azure.net/secrets/<hash>
PS C:\> $cert = New-AzApplicationGatewaySslCertificate -Name "Cert01" -KeyVaultSecretId $secretId
```

Få hemligheten och skapa ett SSL-certifikat med `New-AzApplicationGatewaySslCertificate` .
OBS! om det krävs att Application Gateway synkroniserar certifikatet med ett valv kan du ange secretId.
