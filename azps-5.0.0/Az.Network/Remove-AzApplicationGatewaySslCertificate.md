# Remove-AzApplicationGatewaySslCertificate

## Sammanfattning
Tar bort ett SSL-certifikat från en Azure Application Gateway.

## FRÅGESYNTAXEN

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApplicationGatewaySslCertificate** tar bort ett SSL-certifikat från en Azure Application Gateway.

## BESKRIVS

### Exempel 1: ta bort ett SSL-certifikat från en Programgateway
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Med det första kommandot får programgatewayen namnet ApplicationGateway01 och lagras resultatet i variabeln som heter $AppGW.
Det andra kommandot tar bort SSL-certifikatet som heter Cert02 från Application Gateway som lagras i $AppGW variabel.
