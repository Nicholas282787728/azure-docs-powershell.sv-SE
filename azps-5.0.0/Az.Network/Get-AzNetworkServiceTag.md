# Get-AzNetworkServiceTag

## Sammanfattning
Hämtar listan över service tag information-resurser.

## FRÅGESYNTAXEN

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzNetworkServiceTag** hämtar listan med information om tjänst märknings resurser.

Observera att den Azure region information du anger kommer att användas som referens för versionen (inte som ett filter baserat på plats). Till exempel om du anger `-Location eastus2` att du vill få en lista med service märken med uppgifter om prefix i alla regioner, men begränsat till det moln som ditt abonnemang tillhör (t. ex. offentlig, amerikansk regering, Kina eller Tyskland).
