# New-AzVirtualApplianceSkuProperty

## Sammanfattning
Definiera en virtuell enhet för nätverks enheter för resursen.

## FRÅGESYNTAXEN

```
New-AzVirtualApplianceSkuProperty -VendorName <String> -BundledScaleUnit <String> -MarketPlaceVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Med kommandot New-AzVirtualApplianceSkuProperties definierar du en SKU för nätverks resurser för virtuella enheter.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $var=New-AzVirtualApplianceSkuProperty -VendorName "barracudasdwanrelease" -BundledScaleUnit 1 -MarketPlaceVersion 'latest'
```

Skapa ett objekt för SKU-egenskaper för virtuella enheter som ska användas med New-AzNetworkVirtualAppliance-kommando.
