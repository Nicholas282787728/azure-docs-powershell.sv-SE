# Get-AzsPlatformImage

## Sammanfattning
Returnerar virtuella dator bilder som laddats till plattforms bild lagringen.

## FRÅGESYNTAXEN

### Lista (standard)
```
Get-AzsPlatformImage [-Publisher <String>] [-Offer <String>] [-Sku <String>] [-Location <String>]
 [<CommonParameters>]
```

### Lära
```
Get-AzsPlatformImage -Publisher <String> -Offer <String> -Sku <String> -Version <String> [-Location <String>]
 [<CommonParameters>]
```

### ID
```
Get-AzsPlatformImage -ResourceId <String> [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Returnerar plattforms bilder.

## BESKRIVS

### --------------------------EXEMPEL 1--------------------------
```
Get-AzsPlatformImage
```

Returnerar virtuella dator bilder som är inlästa i Platform image lagring på platsen lokal.

### --------------------------EXEMPEL 2--------------------------
```
Get-AzsPlatformImage -Publisher Canonical -Offer UbuntuServer -Sku 16.04-LTS -Version 0.1.0
```

Skaffa en särskild plattforms bild.
