Hämta listan med erbjudanden för ombud.

## FRÅGESYNTAXEN

### Lista (standard)
```
Get-AzsDelegatedProviderManagedOffer -DelegatedProviderId <String> [-Skip <Int32>] [-Top <Int32>]
 [<CommonParameters>]
```
