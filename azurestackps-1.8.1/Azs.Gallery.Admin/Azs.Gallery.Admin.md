# AZS. Gallery. admin-modul
## Problembeskrivning
För hands version av AzureStack Gallery-administratör som tillhandahåller funktioner för att hantera Galleri objekt på Marketplace.

## AZS. Gallery. admin-cmdletar
### [Add-AzsGalleryItem](Add-AzsGalleryItem.md)
Laddar upp ett tjänst Galleri objekt till lagrings platsen.

### [Get-AzsGalleryItem](Get-AzsGalleryItem.md)
Visar en lista med Galleri objekt.

### [Remove-AzsGalleryItem](Remove-AzsGalleryItem.md)
Ta bort ett specifikt Galleri objekt.
