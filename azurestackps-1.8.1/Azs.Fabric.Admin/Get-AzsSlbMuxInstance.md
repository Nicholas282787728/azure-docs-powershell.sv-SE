# Get-AzsSlbMuxInstance

## Sammanfattning
Returnerar en lista över alla instanser av program varu belastning på en plats.

## FRÅGESYNTAXEN

### Lista (standard)
```
Get-AzsSlbMuxInstance [-Location <String>] [-ResourceGroupName <String>] [-Filter <String>] [-Skip <Int32>]
 [-Top <Int32>] [<CommonParameters>]
```

### Lära
```
Get-AzsSlbMuxInstance [-Name] <String> [-Location <String>] [-ResourceGroupName <String>] [<CommonParameters>]
```

### ID
```
Get-AzsSlbMuxInstance -ResourceId <String> [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Returnerar en lista över alla instanser av program varu belastning på en plats.

## BESKRIVS

### EXEMPEL 1
```
Get-AzsSlbMuxInstance
```

Skaffa alla multiplexor-instanser för belastningsutjämnare för program vara.

### EXEMPEL 2
```
Get-AzsSlbMuxInstance -Name "AzS-SLB01"
```

Skaffa en specifik multiplexor för multiplexor för belastningsutjämnare på en plats med ett namn.
