# Unregister-AzProviderFeature

## Sammanfattning
Avregistrerar en Azure-leverantör i ditt konto.

## FRÅGESYNTAXEN

```
Unregister-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Med cmdleten **Unregistered-AzProviderFeature** avregistrerar du en Azure Provider-funktion på ditt konto.

## BESKRIVS

### Exempel 1: avregistrera en funktion
```
PS C:\>Unregister-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Då avregistreras AllowApplicationSecurityGroups-funktionen för Microsoft. Network från ditt konto.
