## PROBLEMBESKRIVNING
Cmdleten **Get-AzManagementGroupDeploymentWhatIfResult** får arm-mallen What-If resultat för en mall distribution i den angivna hanterings gruppens omfattning. Den returnerar en lista över ändringar som visar vilka resurser som kommer att uppdateras om distributionen tillämpas utan att du behöver göra några ändringar i verkliga resurser. Om du vill ange formatet för det resulterande resultatet kan du använda parametern *ResultFormat* .

## BESKRIVS

### Exempel 1: Hämta ett What-If resultat i hanterings gruppens omfattning
```powershell
PS C:\> Get-AzManagementGroupDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -ManagementGroupId "myManagementGroup" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "FullResourcePayloads"
```

Det här kommandot får en What-If resultat i hanterings gruppens omfattning genom att använda en anpassad mallfil och en parameter fil på disken.
Kommandot använder parametern *location* för att ange var distributions data ska lagras.
Kommandot använder parametern *ManagementGroupId* för att ange hanterings gruppen där mallen ska distribueras.
Kommandot använder parametern *TemplateFile* för att ange en mallfil.
Kommandot använder parametern *TemplateParameterFile* för att ange en mallparameter.
Kommandot använder parametern *ResultFormat* för att ange What-If resultatet ska inkludera fullständiga resurs nytto laster.

### Exempel 2: få ett What-If resultat i hanterings gruppens omfattning med ResourceIdOnly
```powershell
PS C:\> Get-AzManagementGroupDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -ManagementGroupId "myManagementGroup" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "ResourceIdOnly"
```

Det här kommandot får en What-If resultat i hanterings gruppens omfattning genom att använda en anpassad mallfil och en parameter fil på disken.
Kommandot använder parametern *location* för att ange var distributions data ska lagras.
Kommandot använder parametern *ManagementGroupId* för att ange hanterings gruppen där mallen ska distribueras.
Kommandot använder parametern *TemplateFile* för att ange en mallfil.
Kommandot använder parametern *TemplateParameterFile* för att ange en mallparameter.
Kommandot använder parametern *ResultFormat* för att ange What-If-resultatet som endast innehåller resurs-ID.
