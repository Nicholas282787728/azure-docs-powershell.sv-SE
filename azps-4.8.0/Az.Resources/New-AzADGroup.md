# New-AzADGroup

## Sammanfattning
Skapar en ny Active Directory-grupp.

## FRÅGESYNTAXEN

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar en ny Active Directory-grupp. Här är de behörigheter som behövs:

- Azure Active Directory-diagram
  - Directory. ReadWrite. all
- Microsoft Graph
  - Directory. ReadWrite. all
  - PrivilegedAccess. ReadWrite. AzureADGroup

## BESKRIVS

### Exempel 1: skapa en ny AD-grupp

```powershell
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "MyGroupNick"
```

Skapar en ny AD-grupp med namnet "MyGroupDisplayName" och e-postsmek namnet "MyGroupNick".
