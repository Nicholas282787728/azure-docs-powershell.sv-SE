# Remove-AzTenantDeployment

## Sammanfattning
Tar bort en distribution med klient omfattningen och eventuella associerade åtgärder

## FRÅGESYNTAXEN

### RemoveByDeploymentName (standard)
```
Remove-AzTenantDeployment [-Name] <String> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByDeploymentId
```
Remove-AzTenantDeployment -Id <String> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObject
```
Remove-AzTenantDeployment -InputObject <PSDeployment> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzTenantDeployment** tar bort en Azure-distribution för det aktuella klient-scopet och eventuella associerade åtgärder.

## BESKRIVS

### Exempel 1: ta bort en distribution med ett angivet namn
```
PS C:\>Remove-AzTenantDeployment -Name "RolesDeployment"
```

Det här kommandot tar bort distributionen "RolesDeployment" med den aktuella klient organisations omfattningen.

### Exempel 2: skaffa en distribution och ta bort den
```
PS C:\>Get-AzTenantDeployment -Name "RolesDeployment" | Remove-AzTenantDeployment
```

Det här kommandot får distributionen "RolesDeployment" vid den aktuella klient omfattningen och tar bort det.
