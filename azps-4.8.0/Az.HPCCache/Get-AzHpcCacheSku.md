# Get-AzHpcCacheSku

## Sammanfattning
Får alla SKU: er tillgängliga i abonnemanget.

## FRÅGESYNTAXEN

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzHpcCacheSku** returnerar en lista med SKU: er som finns tillgängliga i prenumerationen.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
