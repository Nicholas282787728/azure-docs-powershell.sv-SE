# Remove-AzApiManagementAuthorizationServer

## Sammanfattning
Tar bort en auktoriseringsprincip.

## FRÅGESYNTAXEN

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApiManagementAuthorizationServer** tar bort en Azure API Management Authorization Server.

## BESKRIVS

### Exempel 1: ta bort en auktoriseringsregel
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Det här kommandot tar bort den angivna Server för API-hantering.
Eftersom *Force* -parametern anges krävs ingen bekräftelse.
