# Set-AzSynapseDataset

## Sammanfattning
Skapar eller uppdaterar en data uppsättning i arbets ytan.

## FRÅGESYNTAXEN

### SetByName (standard)
```
Set-AzSynapseDataset -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzSynapseDataset** skapar en DataSet eller uppdaterar en befintlig data uppsättning i arbets ytan.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Set-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset -DefinitionFile "C:\\samples\\Dataset.json"
```

Det här kommandot skapar en data mängd som heter ContosoDataset i arbets ytan med namnet ContosoWorkspace.
Kommandot baserar data uppsättningen på informationen i filen Dataset.js.
