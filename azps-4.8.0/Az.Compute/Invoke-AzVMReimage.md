# Invoke-AzVMReimage

## Sammanfattning
Bild av en virtuell Azure-dator.

## FRÅGESYNTAXEN

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Invoke-AzVMReimage** avbildningar en virtuell Azure-dator.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Det här kommandot återsätter den virtuella datorn som heter VirtualMachine07 i ResourceGroup11.
