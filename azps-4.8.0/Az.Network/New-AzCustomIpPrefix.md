# New-AzCustomIpPrefix

## Sammanfattning
Skapar en CustomIpPrefix-resurs

## FRÅGESYNTAXEN

```
New-AzCustomIpPrefix -Name <String> -ResourceGroupName <String> -Location <String> -Cidr <String>
 [-Zone <String[]>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzCustomIpPrefix** skapar en CustomIpPrefix-resurs.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $myCustomIpPrefix = New-AzCustomIpPrefix -Name $prefixName -ResourceGroupName $rgName -Cidr 40.40.40.0/24 -Location westus
```

Det här kommandot skapar en ny CustomIpPrefix-resurs med namnet $prefixName i resurs gruppen $rgName med en CIDR på 40.40.40.0/24 under västkusten
