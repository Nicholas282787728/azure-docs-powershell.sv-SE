# Get-AzRouteFilterRuleConfig

## Sammanfattning
Hämtar en regel för flödes filter i ett väg filter.

## FRÅGESYNTAXEN

```
Get-AzRouteFilterRuleConfig [-Name <String>] -RouteFilter <PSRouteFilter>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzRouteFilterRuleConfig** hämtar en regel för flödes filter eller en lista med flödes filter regler i ett väg filter.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "MyRouteFilter" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf
```

Det första kommandot får vägen med namnet MyRouteFilter och lagrar det sedan i variabeln $rf.
Det andra kommandot får regel regeln med namnet Rule01 associerat med vägen.
Det tredje kommandot får en lista över regler för flödes filter som är kopplade till det väg filtret.
