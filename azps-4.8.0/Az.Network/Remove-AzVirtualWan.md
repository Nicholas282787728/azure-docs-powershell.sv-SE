# Remove-AzVirtualWan

## Sammanfattning
Tar bort en Azure Virtual WAN.

## FRÅGESYNTAXEN

### ByVirtualWanName (standard)
```
Remove-AzVirtualWan -ResourceGroupName <String> -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanObject
```
Remove-AzVirtualWan -InputObject <PSVirtualWan> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanResourceId
```
Remove-AzVirtualWan -ResourceId <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort en Azure Virtual WAN.

## BESKRIVS

### Exempel 1

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Passthru
```

I det här exemplet skapas ett virtuellt WAN i en resurs grupp och sedan tas det omedelbart bort. Om du inte vill att meddelandet ska visas när det virtuella WANet tas bort använder du flaggan-Force.

### Exempel 2

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -InputObject $virtualWan -Passthru
```

I det här exemplet skapas ett virtuellt WAN i en resurs grupp och sedan tas det omedelbart bort. Denna borttagning sker med det virtuella WAN-objekt som returneras av New-AzVirtualWan.
Om du inte vill att meddelandet ska visas när det virtuella WANet tas bort använder du flaggan-Force.

### Exempel 3

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -ResourceId $virtualWan.Id -Passthru
```

I det här exemplet skapas ett virtuellt WAN i en resurs grupp och sedan tas det omedelbart bort. Den här borttagningen görs med den virtuella WAN Resource-ID som returneras av New-AzVirtualWan.
Om du inte vill att meddelandet ska visas när det virtuella WANet tas bort använder du flaggan-Force.
