# New-AzFirewallApplicationRule

## Sammanfattning
Skapar en regel för brand Väggs program.

## FRÅGESYNTAXEN

### TargetFqdn (standard)
```
New-AzFirewallApplicationRule -Name <String> [-Description <String>] [-SourceAddress <String[]>]
 [-SourceIpGroup <String[]>] -TargetFqdn <String[]> -Protocol <String[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### FqdnTag
```
New-AzFirewallApplicationRule -Name <String> [-Description <String>] [-SourceAddress <String[]>]
 [-SourceIpGroup <String[]>] -FqdnTag <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzFirewallApplicationRule** skapar en program regel för Azure Firewall.

## BESKRIVS

### Exempel 1: skapa en regel för att tillåta all HTTPS-trafik från 10.0.0.0
```powershell
New-AzFirewallApplicationRule -Name "https-rule" -Protocol "https:443" -TargetFqdn "*" -SourceAddress "10.0.0.0"
```

I det här exemplet skapas en regel som tillåter all HTTPS-trafik på port 443 från 10.0.0.0.

### Exempel 2: skapa en regel för att tillåta WindowsUpdate för 10.0.0.0/24 under nätet
```powershell
New-AzFirewallApplicationRule -Name "windows-update-rule" -FqdnTag WindowsUpdate -SourceAddress "10.0.0.0/24"
```

I det här exemplet skapas en regel som tillåter trafik för Windows-uppdateringar för 10.0.0.0/24-domän.
