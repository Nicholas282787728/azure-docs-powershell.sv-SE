# New-AzApplicationGatewayRewriteRuleCondition

## Sammanfattning
Lägger till ett villkor i RewriteRule för en Programgateway.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayRewriteRuleCondition -Variable <String> [-Pattern <String>] [-IgnoreCase] [-Negate]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**AzApplicationGatewayRewriteRuleCondition** -cmdleten skapar ett villkor för att skapa en regel för en Azure Application Gateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayRewriteRuleCondition -Variable "var_request_uri" -Pattern "http" -IgnoreCase
PS C:\> $condition

Variable   : var_request_uri
Pattern    : http
IgnoreCase : True
Negate     : False

PS C:\> $condition | Format-Table

Variable        Pattern IgnoreCase Negate
--------        ------- ---------- ------
var_request_uri http          True  False
```
Det här kommandot skapar ett villkor i en återskrivning-regel och lagrar resultatet i variabeln som heter $condition.
