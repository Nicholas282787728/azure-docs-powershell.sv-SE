# Remove-AzPrivateEndpoint

## Sammanfattning
Tar bort en privat slut punkt.

## FRÅGESYNTAXEN

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzPrivateEndpoint** tar bort en privat slut punkt. 

## BESKRIVS

### Exempel 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

Det här exemplet tar bort en privat slut punkt med namnet MyPrivateEndpoint1.
