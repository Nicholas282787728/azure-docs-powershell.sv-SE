# Set-AzRouteTable

## Sammanfattning
Uppdaterar en routningstabell.

## FRÅGESYNTAXEN

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzRouteTable** uppdaterar en routningstabell.

## BESKRIVS

### Exempel 2: ändra routningstabellen

```
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
False
PS C:\> $rt.DisableBgpRoutePropagation = $true
PS C:\> Set-AzRouteTable -RouteTable $rt
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
True
```

Med det första kommandot hämtas routningstabellen med namnet rtName och lagras i variabeln $rt.
Det andra kommandot visar värdet för DisableBgpRoutePropagation.
Det tredje kommandot uppdaterar värdet för DisableBgpRoutePropagation.
Det fjärde kommandot uppdaterar routningstabellen på servern.
Det femte kommandot uppdaterar väg tabellen och lagrar den i $rt variabeln.
Det sjätte kommandot visar värdet för DisableBgpRoutePropagation.
