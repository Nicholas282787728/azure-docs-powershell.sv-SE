# Test-AzDnsAvailability

## Sammanfattning
Kontrollerar om ett domän namn i cloudapp.azure.com-zonen är tillgängligt för användning.

## FRÅGESYNTAXEN

```
Test-AzDnsAvailability -DomainNameLabel <String> -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Kontrollerar om ett domän namn i cloudapp.azure.com-zonen är tillgängligt för användning.

## BESKRIVS

### Exempel 1: kontrol lera om contoso.westus.cloudapp.azure.com är tillgängligt för användning.
```
Test-AzDnsAvailability -DomainNameLabel contoso -Location westus
```
