# New-AzO365PolicyProperty

## Sammanfattning
Skapa ett princip objekt för Office 365-översikten.

## FRÅGESYNTAXEN

```
New-AzO365PolicyProperty [-Allow] [-Optimize] [-Default] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapa en princip för Office 365-översikten som ska användas med New-AzVpnSite och Update-AzVpnSite cmdletar.
## BESKRIVS

### Exempel 1
```powershell
PS C:\> $policy = New-AzO365PolicyProperty -Allow -Optimize
```

Skapa en policy för Office 365-översikten med översikten som tillåts för att tillåta och optimera trafik kategorin.
