# New-AzFirewallPublicIpAddress

## Sammanfattning
Det här är plats hållaren för den IP-adress som kan användas för multi-pip på Azure Firewall.

## FRÅGESYNTAXEN

```
New-AzFirewallPublicIpAddress [-Address <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Det här är plats hållaren för den IP-adress som kan användas för multi-pip på Azure Firewall.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $publicIp = New-AzFirewallPublicIpAddress -Address 20.2.3.4
```

$publicIp är plats hållaren för IP-adressen 20.2.3.4
