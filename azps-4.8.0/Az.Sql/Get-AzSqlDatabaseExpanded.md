# Get-AzSqlDatabaseExpanded

## Sammanfattning
Hämtar en databas och dess utökade egenskaps värden.

## FRÅGESYNTAXEN

```
Get-AzSqlDatabaseExpanded [-ServerName] <String> [[-DatabaseName] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlDatabaseExpanded** hämtar en databas och dess utökade egenskaps värden.
Denna cmdlet stöds också av SQL Server utsträckt databas tjänst på Azure.

## BESKRIVS

### Exempel 1: Hämta databas objekt med information om tjänst nivå klassificering
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot returnerar den databas som har utökade egenskaper som innehåller information om tjänst nivå klassificering.

### Exempel 2: lista databas objekt med hjälp av filtrering
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database*"
```

Det här kommandot returnerar utökade databasobjektet för alla databaser i Server01 som börjar med "databas".
