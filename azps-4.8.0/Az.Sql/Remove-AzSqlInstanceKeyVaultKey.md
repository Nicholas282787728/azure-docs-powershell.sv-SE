# Remove-AzSqlInstanceKeyVaultKey

## Sammanfattning
Tar bort en Key Vault-tangenten från en SQL-hanterad instans

## FRÅGESYNTAXEN

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (standard)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Remove-AzSqlInstanceKeyVaultKey cmdlet tar bort Key Vault-tangenten från den angivna hanterade instansen. Observera att den SQL-hanterade instansens behörigheter till Key-valvet inte har ändrats. Använd set-AzKeyVaultAccessPolicy för att ändra behörigheter. Observera att denna cmdlet inte gör några ändringar i huvud valvet. Använd Remove-AzureKeyVaultKey om du vill ta bort en Key-valv.
