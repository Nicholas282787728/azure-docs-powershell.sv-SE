# Resume-AzSqlDatabase

## Sammanfattning
Återupptar en SQL Data Warehouse-databas.

## FRÅGESYNTAXEN

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Resume-AzSqlDatabase** återupptar en Azure SQL Data Warehouse-databas.

## BESKRIVS

### Exempel 1: återupptar en Azure SQL Data Warehouse-databas
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot återupptar en upplåst Azure SQL Data Warehouse-databas.
