# Get-AzSqlDatabaseGeoBackupPolicy

## Sammanfattning
Hämtar en databas princip för säkerhets kopiering av databasen.

## FRÅGESYNTAXEN

```
Get-AzSqlDatabaseGeoBackupPolicy [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlDatabaseGeoBackupPolicy** hämtar den geo säkerhets kopierings princip som registrerats för den här databasen.
Det här är en Azure Backup-resurs som används för att definiera säkerhets kopierings lagrings princip.

## BESKRIVS

### Exempel 1

Hämtar en databas princip för säkerhets kopiering av databasen. (automatiskt genererat)

```powershell
<!-- Aladdin Generated Example --> 
Get-AzSqlDatabaseGeoBackupPolicy -DatabaseName db1 -ResourceGroupName MyResourceGroup -ServerName s1
```
