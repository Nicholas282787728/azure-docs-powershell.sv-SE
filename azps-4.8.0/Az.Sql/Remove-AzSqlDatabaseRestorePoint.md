# Remove-AzSqlDatabaseRestorePoint

## Sammanfattning
Tar bort en given återställnings punkt från en SQL-databas.

## FRÅGESYNTAXEN

```
Remove-AzSqlDatabaseRestorePoint -RestorePointCreationDate <DateTime> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzSqlDatabaseRestorePoint** tar bort en återställnings punkt från en Azure SQL-databas.
Denna cmdlet stöds för närvarande av SQL Server DataWarehouse-tjänsten i Azure SQL-databasen.

## BESKRIVS

### Exempel 1: tar bort en återställnings punkt
```
PS C:\>$RestorePointCreationDate = Get-Date "3/11/2017 1:50:00 AM"
PS C:\>Remove-AzSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -RestorePointCreationDate $RestorePointCreationDate
```

Det här kommandot tar bort en återställnings punkt för en Azure SQL-databas skapad.
