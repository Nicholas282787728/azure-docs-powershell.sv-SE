# Get-AzPeeringServiceCountry

## Sammanfattning
Visar de länder som är tillgängliga för peering service.

## FRÅGESYNTAXEN

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Visar de länder som är tillgängliga för peering service.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Lista över tillgängliga länder för peering service.
