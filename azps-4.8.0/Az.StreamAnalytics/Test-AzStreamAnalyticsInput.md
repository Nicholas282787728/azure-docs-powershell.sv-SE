# Test-AzStreamAnalyticsInput

## Sammanfattning
Testar anslutnings statusen för en inmatning.

## FRÅGESYNTAXEN

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Test-AzStreamAnalyticsInput-** cmdleten testar möjligheten att strömma analyser för att ansluta till ett indata.

## BESKRIVS

### Exempel 1: testa anslutnings statusen för en indataströmmen
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Detta testar anslutnings statusen för EntryStream i StreamingJob.
