# Test-AzStreamAnalyticsFunction

## Sammanfattning
Testar om Stream Analytics kan ansluta till en funktion.

## FRÅGESYNTAXEN

```
Test-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Test-AzStreamAnalyticsFunction** cmdlet testar om Azure Stream Analytics kan ansluta till en funktion.

## BESKRIVS

### Exempel 1: testa en Stream Analytics-funktion
```
PS C:\>Test-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Det här kommandot testar anslutnings statusen för funktionen som heter ScoreTweet i jobbet med namnet StreamJob22.
