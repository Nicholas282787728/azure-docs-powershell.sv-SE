# Remove-AzAppConfigurationStore

## Sammanfattning
Tar bort en konfigurations lagring.

## FRÅGESYNTAXEN

### Ta bort (standard)
```
Remove-AzAppConfigurationStore -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzAppConfigurationStore -InputObject <IAppConfigurationIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort en konfigurations lagring.

## BESKRIVS

### Exempel 1: ta bort ett program konfigurations Arkiv
```powershell
PS C:\> Remove-AzAppConfigurationStore -Name appconfig-test03 -ResourceGroupName lucas-manual-test

```

Det här kommandot tar bort ett program konfigurations arkiv.

### Exempel 2: ta bort ett program konfigurations Arkiv
```powershell
PS C:\> Get-AzAppConfigurationStore -Name appconfig-test02 -ResourceGroupName lucas-manual-test | Remove-AzAppConfigurationStore

```

Det här kommandot tar bort ett program konfigurations arkiv.
