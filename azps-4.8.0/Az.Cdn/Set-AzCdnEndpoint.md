# Set-AzCdnEndpoint

## Sammanfattning
Uppdaterar en CDN-slutpunkt.

## FRÅGESYNTAXEN

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzCdnEndpoint** uppdaterar en CDN-slutpunkt (Azure Content Delivery Network).
