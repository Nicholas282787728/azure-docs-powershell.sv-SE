# Get-AzWebAppAccessRestrictionConfig

## Sammanfattning
Hämtar Restiction-konfiguration för ett Azure Web App.

## FRÅGESYNTAXEN

```
Get-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String> [[-SlotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzWebAppAccessRestrictionConfig** får åtkomst begränsnings konfiguration för Azure Web App.

## BESKRIVS

### Exempel 1: Hämta en begränsning för åtkomst begränsningar för webb program från en resurs grupp
```
PS C:\>Get-AzWebAppAccessRestrictionConfig -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Det här kommandot får åtkomst begränsningen config för ett webb program som heter ContosoSite som tillhör resurs gruppens standard-väst.
