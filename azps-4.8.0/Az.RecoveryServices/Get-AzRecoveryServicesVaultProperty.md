# Get-AzRecoveryServicesVaultProperty

## Sammanfattning
Returnerar egenskaperna för ett Recovery Services-valv.

## FRÅGESYNTAXEN

```
Get-AzRecoveryServicesVaultProperty [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzRecoveryServicesVaultProperty** returnerar egenskaperna för ett Recovery Services-valv.

## BESKRIVS

### Exempel 1: Hämta egenskaper för ett valv
```
PS C:\> $vault = Get-AzRecoveryServicesVault -Name "MyVaultName"
PS C:\> $props = Get-AzRecoveryServicesVaultProperty -VaultId $vault.Id
```

Det första kommandot får ett valv objekt och lagrar det sedan i $vault variabel.
Det andra kommandot får egenskaperna för valvet.
