# Export-AzDataLakeStoreChildItemProperty

## Sammanfattning
Exporterar egenskaperna (disk användning och ACL) för hela trädet från den angivna sökvägen till en utgångs Sök väg

## FRÅGESYNTAXEN

```
 [-OutputPath] <String> [-SaveToAdl] [-IncludeFile] [-MaximumDepth <Int32>] [-Concurrency <Int32>]
 [-GetDiskUsage] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Exportera-AzDataLakeStoreChildItemProperty** används för att rapportera ADLS för användning eller/och ACL för den angivna katalogen och under kataloger och filer.

## BESKRIVS

### Exempel 1: få disk användning och ACL-användning för alla under kataloger och filer
```
PS C:\> Export-AzDataLakeStoreChildItemProperty -Account ContosoADL -Path /a -OutputPath "C:\Users\contoso\Desktop\DumpFile.txt" -GetAcl -GetDiskUsage -IncludeFile
```

Få disk användning och ACL-användning för alla under kataloger och filer under/a. IncludeFile säkerställer att användningen rapporteras för filer också

### Exempel 2: Hämta ACL-användning för alla under kataloger och filer med den enhetliga åtkomst kontrol listan dold
```
PS C:\> $fullAcl="user:contoso-userid:--x|user::rwx|other::---|group::rwx"
PS C:\> $newFullAcl = $fullAcl.Split("{|}");
PS C:\> Set-AzDataLakeStoreItemAcl -Account ContosoADL -Path /a -Acl $newFullAcl -Recurse -Debug

PS C:\> Export-AzDataLakeStoreChildItemProperty -Account ContosoADL -Path /a -OutputPath "C:\Users\contoso\Desktop\DumpFile.txt" -GetAcl -HideConsistentAcl -IncludeFile
```

Hämta ACL-användning för alla under kataloger och filer under/a. IncludeFile säkerställer att användningen rapporteras för filer också. HideconsistentAcl i det här fallet visas ACL för/a, inte till barn eftersom alla underordnade har samma ACL som/a. Den här flaggan hoppar över under trädets ACL-utdata om alla dess åtkomst kontrol listor är samma som roten.
