# Remove-AzDataLakeStoreVirtualNetworkRule

## Sammanfattning
Tar bort den angivna regeln för virtuella nätverk i den angivna data Lake Store.

## FRÅGESYNTAXEN

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzDataLakeStoreVirtualNetworkRule** tar bort den angivna regeln för virtuella nätverk i den angivna data Lake Store.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Tar bort den virtuella nätverks regeln "myVNET" från kontot "DLS"
