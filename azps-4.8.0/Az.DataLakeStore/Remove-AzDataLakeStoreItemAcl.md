# Remove-AzDataLakeStoreItemAcl

## Sammanfattning
Rensar en ACL för en fil eller mapp i data Lake Store.

## FRÅGESYNTAXEN

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzDataLakeStoreItemAcl** rensar åtkomst kontrol listan (ACL) för en fil eller mapp i data Lake Store.

## BESKRIVS

### Exempel 1: ta bort en ACL från en mapp
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Det här kommandot tar bort ACL för rot katalogen för ContosoADL-kontot.
