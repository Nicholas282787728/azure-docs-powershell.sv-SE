# New-AzFrontDoorWafManagedRuleOverrideObject

## Sammanfattning
Skapa åsidosättning av hanterade regler

## FRÅGESYNTAXEN

```
New-AzFrontDoorWafManagedRuleOverrideObject -RuleId <String> [-Action <String>] [-Disabled]
 [-Exclusion <PSManagedRuleExclusion[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapa PSAzureManagedRuleOverride-objekt för regel gruppen hanterade WAF åsidosätta objekt.

## BESKRIVS

### Exempel 1
Skapa ett objekt för åsidosättning av hanterade regler för regel 942250 (finns i gruppen SQLI).

```powershell
PS C:\> New-AzFrontDoorWafManagedRuleOverrideObject -RuleId "942250" -Action Log

RuleId EnabledState Action
------ ------------ ------
942250      Enabled    Log
```
