# Get-AzIotSecurityAnalytics

## Sammanfattning
Skaffa IoT Security Analytics

## FRÅGESYNTAXEN

```
Get-AzIotSecurityAnalytics -ResourceGroupName <String> -SolutionName <String> [-Default]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Get-AzIotSecurityAnalytics cmdlet returnerar en uppsättning säkerhets analyser för en specifik IoT-säkerhetslösning
