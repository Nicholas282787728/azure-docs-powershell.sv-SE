# Reset-AzRedisCache

## Sammanfattning
Startar om noder i ett cacheminne.

## FRÅGESYNTAXEN

```
Reset-AzRedisCache [-ResourceGroupName <String>] -Name <String> -RebootType <String> [-ShardId <Int32>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Reset-AzRedisCache** startar om noder för en Azure Redis-cachepost.

## BESKRIVS

### Exempel 1: starta om båda noderna
```
PS C:\>Reset-AzRedisCache -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -RebootType "AllNodes" -Force
```

Med det här kommandot startas båda noderna för cacheminnet som heter RedisCache06.
