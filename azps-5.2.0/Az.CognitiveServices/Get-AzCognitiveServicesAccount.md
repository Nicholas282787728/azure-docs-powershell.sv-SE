# Get-AzCognitiveServicesAccount

## Sammanfattning
Hämtar ett konto.

## FRÅGESYNTAXEN

### ResourceGroupParameterSet
```
Get-AzCognitiveServicesAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzCognitiveServicesAccount** hämtar de etablerade kontona för kognitiva tjänster i resurs gruppen som anges av parametern *ResourceGroupName* .
Om du inte anger parametern *ResourceGroupName* hämtar denna cmdlet alla kognitiva tjänst konton för den aktuella prenumerationen.
