# Connect-AzContainerRegistry

## Sammanfattning
Logga in på en Azure-behållarobjekt.

## FRÅGESYNTAXEN

### WithoutNameAndPasswordParameterSet (standard)
```
Connect-AzContainerRegistry -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### WithNameAndPasswordParameterSet
```
Connect-AzContainerRegistry -Name <String> -UserName <String> -Password <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Logga in på en Azure-behållarobjekt.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName
```

Logga in på ACR utan autentiseringsuppgifter när du redan har loggat in på Azure-kontot.

### Exempel 2
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $RegistryName -Password $AdminPassWord
```

Logga in på ACR med administratörs användar namn/lösen ord när administratörs användaren aktiverade.

### Exempel 3
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $ServicePrincipal -Password $ServicePrincipalPassword
```

Logga in på ACR med ID och lösen ord för tjänstens huvud program.
