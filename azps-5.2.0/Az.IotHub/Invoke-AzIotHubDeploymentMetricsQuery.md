# Invoke-AzIotHubDeploymentMetricsQuery

## Sammanfattning
Kör en IoT Edge Deployment Metric-fråga.

## FRÅGESYNTAXEN

### ResourceSet (standard)
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Utvärdera en anpassad mål-eller system mått definition i en IoT Edge-distribution.
Det finns fördefinierade system värden som beräknas av IoT Hub och inte kan anpassas.
- "Riktat" visar vilka IoT-enheter som matchar distributions villkoren.
- "Använt" visar de riktade enheter för IoT som inte är riktade till en annan distribution av högre prioritet.
- "Rapporterar framgång" visar vilka IoT-enheter som har rapporterat att modulerna har distribuerats.
- "Rapporterar fel" visar vilka IoT-enheter som har rapporterat att en eller flera moduler inte har distribuerats. 
  Om du vill undersöka felet kan du fjärrans luta till dessa enheter och Visa loggfilerna.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Invoke-AzIotHubDeploymentMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "warningLimit"
```

Utvärdera det anpassade måttet ' warningLimit '.

### Exempel 2
```powershell
PS C:\> Invoke-AzIotHubDeployMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "Reporting Success" -MetricType "system"
```

Utvärdera om systemet rapporterar framgång.
