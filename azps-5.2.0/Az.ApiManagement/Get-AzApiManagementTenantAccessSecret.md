# Get-AzApiManagementTenantAccessSecret

## Sammanfattning
Hämtar konfigurations nycklarna för åtkomst till en klient organisation.

## FRÅGESYNTAXEN

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämtar konfigurations nycklarna för åtkomst till en klient organisation.

## BESKRIVS

### Exempel 1: hämta konfigurations nycklar för klient åtkomst
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

Det här kommandot får konfigurations nycklarna för klient åtkomst för angiven kontext.
