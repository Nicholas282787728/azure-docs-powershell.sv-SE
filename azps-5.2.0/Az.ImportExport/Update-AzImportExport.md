# Update-AzImportExport

## Sammanfattning
Uppdaterar specifika egenskaper för ett jobb.
Du kan ringa den här åtgärden för att meddela import-och export tjänsten att hård diskarna som innehåller import-eller export jobbet har levererats till Microsoft Data Center.
Den kan också användas för att avbryta ett befintligt jobb.

## FRÅGESYNTAXEN

### UpdateExpanded (standard)
```
Update-AzImportExport -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-AcceptLanguage <String>] [-BackupDriveManifest] [-CancelRequested] [-DeliveryPackageCarrierName <String>]
 [-DeliveryPackageDriveCount <Int32>] [-DeliveryPackageShipDate <String>]
 [-DeliveryPackageTrackingNumber <String>] [-DriveList <IDriveStatus[]>] [-LogLevel <String>]
 [-ReturnAddressCity <String>] [-ReturnAddressCountryOrRegion <String>] [-ReturnAddressEmail <String>]
 [-ReturnAddressPhone <String>] [-ReturnAddressPostalCode <String>] [-ReturnAddressRecipientName <String>]
 [-ReturnAddressStateOrProvince <String>] [-ReturnAddressStreetAddress1 <String>]
 [-ReturnAddressStreetAddress2 <String>] [-ReturnShippingCarrierAccountNumber <String>]
 [-ReturnShippingCarrierName <String>] [-State <String>] [-Tag <IUpdateJobParametersTags>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzImportExport -InputObject <IImportExportIdentity> [-AcceptLanguage <String>] [-BackupDriveManifest]
 [-CancelRequested] [-DeliveryPackageCarrierName <String>] [-DeliveryPackageDriveCount <Int32>]
 [-DeliveryPackageShipDate <String>] [-DeliveryPackageTrackingNumber <String>] [-DriveList <IDriveStatus[]>]
 [-LogLevel <String>] [-ReturnAddressCity <String>] [-ReturnAddressCountryOrRegion <String>]
 [-ReturnAddressEmail <String>] [-ReturnAddressPhone <String>] [-ReturnAddressPostalCode <String>]
 [-ReturnAddressRecipientName <String>] [-ReturnAddressStateOrProvince <String>]
 [-ReturnAddressStreetAddress1 <String>] [-ReturnAddressStreetAddress2 <String>]
 [-ReturnShippingCarrierAccountNumber <String>] [-ReturnShippingCarrierName <String>] [-State <String>]
 [-Tag <IUpdateJobParametersTags>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Uppdaterar specifika egenskaper för ett jobb.
Du kan ringa den här åtgärden för att meddela import-och export tjänsten att hård diskarna som innehåller import-eller export jobbet har levererats till Microsoft Data Center.
Den kan också användas för att avbryta ett befintligt jobb.

## BESKRIVS

### Exempel 1: uppdatera ImportExport-jobbet efter resurs grupp och Server namn
```powershell
PS C:\> Update-AzImportExport -Name test-job -ResourceGroupName ImportTestRG -DeliveryPackageCarrierName pwsh -DeliveryPackageTrackingNumber pwsh20200000
Location Name     Type
-------- ----     ----
East US  test-job Microsoft.ImportExport/jobs
```

Denna cmdlet uppdaterar ImportExport-jobbet efter resurs grupp och Server namn.

### Exempel 2: uppdatera ImportExport-jobbet efter identitet.
```powershell
PS C:\> Get-AzImportExport -Name test-job -ResourceGroupName ImportTestRG | Update-AzImportExport -CancelRequested
Location Name     Type
-------- ----     ----
East US  test-job Microsoft.ImportExport/jobs
```

Denna cmdlet uppdaterar ImportExport-jobbet efter identitet.

## ANMÄRKNINGAR

ALIAS

KOMPLEXA PARAMETER EGENSKAPER

Skapa en hash-tabell som innehåller lämpliga egenskaper för att lägga till de parametrar som beskrivs nedan. Om du vill ha information om hash-tabeller kör Get-Help about_Hash_Tables.


DRIVELIST <IDriveStatus [] >: lista med enheter som utgör jobbet.
  - `[BitLockerKey <String>]`: Den BitLocker-nyckel som används för att kryptera enheten.
  - `[BytesSucceeded <Int64?>]`: Byte har överförts för enheten.
  - `[CopyStatus <String>]`: Detaljerad status om data överförings processen. Det här fältet returneras inte i svaret förrän enheten är i överförings läget.
  - `[DriveHeaderHash <String>]`: Värdet på enhetens huvud.
  - `[DriveId <String>]`: Enhetens maskin varu serie nummer utan blank steg.
  - `[ErrorLogUri <String>]`: En URI som pekar på blobben som innehåller fel loggen för data överförings åtgärden.
  - `[ManifestFile <String>]`: Den relativa sökvägen för manifest filen på enheten. 
  - `[ManifestHash <String>]`: Den Base16iska MD5-hashen för manifest filen på enheten.
  - `[ManifestUri <String>]`: En URI som pekar på blobben som innehåller enhetens manifest fil. 
  - `[PercentComplete <Int32?>]`: Procent färdigt för enheten. 
  - `[State <DriveState?>]`: Enhetens aktuella status. 
  - `[VerboseLogUri <String>]`: En URI som pekar på blobben som innehåller utförlig loggen för data överförings åtgärden. 

INPUTOBJECT <IImportExportIdentity> : identitets parameter
  - `[Id <String>]`: Sökväg till resurs identitet
  - `[JobName <String>]`: Import-och export jobbets namn.
  - `[LocationName <String>]`: Namnet på platsen. Till exempel West, USA eller västkusten.
  - `[ResourceGroupName <String>]`: Resurs grupp namnet identifierar unikt resurs gruppen i användar abonnemanget.
  - `[SubscriptionId <String>]`: Prenumerations-ID för Azure-användaren.
