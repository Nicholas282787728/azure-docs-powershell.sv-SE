# Remove-AzImportExport

## Sammanfattning
Tar bort ett befintligt jobb.
Det är bara jobb i tillståndet skapa eller slutfört som kan tas bort.

## FRÅGESYNTAXEN

### Ta bort (standard)
```
Remove-AzImportExport -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-AcceptLanguage <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzImportExport -InputObject <IImportExportIdentity> [-AcceptLanguage <String>]
 [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort ett befintligt jobb.
Det är bara jobb i tillståndet skapa eller slutfört som kan tas bort.

## BESKRIVS

### Exempel 1: ta bort ImportExport-jobbet genom resourceGroup och Server namn
```powershell
PS C:\> Remove-AzImportExport -Name test-job -ResourceGroupName ImportTestRG
```

Denna cmdlet tar bort ImportExport-jobbet genom resourceGroup och Server namn.

### Exempel 2: ta bort ImportExport-jobbet efter identitet
```powershell
PS C:\> Get-AzImportExport -Name test-job -ResourceGroupName ImportTestRG | Remove-AzImportExport
 
```

Denna cmdlet tar bort ImportExport-jobbet efter identitet.
