# Suspend-AzAutomationJob

## Sammanfattning
Inaktiverar ett automatiserings jobb.

## FRÅGESYNTAXEN

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **suspend-AzAutomationJob** inaktive ras ett Azure Automation-jobb.
Ange ett automatiserings jobb som körs.
Om du vill återuppta ett avbrutet jobb kan du använda Resume-AzAutomationJob cmdlet.

## BESKRIVS

### Exempel 1: pausa ett jobb
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Det här kommandot avaktiverar det jobb som har angivet ID.
