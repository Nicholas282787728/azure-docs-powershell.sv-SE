# Get-AzAutoscaleSetting

## Sammanfattning
Hämtar inställningar för Autoskala.

## FRÅGESYNTAXEN

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzAutoscaleSetting** hämtar alla inställningar för Autoskala som är kopplade till en resurs grupp eller en viss autoskalningsinställning.
