# Get-AzLogProfile

## Sammanfattning
Hämtar en logg profil.

## FRÅGESYNTAXEN

```
Get-AzLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzLogProfile** hämtar en logg profil.
