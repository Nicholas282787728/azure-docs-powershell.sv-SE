# Remove-AzDataLakeStoreAccount

## Sammanfattning
Tar bort ett data Lake Store-konto permanent.

## FRÅGESYNTAXEN

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzDataLakeStoreAccount** tar bort ett data Lake Store-konto permanent.

## BESKRIVS

### Exempel 1: ta bort ett data Lake Store-konto
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Det här kommandot tar bort kontot som heter ContosoADL från data Lake Store.
