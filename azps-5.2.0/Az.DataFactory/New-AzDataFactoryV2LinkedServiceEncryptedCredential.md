# New-AzDataFactoryV2LinkedServiceEncryptedCredential

## Sammanfattning
Kryptera Credential i den länkade tjänsten med den angivna integrerings körningen.

## FRÅGESYNTAXEN

### ByFactoryName (standard)
```
New-AzDataFactoryV2LinkedServiceEncryptedCredential [-IntegrationRuntimeName] <String>
 [-DefinitionFile] <String> [-Force] [-ResourceGroupName] <String> [-DataFactoryName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryV2LinkedServiceEncryptedCredential [-IntegrationRuntimeName] <String>
 [-DefinitionFile] <String> [-Force] [-DataFactory] <PSDataFactory> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
New-AzDataFactoryV2LinkedServiceEncryptedCredential cmdlet krypterar autentiseringsuppgiften i den länkade tjänsten med den angivna integrerings körnings miljön.

Se till att följande förutsättningar uppfylls:
* Alternativet **fjärråtkomst** är aktiverat i den självvärdbaserade integrerings körningen.
* PowerShell 7,0 eller senare används för att köra cmdleten.

## BESKRIVS

### Exempel 1: kryptera autentiseringsuppgifter i en länkad tjänst

Det här kommandot krypterar autentiseringsuppgifter i filen C:\samples\WikiSample\TaxiDemo1.jspå med integrerings körningen tested-selfhost-IR.

```powershell <!-- Aladdin Generated Example --> 
New-AzDataFactoryV2LinkedServiceEncryptedCredential -DataFactoryName WikiADF -DefinitionFile 'C:\samples\WikiSample\TaxiDemo1.json' -IntegrationRuntimeName 'test-selfhost-ir' -ResourceGroupName MyResourceGroup
```
