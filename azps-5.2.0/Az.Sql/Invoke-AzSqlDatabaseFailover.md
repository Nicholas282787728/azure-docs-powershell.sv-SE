# Invoke-AzSqlDatabaseFailover

## Sammanfattning
Redundans för en databas.

## FRÅGESYNTAXEN

```
Invoke-AzSqlDatabaseFailover [-DatabaseName] <String> [-ReadableSecondary] [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Invoke-AzSqlDatabaseFailover cmdlet redundansväxlingen en Azure SQL-databas. Om databasen är i en elastisk pool kommer det här kommandot att redundansväxla den angivna databasen utan att det påverkar de andra databaserna i samma elastiska pool.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Invoke-AzSqlDatabaseFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot kommer att redundansväxla den primära repliken av databasen med namnet "Database01" på servern med namnet "Server01"

### Exempel 2
```powershell
PS C:\> Invoke-AzSqlDatabaseFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -ReadableSecondary
```

Det här kommandot kommer att redundansväxla den läsbara sekundära repliken av databasen med namnet "Database01" på servern med namnet "Server01"
