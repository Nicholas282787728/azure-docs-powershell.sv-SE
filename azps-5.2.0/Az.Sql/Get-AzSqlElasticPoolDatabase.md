# Get-AzSqlElasticPoolDatabase

## Sammanfattning
Hämtar Elastic databases i en elastisk pool och deras egenskaps värden.

## FRÅGESYNTAXEN

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlElasticPoolDatabase** får Elastic databases i en elastisk pool och deras egenskaps värden.
Du kan ange namnet på en elastisk databas i Azure SQL Database för att Visa egenskaps värden för enbart den databasen.

## BESKRIVS

### Exempel 1: Hämta alla databaser i en elastisk pool
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Det här kommandot får alla databaser i en elastisk pool med namnet ElasticPool01.

### Exempel 2: Hämta alla databaser i en elastisk pool med filter
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

Det här kommandot får alla databaser i en elastisk pool med namnet ElasticPool01 som börjar med "databas".
