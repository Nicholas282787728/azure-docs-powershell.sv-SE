# Set-AzSqlDatabaseGeoBackupPolicy

## Sammanfattning
Ställer in en databas princip för säkerhets kopiering av databasen.

## FRÅGESYNTAXEN

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzSqlDatabaseGeoBackupPolicy** anger den geo säkerhets kopierings princip som registrerats för en databas.
Det här är en Azure Backup-resurs som används för att definiera säkerhets kopierings lagrings princip.
