# Disable-AzSqlInstanceAdvancedDataSecurity

## Sammanfattning
Inaktiverar avancerad data säkerhet för en hanterad instans.

## FRÅGESYNTAXEN

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **disable-AzSqlInstanceAdvancedDataSecurity** inaktiverar avancerad data säkerhet för en hanterad instans.

## BESKRIVS

### Exempel 1 – inaktivera den hanterade instansens avancerade data säkerhet
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Exempel 2 – inaktivera avancerad data säkerhet för servrar från hanterad instans resurs
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
