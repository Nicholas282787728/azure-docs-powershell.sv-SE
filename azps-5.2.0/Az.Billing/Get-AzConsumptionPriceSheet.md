# Get-AzConsumptionPriceSheet

## Sammanfattning
Skaffa pris rapporter för abonnemanget.

## FRÅGESYNTAXEN

```
Get-AzConsumptionPriceSheet [-BillingPeriodName <String>] [-ExpandMeterDetail] [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzConsumptionPriceSheet** får pris listor för abonnemanget.
