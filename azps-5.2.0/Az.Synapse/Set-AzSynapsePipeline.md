# Set-AzSynapsePipeline

## Sammanfattning
Skapar en pipeline på en arbets yta.

## FRÅGESYNTAXEN

### SetByName (standard)
```
Set-AzSynapsePipeline -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapsePipeline -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzSynapsePipeline** skapar en pipeline i arbets ytan.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Set-AzSynapsePipeline -WorkspaceName ContosoWorkspace -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Det här kommandot skapar en pipeline med namnet ContosoPipeline i arbets ytan med namnet ContosoWorkspace.
Kommandot baserar sig på informationen i pipeline.jsi filen.
Filen innehåller information om aktiviteter.

### Exempel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Set-AzSynapsePipeline -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Det här kommandot skapar en pipeline med namnet ContosoPipeline i arbets ytan med namnet ContosoWorkspace till pipeline.
Kommandot baserar sig på informationen i pipeline.jsi filen.
Filen innehåller information om aktiviteter.
