# Export-AzSynapseNotebook

## Sammanfattning
Exporterar notbooks.

## FRÅGESYNTAXEN

### ExportByName (standard)
```
Export-AzSynapseNotebook -WorkspaceName <String> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByObject
```
Export-AzSynapseNotebook -WorkspaceObject <PSSynapseWorkspace> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByInputObject
```
Export-AzSynapseNotebook -InputObject <PSNotebookResource> -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **export-AzSynapseNotebook** exporterar en Azure Synapse-anteckningsbok till en antecknings bok (. ipynb). Namnet på antecknings boken blir namnet på den exporterade filen. Om du anger namnet på en antecknings bok exporterar cmdleten till antecknings boken. Om du inte anger ett namn exporterar cmdleten alla antecknings böcker i arbets ytan.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -OutputFolder "C:\Notebook"
```

Exporterar alla antecknings böcker i ContosoWorkspace till mappen "C:\Notebook".

### Exempel 2
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Exporterar en enskild antecknings bok som heter ContosoNotebook i arbets ytan ContosoWorkspace till mappen "C:\Notebook".

### Exempel 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Export-AzSynapseNotebook -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Exporterar en enskild antecknings bok som heter ContosoNotebook i arbets ytan ContosoWorkspace till mappen "C:\Notebook" genom pipeline.

### Exempel 4
```powershell
PS C:\> $notebook = Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook
PS C:\> $notebook | Export-AzSynapseNotebook -OutputFolder "C:\Notebook"
```

Exporterar en enskild antecknings bok som heter ContosoNotebook i arbets ytan ContosoWorkspace till mappen "C:\Notebook" genom pipeline.
