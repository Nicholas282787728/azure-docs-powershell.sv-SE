# Set-AzSynapseLinkedService

## Sammanfattning
Länkar ett data lager eller en moln tjänst till arbets ytan.

## FRÅGESYNTAXEN

### SetByName (standard)
```
Set-AzSynapseLinkedService -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseLinkedService -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzSynapseLinkedService** länkar ett data lager eller en moln tjänst till arbets ytan.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Set-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService -DefinitionFile "C:\\samples\\LinkedService.json"
```

Det här kommandot skapar en länkad tjänst som heter ContosoLinkedService i arbets ytan med namnet ContosoWorkspace.

### Exempel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Set-AzSynapseLinkedService -Name ContosoLinkedService -DefinitionFile "C:\\samples\\LinkedService.json"
```

Det här kommandot skapar en länkad tjänst som heter ContosoLinkedService i arbets ytan ContosoWorkspace via pipeline.
