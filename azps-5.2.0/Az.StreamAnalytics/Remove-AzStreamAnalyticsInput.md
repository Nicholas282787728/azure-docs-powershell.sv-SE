# Remove-AzStreamAnalyticsInput

## Sammanfattning
Tar bort en inmatning från ett ström analys jobb.

## FRÅGESYNTAXEN

```
Remove-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzStreamAnalyticsInput** asynkront tar bort en inmatning från ett ström analys jobb i Azure.

## BESKRIVS

### EXEMPEL 1: ta bort en indataströmmen från ett jobb
```
PS C:\>Remove-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Det här kommandot tar bort indatavärdet EventStream från StreamingJob.
