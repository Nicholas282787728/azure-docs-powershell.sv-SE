## PROBLEMBESKRIVNING
Cmdleten **Get-AzTenantDeploymentWhatIfResult** får arm-mallen What-If resultat för en mall distribution för den angivna klient organisations omfattningen. Den returnerar en lista över ändringar som visar vilka resurser som kommer att uppdateras om distributionen tillämpas utan att du behöver göra några ändringar i verkliga resurser. Om du vill ange formatet för det resulterande resultatet kan du använda parametern *ResultFormat* .

## BESKRIVS

### Exempel 1: få ett What-If resultat på klient området
```powershell
PS C:\> Get-AzTenantDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "FullResourcePayloads"
```

Det här kommandot får en What-If resultat på klient området genom att använda en anpassad mallfil och en parameter fil på disken.
Kommandot använder parametern *location* för att ange var distributions data ska lagras.
Kommandot använder parametern *TemplateFile* för att ange en mallfil.
Kommandot använder parametern *TemplateParameterFile* för att ange en mallparameter.
Kommandot använder parametern *ResultFormat* för att ange What-If resultatet ska inkludera fullständiga resurs nytto laster.

### Exempel 2: få ett What-If resultat på klient omfattning med ResourceIdOnly
```powershell
PS C:\> Get-AzTenantDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "ResourceIdOnly"
```

Det här kommandot får en What-If resultat på klient området genom att använda en anpassad mallfil och en parameter fil på disken.
Kommandot använder parametern *location* för att ange var distributions data ska lagras.
Kommandot använder parametern *TemplateFile* för att ange en mallfil.
Kommandot använder parametern *TemplateParameterFile* för att ange en mallparameter.
Kommandot använder parametern *ResultFormat* för att ange What-If-resultatet som endast innehåller resurs-ID.
