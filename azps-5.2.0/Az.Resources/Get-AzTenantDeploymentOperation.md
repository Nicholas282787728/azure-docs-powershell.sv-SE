# Get-AzTenantDeploymentOperation

## Sammanfattning
Skaffa distributions åtgärd för distribution med klient-scope

## FRÅGESYNTAXEN

### GetByDeploymentName (standard)
```
Get-AzTenantDeploymentOperation -DeploymentName <String> [-OperationId <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByDeploymentObject
```
Get-AzTenantDeploymentOperation -DeploymentObject <PSDeployment> [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzTenantDeploymentOperation** innehåller alla åtgärder som ingick i distributionen i den aktuella klient omfattningen för att hjälpa dig att identifiera och ge mer information om de exakta operationer som misslyckats för en viss distribution.

## BESKRIVS

### Exempel 1: skaffa distributions åtgärder som fått ett distributions namn
```
PS C:\>Get-AzTenantDeploymentOperation -DeploymentName Deploy01
```

Hämtar distributions åtgärder med namnet "Deploy01" i det aktuella klient scopet.

### Exempel 2: skaffa en distribution och få drift sättnings åtgärder
```
PS C:\>Get-AzTenantDeployment -Name Deploy01 | Get-AzTenantDeploymentOperation
```

Det här kommandot får distributionen "Deploy01" vid den aktuella klient omfattningen och får drift sättnings åtgärder.
