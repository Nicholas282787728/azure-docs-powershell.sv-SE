# Remove-AzRelayNamespace

## Sammanfattning
Tar bort namn området från den angivna resurs gruppen. 

## FRÅGESYNTAXEN

```
Remove-AzRelayNamespace [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzRelayNamespace** tar bort namn området från den angivna resurs gruppen.

## BESKRIVS

### Exempel 1
```
PS C:\> Remove-AzRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -Name TestNameSpace-Relay1
```

Tar bort relä namnområdet `TestNameSpace-Relay1` från den angivna resurs gruppen `Default-ServiceBus-WestUS` .
