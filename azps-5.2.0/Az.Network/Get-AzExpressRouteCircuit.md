# Get-AzExpressRouteCircuit

## Sammanfattning
Hämtar en Azure ExpressRoute-krets från Azure.

## FRÅGESYNTAXEN

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzExpressRouteCircuit** används för att hämta ett ExpressRoute-kretskort från din prenumeration. Det kretsar som returneras kan användas som indata för andra cmdlets som fungerar på ExpressRoute-kretsar.
