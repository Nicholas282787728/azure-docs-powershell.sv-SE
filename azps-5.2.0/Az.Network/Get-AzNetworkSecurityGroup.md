# Get-AzNetworkSecurityGroup

## Sammanfattning
Hämtar en nätverks säkerhets grupp.

## FRÅGESYNTAXEN

### Noexpandering
```
Get-AzNetworkSecurityGroup [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Byggnad
```
Get-AzNetworkSecurityGroup -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzNetworkSecurityGroup** får en Azure Network Security-grupp.
