# Remove-AzApplicationGatewayPrivateLinkConfiguration

## Sammanfattning
Tar bort en privateLink-konfiguration från en Programgateway.

## FRÅGESYNTAXEN

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApplicationGatewayPrivateLinkConfiguration** tar bort en privateLink-konfiguration från en Azure Application Gateway.

## BESKRIVS

### Exempel 1: ta bort en Programgateway PrivateLink konfiguration
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

Det första kommandot får en Programgateway och lagrar den i $AppGw variabeln.
Det andra kommandot tar bort privateLink-konfigurationen som heter privateLinkConfig01 från Application Gateway som lagras i $AppGw.
