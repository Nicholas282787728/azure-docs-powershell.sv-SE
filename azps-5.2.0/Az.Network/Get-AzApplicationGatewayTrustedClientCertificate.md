# Get-AzApplicationGatewayTrustedClientCertificate

## Sammanfattning
Hämtar den betrodda klient certifikat utfärdarens certifikats kedja med ett visst namn från Application Gateway.

## FRÅGESYNTAXEN

```
Get-AzApplicationGatewayTrustedClientCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Get-AzApplicationGatewayTrustedClientCertificate-cmdleten hämtar kedjan med betrodda klient certifikat utfärdare med ett visst namn från Application Gateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClientCert = Get-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName
```

Med det första kommandot hämtas programgatewayen och lagras i $gw variabel. Det andra kommandot får den betrodda klient certifikat utfärdarens certifikat kedja med ett angivet namn från Application Gateway.
