# Remove-AzPrivateDnsZoneGroup

## Sammanfattning
Tar bort en DNS-zonfil.

## FRÅGESYNTAXEN

```
Remove-AzPrivateDnsZoneGroup -ResourceGroupName <String> -PrivateEndpointName <String> -Name <String> [-Force]
 [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzPrivateDnsZoneGroup** tar bort en DNS-zonfil. 

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzPrivateDnsZoneGroup -ResourceGroupName "rg" -PrivateEndpointName "test-pr-endpoint" -name dnsgroup1 
```

Ovan exemplet tar bort en DNS-zon grup med namnet dnsgroup1 from slut punkts test-PR-slut punkt.
