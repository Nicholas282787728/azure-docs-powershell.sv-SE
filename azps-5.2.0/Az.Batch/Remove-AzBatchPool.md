# Remove-AzBatchPool

## Sammanfattning
Tar bort angiven gruppbearbetningssekvens.

## FRÅGESYNTAXEN

```
Remove-AzBatchPool [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzBatchPool** tar bort angiven Azure-grupppool.
Du uppmanas att bekräfta om du inte använder parametern *Force* .

## BESKRIVS

### Exempel 1: ta bort en grupppool utifrån pool-ID
```
PS C:\>Remove-AzBatchPool -Id "MyPool" -BatchContext $Context
```

Det här kommandot tar bort poolen med ID-nummer.
Användaren uppmanas att bekräfta innan borttagningen utförs.

### Exempel 2: ta bort alla batchattribut per tvång
```
PS C:\>Get-AzBatchPool -BatchContext $Context | Remove-AzBatchPool -Force -BatchContext $Context
```

Det här kommandot tar bort alla batchattribut.
Eftersom *Force* -parametern finns ignoreras bekräftelse meddelandet.
