# AZ. modul
## Problembeskrivning
Modul för att skapa, avbryta, ta bort och hämta data i jobb

## Cmdletar för AZ.
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
Hämtar autentiseringsuppgifterna för data i ett visst jobb

### [Get-AzDataBoxJob](Get-AzDataBoxJob.md)
Hämtar information om datavyer

### [New-AzDataBoxJob](New-AzDataBoxJob.md)
Skapar ett nytt data-jobb med de angivna parametrarna

### [Remove-AzDataBoxJob](Remove-AzDataBoxJob.md)
Tar bort datamappen

### [Stopp-AzDataBoxJob](Stop-AzDataBoxJob.md)
Avbryter ett pågående data jobb om jobbet är avbrutet.
