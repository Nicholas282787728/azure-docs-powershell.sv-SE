# New-AzCosmosDBGremlinSpatialSpec

## Sammanfattning
Skapar ett nytt objekt av typen PSSpatialSpec. Det kan skickas som ett parameter värde för set-AzCosmosDBGremlinIndexingPolicy.

## FRÅGESYNTAXEN

```
New-AzCosmosDBGremlinSpatialSpec -Path <String> -Type <String[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar objekt som motsvarar Gremlin API-SpatialSpec.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> New-AzCosmosDBGremlinSpatialSpec -Path "/abc" -Type String
Path Types
---- -----
/abc {String}
```
