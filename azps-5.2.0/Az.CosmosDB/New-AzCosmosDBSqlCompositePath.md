# New-AzCosmosDBSqlCompositePath

## Sammanfattning
Skapar ett nytt objekt av typen PSCompositePath. Det kan skickas som ett parameter värde för set-AzCosmosDBSqlContainer.

## FRÅGESYNTAXEN

```
New-AzCosmosDBSqlCompositePath [-Path <String>] [-Order <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Objekt som motsvarar SQL API-CompositePath.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> New-AzCosmosDBSqlCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
