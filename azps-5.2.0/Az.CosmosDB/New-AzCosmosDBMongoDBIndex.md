# New-AzCosmosDBMongoDBIndex

## Sammanfattning
Skapar ett nytt CosmosDB MongoDB-index.

## FRÅGESYNTAXEN

```
New-AzCosmosDBMongoDBIndex [-TtlInSeconds <Int32>] [-Unique <Boolean>] [-Key <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**New-AzCosmosDBMongoDBIndex** skapar ett nytt CosmosDB MongoDB-index.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> New-AzCosmosDBMongoDBIndex -TtlInSeconds {val} -Unique 1 -Key "key1"
```
