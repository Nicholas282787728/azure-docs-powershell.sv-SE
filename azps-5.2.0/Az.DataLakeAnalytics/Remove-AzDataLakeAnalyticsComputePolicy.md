# Remove-AzDataLakeAnalyticsComputePolicy

## Sammanfattning
Tar bort en angiven beräknings princip för Azure Data Lake Analytics

## FRÅGESYNTAXEN

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Remove-AzDataLakeAnalyticsComputePolicy** tar bort en viss beräknings princip för Azure Data Lake Analytics.

## BESKRIVS

### Exempel 1: ta bort en beräknings princip
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Det här kommandot tar bort den angivna beräknings principen med namnet "policy" i kontot ' contosoadla '.
