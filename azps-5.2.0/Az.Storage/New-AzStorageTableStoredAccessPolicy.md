# New-AzStorageTableStoredAccessPolicy

## Sammanfattning
Skapar en lagrad åtkomst princip för en Azure Storage-tabell.

## FRÅGESYNTAXEN

```
New-AzStorageTableStoredAccessPolicy [-Table] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzStorageTableStoredAccessPolicy** skapar en lagrad åtkomst policy för en Azure Storage-tabell.

## BESKRIVS

### Exempel 1: skapa en lagrad åtkomst princip i en tabell
```
PS C:\>New-AzStorageTableStoredAccessPolicy -Table "MyTable" -Policy "Policy02"
```

Det här kommandot skapar en åtkomst princip med namnet Policy02 i lagrings tabellen tabellen.
