# Get-AzSupportProblemClassification

## Sammanfattning
Få problem klassificeringar för den angivna tjänsten.

## FRÅGESYNTAXEN

### GetByNameParameterSet (standard)
```
Get-AzSupportProblemClassification -ServiceId <String> [-Id <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSupportProblemClassification [-Id <String>] -ServiceObject <PSSupportService>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämtar den aktuella listan över problem klassificering för en Azure-tjänst. Du kan använda tjänst-och problem klassificerings-GUID för att skapa ett nytt support ärende med hjälp av New-AzSupportTicket.

Använd alltid tjänst-och problem klassificerings klasserna som hämtas programmatiskt. I den här övningen får du se till att du har den senaste uppsättningen tjänst-och problem klassificerings-GUID för att skapa en support biljett.
