# Remove-AzWvdRegistrationInfo

## Sammanfattning
Ta bort registrerings informationen för Windows Virtual Desktop.

## FRÅGESYNTAXEN

```
Remove-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Ta bort registrerings informationen för Windows Virtual Desktop.

## BESKRIVS

### Exempel 1: ta bort ett registrerings-token för Windows Virtual Desktop
```powershell
PS C:\> Remove-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

Det här kommandot tar bort ett registrerings-token för Windows virtuella skriv bord i en adresspool.
