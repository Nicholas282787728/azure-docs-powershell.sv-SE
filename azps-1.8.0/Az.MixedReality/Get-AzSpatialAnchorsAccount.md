# Get-AzSpatialAnchorsAccount

## Sammanfattning
Konto för att få avstånd till ankare

## FRÅGESYNTAXEN

### ListParameterSet (standard)
```
Get-AzSpatialAnchorsAccount [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetParameterSet (standard)
```
Get-AzSpatialAnchorsAccount -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzSpatialAnchorsAccount -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Få eller Visa en lista över spatiala ankare i vissa abonnemang och resurs grupper.
