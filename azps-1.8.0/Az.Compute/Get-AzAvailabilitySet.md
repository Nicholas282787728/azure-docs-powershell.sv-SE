# Get-AzAvailabilitySet

## Sammanfattning
Hämtar Azure Availability-uppsättningar i en resurs grupp.

## FRÅGESYNTAXEN

```
Get-AzAvailabilitySet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzAvailabilitySet** hämtar Azure Availability-uppsättningar i en resurs grupp.
Du kan ange namnet på en specifik tillgänglighets uppsättning som du vill hämta.

## BESKRIVS
