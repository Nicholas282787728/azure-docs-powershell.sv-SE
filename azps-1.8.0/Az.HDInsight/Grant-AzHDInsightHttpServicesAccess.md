# Grant-AzHDInsightHttpServicesAccess

## Sammanfattning
Ger åtkomst till klustret via HTTP.

## FRÅGESYNTAXEN

```
Grant-AzHDInsightHttpServicesAccess [-ClusterName] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Grant-AzHDInsightHttpServicesAccess** cmdlet beviljar http-åtkomst till ett Azure HDInsight-kluster med ODBC, Ambari, Oozie och webb tjänster.

## BESKRIVS

### Exempel 1: bevilja HTTP-åtkomst till ett Azure HDInsight-kluster
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzHDInsightHttpServicesAccess `
            -ClusterName $clusterName `
            -HttpCredential $newClusterCreds
```

Det här kommandot beviljar HTTP-åtkomst till det kluster som heter din-Hadoop-001.
