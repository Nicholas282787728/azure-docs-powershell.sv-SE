# Get-AzCognitiveServicesAccountKey

## Sammanfattning
Hämtar API-nycklarna för ett konto.

## FRÅGESYNTAXEN

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzCognitiveServicesAccountKey** hämtar API-nycklarna för ett etablerat konto för kognitiva tjänster.
Ett konto för kognitiva tjänster har två API-nycklar: KEY1 och Key2.
Nycklarna aktiverar interaktion med konto slut punkten för kognitiva tjänster.
Använd New-AzCognitiveServicesAccountKey för att återskapa en nyckeln.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
