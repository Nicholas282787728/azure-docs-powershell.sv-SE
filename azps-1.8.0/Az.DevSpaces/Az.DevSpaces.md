# Modulen AZ. DevSpaces
## Problembeskrivning
Kommandon för att interagera med Azure DevSpaces-styrenheten.

## AZ. DevSpaces-cmdletar
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Skaffa eller lista Azure DevSpaces-styrenheten.

### [New-AzDevSpacesController](New-AzDevSpacesController.md)
Skapa en ny Azure DevSpaces-styrenhet.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
Ta bort en DevSpaces-styrenhet.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Uppdatera DevSpaces-styrenheten för att lägga till taggar.
