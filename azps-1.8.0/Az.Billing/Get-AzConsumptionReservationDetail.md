# Get-AzConsumptionReservationDetail

## Sammanfattning
Få information om reservationer för angivet datum intervall.

## FRÅGESYNTAXEN

```
Get-AzConsumptionReservationDetail -StartDate <DateTime> -EndDate <DateTime> -ReservationOrderId <String>
 [-ReservationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzConsumptionReservationDetail** hämtar information om reservationer för angivet datum intervall.
