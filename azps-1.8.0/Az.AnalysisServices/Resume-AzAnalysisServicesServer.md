# Resume-AzAnalysisServicesServer

## Sammanfattning
Återupptar en instans av Analysis Services-servern

## FRÅGESYNTAXEN

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Resume-AzAnalysisServicesServer cmdleten återupptar en instans av Analysis Services-servern

## BESKRIVS

### Exempel 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Det här kommandot återupptar en pausad server med namnet testserver i resourcegroup testgroup
