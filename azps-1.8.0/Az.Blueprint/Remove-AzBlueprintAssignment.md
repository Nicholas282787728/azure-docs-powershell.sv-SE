# Remove-AzBlueprintAssignment

## Sammanfattning
Ta bort en skiss tilldelning från ett abonnemang.

## FRÅGESYNTAXEN

### DeleteBlueprintAssignmentByName (standard)
```
Remove-AzBlueprintAssignment [[-SubscriptionId] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteBlueprintAssignmentByObject
```
Remove-AzBlueprintAssignment [[-SubscriptionId] <String>] [-InputObject] <PSBlueprintAssignment> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Ta bort en skiss som har tilldelats till ett abonnemang.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzBlueprintAssignment -Name "myAssignment" -Subscription "00000000-1111-0000-1111-000000000000" -Confirm
```

Ta bort den skiss tilldelning som anges av namnet från det angivna abonnemanget. Cmdleten uppmanas att bekräfta innan kommandot körs.
