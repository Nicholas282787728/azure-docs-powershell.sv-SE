# Remove-AzRouteFilter

## Sammanfattning
Tar bort ett väg filter.

## FRÅGESYNTAXEN

```
Remove-AzRouteFilter -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzRouteFilter** tar bort ett väg filter.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
```

Kommandot tar bort väg filtret med namnet RouteFilter01 i resurs gruppen med namnet ResourceGroup01.
