# New-AzApplicationGatewayFirewallCustomRule

## Sammanfattning
Skapar en ny anpassad regel för brand Väggs principen för Application Gateway.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayFirewallCustomRule -Name <String> -Priority <Int32> -RuleType <String>
 -MatchCondition <PSApplicationGatewayFirewallCondition[]> -Action <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**New-AzApplicationGatewayFirewallCustomRule** skapar en anpassad regel för brand Väggs princip.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $customRule = New-AzureRmApplicationGatewayFirewallCustomRule -Name example-rule -Priority 1 -RuleType MatchRule -matchConditons $condtion -Action Allow
```

Kommandot skapar en ny anpassad regel med namnet på exempel regel, prioritet 1 och regel typen MatchRule med villkor som definieras i variabeln condition och åtgärden Tillåt. Den nya anpassade regeln sparas i $customRule.
