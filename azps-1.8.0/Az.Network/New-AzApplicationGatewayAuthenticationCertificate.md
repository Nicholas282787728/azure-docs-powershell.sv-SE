# New-AzApplicationGatewayAuthenticationCertificate

## Sammanfattning
Skapar ett autentiseringscertifikat för en Programgateway.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayAuthenticationCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzApplicationGatewayAuthenticationCertificate** skapar ett autentiseringscertifikat för en Azure Application Gateway.

## BESKRIVS

### Exempel 1: skapa ett autentiseringscertifikat
```
PS C:\> $cert = New-AzApplicationGatewayAuthenticationCertificate -Name "cert01" -CertificateFile "C:\cert.cer"
```

Det första kommandot skapar autentiseringscertifikatet med namnet cert01.
