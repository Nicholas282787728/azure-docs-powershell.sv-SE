# Get-AzApplicationGatewaySslCertificate

## Sammanfattning
Hämtar ett SSL-certifikat för en Programgateway.

## FRÅGESYNTAXEN

```
Get-AzApplicationGatewaySslCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzApplicationGatewaySslCertificate** får ett SSL-certifikat för en Programgateway.

## BESKRIVS

### Exempel 1: skaffa ett specifikt SSL-certifikat
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Cert = Get-AzApplicationGatewaySslCertificate -Name "Cert01" -ApplicationGateway $AppGW
```

Med det första kommandot får programgatewayen namnet ApplicationGateway01 och lagras resultatet i variabeln som heter $AppGW.
Det andra kommandot får SSL-certifikatet som heter Cert01 från den Application Gateway som lagras i variabeln $AppGW.
Kommandot lagrar certifikatet i variabeln som heter $Cert.

### Exempel 2: skaffa en lista över SSL-certifikat
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Certs = Get-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW
```

Med det första kommandot får programgatewayen namnet ApplicationGateway01 och lagras resultatet i variabeln som heter $AppGW.
Det andra kommandot får en lista med SSL-certifikat från den Programgateway som lagras i variabeln $AppGW.
Då sparas resultaten i variabeln $Certs.
