# Set-AzVirtualNetworkTap

## Sammanfattning
Uppdaterar ett virtuellt nätverk.

## FRÅGESYNTAXEN

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzVirtualNetworkTap** uppdaterar ett virtuellt nätverk.

## BESKRIVS

### Exempel 1: Konfigurera ett virtuellt nätverk tryck
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

Kommandot uppdaterar mål-IpConfiguration och uppdaterar det virtuella nätverkets tryckning.
Om det finns några trycknings konfigurationer som refererar till den kommer inte all käll trafik att starta till nya mål uppdateringar för konfiguration efter inlägg.
