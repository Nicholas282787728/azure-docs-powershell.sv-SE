# Get-AzNetworkUsage

## Sammanfattning
Visar nätverks användning för en prenumeration

## FRÅGESYNTAXEN

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Get-AzNetworkUsage cmdlet får gränser och aktuell användning för nätverks resurser.
