# Get-AzStreamAnalyticsOutput

## Sammanfattning
Hämtar de utdata som definierats i ett angivet ström analys jobb eller utdata.

## FRÅGESYNTAXEN

```
Get-AzStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzStreamAnalyticsOutput** visar alla utdata som är definierade i ett angivet Stream analys jobb eller får information om ett visst resultat.

## BESKRIVS

### EXEMPEL 1: Hämta information om jobb utmatningar
```
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Det här kommandot returnerar information om de utdata som definierats i jobbet StreamingJob.

### EXEMPEL 2: Hämta information om ett visst utskrifts jobb
```
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Det här kommandot returnerar information om den utdata som heter output som definierats i jobbet StreamingJob.
