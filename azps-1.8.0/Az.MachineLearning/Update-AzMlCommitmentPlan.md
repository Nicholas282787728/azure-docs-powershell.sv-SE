# Update-AzMlCommitmentPlan

## Sammanfattning
Uppdaterar egenskaper för en befintlig åtagande Plans resurs.

## FRÅGESYNTAXEN

```
Update-AzMlCommitmentPlan -ResourceGroupName <String> -Name <String> -SkuName <String> -SkuTier <String>
 [-SkuCapacity <Int32>] [-Tag <Hashtable>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Uppdaterar en befintlig resurs för åtagande plan. Observera att de flesta egenskaperna för åtagande planen är oföränderliga och inte kan ändras. Egenskaper som kan ändras är bland annat SKU (för att du ska kunna migrera ett åtagande från en SKU till en annan) och taggar.

## BESKRIVS

### Exempel 1: uppdatera en åtagande plan
```
Update-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName" -Tags @{'MyTagKey'='MyTagValue'}
```
