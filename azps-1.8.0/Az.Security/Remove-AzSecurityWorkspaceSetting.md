# Remove-AzSecurityWorkspaceSetting

## Sammanfattning
Tar bort inställningen för säkerhets arbets ytan för det här abonnemanget.

## FRÅGESYNTAXEN

### SubscriptionLevelResource (standard)
```
Remove-AzSecurityWorkspaceSetting -Name <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ID
```
Remove-AzSecurityWorkspaceSetting -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSecurityWorkspaceSetting -InputObject <PSSecurityWorkspaceSetting> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort inställningen för säkerhets arbets ytan för det här abonnemanget.
Den här åtgärden gör att de nyligen installerade säkerhets agenterna kan rapportera till standard arbets ytan för denna susbscription.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Remove-AzSecurityWorkspaceSetting -Name "default"
```

Tar bort inställningen för säkerhets arbets ytan för det här abonnemanget.
