# Set-AzApiManagementGroup

## Sammanfattning
Konfigurerar en API-hanterings grupp.

## FRÅGESYNTAXEN

```
Set-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-Name <String>]
 [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzApiManagementGroup** konfigurerar en API-hanterings grupp.

## BESKRIVS

### Exempel 1: Konfigurera en hanterings grupp
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementGroup -Context $apimContext -Description "Updated Management Group" -Name "Group0001"
```

Det här kommandot konfigurerar en hanterings grupp som heter Group0001.
