# Remove-AzApiManagementProperty

## Sammanfattning
Tar bort en API-egenskap.

## FRÅGESYNTAXEN

```
Remove-AzApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApiManagementProperty** tar bort en Azure API Management- **egenskap**.

## BESKRIVS

### Exempel 1: ta bort en egenskap
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -PassThru
```

Det här kommandot tar bort egenskapen med ID-Property11.
