# Remove-AzDataLakeStoreItem

## Sammanfattning
Tar bort en fil eller mapp i data Lake Store.

## FRÅGESYNTAXEN

```
Remove-AzDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]> [-Recurse] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzDataLakeStoreItem** tar bort en fil eller mapp i data Lake Store.

## BESKRIVS

### Exempel 1: ta bort flera objekt
```
PS C:\>Remove-AzDataLakeStoreItem -AccountName "ContosoADL" -Paths "/File01.txt","/MyFiles/File.csv"
```

Det här kommandot tar bort filerna File01.txt och File.csv från data Lake Store.
