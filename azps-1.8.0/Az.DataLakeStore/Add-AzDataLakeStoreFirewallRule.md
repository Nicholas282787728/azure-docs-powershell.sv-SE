# Add-AzDataLakeStoreFirewallRule

## Sammanfattning
Lägger till en brand Väggs regel till det angivna data Lake Store-kontot.

## FRÅGESYNTAXEN

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Add-AzDataLakeStoreFirewallRule** lägger till en brand Väggs regel i det angivna data Lake Store-kontot.

## BESKRIVS

### Exempel 1: lägga till en ny brand Väggs regel i ett data Lake Store-konto
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Detta skapar en ny brand Väggs regel som heter "min regel" i kontot "ContosoADL" med ett IP-intervall på 127.0.0.1-127.0.0.2
