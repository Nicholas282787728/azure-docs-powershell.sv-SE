# Test-AzDataLakeStoreAccount

## Sammanfattning
Testar förekomsten av ett data Lake Store-konto.

## FRÅGESYNTAXEN

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Test-AzDataLakeStoreAccount-** cmdleten testar att det finns ett data Lake Store-konto.

## BESKRIVS

### Exempel 1: testa ett konto
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Det här kommandot testar om kontot som heter ContosoADL finns.
