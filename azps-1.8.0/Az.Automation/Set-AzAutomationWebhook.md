# Set-AzAutomationWebhook

## Sammanfattning
Ändrar en webhook för en Automation-Runbook.

## FRÅGESYNTAXEN

```
Set-AzAutomationWebhook [-Name] <String> [-IsEnabled] <Boolean> [[-Parameters] <IDictionary>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzAutomationWebhook** ändrar en webhook för en Azure Automation-Runbook.

## BESKRIVS

### Exempel 1: inaktivera en webhook
```
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -IsEnabled $False
```

Det här kommandot inaktiverar en webhook med namnet Webhook01 i Automation-kontot med namnet AutomationAccount01.
