# Resume-AzAutomationJob

## Sammanfattning
Återupptar ett uppehållet automatiserings jobb.

## FRÅGESYNTAXEN

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Resume-AzAutomationJob** återupptar ett inaktiverat Azure Automation-jobb.
Ange det upphävda jobbet.
Om du vill pausa ett jobb kan du använda Suspend-AzAutomationJob cmdlet.

## BESKRIVS

### Exempel 1: återuppta ett avbrutet jobb
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Det här kommandot återupptar jobbet som har angivet ID.
