# Get-AzSqlServerDisasterRecoveryConfiguration

## Sammanfattning
Hämtar en konfiguration för återställning av databas server system.

## FRÅGESYNTAXEN

```
Get-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlServerDisasterRecoveryConfiguration** har en konfiguration för SQL Server System återställning.
