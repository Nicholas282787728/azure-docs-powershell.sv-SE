# Get-AzSqlElasticPoolRecommendedAction

## Sammanfattning
Får du en eller flera rekommenderade åtgärder för en Azure SQL-pool för elastiska pooler.

## FRÅGESYNTAXEN

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlElasticPoolRecommendedAction** har en eller flera rekommenderade åtgärder för en Azure SQL-grupprådgivare.
