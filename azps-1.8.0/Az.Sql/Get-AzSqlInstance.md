# Get-AzSqlInstance

## Sammanfattning
Returnerar information om en Azure SQL-hanterad databas instans.

## FRÅGESYNTAXEN

```
Get-AzSqlInstance [[-Name] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlInstance** returnerar information om ett eller flera Azure SQL-hanterade instanser.
Ange namnet på en instans om du endast vill visa information för den instansen.
