# Disable-AzSqlServerAdvancedThreatProtection

## Sammanfattning
Inaktiverar Avancerat skydd på en server.

## FRÅGESYNTAXEN

```
Disable-AzSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **disable-AzSqlServerAdvancedThreatProtection** inaktiverar Avancerat skydd mot en server.

## BESKRIVS

### Exempel 1 – inaktivera Avancerat skydd för Server
```powershell
PS C:\>  Disable-AzSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```

### Exempel 2 – inaktivera Avancerat skydd för Server från Server resurs
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Disable-AzSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```
