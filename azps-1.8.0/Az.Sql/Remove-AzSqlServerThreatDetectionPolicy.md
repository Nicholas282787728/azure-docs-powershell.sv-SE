# Remove-AzSqlServerThreatDetectionPolicy

## Sammanfattning
Tar bort hot identifierings policyn från en server.

## FRÅGESYNTAXEN

```
Remove-AzSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Remove-AzSqlServerThreatDetectionPolicy cmdlet tar bort hot Detection policyn från en Azure SQL Server.
Om du vill använda den här cmdleten anger du parametrarna ResourceGroupName och ServerName för att identifiera den server som cmdleten tar bort från.

## BESKRIVS

### Exempel 1: ta bort en princip för hot identifiering för en databas
```
PS C:\> Remove-AzSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Det här kommandot tar bort hot identifierings policyn från en server som heter Server01.
