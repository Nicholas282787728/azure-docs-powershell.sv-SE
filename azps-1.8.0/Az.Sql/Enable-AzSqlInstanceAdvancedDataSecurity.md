# Enable-AzSqlInstanceAdvancedDataSecurity

## Sammanfattning
Aktiverar avancerad data säkerhet för en hanterad instans.

## FRÅGESYNTAXEN

```
Enable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Enable-AzSqlInstanceAdvancedDataSecurity** aktiverar avancerad data säkerhet på en hanterad instans.

## BESKRIVS

### Exempel 1-aktivera hanterad instans avancerad data säkerhet
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Exempel 2 – aktivera hanterad instans avancerad data säkerhet från Server resurs
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
