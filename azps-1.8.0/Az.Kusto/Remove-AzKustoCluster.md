# Remove-AzKustoCluster

## Sammanfattning
Tar bort ett befintligt Kusto-kluster.

## FRÅGESYNTAXEN

### ByNameAndResourceGroup (standard)
```
Remove-AzKustoCluster [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzKustoCluster [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKustoCluster [-InputObject] <PSKustoCluster> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort ett befintligt Kusto-kluster.

## BESKRIVS

### Exempel 1 – ta bort ett befintligt Kusto-kluster med namn

```
PS C:\> Remove-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster
```

Kommandot ovan tar bort Kusto-klustret med namnet "mykustocluster" i resurs gruppen "testrg".

### Exempel 2 – ta bort ett befintligt Kusto-kluster genom att rör

```
PS C:\> Get-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster | Remove-AzKustoCluster
```

Kommandot ovan får till Kusto-klustret med namnet "mykustocluster" i resurs gruppen "testrg" med hjälp av `Get-AzKustoCluster` cmdleten och sedan flyttas resultatet till `Remove-AzKustoCluster` för att ta bort det.
