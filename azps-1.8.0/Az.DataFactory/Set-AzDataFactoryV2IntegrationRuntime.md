## PROBLEMBESKRIVNING
Set-AzDataFactoryV2IntegrationRuntime cmdlet uppdaterar en integrations körning med specifika parametrar.

## BESKRIVS

### Exempel 1: Uppdatera beskrivningen av integrations körningen.
```
PS C:\> Set-AzDataFactoryV2IntegrationRuntime -ResourceGroupName 'rg-test-dfv2' -DataFactoryName 'test-df-eu2' -Name 'test-selfhost-ir' `
                                            -Description 'New description'

    Id                : /subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir
    ResourceGroupName : rg-test-dfv2
    DataFactoryName   : test-df-eu2
    Name              : test-selfhost-ir
    Description       : New description
```

Cmdleten uppdaterar beskrivningen av integrations körningen "test-selfhost-IR".

### Exempel 2: dela självvärdbaserade integrerings körningar.
```
PS C:\> Set-AzDataFactoryV2IntegrationRuntime -ResourceGroupName 'rg-test-dfv2' -DataFactoryName 'test-df-eu2' -Name 'test-selfhost-ir' `
                                            -SharedIntegrationRuntimeResourceId '/subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir' -Type "SelfHosted"

    Id                : /subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir
    ResourceGroupName : rg-test-dfv2
    DataFactoryName   : test-df-eu2
    Name              : test-selfhost-ir
    Description       : New description
```

Denna cmdlet lägger till ADF för att använda den delade integrerings körningen. Om du använder en `-SharedIntegrationRuntimeResourceId` parameter `-Type` måste också inkluderas. Observera att data fabriken måste ges tillstånd att använda integrations körningen innan cmdleten körs.
