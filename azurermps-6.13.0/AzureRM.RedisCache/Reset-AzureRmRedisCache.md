# Reset-AzureRmRedisCache

## Sammanfattning
Startar om noder i ett cacheminne.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Reset-AzureRmRedisCache [-ResourceGroupName <String>] -Name <String> -RebootType <String> [-ShardId <Int32>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Reset-AzureRmRedisCache** startar om noder för en Azure Redis-cachepost.

## BESKRIVS

### Exempel 1: starta om båda noderna
```
PS C:\>Reset-AzureRmRedisCache -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -RebootType "AllNodes" -Force
```

Med det här kommandot startas båda noderna för cacheminnet som heter RedisCache06.
