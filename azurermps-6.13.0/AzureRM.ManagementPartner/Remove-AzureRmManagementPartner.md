# Remove-AzureRmManagementPartner

## Sammanfattning
Ta bort Microsoft Partner Network (MPN) ID för den aktuella autentiserade användaren eller tjänstens huvud objekt.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmManagementPartner [-PartnerId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Ta bort Microsoft Partner Network (MPN) ID för den aktuella autentiserade användaren eller tjänstens huvud objekt.
