# Modulen AzureRM. DevSpaces
## Problembeskrivning
Kommandon för att interagera med Azure DevSpaces-styrenheten.

## AzureRM. DevSpaces-cmdletar
### [Get-AzureRmDevSpacesController](Get-AzureRmDevSpacesController.md)
Skaffa eller lista Azure DevSpaces-styrenheten.

### [New-AzureRmDevSpacesController](New-AzureRmDevSpacesController.md)
Skapa en ny Azure DevSpaces-styrenhet.

### [Remove-AzureRmDevSpacesController](Remove-AzureRmDevSpacesController.md)
Ta bort en DevSpaces-styrenhet.

### [Update-AzureRmDevSpacesController](Update-AzureRmDevSpacesController.md)
Uppdatera DevSpaces-styrenheten för att lägga till taggar.
