# Add-AzureRmMlWebServiceRegionalProperty

## Sammanfattning
Skapar egenskaper för webb tjänsten.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Add-AzureRmMlWebServiceRegionalProperty -ResourceGroupName <String> -Name <String> -Region <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar nationella egenskaper för Azure Machine Learning för en befintlig webb tjänst.

## BESKRIVS

### Exempel 1: lägga till nya nationella egenskaper för västra centrala USA

```
Add-AzureRmMlWebServiceRegionalProperty -ResourceGroupName "myresourcegroup" -Name "mywebservicename" -Region westcentralus
```

I det här exemplet kommando skapas en regional egenskap för en webb tjänst i regionen "Västra Central USA".
