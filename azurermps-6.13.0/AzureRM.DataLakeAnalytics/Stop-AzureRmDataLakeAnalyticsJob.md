# Stop-AzureRmDataLakeAnalyticsJob

## Sammanfattning
Avbryter ett jobb.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Stop-AzureRmDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Stop-AzureRmDataLakeAnalyticsJob** avbryter ett Azure Data Lake Analytics-jobb.

## BESKRIVS

### Exempel 1: avbryta ett jobb
```
PS C:\>Stop-AzureRmDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Det här kommandot avbryter jobbet med angivet ID.
