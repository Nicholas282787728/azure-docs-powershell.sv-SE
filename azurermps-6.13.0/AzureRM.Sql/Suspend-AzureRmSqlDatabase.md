# Suspend-AzureRmSqlDatabase

## Sammanfattning
Avbryter en SQL Data Warehouse-databas.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Suspend-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **suspend-AzureRmSqlDatabase** inaktive ras en Azure SQL Data Warehouse-databas.

## BESKRIVS

### Exempel 1: inaktiverar en Azure SQL Data Warehouse-databas
```
PS C:\>Suspend-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot gör att en Active Azure SQL Data Warehouse-databas stoppas.
