# New-AzureRmSqlServerDnsAlias

## Sammanfattning
Det här kommandot skapar ett nytt DNS-alias för Azure SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar ett nytt DNS-alias för Azure SQL Server som pekar på angiven server.

## BESKRIVS

### Exempel 1
```
PS C:\> $serverDNSAlias = New-AzureRmSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Det här kommandot skapar DNS-aliaset för Azure SQL Server med namnet aliasName som pekar på Server namn
