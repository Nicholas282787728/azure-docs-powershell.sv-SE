# Get-AzureRmSqlInstance

## Sammanfattning
Returnerar information om en Azure SQL-hanterad databas instans.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### GetInstanceByResourceGroup (standard)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetInstanceByNameAndResourceGroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSqlInstance** returnerar information om ett eller flera Azure SQL-hanterade instanser.
Ange namnet på en instans om du endast vill visa information för den instansen.
