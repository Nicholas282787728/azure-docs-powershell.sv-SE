# Get-AzureRmSqlDatabaseGeoBackupPolicy

## Sammanfattning
Hämtar en databas princip för säkerhets kopiering av databasen.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmSqlDatabaseGeoBackupPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSqlDatabaseGeoBackupPolicy** hämtar den geo säkerhets kopierings princip som registrerats för den här databasen.
Det här är en Azure Backup-resurs som används för att definiera säkerhets kopierings lagrings princip.
