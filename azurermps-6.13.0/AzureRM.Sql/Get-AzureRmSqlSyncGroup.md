# Get-AzureRmSqlSyncGroup

## Sammanfattning
Returnerar information om synkroniseringsresurser för Azure SQL-databas.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmSqlSyncGroup [[-Name] <String>] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSqlSyncGroup** returnerar information om en eller flera synkroniseringsresurser för Azure SQL-databas.
Ange namnet på en synkroniseringsresurs för att visa information för enbart den synkroniseringsresursen.
