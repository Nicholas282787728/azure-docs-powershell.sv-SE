# Remove-AzureRmSqlServerAuditing

## Sammanfattning
Tar bort granskningen av en SQL-Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmSqlServerAuditing [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmSqlServerAuditing** tar bort granskningen av en Azure SQL-Server.
Om du vill använda denna cmdlet anger du parametrarna *ResourceGroupName* och *servername* för att identifiera servern.
När du har kört denna cmdlet utförs inte granskning av databaserna på Azure SQL Server.
Om kommandot lyckas och du anger parametern *Passthru* returnerar cmdleten ett objekt som beskriver den aktuella gransknings principen och Azure SQL Server-identifierarna.
Server-ID: n innehåller **ResourceGroupName** och **servername**.

## BESKRIVS

### Exempel 1: ta bort granskningen av en Azure SQL-Server
```
PS C:\>Remove-AzureRmSqlServerAuditing -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Det här kommandot tar bort granskningen av alla databaser som finns på Server01 i resurs gruppen.
