# Get-AzureRmSqlDatabaseIndexRecommendations

## Sammanfattning
Hämtar de rekommenderade index åtgärderna för en server eller en databas.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmSqlDatabaseIndexRecommendations -ServerName <String> [-DatabaseName <String>] [-TableName <String>]
 [-IndexRecommendationName <String>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSqlDatabaseIndexRecommendations** får de rekommenderade index åtgärderna för en Azure SQL Database-Server eller-databas.

## BESKRIVS

### Exempel 1: få index rekommendationer för alla databaser på servern
```
PS C:\>Get-AzureRmSqlDatabaseIndexRecommendations -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Det här kommandot returnerar index rekommendationer för alla databaser på servern Server01.

### Exempel 2: Hämta index rekommendationer för en viss databas
```
PS C:\>Get-AzureRmSqlDatabaseIndexRecommendations -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot returnerar index rekommendationer för en viss databas.

### Exempel 3: få en enda index rekommendation efter namn
```
PS C:\>Get-AzureRmSqlDatabaseIndexRecommendations -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -IndexRecommendationName "INDEX_NAME"
```

Det här kommandot returnerar en enkel index rekommendation efter namn.
