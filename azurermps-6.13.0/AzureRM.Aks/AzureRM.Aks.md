# Modulen AzureRM. AKS
## Problembeskrivning
Kommandon för att interagera med Azure Managed Kubernetes-kluster.

## AzureRM. AKS-cmdletar
### [Get-AzureRmAks](Get-AzureRmAks.md)
Lista Kubernetes hanterade kluster.

### [Import-AzureRmAksCredential](Import-AzureRmAksCredential.md)
Importera och slå samman Kubectl config för ett hanterat Kubernetes-kluster.

### [New-AzureRmAks](New-AzureRmAks.md)
Skapa ett nytt hanterat Kubernetes-kluster.

### [Remove-AzureRmAks](Remove-AzureRmAks.md)
Ta bort ett hanterat Kubernetes-kluster.

### [Set-AzureRmAks](Set-AzureRmAks.md)
Uppdatera eller skapa ett hanterat Kubernetes-kluster.

### [Start-AzureRmAksDashboard](Start-AzureRmAksDashboard.md)
Skapa en Kubectl SSH-tunnel till instrument panelen för hanterade kluster.

### [Stopp-AzureRmAksDashboard](Stop-AzureRmAksDashboard.md)
Stoppa den Kubectl SSH-tunnel som skapades i Start-AzureRmKubernetesDashboard.
