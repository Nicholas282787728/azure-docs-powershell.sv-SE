# Move-AzureRmDataLakeStoreItem

## Sammanfattning
Flyttar eller byter namn på en fil eller mapp i data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Move-AzureRmDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-Destination] <DataLakeStorePathInstance> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Move-AzureRmDataLakeStoreItem** flyttar eller byter namn på en fil eller mapp i data Lake Store.

## BESKRIVS

### Exempel 1: flytta och byta namn på ett objekt
```
PS C:\>Move-AzureRmDataLakeStoreItem -AccountName "ContosoADL" -Path "/Original/Path/File.txt" -Destination "/New/Path/RenamedFile.txt"
```

Det här kommandot byter namn på objektet File.txt till RenamedFile.txt och flyttar det till en annan mapp.
