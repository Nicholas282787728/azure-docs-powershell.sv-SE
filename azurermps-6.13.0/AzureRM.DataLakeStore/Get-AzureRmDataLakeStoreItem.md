# Get-AzureRmDataLakeStoreItem

## Sammanfattning
Hämtar information om en fil eller mapp i data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmDataLakeStoreItem** hämtar information om en fil eller mapp i data Lake Store.

## BESKRIVS

### Exempel 1: Hämta information om en fil från data Lake Store
```
PS C:\>Get-AzureRmDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

Det här kommandot får information om filen Test.csv från data Lake Store.
