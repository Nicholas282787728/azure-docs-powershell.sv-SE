# Remove-AzureRmApiManagement

## Sammanfattning
Tar bort en API-hanterings tjänst.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmApiManagement -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmApiManagement** tar bort en Azure API Management-tjänst.

## BESKRIVS

### Exempel 1: ta bort en API-hanterings tjänst
```
PS C:\>Remove-AzureRmApiManagement -ResourceGroupName "ContosoGroup02" -Name "ContosoApi"
```

Det här kommandot tar bort API-hanteringskonsolen med namnet ContosoApi.
