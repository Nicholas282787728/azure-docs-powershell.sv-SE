# Set-AzureBatchJob

## Sammanfattning
Uppdaterar ett batchjobb.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Set-AzureBatchJob [-Job] <PSCloudJob> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureBatchJob** uppdaterar ett Azure-batchjobb.
Använd Get-AzureBatchJob cmdlet för att hämta ett **PSCloudJob** -objekt.
Ändra egenskaperna för objektet och Använd sedan aktuell cmdlet för att bekräfta ändringarna i kommando tjänsten.

## BESKRIVS

### Exempel 1: uppdatera ett jobb
```
PS C:\>$Job = Get-AzureBatchJob -Id "Job17" -BatchContext $Context
PS C:\> $Job.Priority = 1
PS C:\> Set-AzureBatchJob -Job $Job -BatchContext $Context
```

Det första kommandot får en pool genom att använda **Get-AzureBatchJob** och lagrar den sedan i $Job variabel.
Det andra kommandot ändrar prioritets specifikationen för $Job-objektet.
Det sista kommandot uppdaterar kommando tjänsten för att matcha det lokala objektet i $Job.

Om du använder Get-AzureRmBatchAccount cmdlet för att hämta din BatchAccountContext används Azure Active Directory-identifiering när den fungerar tillsammans med kommando tjänsten.
