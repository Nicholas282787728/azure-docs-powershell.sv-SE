# New-AzureRmBatchApplication

## Sammanfattning
Lägger till ett program i angivet batch-konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationId] <String>
 [[-AllowUpdates] <Boolean>] [[-DisplayName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzureRmBatchApplication** lägger till ett program till det angivna Azure Batch-kontot.

## BESKRIVS

### Exempel 1: lägga till ett tomt program i ett batch-konto
```
PS C:\>New-AzureRmBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "Litware" -AllowUpdates $True -DisplayName "Litware Advanced Reticulator"
```

Det här kommandot skapar programmet Litware i ContosoBatch-kontot.
Programmet innehåller inte några paket.
