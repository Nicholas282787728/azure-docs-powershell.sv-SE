# Set-AzureRmWebAppSlot

## Sammanfattning
Ändrar en Azure Web App-plats.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## PROBLEMBESKRIVNING
Cmdleten **set-AzureRmWebApp** anger en Azure Web App-plats.

## BESKRIVS

### Exempel 1
```
PS C:\> Set-AzureRmWebAppSlot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -Slot "Slot001" -HttpLoggingEnabled $true
```

Det här kommandot anger HttpLoggingEnabled till sant för fack Slot001 som hör till Web App-ContosoWebApp som är kopplad till resurs gruppens standard-väst
