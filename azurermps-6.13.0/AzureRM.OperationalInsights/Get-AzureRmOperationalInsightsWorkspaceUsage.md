# Get-AzureRmOperationalInsightsWorkspaceUsage

## Sammanfattning
Hämtar användnings data för en arbets yta.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmOperationalInsightsWorkspaceUsage [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmOperationalInsightsWorkspaceUsage** hämtar användnings data för en arbets yta.
Detta visar hur många data som analyseras av arbets ytan under en viss tids period.

## BESKRIVS

### Exempel 1: Hämta användnings data efter arbets ytans namn
```
PS C:\>Get-AzureRmOperationalInsightsWorkspaceUsage -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace"
```

Det här kommandot får användnings information för arbets ytan med namnet min arbets yta i den angivna resurs gruppen.

### Exempel 2: Hämta användnings data med pipeline
```
PS C:\>Get-AzureRmOperationalInsightsWorkspace -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace" | Get-AzureOperationalInsightsWorkspaceUsage
```

Det här kommandot hämtar arbets ytan med namnet min arbets yta med Get-AzureRmOperationalInsightsWorkspace cmdlet och skickar sedan arbets ytan till den aktuella cmdleten.
Kommandot får användnings informationen för den arbets ytan.
