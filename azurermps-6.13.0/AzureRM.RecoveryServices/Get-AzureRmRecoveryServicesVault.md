# Get-AzureRmRecoveryServicesVault

## Sammanfattning
Hämtar en lista över återställnings tjänstens valv.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmRecoveryServicesVault [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmRecoveryServicesVault** hämtar en lista över återställnings tjänst valv i den aktuella prenumerationen.

## BESKRIVS

### Exempel 1
```
PS C:\> Get-AzureRmRecoveryServicesVault
```

Hämta listan med valv i det valda abonnemanget.

### Exempel 2
```
PS C:\> Get-AzureRmRecoveryServicesVault -ResourceGroupName "resourceGroup"
```

Hämta listan med valv i resurs gruppen i det valda abonnemanget.

### Exempel 3
```
PS C:\> Get-AzureRmRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
```

Hämta valvet i resurs gruppen med angivet namn.
