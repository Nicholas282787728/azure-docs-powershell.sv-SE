# New-AzureRmSignalR

## Sammanfattning
Skapa en signal tjänst.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmSignalR [-ResourceGroupName <String>] [-Name] <String> [-Location <String>] [-Sku <String>]
 [-UnitCount <Int32>] [-Tag <System.Collections.Generic.IDictionary`2[System.String,System.String]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapa en signal tjänst.
Följande värden används för parametrar om de inte anges:
* `ResourceGroupName`: standard resurs gruppen som anges av `Set-AzureRmDefault -ResourceGroupName` .
* `Location`: resurs gruppens plats
* `Sku`: `Standard_S1`

## BESKRIVS

### Skapa en serivce
```powershell
PS C:\> New-AzureRmSignalR -ResourceGroupName myResourceGroup1 -Name mysignalr1 -Location eastus -Sku Standard_S1

HostName                                           Location       ServerPort PublicPort ProvisioningState Version
--------                                           --------       ---------- ---------- ----------------- -------

```
