# Get-AzureRmAutomationSoftwareUpdateConfiguration

## Sammanfattning
Hämtar en lista över konfigurationer för Azure Automation-programuppdatering.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### ByAll (standard)
```
Get-AzureRmAutomationSoftwareUpdateConfiguration [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzureRmAutomationSoftwareUpdateConfiguration -Name <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByVMId
```
Get-AzureRmAutomationSoftwareUpdateConfiguration -AzureVMResourceId <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Get-AzureRmAutomationSoftwareUpdateConfiguration returnerar en lista med konfigurationer för program uppdatering. Om du vill ha en specifik konfiguration för program uppdateringen anger du parametern name. Du kan också lista de konfigurationer för program uppdateringar som riktar sig mot specifik Azure Virtual Machine genom att ange Azure Resource ID för den här virtuella datorn.
