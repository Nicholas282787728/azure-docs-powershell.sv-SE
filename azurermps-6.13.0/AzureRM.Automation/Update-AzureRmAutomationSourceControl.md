# Update-AzureRmAutomationSourceControl

## Sammanfattning
Uppdaterar en käll kontroll för Azure Automation.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Update-AzureRmAutomationSourceControl -Name <String> [-AccessToken <SecureString>] [-FolderPath <String>]
 [-Branch <String>] [-Description <String>] [-AutoSync <Boolean>] [-PublishRunbook <Boolean>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Den Update-AzureRmAutomationSourceControl cmdleten ändrar värdet för ett fält i en käll kontroll i Azure Automation.

## BESKRIVS

### Exempel 1
Med det här kommandot anges fältet PublishRunbook till falskt för Azure Automation Source Control med namnet VSTSNative i kontot som heter devAccount.


```powershell
Update-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                      -AutomationAccountName "devAccount" `
                                      -Name "VSTSNative" `
                                      -PublishRunbook $false

```
