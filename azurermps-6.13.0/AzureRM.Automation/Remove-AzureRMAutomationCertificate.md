# Remove-AzureRmAutomationCertificate

## Sammanfattning
Tar bort ett Automation-certifikat.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmAutomationCertificate [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmAutomationCertificate** tar bort ett certifikat från Azure Automation.

## BESKRIVS

### Exempel 1: ta bort ett certifikat
```
PS C:\>Remove-AzureRmAutomationCertificate -AutomationAccountName "Contoso17" -Name "Cert01" -ResourceGroupName "ResourceGroup01"
```

Det här kommandot tar bort ett certifikat som heter Cert01 i Automation-kontot med namnet Contoso17.
