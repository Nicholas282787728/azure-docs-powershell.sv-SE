# Remove-AzureRmAutomationConnectionType

## Sammanfattning
Tar bort en typ av Automation-anslutning.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmAutomationConnectionType [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmAutomationConnectionType** tar bort en Anslutnings typ från Azure Automation.
Alla anslutningar som är kopplade till den Anslutnings typ du tar bort blir oanvändbara.
Ta bort dem, såvida du inte skapar en ny Anslutnings typ som uppfyller följande villkor: 
- Typen har samma namn som den ursprungliga anslutnings typen. 
- Typen har samma fält definitioner som den ursprungliga anslutnings typen.
Den kan innehålla ytterligare fält.

## BESKRIVS

### Exempel 1: ta bort en Anslutnings typ
```
PS C:\>Remove-AzureRmAutomationConnectionType -AutomationAccountName "Contoso17" -Name "ContosoConnectionType" -ResourceGroupName "ResourceGroup01"
```

Det här kommandot tar bort en Anslutnings typ med namnet ContosoConnectionType i Automation-kontot med namnet Contoso17.
