# Remove-AzureRmAutomationSourceControl

## Sammanfattning
Tar bort en Azure Automation-källkod.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Remove-AzureRmAutomationSourceControl cmdlet tar bort en käll kontroll från Azure Automation.

## BESKRIVS

### Exempel 1
Det här kommandot tar bort Automation Source Control med namnet VSTSNative i kontot som heter devAccount.
Det här kommandot anger parametern *Force* . Därför behöver du inte bekräfta.

```powershell
PS C:\> Remove-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
