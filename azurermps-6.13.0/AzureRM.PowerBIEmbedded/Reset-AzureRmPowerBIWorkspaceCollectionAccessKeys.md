# Reset-AzureRmPowerBIWorkspaceCollectionAccessKeys

## Sammanfattning
Återställer den angivna åtkomst tangenten.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Reset-AzureRmPowerBIWorkspaceCollectionAccessKeys [-ResourceGroupName] <String>
 [-WorkspaceCollectionName] <String> [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Reset-AzureRmPowerBIWorkspaceCollectionAccessKeys** återställer den angivna åtkomst tangenten i din Power BI-arbetsyta.

## BESKRIVS

### Exempel 1: Återställ primär åtkomst-tangenten
```
PS C:\>Reset-AzureRmPowerBIWorkspaceCollectionAccessKeys -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Det här kommandot återställer den primära åtkomst-tangenten för arbets ytan med namnet WCN11 i den angivna resurs gruppen.
