# Get-AzureRmUserAssignedIdentity

## Sammanfattning
Hämtar en användares tilldelade identitet/identiteter.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### SuscriptionParameterSet (standard)
```
Get-AzureRmUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzureRmUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Med **Get-AzureRmUserAssignedIdentity** blir befintliga användare tilldelade identiteter.
