# Modulen AzureRM. Maps
## Problembeskrivning
I det här avsnittet visas hjälp för Azure Maps-cmdletar.

## AzureRM. kartor cmdlets
### [Get-AzureRmMapsAccount](Get-AzureRmMapsAccount.md)
Får kontot.

### [Get-AzureRmMapsAccountKey](Get-AzureRmMapsAccountKey.md)
Hämtar API-nycklarna för ett konto.
De här nycklarna är den autentiseringsmekanism som används vid kommande samtal till Azure Maps.

### [New-AzureRmMapsAccount](New-AzureRmMapsAccount.md)
Skapar ett Azure Maps-konto.

### [New-AzureRmMapsAccountKey](New-AzureRmMapsAccountKey.md)
Återskapar en konto-nyckeln.

### [Remove-AzureRmMapsAccount](Remove-AzureRmMapsAccount.md)
Tar bort ett Azure Maps-konto.
