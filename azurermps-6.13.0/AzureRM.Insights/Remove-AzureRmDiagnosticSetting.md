# Remove-AzureRmDiagnosticSetting

## Sammanfattning
Ta bort en diagnostisk inställning för en resurs.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmDiagnosticSetting -ResourceId <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmDiagnosticSetting** tar bort Diagnostikrapporten för den specifika resursen.
Denna cmdlet implementerar ShouldProcess-mönstret, t. ex. det kan begära bekräftelse från användaren innan de skapar, ändrar eller tar bort resursen.

## BESKRIVS

### Exempel 1: ta bort standardinställningarna för diagnostik (tjänst) för en resurs
```
PS C:\>Remove-AzureRmDiagnosticSetting -ResourceId "Resource01"
```

Det här kommandot tar bort standardinställningarna för diagnostik (tjänst) för resursen som heter Resource01.

### Exempel 2: ta bort standardinställningen för diagnos som identifieras av det angivna namnet för en resurs
```
PS C:\>Remove-AzureRmDiagnosticSetting -ResourceId "Resource01" -Name myDiagSetting
```

Det här kommandot tar bort Diagnostic-inställningen "myDiagSetting" för resursen som heter Resource01.
