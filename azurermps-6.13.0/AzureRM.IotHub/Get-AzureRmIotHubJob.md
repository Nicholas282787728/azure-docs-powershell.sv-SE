# Get-AzureRmIotHubJob

## Sammanfattning
Hämtar information om ett IotHub-jobb.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmIotHubJob [-ResourceGroupName] <String> [-Name] <String> [[-JobId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämtar information om ett IotHub-jobb.
Ett IotHub-jobb skapas när en import-eller export åtgärd utförs med New-AzureRmIotHubExportDevices eller New-AzureRmIotHubImportDevices kommandon.
Du kan antingen Visa alla jobb eller filtrera jobben med jobb-ID.

## BESKRIVS

### Exempel 1 visar alla jobb
```
PS C:\> Get-AzureRmIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Hämtar alla jobb för IotHub med namnet "myiothub"

### Exempel 2 Hämta ett specifikt jobb
```
PS C:\> Get-AzureRmIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub" -JobId 3630fc31-4caa-43e8-a232-ea0577221cb2
```

Hämtar information om jobbet med identifieraren "3630fc31-4caa-43e8-A232-ea0577221cb2" för IotHub med namnet "myiothub"
