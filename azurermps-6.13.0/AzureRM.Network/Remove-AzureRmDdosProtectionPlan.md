# Remove-AzureRmDdosProtectionPlan

## Sammanfattning
Tar bort ett DDoS skydds abonnemang.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Remove-AzureRmDdosProtectionPlan cmdlet tar bort ett DDoS skydds abonnemang.

## BESKRIVS

### Exempel 1: ta bort ett tomt DDoS skydds abonnemang
```
D:\> Remove-AzureRmDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

I det här fallet tar vi bort ett DDoS skydds abonnemang enligt vad som angetts.

### Exempel 2: ta bort ett DDoS skydds abonnemang som är associerat med ett virtuellt nätverk

DDoS skydds planer kan inte tas bort om de är kopplade till ett virtuellt nätverk. Det första steget är att avassociera båda objekten. Därefter behåller vi den nya statusen genom att översätta den lokala variabeln till **set-AzureRmVirtualNetwork**. I det här läget är abonnemanget inte längre associerat med det virtuella nätverket.
Om det är den sista som är kopplad till planen kan vi ta bort DDoS skydds plan genom att använda kommandot Remove-AzureRmDdosProtectionPlan.
