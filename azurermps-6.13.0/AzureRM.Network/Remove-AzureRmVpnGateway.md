# Remove-AzureRmVpnGateway

## Sammanfattning
Remove-AzureRmVpnGateway cmdlet tar bort en Azure VPN gateway. Det här är en gateway som är specifik för den program varu anslutning som definierats i Azure Virtual WAN.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### ByVpnGatewayName (standard)
```
Remove-AzureRmVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayObject
```
Remove-AzureRmVpnGateway -InputObject <PSVpnGateway> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayResourceId
```
Remove-AzureRmVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Remove-AzureRmVpnGateway cmdlet tar bort en Azure VPN gateway. Det här är en gateway som är specifik för den program varu anslutning som definierats i Azure Virtual WAN.

## BESKRIVS

### Exempel 1

```powershell
PS C:\> New-AzureRmResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzureRmVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzureRmVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzureRmVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Remove-AzureRmVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -Passthru
```

I det här exemplet skapas en resurs grupp, virtuellt WAN, virtuellt nav, utbyggbar VPN gateway i Central USA och sedan omedelbart bort. Använd flaggan-Force för att utelämna uppmaningen när du tar bort den virtuella gatewayen.
Detta tar bort VpnGateway och alla VpnConnections som är kopplade till den.

### Exempel 2

```powershell
PS C:\> New-AzureRmResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzureRmVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzureRmVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzureRmVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Get-AzureRmVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" | Remove-AzureRmVpnGateway-Passthru
```

I det här exemplet skapas en resurs grupp, virtuellt WAN, virtuellt nav, utbyggbar VPN gateway i Central USA och sedan omedelbart bort. Denna borttagning sker med PowerShell-rör som använder VpnGateway-objektet som returneras av kommandot Get-AzureRmVpnGateway.
Använd flaggan-Force för att utelämna uppmaningen när du tar bort den virtuella gatewayen.
Detta tar bort VpnGateway och alla VpnConnections som är kopplade till den.
