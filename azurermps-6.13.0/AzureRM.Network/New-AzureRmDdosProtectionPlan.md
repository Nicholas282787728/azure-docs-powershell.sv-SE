# New-AzureRmDdosProtectionPlan

## Sammanfattning
Skapar ett DDoS skydds abonnemang.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmDdosProtectionPlan -ResourceGroupName <String> -Name <String> -Location <String> [-Tag <Hashtable>]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
New-AzureRmDdosProtectionPlan-cmdleten skapar ett DDoS skydds abonnemang.

## BESKRIVS

### Exempel 1: skapa och koppla ett DDoS skydds abonnemang till ett nytt virtuellt nätverk
```
D:\> $ddosProtectionPlan = New-AzureRmDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlanName -Location "West US"
D:\> $subnet = New-AzureRmVirtualNetworkSubnetConfig -Name SubnetName -AddressPrefix 10.0.1.0/24
D:\> $vnet = New-AzureRmvirtualNetwork -Name VnetName -ResourceGroupName ResourceGroupName -Location "West US" -AddressPrefix 10.0.0.0/16 -DnsServer 8.8.8.8 -Subnet $subnet -EnableDdoSProtection -DdosProtectionPlanId $ddosProtectionPlan.Id
```

Först skapar vi ett nytt DDoS skydds abonnemang med kommandot **ny-AzureRmDdosProtectionPlan** .
Sedan skapar vi ett nytt virtuellt nätverk med **New-AzureRmvirtualNetwork** och anger ID för det nya abonnemanget i parametern **DdosProtectionPlanId**. I det här fallet, eftersom vi associerar det virtuella nätverket med en plan, kan vi också ange parametern **EnableDdoSProtection**.

### Exempel 2: skapa och koppla ett DDoS skydds abonnemang till ett befintligt virtuellt nätverk

D:\> $vnet.DdosProtectionPlan = New-Object Microsoft.Azure.Commands.Network.Models.PSResourceId
D:\> $vnet.DdosProtectionPlan.Id = $ddosProtectionPlan.Id
D:\> $vnet.EnableDdosProtection = $true

För det andra får vi den senaste versionen av det virtuella nätverk vi vill associera med abonnemanget. Slutligen sparar vi den nya statusen genom att översätta den lokala variabeln till **set-AzureRmVirtualNetwork**.
