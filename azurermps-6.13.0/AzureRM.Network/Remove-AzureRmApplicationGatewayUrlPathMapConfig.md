# Remove-AzureRmApplicationGatewayUrlPathMapConfig

## Sammanfattning
Tar bort URL-sökvägar till en server del.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmApplicationGatewayUrlPathMapConfig** tar bort URL-sökvägar till en server del.
