# New-AzureRmNetworkWatcherProtocolConfiguration

## Sammanfattning
Skapar ett nytt protokoll konfigurations objekt.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmNetworkWatcherProtocolConfiguration -Protocol <String> [-Method <String>] [-Header <IDictionary>]
 [-ValidStatusCode <Int32[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
New-AzureRmNetworkWatcherProtocolConfiguration cmdlet skapar ett nytt protokoll konfigurations objekt. Det här objektet används för att begränsa protokollets confiuration under en connecitivity-incheckning med de angivna villkoren.
