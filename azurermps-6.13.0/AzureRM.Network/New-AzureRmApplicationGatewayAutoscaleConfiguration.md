# New-AzureRmApplicationGatewayAutoscaleConfiguration

## Sammanfattning
Skapar en Autoskala-konfiguration för programgatewayen.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzureRmApplicationGatewayAutoscaleConfiguration** skapar en konfiguration för automatisk skalning för en Azure Application Gateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $autoscaleConfig = New-AzureRmApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzureRmApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Det första kommandot skapar en Autoskala-konfiguration med minsta kapacitet 3.
Det andra kommandot skapar en Programgateway med den autoskalningsinställning som du konfigurerar.
