# Set-AzureRmVirtualNetworkTap

## Sammanfattning
Anger mål tillståndet för ett virtuellt nätverk tryck.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Set-AzureRmVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Set-AzureRmVirtualNetworkTap** anger mål tillståndet för ett Azure Virtual Network-tryck.

## BESKRIVS

### Exempel 1: Konfigurera ett virtuellt nätverk tryck
```
PS C:\>$vTap = Get-AzureRmVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzureRmVirtualNetworkTap -VirtualNetworkTap $vTap
```

Kommandot uppdaterar mål-IpConfiguration och uppdaterar det virtuella nätverkets tryckning.
Om det finns några trycknings konfigurationer som refererar till den kommer inte all käll trafik att starta till nya mål uppdateringar för konfiguration efter inlägg.
