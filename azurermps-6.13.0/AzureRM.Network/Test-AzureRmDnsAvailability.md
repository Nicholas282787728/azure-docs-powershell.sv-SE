# Test-AzureRmDnsAvailability

## Sammanfattning
Kontrollerar om ett domän namn i cloudapp.azure.com-zonen är tillgängligt för användning.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Test-AzureRmDnsAvailability -DomainNameLabel <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Kontrollerar om ett domän namn i cloudapp.azure.com-zonen är tillgängligt för användning.

## BESKRIVS

### Exempel 1: kontrol lera om contoso.cloudapp.azure.com är tillgängligt för användning.
```
Test-AzureRmDnsAvailability -DomainNameLabel contoso -Location westus
```
