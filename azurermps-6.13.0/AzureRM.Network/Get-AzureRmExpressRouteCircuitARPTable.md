# Get-AzureRmExpressRouteCircuitARPTable

## Sammanfattning
Hämtar ARP-tabellen från en ExpressRoute-krets.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmExpressRouteCircuitARPTable -ResourceGroupName <String> -ExpressRouteCircuitName <String>
 [-PeeringType <String>] -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmExpressRouteCircuitARPTable** hämtar ARP-tabellen från båda gränssnitten av en ExpressRoute-krets. ARP-tabellen tillhandahåller en mappning av IPv4-adress till MAC-adress för en viss peering. Du kan använda ARP-tabellen för att verifiera nivå 2-konfiguration och anslutning.

## BESKRIVS

### Exempel 1: Visa ARP-tabellen för en ExpressRoute-peer
```
Get-AzureRmExpressRouteCircuitARPTable -ResourceGroupName $RG -ExpressRouteCircuitName $CircuitName -PeeringType MicrosoftPeering -DevicePath Primary
```
