# New-AzureRmApplicationGatewayCustomError

## Sammanfattning
Skapar ett anpassat fel med HTTP-statuskod och URL-adressen för anpassad sida 

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmApplicationGatewayCustomError -StatusCode <String> -CustomErrorPageUrl <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzureRmApplicationGatewayCustomError** skapar ett anpassat fel.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $customError403Url = "https://mycustomerrorpages.blob.core.windows.net/errorpages/403-another.htm"
PS C:\> $ce = New-AzureRmApplicationGatewayCustomError -StatusCode HttpStatus403 -CustomErrorPageUrl $customError403Url
```

Det här kommandot skapar det anpassade felet hos HTTP-statuskod 403.
