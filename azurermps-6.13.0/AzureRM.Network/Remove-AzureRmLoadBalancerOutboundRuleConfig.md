# Remove-AzureRmLoadBalancerOutboundRuleConfig

## Sammanfattning
Tar bort en konfiguration för utgående regel från en belastningsutjämnare.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmLoadBalancerOutboundRuleConfig** tar bort en konfiguration för utgående regel från en Azure-belastningsutjämnare.

## BESKRIVS

### Exempel 1: ta bort en regel för utgående trafik från en Azure-belastningsutjämnare
```powershell
PS C:\>$slb = Get-AzureRmLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Remove-AzureRmLoadBalancerOutboundRuleConfig -Name "RuleName" -LoadBalancer $slb
PS C:\>Set-AzureRmLoadBalancer -LoadBalancer $slb
```

Det första kommandot får belastningsutjämnaren som är kopplad till den utgående regel konfiguration som du vill ta bort och sedan spara den i $slb variabel.
Det andra kommandot tar bort den associerade konfigurationen för utgående trafik från belastningsutjämnaren.
Det tredje kommandot uppdaterar belastningsutjämnaren.
