# Remove-AzureRmApplicationGatewayAutoscaleConfiguration

## Sammanfattning
Tar bort autoskalans konfiguration från en Programgateway.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmApplicationGatewayAutoscaleConfiguration** tar bort autoskalans konfiguration från en befintlig Programgateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $gw = Get-AzureRmApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzureRmApplicationGateway -ApplicationGateway $gw
```

Med det första kommandot hämtas programgatewayen och lagras i $gw variabel.
Det andra kommandot tar bort autoskalans konfiguration från applicationg Gateway.
Det tredje kommandot uppdaterar programgatewayen på Azure.
