# Remove-AzureRmPolicySetDefinition

## Sammanfattning
Tar bort en uppsättning definitioner för principer.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

```

### IdParameterSet

```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmPolicySetDefinition** tar bort en princip definition.

## BESKRIVS

### Exempel 1: ta bort princip uppsättnings definition efter resurs-ID
```
PS C:\> $PolicySetDefinition = Get-AzureRmPolicySetDefinition -ResourceId '/subscriptions/mySub/Microsoft.Authorization/policySetDefinitions/myPSSetDefinition'
PS C:\> Remove-AzureRmPolicySetDefinition -Id $PolicySetDefinition.ResourceId -Force
```

Det första kommandot får en princip uppsättnings definition med hjälp av Get-AzureRmPolicySetDefinition cmdlet.
Kommandot sparar det i $PolicySetDefinition variabel.
Det andra kommandot tar bort princip uppsättnings definitionen som identifieras av egenskapen **ResourceID** för $PolicySetDefinition.
