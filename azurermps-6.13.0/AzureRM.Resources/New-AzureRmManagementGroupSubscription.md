# New-AzureRmManagementGroupSubscription

## Sammanfattning
Lägger till ett abonnemang i en hanterings grupp.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmManagementGroupSubscription [-GroupName] <String> [-SubscriptionId] <Guid> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzureRMManagementGroupSubscription** lägger till en prenumeration i en hanterings grupp.

## BESKRIVS

### Exempel 1: lägga till ett abonnemang i en hanterings grupp
```
PS C:\> New-AzureRMManagementGroupSubscription -GroupName "TestGroup" -SubscriptionId 2120692d-35c3-44c8-81f5-631fa7351726
```
