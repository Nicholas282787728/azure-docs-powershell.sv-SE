# Wait-AzDataLakeAnalyticsJob

## Sammanfattning
Väntar på att ett jobb ska slutföras.

## FRÅGESYNTAXEN

```
Wait-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [[-WaitIntervalInSeconds] <Int32>]
 [[-TimeoutInSeconds] <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **wait-AzDataLakeAnalyticsJob** väntar på att ett Azure Data Lake Analytics-jobb ska slutföras.

## BESKRIVS

### Exempel 1: vänta tills jobbet är klart
```
PS C:\>Wait-AzDataLakeAnalyticsJob -Account "ContosoAdlAccount" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Följande kommando väntar på jobbet med det ID som ska slutföras.
