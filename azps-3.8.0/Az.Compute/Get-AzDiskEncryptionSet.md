# Get-AzDiskEncryptionSet

## Sammanfattning
Hämta eller Visa disk krypterings uppsättningar.

## FRÅGESYNTAXEN

```
Get-AzDiskEncryptionSet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämta eller Visa disk krypterings uppsättningar.

## BESKRIVS
