# Add-AzServiceFabricNode

## Sammanfattning
Lägga till noder till den specifika nodtypen i klustret.

## FRÅGESYNTAXEN

```
Add-AzServiceFabricNode -NumberOfNodesToAdd <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Använd **Add-AzServiceFabricNode** för att lägga till noder till den specifika nodtypen. Du behöver bara ange antalet noder som du vill lägga till i en nodtyp.

## BESKRIVS

### Exempel 1
```powershell
PS c:> Add-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Det här kommandot lägger till 2 noder till nodtypen ' N1 '.
