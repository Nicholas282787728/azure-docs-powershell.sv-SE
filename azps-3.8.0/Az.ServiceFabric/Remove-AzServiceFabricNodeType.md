# Remove-AzServiceFabricNodeType

## Sammanfattning
Ta bort en fullständig nodtyp från ett kluster.

## FRÅGESYNTAXEN

```
Remove-AzServiceFabricNodeType [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Använd **Remove-AzServiceFabricNodeType** för att ta bort alla noder från en viss nodtyp och nodtypen från ett kluster. Det går inte att använda det här kommandot för att ta bort typen primär nodtyp.

## BESKRIVS

### Exempel 1
```powershell
PS c:> Remove-AzServiceFabricNodeType -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1'
```

Det här kommandot tar bort NodeType ' NT1 ' från klustret.
