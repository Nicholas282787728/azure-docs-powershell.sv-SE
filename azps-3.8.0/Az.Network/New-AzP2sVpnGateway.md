# New-AzP2sVpnGateway

## Sammanfattning

## PROBLEMBESKRIVNING
Med **New-AzP2sVpnGateway** cmdlet kan du skapa en ny P2SVpnGateway under VirtualHub för att peka på webbplats anslutningar från peka på webbplats klienter till Azure VirtualWan.
