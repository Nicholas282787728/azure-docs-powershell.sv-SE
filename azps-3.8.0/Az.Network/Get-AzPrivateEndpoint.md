# Get-AzPrivateEndpoint

## Sammanfattning
Skaffa en privat slut punkt

## FRÅGESYNTAXEN

### Noexpandering (standard)
```
Get-AzPrivateEndpoint [-Name <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Byggnad
```
Get-AzPrivateEndpoint -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzPrivateEndpoint** hämtar en eller flera privata slut punkter.
