# Remove-AzApplicationGatewayUrlPathMapConfig

## Sammanfattning
Tar bort URL-sökvägar till en server del.

## FRÅGESYNTAXEN

```
Remove-AzApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApplicationGatewayUrlPathMapConfig** tar bort URL-sökvägar till en server del.

## BESKRIVS

### Exempel 1: ta bort en URL-mappsökväg från en Programgateway
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway $appgw -Name "map01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Med det första kommandot får programgatewayen namnet appGwName och lagras resultatet i $appgw variabel.
Det andra kommandot tar bort URL-sökvägen som heter map01 från Application Gateway.
Det tredje kommandot uppdaterar programgatewayen.
