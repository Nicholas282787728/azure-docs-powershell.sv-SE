# Add-AzVirtualHubRoute

## Sammanfattning
Skapar ett VirtualHubRoute-objekt som kan skickas som parameter till kommandot Add-AzVirtualHubRouteTable. 

## FRÅGESYNTAXEN

```
Add-AzVirtualHubRoute -Destination <String[]> -DestinationType <String> -NextHop <String[]>
 -NextHopType <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar ett VirtualHubRoute-objekt.

## BESKRIVS

Med kommandot ovan skapas ett VirtualHubRoute-objekt som sedan kan läggas till i en VirtualHubRouteTable-resurs och ställas in på en VirtualHub.
