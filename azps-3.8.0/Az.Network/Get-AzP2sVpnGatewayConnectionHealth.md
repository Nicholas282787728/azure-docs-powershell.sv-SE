# Get-AzP2sVpnGatewayConnectionHealth

## Sammanfattning
Hämtar hälso information om aggregared för webbplats anslutningar från P2SVpnGateway.

## FRÅGESYNTAXEN

### ByP2SVpnGatewayName (standard)
```
Get-AzP2sVpnGatewayConnectionHealth [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Get-AzP2sVpnGatewayConnectionHealth -InputObject <PSP2SVpnGateway> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Get-AzP2sVpnGatewayConnectionHealth -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Med cmdleten **Get-AzP2sVpnGatewayConnectionHealth** kan du hämta den aktuella aggregerade hälsan för Point to Site Connections från P2SVpnGateway. Aggregerad hälsa visar antalet peka på webbplats klienter som är anslutna till P2SVpnGateway, totalt inkommande och utgående byte via P2SVpnGateway och tilldelade IP-adresser för anslutna punkter till klient klienter.
