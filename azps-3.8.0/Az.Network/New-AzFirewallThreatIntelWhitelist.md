# New-AzFirewallThreatIntelWhitelist

## Sammanfattning
Skapa en ny Threat Intelligence-Lägg för Azure-brandväggen

## FRÅGESYNTAXEN

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**New-AzFirewallThreatIntelWhitelist-** cmdleten skapar ett hot Intel Lägg-objekt som kan användas när du skapar eller ställer in en Azure Firewall.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

I det här exemplet skapas ett hot Intel Lägg som innehåller en FQDN-Lägg med två poster och en IP-adress Lägg med två poster
