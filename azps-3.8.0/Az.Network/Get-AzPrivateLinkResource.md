# Get-AzPrivateLinkResource

## Sammanfattning
Hämtar en privat länk resurs.

## FRÅGESYNTAXEN

### ByPrivateLinkResourceId (standard)
```
Get-AzPrivateLinkResource -PrivateLinkResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzPrivateLinkResource** hämtar alla länk resurser tillhör PrivateLinkResource.

## BESKRIVS

### Exempel 1
```
Get-AzPrivateLinkResource -PrivateLinkResourceId '/subscriptions/00000000-0000-0000-0000-000000000000/resourceGroups/TestResourceGroup/providers/Microsoft.Sql/servers/mySql'
```

I det här exemplet listas alla nbelong för privata länkar till SQL Server med namnet mySql.
