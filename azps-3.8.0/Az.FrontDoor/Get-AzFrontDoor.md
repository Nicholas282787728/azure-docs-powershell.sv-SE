# Get-AzFrontDoor

## Sammanfattning
Få belastnings utjämning på främre dörren

## FRÅGESYNTAXEN

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Med **Get-AzFrontDoor** cmdletGet får du alla befintliga front dörrar i det aktuella abonnemanget som innehåller information.
