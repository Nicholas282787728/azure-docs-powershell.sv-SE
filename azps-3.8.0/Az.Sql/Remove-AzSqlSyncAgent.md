# Remove-AzSqlSyncAgent

## Sammanfattning
Tar bort en Azure SQL Sync-agent.

## FRÅGESYNTAXEN

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzSqlSyncAgent** tar bort en Azure SQL Sync-agent.

## BESKRIVS

### Exempel 1: ta bort en synkroniseringsresurs
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Det här kommandot tar bort Azure SQL Sync-agenten med namnet syncAgent01.
