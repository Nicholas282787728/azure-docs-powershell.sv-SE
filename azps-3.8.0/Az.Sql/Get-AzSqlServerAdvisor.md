# Get-AzSqlServerAdvisor

## Sammanfattning
Får en eller flera rådgivare till en Azure SQL Server.

## FRÅGESYNTAXEN

```
Get-AzSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSqlServerAdvisor** hämtar en eller flera Azure SQL Server-rådgivare för en Azure SQL Server.
