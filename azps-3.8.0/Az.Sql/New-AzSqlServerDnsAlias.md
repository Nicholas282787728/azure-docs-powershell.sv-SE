# New-AzSqlServerDnsAlias

## Sammanfattning
Det här kommandot skapar ett nytt DNS-alias för Azure SQL Server.

## FRÅGESYNTAXEN

```
New-AzSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar ett nytt DNS-alias för Azure SQL Server som pekar på angiven server.

## BESKRIVS

### Exempel 1
```
PS C:\> $serverDNSAlias = New-AzSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Det här kommandot skapar DNS-aliaset för Azure SQL Server med namnet aliasName som pekar på Server namn
