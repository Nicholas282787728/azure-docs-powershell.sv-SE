# New-AzSqlSyncAgentKey

## Sammanfattning
Skapar en Azure SQL Sync-agenttjänsten.

## FRÅGESYNTAXEN

```
New-AzSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzSqlSyncAgentKey** skapar en Azure SQL Sync-agenttjänsten.

## BESKRIVS

### Exempel 1: skapa en synkroniseringsresurs för en Azure SQL-synkroniseringsklient.
```
PS C:\> New-AzSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Det här kommandot skapar en synkroniseringsresurs för en Azure SQL-synkroniseringsresurs.
