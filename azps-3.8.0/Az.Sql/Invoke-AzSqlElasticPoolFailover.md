# Invoke-AzSqlElasticPoolFailover

## Sammanfattning
Redundansväxlas en elastisk pool.

## FRÅGESYNTAXEN

```
Invoke-AzSqlElasticPoolFailover [-ElasticPoolName] <String> [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Invoke-AzSqlElasticPoolFailover cmdlet redundans en elastisk pool. Failover sker i alla databaser i den elastiska poolen när denna cmdlet har körts.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Invoke-AzSqlElasticPoolFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Det här kommandot kommer att redundansväxla den elastiska poolen med namnet "ElasticPool01" på servern med namnet "Server01".  Det innebär att failover sker i alla databaser i den elastiska poolen med namnet "ElasticPool01".
