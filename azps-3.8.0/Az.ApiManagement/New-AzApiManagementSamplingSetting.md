# New-AzApiManagementSamplingSetting

## Sammanfattning
Skapa en ny samplings inställning för diagnostiken

## FRÅGESYNTAXEN

```
New-AzApiManagementSamplingSetting [-SamplingType <String>] [-SamplingPercentage <Double>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdlet **New-AzApiManagementSamplingSetting** skapar en ny samplings inställning för diagnostiken

## BESKRIVS

### Exempel 1: skapa en grundläggande exempel inställning
```powershell
PS C:\> New-AzApiManagementSamplingSetting -SamplingType fixed -Percentage 100

SamplingType Percentage
------------ ----------
fixed               100
```

Skapar en samplings inställning av `Fixed` typen med loggning för 100% av begäran/svar
