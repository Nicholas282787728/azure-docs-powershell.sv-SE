# New-AzApiManagementProperty

## Sammanfattning
Skapar en ny egenskap.

## FRÅGESYNTAXEN

```
New-AzApiManagementProperty -Context <PsApiManagementContext> [-PropertyId <String>] -Name <String>
 -Value <String> [-Secret] [-Tag <String[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzApiManagementProperty** skapar en Azure API Management- **egenskap**.

## BESKRIVS

### Exempel 1: skapa en egenskap som innehåller Taggar
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>$Tags = 'sdk', 'powershell'
PS C:\> New-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -Name "Property Name" -Value "Property Value" -Tags $Tags
```

Det första kommandot tilldelar två värden till $Tags variabel.
Det andra kommandot skapar en egenskap och tilldelar strängarna i $Tags som taggar på egenskapen.

### Exempel 2: skapa en egenskap som har ett hemligt värde
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementProperty -Context $apimContext -PropertyId "Property12" -Name "Secret Property" -Value "Secret Property Value" -Secret
```

Det här kommandot skapar en **egenskap** som har ett krypterat värde.
