# Remove-AzApiManagement

## Sammanfattning
Tar bort en API-hanterings tjänst.

## FRÅGESYNTAXEN

```
Remove-AzApiManagement -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzApiManagement** tar bort en Azure API Management-tjänst.

## BESKRIVS

### Exempel 1: ta bort en API-hanterings tjänst
```
PS C:\>Remove-AzApiManagement -ResourceGroupName "ContosoGroup02" -Name "ContosoApi"
```

Det här kommandot tar bort API-hanteringskonsolen med namnet ContosoApi.
