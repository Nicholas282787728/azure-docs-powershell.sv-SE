# Get-AzDataLakeGen2Item

## Sammanfattning
Hämtar information om en fil eller katalog i ett fil system.

## FRÅGESYNTAXEN

```
Get-AzDataLakeGen2Item [-FileSystem] <String> [-Path <String>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzDataLakeGen2Item** hämtar information om en fil eller katalog i ett fil system i ett Azure Storage-konto.
Denna cmdlet fungerar bara om hierarkisk namnrymd är aktiverat för lagrings kontot. Den här typen av konto kan skapas med kör "New-AzStorageAccount"-cmdlet med "-EnableHierarchicalNamespace $true".
