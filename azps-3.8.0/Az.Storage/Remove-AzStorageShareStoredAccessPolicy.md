# Remove-AzStorageShareStoredAccessPolicy

## Sammanfattning
Tar bort en lagrad åtkomst princip från en lagrings plats.

## FRÅGESYNTAXEN

```
Remove-AzStorageShareStoredAccessPolicy [-ShareName] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzStorageShareStoredAccessPolicy** tar bort en lagrad åtkomst princip från en Azure Storage-resurs.

## BESKRIVS

### Exempel 1: ta bort en lagrad åtkomst princip från en Azure-lagringsenhet
```
PS C:\>Remove-AzStorageShareStoredAccessPolicy -ShareName "ContosoShare" -Policy "GeneralPolicy"
```

Det här kommandot tar bort en lagrad åtkomst princip med namnet GeneralPolicy från ContosoShare.
