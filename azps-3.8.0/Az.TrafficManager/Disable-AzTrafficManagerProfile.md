# Disable-AzTrafficManagerProfile

## Sammanfattning
Inaktiverar en Traffic Manager-profil.

## FRÅGESYNTAXEN

### Fält
```
Disable-AzTrafficManagerProfile -Name <String> -ResourceGroupName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Serverobjektet
```
Disable-AzTrafficManagerProfile -TrafficManagerProfile <TrafficManagerProfile> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **disable-AzTrafficManagerProfile** inaktiverar en Azure Traffic Manager-profil.
Du kan ange profil objekt med hjälp av pipeline eller som ett parameter värde.
Alternativt kan du ange profilen med hjälp av parametrarna *Name* och *ResourceGroupName* .

## BESKRIVS

### Exempel 1: inaktivera en profil som anges med namn
```
PS C:\>Disable-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11"
```

Det här kommandot inaktiverar profilen med namnet ContosoProfile i ResourceGroup11.
Med kommandot uppmanas du att bekräfta.

### Exempel 2: inaktivera en profil med hjälp av pipeline
```
PS C:\>Get-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11" | Disable-AzTrafficManagerProfile -Force
```

Det här kommandot får profilen med namnet ContosoProfile i ResourceGroup11.
Då överförs den profilen till cmdleten **disable-AzTrafficManagerProfile** med hjälp av pipeline-operatorn.
Denna cmdlet inaktiverar profilen.
Kommandot anger parametern *Force* .
Därför behöver du inte bekräfta.
