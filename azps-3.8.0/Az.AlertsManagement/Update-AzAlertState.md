# Update-AzAlertState

## Sammanfattning
Uppdaterar aviserings tillståndet

## FRÅGESYNTAXEN

### ByAlertId (standard)
```
Update-AzAlertState -AlertId <String> -State <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Update-AzAlertState -State <String> -InputObject <PSAlert> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Update-AzAlertState** cmdlet uppdaterar aviserings tillståndet.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Update-AzAlertState -AlertId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" -State "Closed"
```

Denna cmdlet uppdaterar aviserings tillståndet till stängt.
