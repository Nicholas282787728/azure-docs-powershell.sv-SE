# Set-AzDtlAllowedVMSizesPolicy

## Sammanfattning
Ställer in en policy för tillåten virtuell dator storlek för en laboratorie i DevTest-labb.

## FRÅGESYNTAXEN

### Aktivera (standard)
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Aktiverar
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzDtlAllowedVMSizesPolicy** anger den tillåtna principen för den virtuella dator storleken, som anger en lista över storlekar för virtuell dator som är tillåtna i en labb miljö.
Cmdleten använder den angivna resurs gruppen och namnet på laboratoriet för att ange principen.
