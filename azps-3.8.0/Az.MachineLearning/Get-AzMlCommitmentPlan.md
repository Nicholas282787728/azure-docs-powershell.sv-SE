# Get-AzMlCommitmentPlan

## Sammanfattning
Hämtar den sammanfattande informationen för ett eller flera utfästelser.

## FRÅGESYNTAXEN

```
Get-AzMlCommitmentPlan [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämtar information om åtagande planen.
Beroende på vilka parametrar som skickas returnerar cmdleten ett specifikt åtagande, en samling av åtagande planer för en viss resurs grupp inom det aktuella abonnemanget, eller en samling av åtagande planer inom det aktuella abonnemanget.

## BESKRIVS

### Exempel 1: skaffa en särskild åtagande plan
```
Get-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName"
```

### Exempel 2: Hämta alla åtagande Plans resurser i aktuell prenumeration
```
Get-AzMlCommitmentPlan
```

### Exempel 3: samla alla utfästelser i det aktuella abonnemanget och resurs gruppen
```
Get-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup"
```
