# Update-AzRecoveryServicesAsrServicesProvider

## Sammanfattning
Uppdateringar (uppdatera Server) informationen från Azure Site Recovery Services-tjänsten.

## FRÅGESYNTAXEN

```
Update-AzRecoveryServicesAsrServicesProvider -InputObject <ASRRecoveryServicesProvider>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Update-AzRecoveryServicesAsrServicesProvider** uppdaterar informationen från Azure Site Recovery Services-tjänsten. Du kan använda denna cmdlet för att utlösa en uppdatering av informationen från Recovery Services-leverantören.

## BESKRIVS

### Exempel 1
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrServicesProvider -InputObject $ServicesProvider
```

Startar uppdateringen av informationen från den angivna ASR-leverantören och returnerar det ASR-jobb som används för att spåra åtgärden.
