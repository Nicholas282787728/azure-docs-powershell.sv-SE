# Get-AzDeletedWebApp

## Sammanfattning
Tar bort webb program i prenumerationen.

## FRÅGESYNTAXEN

```
Get-AzDeletedWebApp [[-ResourceGroupName] <String>] [[-Name] <String>] [[-Slot] <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzDeletedWebApp** returnerar alla borttagna webb program i prenumerationen. Borttagna appar kan filtreras efter resurs grupp, namn och plats. Det kan finnas fler än en borttagen app med samma namn och resurs grupp. Kontrol lera DeletionTime för att skilja på borttagna appar som har samma namn.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzDeletedWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Det här kommandot hämtar de borttagna apparna som heter ContosoSite som tillhör resurs gruppens standard-väst.
