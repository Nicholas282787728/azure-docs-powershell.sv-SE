# Remove-AzAutomationModule

## Sammanfattning
Tar bort en modul från Automation.

## FRÅGESYNTAXEN

```
Remove-AzAutomationModule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzAutomationModule** tar bort en modul från ett Automation-konto i Azure Automation.

## BESKRIVS

### Exempel 1: ta bort en modul
```
PS C:\>Remove-AzAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Det här kommandot tar bort en modul som heter ContosoModule från Automation-kontot med namnet Contoso17.
