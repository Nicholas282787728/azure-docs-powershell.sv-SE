# Save-AzTenantDeploymentTemplate

## Sammanfattning
Sparar en distributionsmall till en fil.

## FRÅGESYNTAXEN

### SaveByDeploymentName (standard)
```
Save-AzTenantDeploymentTemplate -DeploymentName <String> [-Path <String>] [-Force] [-ApiVersion <String>]
 [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SaveByDeploymentObject
```
Save-AzTenantDeploymentTemplate -DeploymentObject <PSDeployment> [-Path <String>] [-Force]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Save-AzTenantDeploymentTemplate**  sparar en distributionsmall till en JSON-fil för en distribution med klient omfattning.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Save-AzTenantDeploymentTemplate -DeploymentName "TestDeployment"
```

Det här kommandot får distributions mal len från TestDeployment och sparar den som en JSON-fil i den aktuella katalogen.

### Exempel 2: Hämta en distribution och spara dess mall
```
PS C:\>Get-AzTenantDeploymentTemplate -Name "RolesDeployment" | Save-AzTenantDeploymentTemplate
```

Det här kommandot får distributionen "RolesDeployment" vid den aktuella klient organisations omfattningen och sparar mallen.
