# Get-AzPolicySetDefinition

## Sammanfattning
Hämtar definitioner för princip uppsättningar.

## FRÅGESYNTAXEN

### NameParameterSet (standard)
```
Get-AzPolicySetDefinition [-Name <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ManagementGroupNameParameterSet
```

 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionIdParameterSet
```

 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdParameterSet

```

### BuiltinFilterParameterSet
```

```

### CustomFilterParameterSet

```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzPolicySetDefinition** hämtar en samling princip uppsättnings definitioner eller en specifik definition för princip uppsättning som identifieras med namn eller ID.

## BESKRIVS

### Exempel 1: Hämta alla definitioner för princip uppsättningar
```
PS C:\> Get-AzPolicySetDefinition
```

Det här kommandot får alla definitioner för princip uppsättningar.

### Exempel 2: Hämta definitionen av princip uppsättning från aktuell prenumeration efter namn
```
PS C:\> Get-AzPolicySetDefinition -Name 'VMPolicySetDefinition'
```

Det här kommandot hämtar princip uppsättnings definitionen med namnet VMPolicySetDefinition från den aktuella standard prenumerationen.

### Exempel 3: Hämta princip uppsättnings definition från abonnemang efter namn
```
PS C:\> Get-AzPolicySetDefinition -Name 'VMPolicySetDefinition' -subscriptionId '3bf44b72-c631-427a-b8c8-53e2595398ca'
```

Det här kommandot hämtar princip definitionen med namnet VMPolicySetDefinition från prenumerationen med ID 3bf44b72-c631-427a-b8c8-53e2595398ca.

### Exempel 4: Hämta alla definitioner för anpassade principer uppsättning från en hanterings grupp
```
PS C:\> Get-AzPolicySetDefinition -ManagementGroupName 'Dept42' -Custom
```

Det här kommandot får alla anpassade princip uppsättnings definitioner från hanterings gruppen med namnet Dept42.

### Exempel 5: Hämta princip uppsättnings definitioner från en viss kategori
```
PS C:\> Get-AzPolicySetDefinition | where-object {$_.Properties.metadata.category -eq "Virtual Machine"}
```

Det här kommandot får alla definitioner för princip uppsättningar i kategorin "virtuell dator".
