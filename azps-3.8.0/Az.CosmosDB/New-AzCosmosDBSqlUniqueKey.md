# New-AzCosmosDBSqlUniqueKey

## Sammanfattning
Skapar ett nytt CosmosDB SQL UniqueKey-objekt.

## FRÅGESYNTAXEN

```
New-AzCosmosDBSqlUniqueKey -Path <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**New-AzCosmosDBSqlUniqueKey-** cmdleten skapar ett nytt objekt av typen PSUniqueKey.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> New-AzCosmosDBSqlUniqueKey -Path {path}

Path
----
{path}
```
