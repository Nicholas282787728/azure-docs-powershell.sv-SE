# Get-AzCosmosDBSqlContainer

## Sammanfattning
Hämtar SQL-containern för CosmosDB.

## FRÅGESYNTAXEN

### ByNameParameterSet (standard)
```
Get-AzCosmosDBSqlContainer -ResourceGroupName <String> [-Name <String>] [-Detailed]
 [-DefaultProfile <IAzureContextContainer>] -AccountName <String> -DatabaseName <String> [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBSqlContainer [-Name <String>] -InputObject <PSSqlDatabaseGetResults> [-Detailed]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzCosmosDBSqlContainer** hämtar listan över alla befintliga CosmosDB SQL-behållare för ett givet ResourceGroupName, AccountName och databasename och hämtar en enda CosmosDB SQL-behållare för ett givet ResourceGroupName, AccountName, databasename och ContainerName.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzCosmosDBSqlContainer -AccountName {accountName} -ResourceGroupName {resourceGroupName} -DatabaseName {databaseName}

Name                     : {containerName1}
Id                       : Id
Resource                 : Microsoft.Azure.Commands.CosmosDB.Models.PSSqlContainerGetPropertiesResource
```
