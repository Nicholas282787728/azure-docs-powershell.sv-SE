# Set-AzCosmosDBSqlDatabase

## Sammanfattning
Skapar en ny eller uppdaterar en befintlig CosmosDB SQL-databas.

## FRÅGESYNTAXEN

### ByNameParameterSet (standard)
```
Set-AzCosmosDBSqlDatabase -ResourceGroupName <String> -AccountName <String> -Name <String>
 [-Throughput <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Set-AzCosmosDBSqlDatabase -Name <String> [-Throughput <Int32>] -InputObject <PSDatabaseAccount>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzCosmosDBSqlDatabase** skapar en ny eller uppdaterar en befintlig CosmosDB SQL-databas.
