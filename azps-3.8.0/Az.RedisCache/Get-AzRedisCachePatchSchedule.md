# Get-AzRedisCachePatchSchedule

## Sammanfattning
Får ett uppdaterings schema.

## FRÅGESYNTAXEN

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzRedisCachePatchSchedule** får ett uppdaterings schema för ett cacheminne i Azure Redis-cachen.

## BESKRIVS

### Exempel 1: få uppdaterings schema
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Det här kommandot får ett uppdaterings schema från cachen som heter RedisCache06.
