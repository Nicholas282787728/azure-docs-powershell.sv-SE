# Get-AzPowerBIEmbeddedCapacity

## Sammanfattning
Hämtar information om en inbäddad PowerBI kapacitet.

## FRÅGESYNTAXEN

### ByCapacityOrResourceGroupOrSubscription (standard)
```
Get-AzPowerBIEmbeddedCapacity [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzPowerBIEmbeddedCapacity -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Get-AzPowerBIEmbeddedCapacity cmdlet får information om en inbäddad kapacitet för PowerBI.
