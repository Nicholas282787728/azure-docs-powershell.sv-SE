# Reset-AzPowerBIWorkspaceCollectionAccessKey

## Sammanfattning
Återställer den angivna åtkomst tangenten.

## FRÅGESYNTAXEN

```
Reset-AzPowerBIWorkspaceCollectionAccessKey [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Reset-AzPowerBIWorkspaceCollectionAccessKey** återställer den angivna åtkomst tangenten i din Power BI-arbetsyta.

## BESKRIVS

### Exempel 1: Återställ primär åtkomst-tangenten
```
PS C:\>Reset-AzPowerBIWorkspaceCollectionAccessKey -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Det här kommandot återställer den primära åtkomst-tangenten för arbets ytan med namnet WCN11 i den angivna resurs gruppen.
