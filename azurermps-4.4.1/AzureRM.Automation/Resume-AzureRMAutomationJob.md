# Resume-AzureRmAutomationJob

## Sammanfattning
Återupptar ett uppehållet automatiserings jobb.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Resume-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Resume-AzureRmAutomationJob** återupptar ett inaktiverat Azure Automation-jobb.
Ange det upphävda jobbet.

Om du vill pausa ett jobb kan du använda Suspend-AzureRmAutomationJob cmdlet.

## BESKRIVS

### Exempel 1: återuppta ett avbrutet jobb
```
PS C:\>Resume-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Det här kommandot återupptar jobbet som har angivet ID.
