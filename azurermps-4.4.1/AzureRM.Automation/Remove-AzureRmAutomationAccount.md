# Remove-AzureRmAutomationAccount

## Sammanfattning
Tar bort ett Automation-konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmAutomationAccount** tar bort ett Azure Automation-konto från en resurs grupp.

Mer information om automatiserings konton finns i New-AzureRmAutomationAccount cmdlet.

## BESKRIVS

### Exempel 1: ta bort ett Automation-konto
```
PS C:\>Remove-AzureRmAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

Det här kommandot tar bort ett Automation-konto med namnet ContosoAutomationAccount utan att fråga efter användar verifiering.
