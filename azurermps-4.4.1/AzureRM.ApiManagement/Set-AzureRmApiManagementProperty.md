# Set-AzureRmApiManagementProperty

## Sammanfattning
Ändrar en API-egenskap.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Set-AzureRmApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-Name <String>]
 [-Value <String>] [-Secret <Boolean>] [-Tags <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureRmApiManagementProperty** ändrar en egenskap för Azure API Management.

## BESKRIVS

### Exempel 1: ändra taggarna för en egenskap
```
PS C:\>$Tags = 'sdk', 'powershell'
PS C:\> Set-AzureRmApiManagementProperty -Context $ApimContext -PropertyId "Property11" -Tags $Tags -PassThru
```

Det första kommandot tilldelar två värden till $Tags variabel.

Det andra kommandot ändrar egenskapen som har ID-Property11.
Kommandot tilldelar strängarna i $Tags som taggar på egenskapen.

### Exempel 2: ändra en egenskap så att den har ett hemligt värde
```
PS C:\>Set-AzureRmApiManagementProperty -Context $ApimContext -PropertyId "Property12" -Secret $True -PassThru
```

Det här kommandot ändrar egenskapen som är krypterad.
