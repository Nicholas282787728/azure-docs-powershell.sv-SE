# Remove-AzureRmApiManagementPolicy

## Sammanfattning
Tar bort API-hanteringskonsolen från ett angivet scope.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### Klient organisations nivå (standard)
```
Remove-AzureRmApiManagementPolicy -Context <PsApiManagementContext> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Produkt nivå
```
Remove-AzureRmApiManagementPolicy -Context <PsApiManagementContext> -ProductId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### API-nivå
```
Remove-AzureRmApiManagementPolicy -Context <PsApiManagementContext> -ApiId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Drifts nivå
```
Remove-AzureRmApiManagementPolicy -Context <PsApiManagementContext> -ApiId <String> -OperationId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmApiManagementPolicy** tar bort API-hanterings principen från angivet scope.

## BESKRIVS

### Exempel 1: ta bort principen för klient organisations nivå
```
PS C:\>Remove-AzureRmApiManagementPolicy -Context $APImContext
```

Det här kommandot tar bort policyn för klient organisations nivå från API-hanteringen.

### Exempel 2: ta bort produktens policy princip
```
PS C:\>Remove-AzureRmApiManagementPolicy -Context $APImContext -ProductId "0123456789"
```

Det här kommandot tar bort produktens princip från API-hanteringen.

### Exempel 3: ta bort API-scope-principen
```
PS C:\>Remove-AzureRmApiManagementPolicy -Context $APImContext -ApiId "9876543210"
```

Det här kommandot tar bort API-scope-principen från API-hanteringen.

### Exempel 4: ta bort åtgärdens scope-princip
```
PS C:\>Remove-AzureRmApiManagementPolicy -Context $APImContext -ApiId "9876543210" -OperationId "777"
```

Det här kommandot tar bort åtgärdens omfattning från API-hanteringen.
