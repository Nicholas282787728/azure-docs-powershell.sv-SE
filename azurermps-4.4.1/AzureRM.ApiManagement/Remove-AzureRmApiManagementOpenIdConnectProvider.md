# Remove-AzureRmApiManagementOpenIdConnectProvider

## Sammanfattning
Tar bort en OpenID Connect-leverantör.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 -OpenIdConnectProviderId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmApiManagementOpenIdConnectProvider** tar bort en OpenID Connect-leverantör för Azure API Management.

## BESKRIVS

### Exempel 1: ta bort en leverantör
```
PS C:\>Remove-AzureRmApiManagementOpenIdConnectProvider -Context $ApimContext -OpenIdConnectProviderId "OICProvicer01" -PassThru
```

Det här kommandot tar bort en Provider med ID-OICProvicer01.
