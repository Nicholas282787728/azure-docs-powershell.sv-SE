# Remove-AzureRmSqlSyncAgent

## Sammanfattning
Tar bort en Azure SQL Sync-agent.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmSqlSyncAgent** tar bort en Azure SQL Sync-agent.

## BESKRIVS

### Exempel 1: ta bort en synkroniseringsresurs
```
PS C:\>Remove-AzureRmSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Det här kommandot tar bort Azure SQL Sync-agenten med namnet syncAgent01.
