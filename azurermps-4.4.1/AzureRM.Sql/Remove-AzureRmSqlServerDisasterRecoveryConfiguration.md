# Remove-AzureRmSqlServerDisasterRecoveryConfiguration

## Sammanfattning
Tar bort en konfiguration för en SQL Database Server System återställning.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmSqlServerDisasterRecoveryConfiguration** tar bort en konfiguration för SQL Database Server System återställning.
