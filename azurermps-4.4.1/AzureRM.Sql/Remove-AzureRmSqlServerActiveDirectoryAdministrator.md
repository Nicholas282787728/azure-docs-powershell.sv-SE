# Remove-AzureRmSqlServerActiveDirectoryAdministrator

## Sammanfattning
Tar bort en Azure AD-administratör för SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmSqlServerActiveDirectoryAdministrator [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmSqlServerActiveDirectoryAdministrator** tar bort en Azure Active Directory (Azure AD-administratör) för AzureSQL-servern i den aktuella prenumerationen.

## BESKRIVS

### Exempel 1: ta bort en administratör
```
PS C:\>Remove-AzureRmSqlServerActiveDirectoryAdministrator -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
Confirm 
Are you sure you want to remove the Azure Sql Server Active Directory Administrator on server 'Server01'? 
[Y] Yes [A] Yes to All [N] No [L] No to All [S] Suspend [?] Help (default is "Y"): Y 

ResourceGroupName ServerName DisplayName ObjectId 
----------------- ---------- ----------- -------- 
ResourceGroup01   Server01   DBAs        40b79501-b343-44ed-9ce7-da4c8cc7353b
```

Det här kommandot tar bort Azure AD-administratören för servern som heter Server01 associerad med resurs gruppen ResourceGroup01.
