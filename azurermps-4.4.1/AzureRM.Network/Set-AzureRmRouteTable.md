# Set-AzureRmRouteTable

## Sammanfattning
Anger mål tillstånd för en väg tabell.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Set-AzureRmRouteTable -RouteTable <PSRouteTable> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureRmRouteTable** anger mål tillståndet för en Azure route-tabell.
