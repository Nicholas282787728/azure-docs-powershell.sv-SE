# Remove-AzureRmDataLakeAnalyticsComputePolicy

## Sammanfattning
Tar bort en angiven beräknings princip för Azure Data Lake Analytics

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Remove-AzureRmDataLakeAnalyticsComputePolicy** tar bort en viss beräknings princip för Azure Data Lake Analytics.

## BESKRIVS

### Exempel 1: ta bort en beräknings princip
```
PS C:\>Remove-AzureRmDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Det här kommandot tar bort den angivna beräknings principen med namnet "policy" i kontot ' contosoadla '.
