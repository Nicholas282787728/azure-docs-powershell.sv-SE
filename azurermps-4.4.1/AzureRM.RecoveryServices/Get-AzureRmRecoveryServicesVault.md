# Get-AzureRmRecoveryServicesVault

## Sammanfattning
Hämtar en lista över återställnings tjänstens valv.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmRecoveryServicesVault [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmRecoveryServicesVault** hämtar en lista över återställnings tjänst valv i den aktuella prenumerationen.
