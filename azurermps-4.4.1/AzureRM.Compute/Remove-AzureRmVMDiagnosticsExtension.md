# Remove-AzureRmVMDiagnosticsExtension

## Sammanfattning
Tar bort tillägget diagnostik från en virtuell dator.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmVMDiagnosticsExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmVMDiagnosticsExtension** tar bort ett Azure Diagnostics-tillägg från en virtuell dator.
Du måste skicka utdata från denna cmdlet till Update-AzureRmVM cmdlet för att implementera ändringarna.

## BESKRIVS

### Exempel 1: ta bort tillägget diagnostik från en virtuell dator
```
PS C:\> Remove-AzureRmVMDiagnosticsExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" | Update-AzureRmVM
```

Det här kommandot tar bort tillägget från en virtuell dator med namnet ContosoVM22.
Kommandot skickar resultatet till Update-AzureRmVM cmdlet med hjälp av pipeline-operatorn.
Det kommandot uppdaterar den virtuella datorn.
