# Get-AzureRmVMADDomainExtension

## Sammanfattning
Hämtar information om ett AD-domännätverk.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmVMADDomainExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmVMADDomainExtension** hämtar information om den angivna Azure Active Directory (AD)-domän tillägget.
