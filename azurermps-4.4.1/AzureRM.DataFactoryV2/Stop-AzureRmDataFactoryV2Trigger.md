# Stop-AzureRmDataFactoryV2Trigger

## Sammanfattning

Stoppar en utlösare i en data fabrik.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### ByFactoryName (standard)
```
Stop-AzureRmDataFactoryV2Trigger [-Name] <String> [-ResourceGroupName] <String> [-DataFactoryName] <String>
 [-Force] [-WhatIf] [-Confirm]
```

### ByInputObject
```
Stop-AzureRmDataFactoryV2Trigger [-InputObject] <PSTrigger> [-Force] [-WhatIf] [-Confirm]
```

### ByResourceId
```
Stop-AzureRmDataFactoryV2Trigger [-ResourceId] <String> [-Force] [-WhatIf] [-Confirm]
```

## PROBLEMBESKRIVNING

Cmdleten **Stop-AzureRmDataFactoryV2Trigger** stoppar en utlösare i en data fabrik. Om utlösaren är i läget startat stoppar cmdleten utlösaren och är inte längre aktive rad. Om utlösaren redan är i tillståndet ' stoppad har denna cmdlet ingen effekt. Om parametern Force anges frågar inte cmdleten innan utlösaren stoppas.


## BESKRIVS

### Exempel 1: stoppa en utlösare

```
Stop-AzureRmDataFactoryV2Trigger -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -TriggerName "ScheduledTrigger"

Confirm
Are you sure you want to stop trigger 'ScheduledTrigger' in data factory 'TestFactory'?
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
True
```

Stoppar en utlösare som heter "ScheduledTrigger" i data fabriken "WikiADF".
