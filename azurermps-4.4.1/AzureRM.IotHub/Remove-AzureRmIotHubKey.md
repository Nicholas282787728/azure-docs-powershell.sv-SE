# Remove-AzureRmIotHubKey

## Sammanfattning
Tar bort en IotHub-tangenten.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Tar bort en IotHub-tangenten.
Om det finns flera tangenter med samma namn tas den första i listan bort.

## BESKRIVS

### Exempel 1 ta bort en IotHub
```
PS C:\> Remove-AzureRmIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner1"
```

Tar bort den iothubowner1 från IotHub som heter "myiothub"
