# New-AzureRmAnalysisServicesServer

## Sammanfattning
Skapar en ny Analysis Services-server

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmAnalysisServicesServer [-ResourceGroupName] <String> [-Name] <String> [-Location] <String>
 [-Sku] <String> [[-Tag] <Hashtable>] [[-Administrator] <String>] [[-BackupBlobContainerUri] <String>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
New-AzureRmAnalysisServicesServer-cmdleten skapar en ny Analysis Services-server

## BESKRIVS

### Exempel 1
```
PS C:\> New-AzureRmAnalysisServicesServer -ResourceGroupName "testresourcegroup" -Name "testserver" -Location "West-US" -Sku "S1"
```

Skapar en server med namnet testserver i Azure-regionen West-US och i resurs grupp testresrourcegroup. SKU-nivån för servern blir S1.
