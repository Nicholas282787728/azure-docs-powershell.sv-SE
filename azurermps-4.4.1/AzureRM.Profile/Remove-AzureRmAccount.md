# Remove-AzureRmAccount

## Sammanfattning
Ta bort alla autentiseringsuppgifter och kontexter som är kopplade till det angivna kontot.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### ContextName (standard)
```
Remove-AzureRmAccount [-ContextName <String>] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Användar
```
Remove-AzureRmAccount [-Username] <String> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ServicePrincipal
```
Remove-AzureRmAccount -ApplicationId <String> -TenantId <String> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountObject
```
Remove-AzureRmAccount [-InputObject] <PSAzureRmAccount> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ContextObject
```
Remove-AzureRmAccount [-AzureContext] <PSAzureContext> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Ta bort alla autentiseringsuppgifter och kontexter som är kopplade till det angivna kontot.  När du har kört denna cmdlet måste du logga in igen med Add-AzureRmAccount

## BESKRIVS

### Logga ut från curent-kontot
```
PS C:\> Remove-AzureRmAccount
```

Loggar ut det konto som är kopplat till den aktuella kontexten.

### Logga ut från det konto som är kopplat till en viss kontext
```
PS C:\> Get-AzureRmContext "Work" | Remove-AzureRmAccount -Scope CurrentUser
```

Loggar ut det konto som är kopplat till angiven kontext (med namnet "arbete"). Eftersom "CurrentUser"-området används kommer alla autentiseringsuppgifter och kontexter att tas bort permanent.

### Logga ut en viss användare
```
PS C:\> Remove-AzureRmAccount -Username 'user1@contoso.org'
```

Loggar ut den användare user1@contoso.org som är associerad med den här användaren kommer att tas bort.
