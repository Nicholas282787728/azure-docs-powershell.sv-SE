# Remove-AzureRmServiceBusNamespaceAuthorizationRule

## Sammanfattning
Tar bort auktoriseringsregeln för ett Service Bus-namnområde från den angivna resurs gruppen.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmServiceBusNamespaceAuthorizationRule -ResourceGroupName <String> -Namespace <String>
 -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmServiceBusNamespaceAuthorizationRule** tar bort auktoriseringsregeln för ett Service Bus-namnområde för den angivna resurs gruppen.

## BESKRIVS

### Exempel 1
```
PS C:\> Remove-AzureRmServiceBusNamespaceAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -AuthorizationRuleName AuthoRule1
```

Tar bort auktoriseringsregeln `SBAuthoRule1` för namn område `SB-Example1` från den angivna resurs gruppen.
