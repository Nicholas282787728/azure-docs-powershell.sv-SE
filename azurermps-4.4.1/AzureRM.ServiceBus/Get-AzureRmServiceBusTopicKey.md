# Get-AzureRmServiceBusTopicKey

## Sammanfattning
Hämtar de primära och sekundära anslutnings strängarna för Service Bus-ämnet.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmServiceBusTopicKey [-ResourceGroup] <String> -Namespace <String> -Topic <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmServiceBusTopicKey** returnerar primära och sekundära anslutnings strängar för den angivna tjänst Bus artikeln.

## BESKRIVS

### Exempel 1
```
PS C:\> Get-AzureRmServiceBusTopicKey -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -TopicName SB-Topic_exampl1 -AuthorizationRuleName SBTopicAuthoRule1
```

Returnerar de primära och sekundära anslutnings strängarna för den angivna tjänst Bus artikeln.
