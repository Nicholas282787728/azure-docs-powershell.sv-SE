# Get-AzureRmServiceBusNamespaceAuthorizationRule

## Sammanfattning
Hämtar en beskrivning av den angivna auktoriseringsregeln för ett angivet namn område. 

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmServiceBusNamespaceAuthorizationRule [-ResourceGroup] <String> -Namespace <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmServiceBusNamespaceAuthorizationRule** hämtar beskrivningen av den angivna auktoriseringsregeln i det angivna namn området.

## BESKRIVS

### Exempel 1
```
PS C:\> Get-AzureRmServiceBusNamespaceAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -AuthorizationRuleName AuthoRule1
```

Returnerar den angivna beskrivningen för auktoriseringsregeln för ett angivet namn område.
