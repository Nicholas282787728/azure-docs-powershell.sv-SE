# Disable-AzureBatchJobSchedule

## Sammanfattning
Inaktiverar schemaläggning av batchjobb.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Disable-AzureBatchJobSchedule [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **disable-AzureBatchJobSchedule** inaktiverar ett Azure Batch-schema.
Om du inaktiverar ett schema skapas inte jobb enligt det schemat.
Du kan aktivera ett inaktiverat schema senare.

## BESKRIVS

### Exempel 1: inaktivera en finplanera
```
PS C:\>Disable-AzureBatchJobSchedule -Id "JobSchedule17" -BatchContext $Context
```

Det här kommandot inaktiverar projektschemat med ID-JobSchedule17.
Använd cmdleten **Get-AzureRmBatchAccountKeys** för att tilldela en kontext till variabeln $context.
