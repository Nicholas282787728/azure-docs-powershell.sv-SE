# Remove-AzureBatchPool

## Sammanfattning
Tar bort angiven gruppbearbetningssekvens.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureBatchPool [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureBatchPool** tar bort angiven Azure-grupppool.
Du uppmanas att bekräfta om du inte använder parametern *Force* .

## BESKRIVS

### Exempel 1: ta bort en grupppool utifrån pool-ID
```
PS C:\>Remove-AzureBatchPool -Id "MyPool" -BatchContext $Context
```

Det här kommandot tar bort poolen med ID-nummer.
Användaren uppmanas att bekräfta innan borttagningen utförs.

### Exempel 2: ta bort alla batchattribut per tvång
```
PS C:\>Get-AzureBatchPool -BatchContext $Context | Remove-AzureBatchPool -Force -BatchContext $Context
```

Det här kommandot tar bort alla batchattribut.
Eftersom *Force* -parametern finns ignoreras bekräftelse meddelandet.
