# Remove-AzureRmBatchApplicationPackage

## Sammanfattning
Tar bort en post för programpaket och den binära filen.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationId] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmBatchApplicationPackage** tar bort en post för programpaketet och den binära filen från ett Azure Batch-konto.

## BESKRIVS

### Exempel 1: ta bort ett programpaket från ett batch-konto
```
PS C:\>Remove-AzureRmBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "litware" -ApplicationVersion "1.0"
```

Det här kommandot tar bort version 1,0 av Litware-programmet från ContosoBatchGroup-kontot.
Kommandot tar bort både paket posten och blobben som innehåller den binära filen.
