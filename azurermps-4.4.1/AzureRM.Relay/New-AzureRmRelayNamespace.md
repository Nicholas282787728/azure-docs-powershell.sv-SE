# New-AzureRmRelayNamespace

## Sammanfattning
Skapar ett nytt relä namn.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
New-AzureRmRelayNamespace [-ResourceGroupName] <String> -Name <String> [-Location] <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzureRmRelayNamespace** skapar ett nytt relä namn. När namn rymds resursens manifest har skapats är det oföränderligt.

## BESKRIVS

### Exempel 1
```
PS C:\> New-AzureRmRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Location "West US" -Tag @{Tag1="Tag1Value"}

Name               : TestNameSpace-Relay1
```

Skapar ett nytt relä namn område i den angivna resurs gruppen.
