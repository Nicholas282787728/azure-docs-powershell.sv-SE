# Remove-AzureRmSiteRecoveryNetworkMapping

## Sammanfattning
Tar bort en nätverks mappning från det aktuella återställnings valvet för webbplatsen.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmSiteRecoveryNetworkMapping -NetworkMapping <ASRNetworkMapping>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRMSiteRecoveryNetworkMapping** tar bort en nätverks mappning från det aktuella Azure Site Recovery-valvet.
