# Get-AzOperationalInsightsIntelligencePack

## Sammanfattning
Hämtar tillgängliga intelligens paket.

## FRÅGESYNTAXEN

```
Get-AzOperationalInsightsIntelligencePack [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzOperationalInsightsIntelligencePack** hämtar de tillgängliga intelligens paketen.

## BESKRIVS

### Exempel 1: skaffa paketlösningar
```
PS C:\>Get-AzOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Det här kommandot får de tillgängliga intelligens-paketen.
