# Remove-AzVirtualNetworkTap

## Sammanfattning
Tar bort ett virtuellt nätverk tryck.

## FRÅGESYNTAXEN

### RemoveByNameParameterSet (standard)
```
Remove-AzVirtualNetworkTap -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzVirtualNetworkTap -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzVirtualNetworkTap -InputObject <PSVirtualNetworkTap> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzVirtualNetworkTap** tar bort ett Azure Virtual Network-tryck.

## BESKRIVS

### Exempel 1: ta bort ett virtuellt nätverk tryck
```
PS C:\>Remove-AzNetworkInterface -Name "VirtualNetworkTap1" -ResourceGroup "ResourceGroup1"
```

Det här kommandot tar bort VirtualNetworkTap1 i ResourceGroup1.
Eftersom *Force* -parametern inte används uppmanas användaren att bekräfta åtgärden.
