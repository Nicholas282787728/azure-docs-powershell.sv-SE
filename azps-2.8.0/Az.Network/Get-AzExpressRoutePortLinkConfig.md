# Get-AzExpressRoutePortLinkConfig

## Sammanfattning
Hämtar en ExpressRoutePort.

## FRÅGESYNTAXEN

### ResourceNameParameterSet (standard)
```
Get-AzExpressRoutePortLinkConfig -ExpressRoutePort <PSExpressRoutePort> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePortLinkConfig -ResourceId <String> -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzExpressRoutePortLinkConfig** hämtar konfigurationen för en länk till en ExpressRoutePort.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -Name Link1
```

Hämtar link1-konfigurationen för ExpressRoutePort $erport

### Exempel 2
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -ResourceId $id
```

Hämtar konfigurationen för en länk med ResourceId-$id i ExpressRoutePort $erport
