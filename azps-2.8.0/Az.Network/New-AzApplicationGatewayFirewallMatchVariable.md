# New-AzApplicationGatewayFirewallMatchVariable

## Sammanfattning
Skapar en träff-variabel för brand Väggs villkor.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayFirewallMatchVariable -VariableName <String> [-Selector <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**New-AzApplicationGatewayFirewallMatchVariable** skapar en match-variabel för brand Väggs villkor.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $variable = New-AzApplicationGatewayFirewallMatchVariable -VariableName RequestHeaders -Selector Content-Length
```

Kommandot skapar en ny matcha-variabel med namnet på begärandehuvuden och väljaren för innehålls längd. Den nya matcha-variabeln sparas i $variable.
