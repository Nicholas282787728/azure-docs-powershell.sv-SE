# New-AzApplicationGatewayRewriteRuleHeaderConfiguration

## Sammanfattning
Skapar en omskrivnings huvud konfiguration för en Programgateway.

## FRÅGESYNTAXEN

```
New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName <String> [-HeaderValue <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**AzApplicationGatewayRewriteRuleHeaderConfiguration** -cmdleten skapar en åtgärd för att skriva om en regel för en Azure Application Gateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $hc = New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName abc -HeaderValue def
```

Det här kommandot skapar en rapport huvud konfiguration och lagrar resultatet i variabeln som heter $hc.
