# Get-AzExpressRouteCrossConnectionArpTable

## Sammanfattning
Hämtar ARP-tabellen från en ExpressRoute-kors anslutning.

## FRÅGESYNTAXEN

### SpecifyByParameterValues
```
Get-AzExpressRouteCrossConnectionArpTable -ResourceGroupName <String> -CrossConnectionName <String>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### SpecifyByReference
```
Get-AzExpressRouteCrossConnectionArpTable -ExpressRouteCrossConnection <PSExpressRouteCrossConnection>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzExpressRouteCrossConnectionARPTable** hämtar ARP-tabellen från båda gränssnitten av en ExpressRoute-kors anslutning. ARP-tabellen tillhandahåller en mappning av IPv4-adress till MAC-adress för en viss peering. Du kan använda ARP-tabellen för att verifiera nivå 2-konfiguration och anslutning.

## BESKRIVS

### Exempel 1: Visa ARP-tabellen för en ExpressRoute-peer
```
Get-AzExpressRouteCrossConnectionARPTable -ResourceGroupName $RG -ExpressRouteCrossConnectionName $CrossConnectionName -PeeringType MicrosoftPeering -DevicePath Primary
```
