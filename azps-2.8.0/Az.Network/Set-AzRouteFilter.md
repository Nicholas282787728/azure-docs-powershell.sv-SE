# Set-AzRouteFilter

## Sammanfattning
Uppdaterar ett väg filter.

## FRÅGESYNTAXEN

```
Set-AzRouteFilter -RouteFilter <PSRouteFilter> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzApplicationGateway** uppdaterar ett väg filter

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Set-AzRouteFilter -RouteFilter $rf
```

Det här kommandot uppdaterar Route-filtret med inställningarna i variabeln $rf.
