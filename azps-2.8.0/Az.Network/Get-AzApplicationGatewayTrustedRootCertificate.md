# Get-AzApplicationGatewayTrustedRootCertificate

## Sammanfattning
Hämtar det betrodda rot certifikatet med ett angivet namn från Application Gateway.

## FRÅGESYNTAXEN

```
Get-AzApplicationGatewayTrustedRootCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzApplicationGatewayTrustedRootCertificate** hämtar ett betrott rot certifikat med ett specifikt namn från Application Gateway.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedRootCert = Get-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\rootCA.cer"
```

Med det första kommandot hämtas programgatewayen och lagras i $gw variabel.
Det andra kommandot får det betrodda rot certifikatet med ett angivet namn från Application Gateway.
