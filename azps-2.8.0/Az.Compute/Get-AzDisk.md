# Get-AzDisk

## Sammanfattning
Hämtar egenskaperna för en hanterad disk.

## FRÅGESYNTAXEN

```
Get-AzDisk [[-ResourceGroupName] <String>] [[-DiskName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzDisk** hämtar egenskaperna för en hanterad disk.

## BESKRIVS

### Exempel 1
```
PS C:\> Get-AzDisk -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'

Sku                : Microsoft.Azure.Management.Compute.Models.DiskSku

                     te/disks/Disk01
Name               : Disk01

Location           : westus2

```

Det här kommandot får egenskaperna för disken "Disk01" i resurs gruppen "ResourceGroup01".
