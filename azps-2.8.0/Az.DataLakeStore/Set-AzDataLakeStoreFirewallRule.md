# Set-AzDataLakeStoreFirewallRule

## Sammanfattning
Ändrar den angivna brand Väggs regeln i den angivna data Lake Store.

## FRÅGESYNTAXEN

```
Set-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [[-StartIpAddress] <String>]
 [[-EndIpAddress] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzDataLakeStoreFirewallRule** ändrar den angivna brand Väggs regeln i den angivna data Lake Store.

## BESKRIVS

### Exempel 1: Uppdatera start-och slut adress för en brand Väggs regel
```
PS C:\> Set-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Uppdaterar brand Väggs regeln "MyFirewallRule" i kontot "ContosoADL" för att ha ett område med 127.0.0.1-127.0.0.2
