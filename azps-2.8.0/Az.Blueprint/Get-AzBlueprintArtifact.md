# Get-AzBlueprintArtifact

## Sammanfattning
Hämta artefakter från en skiss definition.

## FRÅGESYNTAXEN

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämta artefakter från en skiss definition. Om du inte anger en version av ritnings definitionen hämtas utkastet. Om det inte finns någon utkast version returneras den senaste publicerade utkasts definitionen.
