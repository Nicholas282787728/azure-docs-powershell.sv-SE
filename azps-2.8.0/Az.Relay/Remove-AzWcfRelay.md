# Remove-AzWcfRelay

## Sammanfattning
Tar bort WcfRelay från det angivna relä namn området.

## FRÅGESYNTAXEN

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzWcfRelay** tar bort WcfRelay från det angivna relä namn området.

## BESKRIVS

### Exempel 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Tar bort WcfRelay `TestWCFRelay1` från namn området `TestNameSpace-Relay1` .
