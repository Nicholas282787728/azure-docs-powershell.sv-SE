# Unregister-AzStorageSyncServer

## Sammanfattning
Varning! om du avregistrerar en server kommer alla Server slut punkter att tas bort från den här servern. Det här kommandot avregistrerar en server från tjänsten Storage Sync.

## FRÅGESYNTAXEN

### StringParameterSet (standard)
```
Unregister-AzStorageSyncServer [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-ServerId] <Guid> [-Force] [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Unregister-AzStorageSyncServer [-InputObject] <PSRegisteredServer> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Unregister-AzStorageSyncServer [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Det här kommandot avregistrerar en server från Storage Sync-tjänsten. Varning! om du avregistrerar en server kommer alla Server slut punkter att tas bort från den här servern. Den bör bara anropas om du är säker på att det inte finns någon sökväg på den här servern längre.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $RegisteredServer = Get-AzStorageSyncServer -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName"
PS C:\> Unregister-AzStorageSyncServer -Force -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -ServerId $RegisteredServer.ServerId
```

Det här kommandot avregistrerar servern och skapar borttagning av alla Server slut punkter på servern.
