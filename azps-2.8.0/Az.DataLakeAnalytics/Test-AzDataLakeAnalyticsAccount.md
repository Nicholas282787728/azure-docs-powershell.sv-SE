# Test-AzDataLakeAnalyticsAccount

## Sammanfattning
Kontrollerar om det finns ett data Lake Analytics-konto.

## FRÅGESYNTAXEN

```
Test-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Testet AzDataLakeAnalyticsAccount** söker efter ett data Lake Analytics-konto.

## BESKRIVS

### Exempel 1: testa om ett konto finns
```
PS C:\>Test-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Det här kommandot testar om kontot som heter ContosoAdlAccount finns.
