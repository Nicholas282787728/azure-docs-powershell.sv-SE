# Set-AzDataLakeAnalyticsDataSource

## Sammanfattning
Ändrar informationen för en data källa för ett data Lake Analytics-konto.

## FRÅGESYNTAXEN

```
Set-AzDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-AccessKey] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzDataLakeAnalyticsDataSource** ändrar informationen för en data källa för ett Azure Data Lake Analytics-konto.

## BESKRIVS

### Exempel 1: ändra åtkomst tangenten för en data Källa
```
PS C:\>Set-AzDataLakeAnalyticsDataSource -Account "ContosoAdlAccount" -Blob "contosowasb" -AccessKey "...newaccesskey..."
```

Det här kommandot ändrar åtkomst-tangenten som är lagrad för en Azure Blob Storage-datakälla.
