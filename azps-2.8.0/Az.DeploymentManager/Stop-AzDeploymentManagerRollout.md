# Stop-AzDeploymentManagerRollout

## Sammanfattning
Avslutar installationen.

## FRÅGESYNTAXEN

### Interaktivt (standard)
```
Stop-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ID
```
Stop-AzDeploymentManagerRollout [-ResourceId] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Stop-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Stop-AzDeploymentManagerRollout** stoppar en pågående installation och returnerar ett objekt som representerar det aktuella tillståndet för installationen.
Ange utgivningen med dess namn och resurs grupps namn. Du kan också ange ett installations objekt eller ResourceId.

Observera att när en introduktion har stoppats kan den inte återupptas eller startas om. Du kan bara skapa en ny installation.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Stop-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout -SkipSucceeded
```

Det här kommandot stoppar en installation med namnet ContosoRollout i ContosoResourceGroup. 

### Exempel 2: stoppa en lansering med resurs-ID
```powershell
PS C:\> Restart-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Det här kommandot stoppar en installation med namnet ContosoRollout i ContosoResourceGroup.

### Exempel 3: stoppa en installation med hjälp av installations objekt.
```powershell
PS C:\> Get-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Det här kommandot stoppar en installation vars namn och ResourceGroup matchar $rolloutObjectens namn och ResourceGroupName-egenskaper.
