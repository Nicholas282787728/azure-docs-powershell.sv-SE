# Get-AzServiceBusTopic

## Sammanfattning
Returnerar en beskrivning för den angivna tjänst Bus artikeln.

## FRÅGESYNTAXEN

```
Get-AzServiceBusTopic [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzServiceBusTopic** returnerar en beskrivning för det angivna Service Bus-namnområdet.

## BESKRIVS


Returnerar beskrivningen av angivet avsnitt för angivet Service Bus-namnområde.

### Exempel 2
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Returnerar en lista med ämnen för angivet Service Bus-namnrymd. Som standard returneras 100-avsnitt, om fler än 100 ämnen returneras, Använd-MaxCount-parameter.

### Exempel 3
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Returnerar en lista med de första 150-avsnitten för angivet Service Bus-namnområde.
