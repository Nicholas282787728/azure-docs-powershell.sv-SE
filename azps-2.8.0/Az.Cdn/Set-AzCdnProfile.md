# Set-AzCdnProfile

## Sammanfattning
Uppdaterar en CDN-profil.

## FRÅGESYNTAXEN

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzCdnProfile** uppdaterar en CDN-profil.
