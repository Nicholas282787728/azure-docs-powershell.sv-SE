# Get-AzCdnProfileSsoUrl

## Sammanfattning
Hämtar URL-adressen för enkel inloggning för en CDN-profil.

## FRÅGESYNTAXEN

### ByFieldsParameterSet (standard)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzCdnProfileSsoUrl** hämtar en enkel inloggnings-URL för Azure Content Delivery Network (CDN)-profilen.
Med den här webb adressen kan användarna ansluta till en extra Portal och använda ytterligare funktioner i CDN.
