# Get-AzRecoveryServicesAsrAlertSetting

## Sammanfattning
Hämtar de konfigurerade aviserings inställningarna för Azure Site Recovery för valvet.

## FRÅGESYNTAXEN

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzRecoveryServicesAsrAlertSetting** hämtar de konfigurerade aviserings inställningarna för Azure Site Recovery för valvet.

## BESKRIVS

### Exempel 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Få aviserings-och aviserings inställningar för Azure Site Recovery.
