# Disable-AzHDInsightOperationsManagementSuite

## Sammanfattning
Inaktiverar Operations Management Suite (OMS) i ett HDInsight-kluster och relevanta loggar slutar flöda till den OMS-arbetsyta som anges vid aktivering.

## FRÅGESYNTAXEN

```
Disable-AzHDInsightOperationsManagementSuite [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **disable-AzHDInsightOperationsManagementSuite** inaktiverar Operations Management Suite (OMS) i ett Azure HDInsight-kluster.

## BESKRIVS

### Exempel 1
```
PS C:\> Disable-AzHDInsightOMS -Name testcluster

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Operations Management Suite (OMS) inaktive ras i HDInsight-klustret och relevanta loggar slutar flöda till OMS-arbetsytan.

### Exempel 2
```
PS C:\> Disable-AzHDInsightOMS -Name testcluster -ResourceGroupName testrg

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Operations Management Suite (OMS) inaktive ras i HDInsight-klustret och relevanta loggar slutar flöda till OMS-arbetsytan.
