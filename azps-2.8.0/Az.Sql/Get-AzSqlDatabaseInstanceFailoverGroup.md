# Get-AzSqlDatabaseInstanceFailoverGroup

## Sammanfattning
Hämtar eller visar grupper med instansen failover.

## FRÅGESYNTAXEN

```
Get-AzSqlDatabaseInstanceFailoverGroup [[-Name] <String>] [-ResourceGroupName] <String>
 [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Hämtar en specifik failover-grupp eller visar instanserna för redundans i en region under användarens prenumeration.

Antingen används en region i gruppen för att köra kommandot. De returnerade värdena reflekterar statusen för de hanterade instanserna i det området med avseende på gruppen för förekomster av redundans.

## BESKRIVS

### Exempel 1
```
PS C:\> $failoverGroups = Get-AzSqlDatabaseInstanceFailoverGroup -ResourceGroupName rg -Location location
Output:
{
    ResourceGroupName                     : rg
    Location                              : East US
    Name                                  : fg
    PartnerResourceGroupName              : rg
    PartnerRegion                         : West US
    PrimaryManagedInstanceName            : managedInstance1
    PartnerManagedInstanceName            : managedInstance2
    ReplicationRole                       : Primary
    ReplicationState                      : CATCH_UP
    ReadWriteFailoverPolicy               : Automatic
    FailoverWithDataLossGracePeriodHours  : 1
    ReadOnlyFailoverPolicy                : Disabled
    Id                                    : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/rg/providers/Microsoft.Sql/locations/eastus/instanceFailoverGroups/fg
}
```

Visar en lista över alla failover-grupper i regionen

### Exempel 2
```
PS C:\> $failoverGroup = Get-AzSqlDatabaseInstanceFailoverGroup -ResourceGroupName rg -Location location -Name fg
Output:
```

Skaffa en specifik failover-grupp för förekomst.
