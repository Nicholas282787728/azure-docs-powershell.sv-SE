# Enable-AzSqlServerAdvancedDataSecurity

## Sammanfattning
Aktiverar avancerad data säkerhet på en server.

## FRÅGESYNTAXEN

```
Enable-AzSqlServerAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Enable-AzSqlServerAdvancedDataSecurity** aktiverar avancerad data säkerhet på en server. Avancerad data säkerhet är ett enhetligt säkerhets paket som innehåller data klassificering, sårbarhets utvärdering och Avancerat skydd för din server. (Ett nytt lagrings konto skapas automatiskt för att spara säkerhets utvärderingar. Om du har skapat ett lagrings konto för detta ändamål används det i stället.

## BESKRIVS

### Exempel 1-aktivera avancerad data säkerhet för servrar
```powershell
PS C:\>  Enable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Exempel 2 – aktivera avancerad data säkerhet för servrar från Server resurs
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
