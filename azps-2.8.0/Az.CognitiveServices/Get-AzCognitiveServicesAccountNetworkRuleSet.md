# Get-AzCognitiveServicesAccountNetworkRuleSet

## Sammanfattning
Hämta egenskapen NetworkRule för ett konto för kognitiva tjänster

## FRÅGESYNTAXEN

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzCognitiveServicesAccountNetworkRuleSet** hämtar egenskapen NetworkRule för ett kognitivt Services-konto

## BESKRIVS

### Exempel 1: get NetworkRule-egenskapen för ett angivet kognitivt-konto
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

Det här kommandot får egenskapen NetworkRule för ett angivet kognitivt-konto
