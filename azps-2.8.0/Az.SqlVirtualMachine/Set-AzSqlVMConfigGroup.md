# Set-AzSqlVMConfigGroup

## Sammanfattning
Ange informationen i relation till en virtuell SQL-dator grupp i en konfiguration för SQL-dator.

## FRÅGESYNTAXEN

```
Set-AzSqlVMConfigGroup [-SqlVM] <AzureSqlVMModel> [-SqlVMGroup] <AzureSqlVMGroupModel>
 -ClusterOperatorAccountPassword <SecureString> -SqlServiceAccountPassword <SecureString>
 [-ClusterBootstrapAccountPassword <SecureString>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Den Set-AzSqlVMConfigGroup cmdleten anger den information som behövs för att ansluta till en virtuell SQL-dator grupp för en konfiguration för SQL-dator.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> $config = Set-AzSqlVMConfigGroup -SqlVM $config -SqlVMGroup $group -ClusterOperatorAccountPassword 'password' -SqlServiceAccountPassword 'password'
```

Uppdatera grupp informationen för en konfiguration av en virtuell SQL-dator.
