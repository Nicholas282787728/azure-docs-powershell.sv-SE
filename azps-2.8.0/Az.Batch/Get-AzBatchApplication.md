# Get-AzBatchApplication

## Sammanfattning
Hämtar information om det angivna programmet.

## FRÅGESYNTAXEN

```
Get-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [[-ApplicationId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzBatchApplication** hämtar information om ett program i ett Azure Batch-konto.

## BESKRIVS

### Exempel 1: Visa programmen i ett batch-konto
```
PS C:\>Get-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup"
ApplicationId AllowUpdates DisplayName

------------- ------------ ----------------------------

litware       False        Litware Advanced Reticulator
```

Det här kommandot visar alla program i ContosoBatch-kontot.
