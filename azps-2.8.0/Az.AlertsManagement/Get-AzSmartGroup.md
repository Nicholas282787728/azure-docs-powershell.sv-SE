# Get-AzSmartGroup

## Sammanfattning
Hämtar smarta grupp information

## FRÅGESYNTAXEN

### SmartGroupsListByFilter (standard)
```
Get-AzSmartGroup [-SortBy <String>] [-SortOrder <String>] [-TimeRange <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SmartGroupById
```
Get-AzSmartGroup -SmartGroupId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzSmartGroup** hämtar Smart Group-information.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzSmartGroup -TimeRange "1h"
```

Lista alla smarta grupper som har skapats under den senaste 1 timmen. Använd Format-List för att få information om varje smart grupp i listan.

### Exempel 2
```powershell
PS C:\> Get-AzSmartGroup -SmartGroupId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" | Format-List
```

Få smart grupp information utifrån ID (GUID) eller resurs-ID (fullständigt ARM-ID)
