# Test-AzureRmDataLakeStoreAccount

## Sammanfattning
Testar förekomsten av ett data Lake Store-konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Test-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Test-AzureRmDataLakeStoreAccount-** cmdleten testar att det finns ett data Lake Store-konto.

## BESKRIVS

### Exempel 1: testa ett konto
```
PS C:\>Test-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Det här kommandot testar om kontot som heter ContosoADL finns.
