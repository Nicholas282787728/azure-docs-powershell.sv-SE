# Modulen AzureRM. LocationBasedServices
## Problembeskrivning
I det här avsnittet visas hjälp för cmdlets för Azure location-baserade tjänster.

## AzureRM. LocationBasedServices-cmdletar
### [Get-AzureRmLocationBasedServicesAccount](Get-AzureRmLocationBasedServicesAccount.md)
Får kontot.

### [Get-AzureRmLocationBasedServicesAccountKey](Get-AzureRmLocationBasedServicesAccountKey.md)
Hämtar API-nycklarna för ett konto. De här nycklarna är den autentiseringsmekanism som används vid efterföljande samtal till Azure location-baserade tjänster.

### [New-AzureRmLocationBasedServicesAccount](New-AzureRmLocationBasedServicesAccount.md)
Skapar ett konto för platsbaserade tjänster.

### [New-AzureRmLocationBasedServicesAccountKey](New-AzureRmLocationBasedServicesAccountKey.md)
Återskapar en konto-nyckeln.

### [Remove-AzureRmLocationBasedServicesAccount](Remove-AzureRmLocationBasedServicesAccount.md)
Tar bort ett konto för platsbaserade tjänster.
