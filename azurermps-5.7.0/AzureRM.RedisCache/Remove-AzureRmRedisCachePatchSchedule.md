# Remove-AzureRmRedisCachePatchSchedule

## Sammanfattning
Tar bort uppdaterings schema.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmRedisCachePatchSchedule** tar bort uppdaterings schema från cachen i Azure Redis-cachen.

## BESKRIVS

### Exempel 1: ta bort uppdaterings schema
```
PS C:\>Remove-AzureRmRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Det här kommandot tar bort uppdaterings schema från cachen som heter RedisCache06.
