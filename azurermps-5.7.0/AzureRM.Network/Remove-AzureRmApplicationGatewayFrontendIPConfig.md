# Remove-AzureRmApplicationGatewayFrontendIPConfig

## Sammanfattning
Tar bort en front-IP-konfiguration från en Programgateway.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmApplicationGatewayFrontendIPConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmApplicationGatewayFrontendIPConfig** tar bort nät ports-IP från en Azure Application Gateway.

## BESKRIVS

### Exempel 1: ta bort en front-IP-konfiguration
```
PS C:\>$AppGw = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzureRmApplicationGatewayFrontendIPConfig -ApplicationGateway $AppGw -Name "FrontEndIP02"
```

Det första kommandot får en Programgateway som heter ApplicationGateway01 och lagrar den i $AppGw variabeln.

Det andra kommandot tar bort front-IP-konfigurationen som heter FrontEndIP02 från Application Gateway som lagras i $AppGw.
