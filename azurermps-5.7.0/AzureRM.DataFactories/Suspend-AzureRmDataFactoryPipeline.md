# Suspend-AzureRmDataFactoryPipeline

## Sammanfattning
Inaktive ras en rörledning i Azure Data Factory.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### ByFactoryName (standard)
```
Suspend-AzureRmDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Suspend-AzureRmDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **suspend-AzureRmDataFactoryPipeline** inaktive ras en pipeline i Azure Data Factory.
Du kan återuppta pipeline genom att använda Resume-AzureRmDataFactoryPipeline cmdlet.

## BESKRIVS

### Exempel 1: pausa en rörledning
```
PS C:\>Suspend-AzureRmDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikiSample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to suspend pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Det här kommandot gör att pipelinen med namnet DPWikiSample i data fabriken med namnet WikiADF.
Kommandot returnerar ett värde med $True.
