# Remove-AzureRmEnvironment

## Sammanfattning
Tar bort slut punkter och metadata för anslutning till en viss Azure-instans.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmEnvironment [-Name] <String> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Remove-AzureRmEnvironment cmdlet tar bort slut punkter och metadatainformation för anslutning till en viss Azure-instans.
