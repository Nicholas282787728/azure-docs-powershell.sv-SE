# Remove-AzureRmStreamAnalyticsInput

## Sammanfattning
Tar bort en inmatning från ett ström analys jobb.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmStreamAnalyticsInput** asynkront tar bort en inmatning från ett ström analys jobb i Azure.

## BESKRIVS

### EXEMPEL 1: ta bort en indataströmmen från ett jobb
```
PS C:\>Remove-AzureRmStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Det här kommandot tar bort indatavärdet EventStream från StreamingJob.
