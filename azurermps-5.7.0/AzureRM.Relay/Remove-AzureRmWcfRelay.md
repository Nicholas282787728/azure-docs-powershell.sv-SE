# Remove-AzureRmWcfRelay

## Sammanfattning
Tar bort WcfRelay från det angivna relä namn området.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmWcfRelay** tar bort WcfRelay från det angivna relä namn området.

## BESKRIVS

### Exempel 1
```
PS C:\> Remove-AzureRmWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Tar bort WcfRelay `TestWCFRelay1` från namn området `TestNameSpace-Relay1` .
