# Remove-AzureRmServiceBusTopic

## Sammanfattning
Tar bort avsnittet från det angivna Service Bus-namnområdet.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmServiceBusTopic [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmServiceBusTopic** tar bort avsnittet från det angivna tjänst buss namn området.

## BESKRIVS

### Exempel 1
```
PS C:\> Remove-AzureRmServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -TopicName SB-Topic_exampl1
```

Tar bort avsnittet `SB-Topic_exampl1` från namn området `SB-Example1` .
