# Get-AzureRmSiteRecoveryVaultSettings

## Sammanfattning
Hämtar inställningar för det aktuella återställnings valvet för webbplatsen.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmSiteRecoveryVaultSettings [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSiteRecoveryVaultSettings** hämtar inställningar för det aktuella Azure Site Recovery-valvet.
