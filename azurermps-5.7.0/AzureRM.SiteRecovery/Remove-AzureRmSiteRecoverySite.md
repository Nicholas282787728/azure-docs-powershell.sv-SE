# Remove-AzureRmSiteRecoverySite

## Sammanfattning
Tar bort en webbplats återställnings webbplats från det aktuella valvet.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmSiteRecoverySite -Site <ASRSite> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmSiteRecoverySite** tar bort en Azure Site Recovery-webbplats från det aktuella valvet.
