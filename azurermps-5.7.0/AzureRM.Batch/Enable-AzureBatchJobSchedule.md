# Enable-AzureBatchJobSchedule

## Sammanfattning
Aktiverar ett batchjobb.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Enable-AzureBatchJobSchedule [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Enable-AzureBatchJobSchedule** aktiverar ett Azure batch-jobb.
När du har aktiverat en jobb schema kan jobb skapas enligt det schemat.

## BESKRIVS

### Exempel 1: aktivera ett jobb schema
```
PS C:\>Enable-AzureBatchJobSchedule -Id "JobSchedule17" -BatchContext $Context
```

Det här kommandot aktiverar det jobb schema som har ID-JobSchedule17.
Använd cmdleten **Get-AzureRmBatchAccountKeys** för att tilldela en kontext till variabeln $context.

När du använder autentisering med delad nycklar används den primära åtkomst tangenten som standard. Om du vill ändra den nyckel som ska användas ställer du in egenskapen BatchAccountContext.
