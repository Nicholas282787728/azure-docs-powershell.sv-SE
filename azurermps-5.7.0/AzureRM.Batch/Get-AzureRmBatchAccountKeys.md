# Get-AzureRmBatchAccountKeys

## Sammanfattning
Hämtar nycklarna till ett batch-konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmBatchAccountKeys [-AccountName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmBatchAccountKeys** hämtar nycklarna för ett Azure Batch-konto i det aktuella abonnemanget.
