# Resume-AzureRmAnalysisServicesServer

## Sammanfattning
Återupptar en instans av Analysis Services-servern

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Resume-AzureRmAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Resume-AzureRmAnalysisServicesServer cmdleten återupptar en instans av Analysis Services-servern

## BESKRIVS

### Exempel 1
```
PS C:\> Resume-AzureRmAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Det här kommandot återupptar en pausad server med namnet testserver i resourcegroup testgroup
