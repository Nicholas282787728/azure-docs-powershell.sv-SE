## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmResource** får Azure-resurser.

## BESKRIVS

### Exempel 1: få alla resurser av en viss typ
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType microsoft.web/sites
```

Det här kommandot får en resurs av typen Microsoft. Web/Sites under ResourceGroup11.

### Exempel 2: Hämta en resurs med namn
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceName ContosoWebsite
```

Med det här kommandot får du resursen namnet ContosoWebsite under ResourceGroup11.

### Exempel 3: Visa alla status för lagrings konton i en resurs grupp
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType Microsoft.ClassicStorage/storageAccounts -ExpandProperties |
   Select * -Expand Properties |
   Sort Name |
   Format-Table Name,Status*
```
