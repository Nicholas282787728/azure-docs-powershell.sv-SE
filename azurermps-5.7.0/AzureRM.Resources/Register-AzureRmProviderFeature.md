# Register-AzureRmProviderFeature

## Sammanfattning
Registrerar en Azure-leverantör i ditt konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Register-AzureRmProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Register-AzureRmProviderFeature** cmdlet registrerar en Azure Provider-funktion på ditt konto.

## BESKRIVS

### Exempel 1: registrera en funktion
```
PS C:\>Register-AzureRmProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Då läggs AllowApplicationSecurityGroups-funktionen till i ditt konto.
