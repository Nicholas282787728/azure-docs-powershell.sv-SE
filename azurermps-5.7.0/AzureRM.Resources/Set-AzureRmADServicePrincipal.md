# Set-AzureRmADServicePrincipal

## Sammanfattning
Uppdaterar ett befintligt objekt i Azure Active Directory-tjänsten.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

### SpObjectIdWithDisplayNameParameterSet (standard)
```
Set-AzureRmADServicePrincipal -ObjectId <String> -DisplayName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SPNWithDisplayNameParameterSet
```
Set-AzureRmADServicePrincipal -ServicePrincipalName <String> -DisplayName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Uppdaterar ett befintligt objekt i Azure Active Directory-tjänsten. Använd New-AzureRmADSpCredential cmdlet för att uppdatera autentiseringsuppgifterna som är associerade med den här tjänstens huvud säkerhets objekt. Använd Set-AzureRmADApplication cmdlet för att uppdatera egenskaperna för det underliggande programmet.

## BESKRIVS

### Exempel 1
```
Set-AzureRmADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 -DisplayName "UpdatedNameForSp"
```

Uppdaterar visnings namnet för tjänstens huvud konto med angivet objekt-ID.

### Exempel 2
```
Set-AzureRmADServicePrincipal -ServicePrincipalName "http://MyApp1" -DisplayName "UpdatedNameforSp"
```

Uppdaterar visnings namnet för tjänstens huvud namn med angivet SPN.
