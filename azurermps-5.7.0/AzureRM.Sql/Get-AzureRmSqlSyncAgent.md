# Get-AzureRmSqlSyncAgent

## Sammanfattning
Returnerar information om Azure SQL Sync-agenter.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSqlSyncAgent** returnerar information om en eller flera Sync-agenter för Azure SQL.
Ange namnet på en synkroniseringsresurs för att visa information för enbart den synkroniseringsklienten.
