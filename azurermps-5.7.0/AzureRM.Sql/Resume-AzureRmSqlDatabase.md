# Resume-AzureRmSqlDatabase

## Sammanfattning
Återupptar en SQL Data Warehouse-databas.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Resume-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Resume-AzureRmSqlDatabase** återupptar en Azure SQL Data Warehouse-databas.

## BESKRIVS

### Exempel 1: återupptar en Azure SQL Data Warehouse-databas
```
PS C:\>Resume-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot återupptar en upplåst Azure SQL Data Warehouse-databas.
