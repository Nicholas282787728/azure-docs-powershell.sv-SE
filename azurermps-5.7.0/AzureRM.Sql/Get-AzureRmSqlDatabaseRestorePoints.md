# Get-AzureRmSqlDatabaseRestorePoints

## Sammanfattning
Hämtar de distinkta återställnings punkter från vilka ett SQL-datalager kan återställas.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmSqlDatabaseRestorePoints [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSqlDatabaseRestorePoints** hämtar de distinkta återställnings punkter som ett Azure SQL data-lager kan återställas från.
För en Azure SQL-databas är återställnings fönstret kontinuerligt.
Det innebär att alla tidpunkter i databasens bevarande period kan användas som återställnings punkt.

Denna cmdlet stöds också av SQL Server utsträckt databas tjänst på Azure.

## BESKRIVS

### Exempel 1: Hämta alla återställnings punkter
```
PS C:\>Get-AzureRmSqlDatabaseRestorePoints -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : CONTINUOUS
RestorePointCreationDate : 
EarliestRestoreDate      : 8/12/2015 12:00:00 AM
RestorePointLabel        : RestorePoint01
```

Det här kommandot returnerar alla tillgängliga återställnings punkter för den Azure SQL-databas som heter Database01.
