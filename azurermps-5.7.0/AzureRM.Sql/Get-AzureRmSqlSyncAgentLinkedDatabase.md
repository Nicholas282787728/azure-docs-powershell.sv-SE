# Get-AzureRmSqlSyncAgentLinkedDatabase

## Sammanfattning
Returnerar information om SQL Server-databaser som är länkade av en synkroniseringsklient.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureRmSqlSyncAgentLinkedDatabase [-ServerName] <String> [-SyncAgentName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmSqlSyncAgentLinkedDatabases** returnerar information om SQL Server-databaser som är länkade av en synkroniseringsresurs.

## BESKRIVS

### Exempel 1: Hämta de länkade SQL Server-databaserna för en Azure SQL Sync-agent.
```
PS C:\> Get-AzureRmSqlSyncAgentLinkedDatabases -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01" | Format-List
SeverName                 : sever01
DatabaseId                : databaseId
DatabaseName              : database01
DatabaseType              : SQLServerDatabase
Description               : 
UserName                  : myAccount
```

Det här kommandot returnerar de länkade SQL Server-databaserna som är länkade av en Azure SQL Sync-agent.
