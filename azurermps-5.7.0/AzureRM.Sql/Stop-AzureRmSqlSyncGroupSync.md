# Stop-AzureRmSqlSyncGroupSync

## Sammanfattning
Stoppar en synkronisering av en synkroniseringsresurs.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Stop-AzureRmSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Stop-AzureRmSqlSyncGroupSync** stoppar synkroniseringen av en synkroniseringsresurs.

## BESKRIVS

### Exempel 1
```
PS C:\> Stop-AzureRmSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Det här kommandot stoppar den synkronisering som pågår för gruppen Sync-mysg.
