# Set-AzureRmDataLakeAnalyticsDataSource

## Sammanfattning
Ändrar informationen för en data källa för ett data Lake Analytics-konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Set-AzureRmDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-AccessKey] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureRmDataLakeAnalyticsDataSource** ändrar informationen för en data källa för ett Azure Data Lake Analytics-konto.

## BESKRIVS

### Exempel 1: ändra åtkomst tangenten för en data Källa
```
PS C:\>Set-AzureRmDataLakeAnalyticsDataSource -Account "ContosoAdlAccount" -Blob "contosowasb" -AccessKey "...newaccesskey..."
```

Det här kommandot ändrar åtkomst-tangenten som är lagrad för en Azure Blob Storage-datakälla.
