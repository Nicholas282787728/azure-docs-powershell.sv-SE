# Get-AzsStorageSettings

## Sammanfattning
Returnerar inställningarna för lagringsprovider för lagrings resursen.

## FRÅGESYNTAXEN

```
Get-AzsStorageSettings [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Returnerar inställningarna för lagringsprovider för lagrings resursen.

## BESKRIVS

### Exempel 1:
```powershell
PS C:\> Get-AzsStorageSettings
```

Hämta lagrings inställningarna.
