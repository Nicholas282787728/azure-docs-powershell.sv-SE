# Get-AzWebApp

## Sammanfattning
Hämtar Azure Web Apps i den angivna resurs gruppen.

## FRÅGESYNTAXEN

### S
```
Get-AzWebApp [[-ResourceGroupName] <String>] [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Get-AzWebApp [-AppServicePlan] <AppServicePlan> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S3
```
Get-AzWebApp [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzWebApp** hämtar information om en Azure Web App.

## BESKRIVS

### Exempel 1: Hämta ett webb program från en resurs grupp
```
PS C:\>Get-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Det här kommandot hämtar webb programmet som heter ContosoSite som tillhör resurs gruppens standard-väst.
