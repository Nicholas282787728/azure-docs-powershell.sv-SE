# Set-AzRouteTable

## Sammanfattning
Anger mål tillstånd för en väg tabell.

## FRÅGESYNTAXEN

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzRouteTable** anger mål tillståndet för en Azure route-tabell.
