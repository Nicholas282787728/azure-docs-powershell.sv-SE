# Set-AzVirtualNetworkGatewayConnectionSharedKey

## Sammanfattning
Konfigurerar den delade nyckeln för anslutningen för virtuell nätverksgateway.

## FRÅGESYNTAXEN

```
Set-AzVirtualNetworkGatewayConnectionSharedKey -Name <String> -ResourceGroupName <String> -Value <String>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzVirtualNetworkGatewayConnectionSharedKey** konfigurerar den delade nyckeln för den virtuella Nätverksgatewayen.
