# Get-AzApplicationGatewayAuthenticationCertificate

## Sammanfattning
Hämtar ett autentiseringscertifikat för en Programgateway.

## FRÅGESYNTAXEN

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>]
 -ApplicationGateway <PSApplicationGateway> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzApplicationGatewayAuthenticationCertificate** hämtar ett autentiseringscertifikat för en Azure Application Gateway.
