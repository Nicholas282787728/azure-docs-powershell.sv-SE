# Get-AzNetworkSecurityGroup

## Sammanfattning
Hämtar en nätverks säkerhets grupp.

## FRÅGESYNTAXEN

### Noexpandering
```
Get-AzNetworkSecurityGroup [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Byggnad
```
Get-AzNetworkSecurityGroup -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzNetworkSecurityGroup** får en Azure Network Security-grupp.

## BESKRIVS

### 1: Hämta en befintlig nätverks säkerhets grupp
```
Get-AzNetworkSecurityGroup -Name  nsg1 -ResourceGroupName "rg1"
```

Det här kommandot returnerar innehållet i Azure nätverks säkerhets gruppen "nsg1" i resurs gruppen "RG1"
