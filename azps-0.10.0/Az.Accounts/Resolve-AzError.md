# Resolve-AzError

## Sammanfattning
Visa detaljerad information om PowerShell-fel med utökad information för Azure PowerShell-fel.

## FRÅGESYNTAXEN

### AnyErrorParameterSet (standard)
```
Resolve-AzError [[-Error] <ErrorRecord[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### LastErrorParameterSet
```
Resolve-AzError [-Last] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Matchar och visar detaljerad information om fel i den aktuella PowerShell-sessionen, inklusive var felet uppstod i skript, stack spårning och alla inre och sammansatta undantag. För Azure PowerShell-fel finns mer information i fel sökning av tjänst problem, inklusive fullständig information om begäran och Server svaret som orsakade felet.
