# Revoke-AzSnapshotAccess

## Sammanfattning
Återkallar en åtkomst till en ögonblicks bild.

## FRÅGESYNTAXEN

```
Revoke-AzSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **REVOKE-AzSnapshotAccess** avvisar en åtkomst till en ögonblicks bild.

## BESKRIVS

### Exempel 1
```
PS C:\> Revoke-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

Återkalla åtkomsten till ögonblicks bilden med namnet "Snapshot01" i resurs gruppen med namnet "ResourceGroup01"
