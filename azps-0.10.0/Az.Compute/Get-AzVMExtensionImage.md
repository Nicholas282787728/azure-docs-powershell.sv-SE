# Get-AzVMExtensionImage

## Sammanfattning
Hämtar alla versioner för ett Azure-tillägg.

## FRÅGESYNTAXEN

```
Get-AzVMExtensionImage -Location <String> -PublisherName <String> -Type <String>
 [-FilterExpression <String>] [-Version <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzVMExtensionImage** hämtar alla versioner av ett Azure-tillägg.

## BESKRIVS

### Exempel 1: Hämta versionerna av en tilläggs bild
```
PS C:\> Get-AzVMExtensionImage -Location "Central US" -PublisherName "Fabrikam" -Type "FabrikamEndpointProtection"
```

Det här kommandot får alla versioner av tilläggs bilden för den angivna platsen, utgivaren och typen.
