# Remove-AzContainerService

## Sammanfattning
Tar bort en behållar tjänst.

## FRÅGESYNTAXEN

```
Remove-AzContainerService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzContainerService** tar bort en behållar tjänst från ditt Azure-konto.

## BESKRIVS

### Exempel 1: ta bort en behållar tjänst
```
PS C:\> Remove-AzContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17"
```

Det här kommandot tar bort behållar tjänsten med namnet CSResourceGroup17.
