# Remove-AzVMSqlServerExtension

## Sammanfattning
Tar bort ett SQL Server-tillägg från en virtuell dator.

## FRÅGESYNTAXEN

```
Remove-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzVMSqlServerExtension** tar bort ett AzureSQL-tillägg från en virtuell dator.

## BESKRIVS

### Exempel 1: ta bort ett SQL Server-tillägg
```
PS C:\> Remove-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" -Name "SqlIaaSAgent"
```

Det här kommandot tar bort ett SQL Server-tillägg från den virtuella datorn med namnet ContosoVM22.
