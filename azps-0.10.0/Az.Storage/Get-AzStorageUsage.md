# Get-AzStorageUsage

## Sammanfattning
Hämtar lagrings resursanvändningen för det aktuella abonnemanget.

## FRÅGESYNTAXEN

```
Get-AzStorageUsage [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzStorageUsage** hämtar resursanvändningen för Azure Storage för det aktuella abonnemanget.

## BESKRIVS

### Exempel 1: skaffa användning av lagrings resurser
```
PS C:\>Get-AzStorageUsage
```

Det här kommandot får användning av lagrings resurserna för det aktuella abonnemanget.
