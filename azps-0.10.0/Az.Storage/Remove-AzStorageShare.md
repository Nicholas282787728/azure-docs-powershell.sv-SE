# Remove-AzStorageShare

## Sammanfattning
Tar bort en fil resurs.

## FRÅGESYNTAXEN

### Resurs namn (standard)
```
Remove-AzStorageShare [-Name] <String> [-IncludeAllSnapshot] [-Force] [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Resurserna
```
Remove-AzStorageShare [-Share] <CloudFileShare> [-IncludeAllSnapshot] [-Force] [-PassThru]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzStorageShare** tar bort en fil resurs.

## BESKRIVS

### Exempel 1: ta bort en fil resurs
```
PS C:\>Remove-AzStorageShare -Name "ContosoShare06"
```

Det här kommandot tar bort fil resursen som heter ContosoShare06.

### Exempel 2: ta bort en fil resurs och alla dess stillbilder
```
PS C:\>Remove-AzStorageShare -Name "ContosoShare06" -IncludeAllSnapshot
```

Det här kommandot tar bort fil resursen med namnet ContosoShare06 och alla dess stillbilder.
