# Register-AzProviderFeature

## Sammanfattning
Registrerar en Azure-leverantör i ditt konto.

## FRÅGESYNTAXEN

```
Register-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
**Register-AzProviderFeature** cmdlet registrerar en Azure Provider-funktion på ditt konto.

## BESKRIVS

### Exempel 1: registrera en funktion
```
PS C:\>Register-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Då läggs AllowApplicationSecurityGroups-funktionen till i ditt konto.
