# New-AzADGroup

## Sammanfattning
Skapar en ny Active Directory-grupp.

## FRÅGESYNTAXEN

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Skapar en ny Active Directory-grupp.

## BESKRIVS

### Exempel 1 – Skapa en ny AD-grupp

```
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Skapar en ny AD-grupp med namnet "MyGroupDisplayName" och "e-postnamn" myemail@domain.com ".
