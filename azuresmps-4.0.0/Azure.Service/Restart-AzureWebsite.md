# Restart-AzureWebsite

## Sammanfattning
Stoppar och startar sedan om den angivna webbplatsen.

## FRÅGESYNTAXEN

```
Restart-AzureWebsite [-PassThru] [-Name <String>] [-Slot <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

Cmdleten **restart-AzureWebsite** stoppar och startar sedan om den angivna webbplatsen.

## BESKRIVS

### Exempel 1: starta om en webbplats
```
PS C:\> Restart-AzureWebsite -Name MyWebsite
```

I det här exemplet startas en webbplats som heter min webbplats.
