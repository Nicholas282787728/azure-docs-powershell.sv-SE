# Remove-AzureRouteTable

## Sammanfattning
Tar bort en routningstabell.

## FRÅGESYNTAXEN

```
Remove-AzureRouteTable -Name <String> [-PassThru] [-Force] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRouteTable** tar bort en routningstabell från din prenumeration.
Om en routningstabell är kopplad till ett undernät kan du inte ta bort tabellen.

## BESKRIVS

### Exempel 1: ta bort en routningstabellen
```
PS C:\> Remove-AzureRouteTable -Name "PublicRouteTable"
```

Det här kommandot tar bort routningstabellen som heter PublicRouteTable.
