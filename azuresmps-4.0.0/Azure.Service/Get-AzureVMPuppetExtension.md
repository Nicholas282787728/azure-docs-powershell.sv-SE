# Get-AzureVMPuppetExtension

## Sammanfattning
Hämtar Puppet-tillägget som används på en virtuell dator.

## FRÅGESYNTAXEN

```
Get-AzureVMPuppetExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureVMPuppetExtension** hämtar fil namns tillägget Puppet på en virtuell dator.

## BESKRIVS

### Exempel 1: få tillägget Puppet installerat på en virtuell dator
```
PS C:\> Get-AzureVMPuppetExtension -VM $VM
```

Det här kommandot får Puppet-tillägget kopplat till en virtuell dator.
