# Remove-AzureVMSqlServerExtension

## Sammanfattning
Tar bort en Azure Virtual Machine SQL Server-tillägg från ett virtuellt dator objekt.

## FRÅGESYNTAXEN

```
Remove-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureVMSqlServerExtension** tar bort ett SQL Server-tillägg för Azure Virtual Machine från ett virtuellt dator objekt.
