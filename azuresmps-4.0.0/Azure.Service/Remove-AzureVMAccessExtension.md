# Remove-AzureVMAccessExtension

## Sammanfattning
Tar bort tillägget VMAccess som används på en virtuell dator.

## FRÅGESYNTAXEN

```
Remove-AzureVMAccessExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureVMAccessExtension** tar bort tillägget VMAccess som används på en virtuell dator.

## BESKRIVS

### Exempel 1: ta bort ett VMAccess-tillägg från en virtuell dator
```
PS C:\> Remove-AzureVMAccessExtension -VM $VM;
```

Det här kommandot tar bort ett VMAccess-tillägg från en virtuell dator.
