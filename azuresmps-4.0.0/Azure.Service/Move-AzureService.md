# Move-AzureService

## Sammanfattning
Migrerar en moln tjänst till Azure Resource Manager-stacken.

## FRÅGESYNTAXEN

### AbortMigrationParameterSet

 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

```

## PROBLEMBESKRIVNING
Cmdleten **Move-AzureService** migrerar en moln tjänst och en distribution i den tjänsten till en resurs grupp i Azure Resource Manager-stacken.

## BESKRIVS

### Exempel 1: förbereda tjänste migration
```
PS C:\> Move-AzureService -Prepare -ServiceName "ContosoService" -DeploymentName "ContosoVM" -CreateNewVirtualNetwork
```

Det här kommandot förbereder tjänsten ContosoService för migrering till Azure Resource Manager-stacken.
Migreringen inkluderar distributionen med namnet ContosoVM.

### Exempel 2: starta tjänste migration
```
PS C:\> Move-AzureService -Commit -ServiceName "ContosoService" -DeploymentName "ContosoVM"
```

Det här kommandot startar migrering av tjänsten med namnet ContosoService till Azure Resource Manager-stacken.
Migreringen inkluderar distributionen med namnet ContosoVM.

### Exempel 3: Avbryt migrering av tjänst
```
PS C:\> Move-AzureService -Abort -ServiceName "ContosoService" -DeploymentName "ContosoVM"
```

Det här kommandot avbryter migrering av tjänsten som heter ContosoService till Azure Resource Manager-stacken.

### Exempel 4: förbereda tjänste migrering till ett befintligt virtuellt nätverk
```
PS C:\> Move-AzureService -Prepare -ServiceName "ContosoService" -DeploymentName "ContosoVM" -UseExistingVirtualNetwork -VirtualNetworkResourceGroupName "VnetRG" -VirtualNetworkName "ContosoVNET" -SubnetName "ContosoSubnet"
```

Det här kommandot förbereder tjänsten ContosoService för migrering till Azure Resource Manager-stacken.
Migreringen inkluderar distributionen med namnet ContosoVM.
Migreringen använder ett virtuellt nätverk som redan har skapats.

### Exempel 5: verifiera tjänst migration
```
PS C:\> Move-AzureService -Validate -ServiceName "ContosoService" -DeploymentName "ContosoVM" -CreateNewVirtualNetwork
```

Det här kommandot verifierar migrering för tjänsten som heter ContosoService till Azure Resource Manager-stacken.
Migreringen inkluderar distributionen med namnet ContosoVM.

### Exempel 6: verifiera tjänst migrering till ett befintligt virtuellt nätverk
```
PS C:\> Move-AzureService -Validate -ServiceName "contosoService" -DeploymentName "contosoVM" -UseExistingVirtualNetwork -VirtualNetworkResourceGroupName "vnetRG" -VirtualNetworkName "contosoVNET" -SubnetName "contosoSubnet"
```

Det här kommandot verifierar migrering för tjänsten som heter ContosoService till Azure Resource Manager-stacken.
Migreringen inkluderar distributionen med namnet ContosoVM.
Migreringen använder ett virtuellt nätverk som redan har skapats.
