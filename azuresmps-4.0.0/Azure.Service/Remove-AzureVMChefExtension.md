# Remove-AzureVMChefExtension

## Sammanfattning
Tar bort den kock-anknytning som tillämpas på den virtuella datorn.

## FRÅGESYNTAXEN

```
Remove-AzureVMChefExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureVMChefExtension** tar bort kock-tillägget som används på den virtuella datorn.

## BESKRIVS

### Exempel 1: ta bort kock-tillägget som används på den angivna virtuella datorn
```
PS C:\> Remove-AzureVMChefExtension -VM $VM;
```

Det här kommandot tar bort kock-tillägget som används på den virtuella datorn.
