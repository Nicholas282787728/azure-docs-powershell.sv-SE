# New-AzureSiteRecoverySite

## Sammanfattning
Skapar en webbplats återställnings webbplats.

## FRÅGESYNTAXEN

```
New-AzureSiteRecoverySite -Name <String> [-Vault <ASRVault>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzureSiteRecoverySite** skapar en Azure Site Recovery-webbplats i det aktuella valvet.

## BESKRIVS

### Exempel 1: skapa en webbplats återställnings webbplats
```
PS C:\> New-AzureSiteRecoverySite -Name "RecoverySite07"
```

Det här kommandot skapar en webbplats återställnings webbplats med namnet RecoverySite07.
