# Remove-AzureAutomationModule

## Sammanfattning

Tar bort en modul från Automation.

## FRÅGESYNTAXEN

```
Remove-AzureAutomationModule -Name <String> [-Force] -AutomationAccountName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Cmdleten **Remove-AzureAutomationModule** tar bort ett Automation-konto från Microsoft Azure Automation.

## BESKRIVS

### Exempel 1: ta bort en modul
```
PS C:\> Remove-AzureAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule"
```

Det här kommandot tar bort en modul som heter ContosoModule från Automation-kontot med namnet Contoso17.
