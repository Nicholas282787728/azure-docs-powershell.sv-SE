# Get-AzureDeployment

## Sammanfattning
Hämtar information om en distribution.

## FRÅGESYNTAXEN

```
Get-AzureDeployment [-ServiceName] <String> [[-Slot] <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureDeployment** hämtar information om en Azure-distribution.
Ange namnet på Azure-tjänsten och distributionen.

## BESKRIVS

### Exempel 1: få information om en produktions distribution
```
PS C:\> Get-AzureDeployment -ServiceName "ContosoService"
```

Det här kommandot returnerar information om distributionen för tjänsten som heter ContosoService.
Det här kommandot anger inte en plats.
Därför använder kommandot standardvärdet för produktion.

### Exempel 2: få information om en implementerings distribution
```
PS C:\> Get-AzureDeployment -ServiceName "ContosoService" -Slot "Staging"
```

Det här kommandot returnerar detaljerna för mellanlagrings distributionen av ContosoService.
