# Remove-AzureNetworkSecurityGroupFromSubnet

## Sammanfattning
Avkopplar en nätverks säkerhets grupp från ett undernät.

## FRÅGESYNTAXEN

```
Remove-AzureNetworkSecurityGroupFromSubnet -Name <String> -VirtualNetworkName <String> -SubnetName <String>
 [-Force] [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureNetworkSecurityGroupFromSubnet** kopplas bort en Azure nätverks säkerhets grupp från ett undernät.
