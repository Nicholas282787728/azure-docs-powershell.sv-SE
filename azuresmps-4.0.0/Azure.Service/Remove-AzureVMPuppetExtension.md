# Remove-AzureVMPuppetExtension

## Sammanfattning
Tar bort tillägget Puppet som används på en virtuell dator.

## FRÅGESYNTAXEN

```
Remove-AzureVMPuppetExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureVMPuppetExtension** tar bort tillägget Puppet som används på en virtuell dator.

## BESKRIVS

### Exempel 1: ta bort Puppet-tillägget som används på en virtuell dator
```
PS C:\> Remove-AzureVMPuppetExtension -VM $VM;
```

Det här kommandot tar bort tillägget Puppet som används på den angivna virtuella datorn.
