# Add-AzureDns

## Sammanfattning
Lägger till en DNS-server till en Azure-tjänst.

## FRÅGESYNTAXEN

```
Add-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Add-AzureDns** lägger till en DNS-server till en Azure-tjänst.

## BESKRIVS

### Exempel 1: lägga till en DNS-Server
```
PS C:\> Add-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.4 -Name "Dns07"
```

Det här kommandot lägger till DNS-servern som har adressen 10.1.2.4 till ContosoService.
