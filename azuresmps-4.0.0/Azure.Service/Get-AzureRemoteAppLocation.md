# Get-AzureRemoteAppLocation

## Sammanfattning
Här listas de platser som Azure RemoteApp har stöd för.

## FRÅGESYNTAXEN

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRemoteAppLocation** visar de platser som stöds av Azure RemoteApp.
