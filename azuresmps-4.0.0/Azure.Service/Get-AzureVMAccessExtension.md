# Get-AzureVMAccessExtension

## Sammanfattning
Hämtar VMAccess-tillägget som används på en virtuell dator.

## FRÅGESYNTAXEN

```
Get-AzureVMAccessExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureVMAccessExtension** hämtar fil namns tillägget VMAccess på en virtuell dator.

## BESKRIVS

### Exempel 1: Hämta VMAccess-tillägget för en virtuell dator
```
PS C:\> Get-AzureVMAccessExtension -VM $VM;
```

Det här kommandot får VMAccess-tillägget för en virtuell dator.
