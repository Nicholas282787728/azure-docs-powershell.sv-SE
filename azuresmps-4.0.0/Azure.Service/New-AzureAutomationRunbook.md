# New-AzureAutomationRunbook

## Sammanfattning

Skapar en Runbook.

## FRÅGESYNTAXEN

### ByRunbookName (standard)
```
New-AzureAutomationRunbook -Name <String> [-Description <String>] [-Tags <String[]>]
 -AutomationAccountName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### ByPath
```
New-AzureAutomationRunbook -Path <String> [-Description <String>] [-Tags <String[]>]
 -AutomationAccountName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Cmdleten **New-AzureAutomationRunbook** skapar en ny, Tom Microsoft Azure Automation-Runbook.
Ange ett namn för att skapa en ny Runbook.

Du kan också ange sökvägen till en Windows PowerShell-skriptfil (. ps1) för att importera en Runbook.
Skriptet som ska importeras måste innehålla en enkel Windows PowerShell-arbetsflöde.
Namnet på den här Windows PowerShell-arbetsflödet blir namnet på runbooken.

## BESKRIVS

### Exempel 1: skapa en Runbook
```
PS C:\> New-AzureAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook02"
```

Det här kommandot skapar en ny Runbook med namnet Runbook02 i Automation-kontot med namnet Contoso17.
