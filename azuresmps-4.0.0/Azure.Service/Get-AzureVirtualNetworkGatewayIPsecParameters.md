# Get-AzureVirtualNetworkGatewayIPsecParameters

## Sammanfattning
Hämtar IPsec-parametrar för en Azure Virtual Network Gateway.

## FRÅGESYNTAXEN

```
Get-AzureVirtualNetworkGatewayIPsecParameters -GatewayId <String> -ConnectedEntityId <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureVirtualNetworkGatewayIPsecParameters** hämtar IPSec-parametrarna för en Azure Virtual Network Gateway.
