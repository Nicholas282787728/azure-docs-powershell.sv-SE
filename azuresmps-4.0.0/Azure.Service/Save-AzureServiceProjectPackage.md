# Save-AzureServiceProjectPackage

## Sammanfattning
Paketerar tjänste projektet i Azure Cloud-paketet.

## FRÅGESYNTAXEN

```
Save-AzureServiceProjectPackage [-Local] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

**AzureServiceProjectPackage** cmdlet-paket till ett Azure Cloud-paket (*. cspkg).

## BESKRIVS

### Exempel 1: skapa ett paket med service projekt
```
PS C:\> Save-AzureServiceProjectPackage
```

I det här exemplet skapas ett *. cspgk för ett tjänst projekt med namnet MyAzureServiceProject.
