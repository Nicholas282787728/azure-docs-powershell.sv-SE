# Get-AzureVMCustomScriptExtension

## Sammanfattning
Hämtar information från ett anpassat skript tillägg för en virtuell Azure-dator.

## FRÅGESYNTAXEN

```
Get-AzureVMCustomScriptExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureVMCustomScriptExtension** hämtar information från ett anpassat skript tillägg för en virtuell Azure-dator.

## BESKRIVS

### Exempel 1: Hämta ett skript tillägg för en Azure Virtual Machine
```
PS C:\> Get-AzureVMCustomScriptExtension -VM $VM;
```

Det här kommandot får ett skript tillägg för Azure Virtual Machine lagrat i variabeln $VM.
