# Get-AzureVirtualNetworkGateway

## Sammanfattning
Hämtar en virtuell nätverksgateway.

## FRÅGESYNTAXEN

```
Get-AzureVirtualNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureVirtualNetworkGateway** får en Azure Virtual Network Gateway.
