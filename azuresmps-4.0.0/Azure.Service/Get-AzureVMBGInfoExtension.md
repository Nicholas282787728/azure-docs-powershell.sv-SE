# Get-AzureVMBGInfoExtension

## Sammanfattning
Hämtar BGInfo-tillägget som används på en virtuell dator.

## FRÅGESYNTAXEN

```
Get-AzureVMBGInfoExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureVMBGInfoExtension** hämtar fil namns tillägget BgInfo på en virtuell dator.

## BESKRIVS

### Exempel 1: få tillägget BGInfo installerat på en angiven virtuell dator
```
PS C:\> Get-AzureVMBGInfoExtension -VM $VM;
```

Det här kommandot får tillägget BGInfo som används på en angiven virtuell dator
