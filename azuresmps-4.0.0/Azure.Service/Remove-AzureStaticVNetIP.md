# Remove-AzureStaticVNetIP

## Sammanfattning
Tar bort den statiska IP-postinformationen från ett virtuellt dator objekt.

## FRÅGESYNTAXEN

```
Remove-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureStaticVNetIP** tar bort det statiska virtuella nätverkets IP-adress från ett virtuellt dator objekt.
