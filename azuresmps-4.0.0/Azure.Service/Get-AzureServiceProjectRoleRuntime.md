# Get-AzureServiceProjectRoleRuntime

## Sammanfattning
Få de program vara som är tillgängliga för installation i en roll.

## FRÅGESYNTAXEN

```
Get-AzureServiceProjectRoleRuntime [-Runtime <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

Hämtar de program vara som är tillgängliga för installation i en roll.
