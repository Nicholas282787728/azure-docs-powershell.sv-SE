# Add-AzureDisk

## Sammanfattning
Lägger till en disk i Azure-lagringsplatsen.

## FRÅGESYNTAXEN

```
Add-AzureDisk [-DiskName] <String> [-MediaLocation] <String> [-Label <String>] [-OS <String>]
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Add-AzureDisk** lägger till en disk i Azure disk-lagringsplatsen i den aktuella prenumerationen.
Denna cmdlet kan lägga till en system diskett eller en data skiva.
Om du vill lägga till en system diskett anger du en operativ system typ genom att använda *OS* -parametern.

## BESKRIVS

### Exempel 1: lägga till en start diskett med operativ systemet Windows
```
PS C:\> Add-AzureDisk -DiskName "MyWinDisk" -MediaLocation "http://contosostorage.blob.core.azure.com/vhds/winserver-system.vhd" -Label "StartupDisk" -OS "Windows"
```

Det här kommandot lägger till en system disk till lagrings platsen.
System disken använder operativ systemet Windows.

### Exempel 2: lägga till en data disk
```
PS C:\> Add-AzureDisk -DiskName "MyDataDisk" -MediaLocation "http://yourstorageaccount.blob.core.azure.com/vhds/winserver-data.vhd" -Label "DataDisk"
```

Det här kommandot lägger till en data disk.

### Exempel 3: lägga till en system disk för Linux
```
PS C:\> Add-AzureDisk -DiskName "MyLinuxDisk" -MediaLocation "http://yourstorageaccount.blob.core.azure.com/vhds/linuxsys.vhd" -OS "Linux"
```

Det här kommandot lägger till en system disk för Linux.
