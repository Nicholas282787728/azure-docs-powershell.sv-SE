# Get-AzureAutomationAccount

## Sammanfattning

Får Azure Automation-konton.

## FRÅGESYNTAXEN

```
Get-AzureAutomationAccount [-Name <String>] [-Location <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Cmdleten **Get-AzureAutomationAccount** hämtar Microsoft Azure Automation-kontona för din prenumeration.
Ett Automation-konto är en behållare för Automation-resurser som är isolerade från andra Automation-konton.
Automatiserings resurser inkluderar Runbooks, jobb och till gångar.

## BESKRIVS

### Exempel 1: skaffa ett Automation-konto
```
PS C:\> Get-AzureAutomationAccount -Name "Contoso17"
```

Det här kommandot får Automation-kontot Contoso17.
