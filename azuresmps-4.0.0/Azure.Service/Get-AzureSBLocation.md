# Get-AzureSBLocation

## Sammanfattning
Hämtar tjänst bussens plats.

## FRÅGESYNTAXEN

```
Get-AzureSBLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

Cmdleten **Get-AzureSBLocation** returnerar platserna som tjänst bussen är tillgänglig från.

[!INCLUDE [sb-deprecation.md](../include/sb-deprecation.md)]

## BESKRIVS

### Exempel 1: skaffa Service Bus-platsen
```
PS C:\> Get-AzureSBLocation
```

I det här exemplet får du platsen för Service Bus.
