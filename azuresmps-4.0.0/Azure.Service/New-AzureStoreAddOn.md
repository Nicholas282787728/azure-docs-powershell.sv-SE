# New-AzureStoreAddOn

## Sammanfattning
Köper en ny tilläggs instans.

## FRÅGESYNTAXEN

```
New-AzureStoreAddOn -Name <String> -AddOn <String> -Plan <String> -Location <String> [-PromotionCode <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

Köper en ny tilläggs instans från Azure Store.

## BESKRIVS

### Exempel 1
```
PS C:\> New-AzureStoreAddOn -Name MyAddOn -AddOn AddonId -Plan PlanId -Location "West US"
```

Det här exemplet köper ett tillägg med namnet ' t-AddOn med en PlanId i västra amerikanska platsen.

### Exempel 2
```
PS C:\> New-AzureStoreAddOn -Name MyAddOn -AddOn AddonId -Plan PlanId -Location "West US" -PromotionCode MyPromoCode
```

I det här exemplet används en kampanj kod för att köpa ett tillägg.
