# Remove-AzureLocalNetworkGateway

## Sammanfattning
Tar bort en Azure-lokal nätverksgateway.

## FRÅGESYNTAXEN

```
Remove-AzureLocalNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureLocalNetworkGateway** tar bort en Azure Local Network Gateway.
