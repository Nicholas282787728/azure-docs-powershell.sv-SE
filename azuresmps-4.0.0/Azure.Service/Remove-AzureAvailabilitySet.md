# Remove-AzureAvailabilitySet

## Sammanfattning
Tar bort en tillgänglighets uppsättning från en virtuell Azure-dator.

## FRÅGESYNTAXEN

```
Remove-AzureAvailabilitySet -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureAvailabilitySet** tar bort en tillgänglighets uppsättning från en virtuell Azure-dator.
