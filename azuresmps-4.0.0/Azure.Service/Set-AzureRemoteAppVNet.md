# Set-AzureRemoteAppVNet

## Sammanfattning
Anger egenskaperna för ett virtuellt Azure RemoteApp-nätverk.

## FRÅGESYNTAXEN

```
Set-AzureRemoteAppVNet -VNetName <String> [-VirtualNetworkAddressSpace <String[]>]
 [-LocalNetworkAddressSpace <String[]>] [-DnsServerIpAddress <String[]>] [-VpnDeviceIpAddress <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureRemoteAppVNet** anger egenskaperna för ett Azure RemoteApp-nätverk.

## BESKRIVS

### Exempel 1: Ange egenskaper för ett virtuellt nätverk
```
PS C:\> Set-AzureRemoteAppVnet -VNetName "ContosoVNet" -DnsServerIpAddress "131.253.33.200" -LocalNetworkAddressSpace "192.168.0.0/24" -VirtualNetworkAddressSpace "10.10.0.0/24" -VpnDeviceIpAddress "131.253.33.200"
```

Det här kommandot anger egenskaperna för ett virtuellt nätverk med namnet ContosoVNet.
