# Remove-AzureSchedulerJob

## Sammanfattning
Tar bort ett jobb i schemaläggare.

## FRÅGESYNTAXEN

```
Remove-AzureSchedulerJob [-Force] [-Location <String>] -JobCollectionName <String> -JobName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

Cmdleten **Remove-AzureSchedulerJob** tar bort ett jobb för schemaläggare.

## BESKRIVS

### Exempel 1: ta bort ett jobb i schemaläggare
```
PS C:\> Remove-AzureSchedulerJob -Location "North Central US" -JobCollectionName "JobCollection01" -JobName "Job17"
```

Det här kommandot tar bort jobbet som heter Job17.
Jobbet ingår i jobb samlingen med namnet JobCollection01 och finns i den angivna platsen.
