# Remove-AzureRemoteAppUserDisk

## Sammanfattning
Tar bort en användares användar disk från en Azure RemoteApp-samling.

## FRÅGESYNTAXEN

```
Remove-AzureRemoteAppUserDisk [-CollectionName] <String> [-UserUpn] <String> [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRemoteAppUserDisk** tar bort användarens disk från en Azure RemoteApp-samling.

## BESKRIVS

### Exempel 1: ta bort en användar diskett
```
PS C:\> Remove-AzureRemoteAppUserDisk -CollectionName "Contoso01" -UserUpn "PattiFuller@contoso.com"
```

Det här kommandot tar bort användar disken för en Azure Active Directory-användare som har UPN PattiFuller@contoso.com från samlingen Contoso01.
