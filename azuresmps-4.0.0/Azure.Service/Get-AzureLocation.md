# Get-AzureLocation

## Sammanfattning
Hämtar tillgängliga data Center platser för det aktuella Azure-abonnemanget.

## FRÅGESYNTAXEN

```
Get-AzureLocation [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureLocation** hämtar en lista över tillgängliga Azure Data Center och deras egenskaper för det aktuella Azure-abonnemanget.
Innan du kör denna cmdlet måste du ange ditt nuvarande abonnemang med hjälp av Select-AzureSubscription och Set-AzureSubscription cmdletar.

## BESKRIVS

### Exempel 1: hitta platser
```
PS C:\> Get-AzureLocation
```

Det här kommandot får en lista över tillgängliga data Center och deras egenskaper för det aktuella abonnemanget.
