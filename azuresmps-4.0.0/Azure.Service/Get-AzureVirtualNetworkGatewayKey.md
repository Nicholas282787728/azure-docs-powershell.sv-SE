# Get-AzureVirtualNetworkGatewayKey

## Sammanfattning
Hämtar nyckeln till en Azure Virtual Network Gateway.

## FRÅGESYNTAXEN

```
Get-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureVirtualNetworkGatewayKey** hämtar nyckeln till en Azure Virtual Network Gateway.
