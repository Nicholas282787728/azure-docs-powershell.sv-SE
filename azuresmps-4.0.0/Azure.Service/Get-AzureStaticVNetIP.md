# Get-AzureStaticVNetIP

## Sammanfattning
Hämtar information om statisk IP-adress från ett virtuellt dator objekt, om det finns något.

## FRÅGESYNTAXEN

```
Get-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureStaticVNetIP** hämtar den statiska virtuell nätverks informationen (VNet) från ett virtuellt dator objekt, om det finns något.
