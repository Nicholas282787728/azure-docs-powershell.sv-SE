# Get-AzureRemoteAppStartMenuProgram

## Sammanfattning
Visar program för Start-menyn i en Azure RemoteApp-samling.

## FRÅGESYNTAXEN

```
Get-AzureRemoteAppStartMenuProgram [-CollectionName] <String> [[-ProgramName] <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRemoteAppStartMenuProgram** listar Start-menyns program i den mallfil som en Azure RemoteApp-samling använder.

## BESKRIVS

### Exempel 1: lista med Start-menyns program för en samling
```
PS C:\> Get-AzureRemoteAppStartMenuProgram -CollectionName "ContosoApps"
```

Det här kommandot returnerar listan med program för Start-menyn för samlingen ContosoApps.
