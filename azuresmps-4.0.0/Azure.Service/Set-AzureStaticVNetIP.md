# Set-AzureStaticVNetIP

## Sammanfattning
Anger den statiska IP-adressen för ett virtuellt dator objekt.

## FRÅGESYNTAXEN

```
Set-AzureStaticVNetIP [-IPAddress] <String> -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureStaticVNetIP** anger den statiska virtuell nätverks informationen (VNet) för ett virtuellt dator objekt.

## BESKRIVS

### Exempel 1: Ange virtuell nätverks-IP-adress kopplad till en virtuell dator
```
PS C:\> # Prerequisite: VNet has been set up with SubNet
          # Set-AzureVNetConfig -ConfigurationPath $vnetConfigPath;

          $vm = New-AzureVMConfig -Name $vmname -ImageName $img -InstanceSize $sz | Add-AzureProvisioningConfig -Windows -Password $pwd -AdminUsername $usr;
          $vm = Set-AzureSubNet -VM $vm -SubNetNames $sn;
          Set-AzureStaticVNetIP -IPAddress $ip -VM $vm;
          New-AzureVM -ServiceName $svc -VMs $vm -VNetName $vnetName -Location $loc;
```

Det första kommandot anger konfigurations Sök vägen för ett virtuellt nätverk.

Det andra kommandot skapar en konfiguration för en virtuell dator.

Det tredje kommandot ställer in under nätet för den virtuella datorn.

Det fjärde kommandot anger den virtuella datorns IP-adress.

Det femte kommandot skapar en virtuell dator med den virtuella datorn.
