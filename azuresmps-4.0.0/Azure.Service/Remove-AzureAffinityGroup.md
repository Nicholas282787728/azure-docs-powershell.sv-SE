# Remove-AzureAffinityGroup

## Sammanfattning
Tar bort en tillhörighets grupp i ett abonnemang.

## FRÅGESYNTAXEN

```
Remove-AzureAffinityGroup [-Name] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureAffinityGroup** tar bort en Azure tillhörighets grupp i det aktuella abonnemanget.
Du kan inte ta bort en tillhörighets grupp som har medlemmar.
Du måste först ta bort alla medlemmar i en tillhörighets grupp.

## BESKRIVS

### Exempel 1: ta bort en tillhörighets grupp
```
PS C:\> Remove-AzureAffinityGroup -Name "South01"
```

Det här kommandot tar bort South01-tillhörighets gruppen i det aktuella abonnemanget.
