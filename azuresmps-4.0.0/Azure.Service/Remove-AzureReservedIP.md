# Remove-AzureReservedIP

## Sammanfattning
Tar bort en reserverad IP-adress utifrån dess namn.

## FRÅGESYNTAXEN

```
Remove-AzureReservedIP [-ReservedIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureReservedIP** tar bort en reserverad IP-adress utifrån dess namn.

## BESKRIVS

### Exempel 1: ta bort en reserverad IP-adress utifrån dess namn
```
PS C:\> Remove-AzureReservedIP -ReservedIPName $name
```

Det här kommandot tar bort en reserverad IP-adress utifrån dess namn.
