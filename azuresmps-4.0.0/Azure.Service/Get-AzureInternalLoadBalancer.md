# Get-AzureInternalLoadBalancer

## Sammanfattning
Hämtar information om intern belastnings Utjämnings konfiguration.

## FRÅGESYNTAXEN

```
Get-AzureInternalLoadBalancer [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureInternalLoadBalancer** hämtar information om den interna belastningsutjämnaren för en Azure-tjänst.

## BESKRIVS

### Exempel 1: få information för en intern belastningsutjämnare
```
PS C:\> Get-AzureService -ServiceName "ContosoService" | Get-AzureInternalLoadBalancer
```

Det här kommandot får tjänsten som heter ContosoService med hjälp av cmdleten **Get-AzureService** .
Kommandot överför denna tjänst till den aktuella cmdleten med hjälp av pipeline-operatorn.
Den aktuella cmdleten hämtar information för den interna belastningsutjämnaren för den tjänsten.
