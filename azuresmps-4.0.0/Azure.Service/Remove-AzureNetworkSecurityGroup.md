# Remove-AzureNetworkSecurityGroup

## Sammanfattning
Tar bort en nätverks säkerhets grupp.

## FRÅGESYNTAXEN

```
Remove-AzureNetworkSecurityGroup -Name <String> [-Force] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureNetworkSecurityGroup** tar bort en Azure Network Security-grupp från din prenumeration.
