# Remove-AzureVMBGInfoExtension

## Sammanfattning
Tar bort tillägget BGInfo som används på en virtuell dator.

## FRÅGESYNTAXEN

```
Remove-AzureVMBGInfoExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureVMBGInfoExtension** tar bort tillägget BgInfo som används på en virtuell dator.

## BESKRIVS

### Exempel 1: ta bort BGInfo-tillägget på en virtuell dator
```
PS C:\> Remove-AzureVMBGInfoExtension -VM $VM;
```

Det här kommandot tar bort tillägget BGInfo som tillämpas på en virtuell dator.
