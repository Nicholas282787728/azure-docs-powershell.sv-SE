# Stop-AzureService

## Sammanfattning
Stoppar den aktuella värdbaserade tjänsten.

## FRÅGESYNTAXEN

```
Stop-AzureService [-ServiceName <String>] [-Slot <String>] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

Cmdleten **Stop-AzureService** stoppar den aktuella värdbaserade tjänsten på platsen i Windows Azure.
Om ingen plats anges stoppar cmdleten tjänsten i produktions platsen.
