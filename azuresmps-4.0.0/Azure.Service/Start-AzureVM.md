# Start-AzureVM

## Sammanfattning
Startar en virtuell Azure-dator.

## FRÅGESYNTAXEN

### ByName (standard)
```
Start-AzureVM [-Name] <String[]> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

### Flöde
```
Start-AzureVM -VM <IPersistentVM[]> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Start-AzureVM** startar en virtuell Azure-dator.

## BESKRIVS

### Exempel 1: starta en virtuell dator
```
PS C:\> Start-AzureVM -ServiceName "ContosoService03" -Name "VirtualMachine04"
```

Det här kommandot startar den virtuella datorn med namnet VirtualMachine04 som körs i Azure-tjänsten med namnet ContosoService03.

### Exempel 2: starta en virtuell dator med ett virtuellt dator objekt
```
PS C:\> Get-AzureVM -ServiceName "ContosoService03" -Name "DatabaseServer" | Start-AzureVM
```

Det här kommandot hämtar det virtuella datorobjektet för den virtuella datorn vars namn är DatabaseServer och begär sedan att starta det.
