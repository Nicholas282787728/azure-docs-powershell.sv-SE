# Set-AzureStorSimpleAccessControlRecord

## Sammanfattning
Uppdaterar IQN för en åtkomst kontroll post.

## FRÅGESYNTAXEN

```
Set-AzureStorSimpleAccessControlRecord -ACRName <String> -IQNInitiatorName <String> [-WaitForComplete]
 [-NewName <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureStorSimpleAccessControlRecord** uppdaterar iSCSI-kvalificerat namn (IQN) för en befintlig åtkomst kontroll post.

## BESKRIVS
