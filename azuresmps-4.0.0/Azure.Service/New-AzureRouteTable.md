# New-AzureRouteTable

## Sammanfattning
Skapar en routningstabell.

## FRÅGESYNTAXEN

```
New-AzureRouteTable -Name <String> -Location <String> [-Label <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **New-AzureRouteTable** skapar en routningstabell på en angiven plats.
Du kan lägga till användardefinierade vägar i routningstabellen.
Du kan koppla väg tabellen till ett undernät i ett virtuellt nätverk.
