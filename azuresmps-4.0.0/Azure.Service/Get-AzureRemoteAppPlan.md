# Get-AzureRemoteAppPlan

## Sammanfattning
Visar Azure RemoteApp-abonnemang.

## FRÅGESYNTAXEN

```
Get-AzureRemoteAppPlan [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRemoteAppPlan** visar Azure RemoteApp-abonnemangen.

## BESKRIVS
