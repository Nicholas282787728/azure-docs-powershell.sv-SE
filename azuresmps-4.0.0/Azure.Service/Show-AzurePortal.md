# Show-AzurePortal

## Sammanfattning
Visa Azure-hanteringskonsolen.

## FRÅGESYNTAXEN

```
Show-AzurePortal [-Name <String>] [-Realm <String>] [-Environment <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
I det här avsnittet beskrivs cmdleten i 0.8.10-versionen av Microsoft Azure PowerShell-modulen.
För att få den version av modulen du använder, gå till Azure PowerShell-konsolen och skriv `(Get-Module -Name Azure).Version` .

Cmdleten **show-AzurePortal** visar Azure Management-portalen.

## BESKRIVS

### Exempel 1: Visa information om en webbplats
```
PS C:\> Show-AzurePortal -Name mySite
```

I det här exemplet öppnas en webbläsare på Azure-portalen med information om en webbplats som heter min webbplats.
