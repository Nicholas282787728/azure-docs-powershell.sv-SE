# Set-AzureStorageQueueStoredAccessPolicy

## Sammanfattning
Ställer in en lagrad åtkomst policy för en Azure Storage-kö.

## FRÅGESYNTAXEN

```
Set-AzureStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-NoStartTime] [-NoExpiryTime] [-Context <IStorageContext>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureStorageQueueStoredAccessPolicy** anger en lagrad åtkomst princip för en Azure Storage-kö.

## BESKRIVS

### Exempel 1: Ange en lagrad åtkomst princip i kön
```
PS C:\> Set-AzureStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy07"
```

Det här kommandot anger en åtkomst princip med namnet Policy07 för lagringsmappen som heter min kö.
