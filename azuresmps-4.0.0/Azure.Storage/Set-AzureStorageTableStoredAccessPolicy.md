# Set-AzureStorageTableStoredAccessPolicy

## Sammanfattning
Anger den lagrade åtkomst principen för en Azure Storage-tabell.

## FRÅGESYNTAXEN

```
Set-AzureStorageTableStoredAccessPolicy [-Table] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-NoStartTime] [-NoExpiryTime] [-Context <IStorageContext>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureStorageTableStoredAccessPolicy** anger den lagrade åtkomst principen för en Azure Storage-tabell.

## BESKRIVS

### Exempel 1: Ange en lagrad åtkomst princip i en tabell med fullständig behörighet
```
PS C:\>Set-AzureStorageTableStoredAccessPolicy -Table "MyTable" -Policy "Policy08"
```

Det här kommandot ställer in en åtkomst princip med namnet Policy08 för lagrings tabellen tabellen.
