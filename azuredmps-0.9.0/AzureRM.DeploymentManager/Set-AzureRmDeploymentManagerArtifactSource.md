# Set-AzureRmDeploymentManagerArtifactSource

## Sammanfattning
Uppdaterar en artefakt källa.

## FRÅGESYNTAXEN

```
Set-AzureRmDeploymentManagerArtifactSource [-ArtifactSource] <PSArtifactSource>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **set-AzureRmDeploymentManagerArtifactSource** uppdaterar en artefakt källa med angivet artefakt objekt.
Cmdleten returnerar det uppdaterade ArtifactSource-objektet.

## BESKRIVS

### Exempel 1
```powershell
PS C:\> Get-AzureRmDeploymentManagerArtifactSource -ArtifactSource $artifactSourceObject
```

Det här kommandot uppdaterar en artefakt källa vars namn och ResourceGroup matchar $artifactSourceObjectens namn och ResourceGroupName-egenskaper.
Artefakt källan uppdaterades till de egenskaper som angavs i $artifactSourceObject.
