# Remove-AzureRmDeploymentManagerStep

## Sammanfattning
Tar bort ett steg.

## FRÅGESYNTAXEN

### Interaktivt (standard)
```
Remove-AzureRmDeploymentManagerStep [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ID
```
Remove-AzureRmDeploymentManagerStep [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzureRmDeploymentManagerStep [-Step] <PSStepResource> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmDeploymentManagerStep** tar bort ett steg.
Ange steget med dess namn och resurs gruppens namn. Du kan också ange Step-ID eller ResourceId.

## BESKRIVS
### Exempel 1: ta bort ett steg
```powershell
PS C:\> Remove-AzureRmDeploymentManagerStep -ResourceGroupName ContosoResourceGroup -Name ContosoService1WaitStep
```

Det här kommandot tar bort ett steg som heter ContosoService1WaitStep i ContosoResourceGroup.

### Exempel 2: ta bort ett steg med resurs-ID
```powershell
PS C:\> Remove-AzureRmDeploymentManagerStep -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/steps/ContosoService1WaitStep"
```

Det här kommandot tar bort ett steg som heter ContosoService1WaitStep i ContosoResourceGroup.

### Exempel 3: ta bort ett steg med ett objekt som returneras av New-AzureRmDeploymentManagerStep
```powershell
PS C:\> Remove-AzureRmDeploymentManagerStep -Step $stepObject
```

 Det här kommandot tar bort ett steg vars namn och ResourceGroup matchar $stepObjectens namn och ResourceGroupName-egenskaper.
