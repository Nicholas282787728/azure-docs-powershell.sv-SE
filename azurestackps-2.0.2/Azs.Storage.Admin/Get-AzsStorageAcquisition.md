# Get-AzsStorageAcquisition

## Sammanfattning
Returnerar en lista med BLOB-förvärv.

## FRÅGESYNTAXEN

```
Get-AzsStorageAcquisition [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Returnerar en lista med BLOB-förvärv.

## BESKRIVS

### Exempel 1:
```powershell
PS C:\> Get-AzsStorageAcquisition
```

Hämta listan över BLOB-Acquistions.
