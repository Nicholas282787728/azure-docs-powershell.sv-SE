# <a name="overview-of-the-azure-powershell-service-management-module"></a>Översikt över Azure PowerShell Service Management-modulen

Azure Service Management är den klassiska distributionsmodellen. Den här modulen låter dig arbeta med klassiska Azure-distributioner som inte har konverterats till Azure Resource Manager. Alla nya distributioner bör använda Azure Resource Manager-metoderna. Om du vill använda cmdletarna måste du installera och konfigurera Azure PowerShell för att ansluta det till ditt konto. Mer information finns i [Installera och konfigurera Azure PowerShell](install-azure-ps.md).
