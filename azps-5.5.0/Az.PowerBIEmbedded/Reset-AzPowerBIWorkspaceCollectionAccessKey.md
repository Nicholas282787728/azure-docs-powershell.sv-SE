# Reset-AzPowerBIWorkspaceCollectionAccessKey

## SYNOPSIS
Återställer den angivna snabbtangenten.

## SYNTAX

```
Reset-AzPowerBIWorkspaceCollectionAccessKey [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Reset-AzPowerBIWorkspaceCollectionAccessKey** återställer den angivna snabbtangenten i din Power BI-arbetsyta.

## EXEMPEL

### Exempel 1: Återställ primäråtkomstnyckeln
```
PS C:\>Reset-AzPowerBIWorkspaceCollectionAccessKey -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Med det här kommandot återställs den primära åtkomstnyckeln för arbetsytesamlingen med namnet WCN11 i den angivna resursgruppen.
