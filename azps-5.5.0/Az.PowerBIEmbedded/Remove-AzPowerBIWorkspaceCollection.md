# Remove-AzPowerBIWorkspaceCollection

## SYNOPSIS
Tar bort en samling med Power BI-arbetsytor.

## SYNTAX

```
Remove-AzPowerBIWorkspaceCollection [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzPowerBIWorkspaceCollection** tar bort en samling arbetsytor i Power BI från din Azure-prenumeration och resursgrupp.

## EXEMPEL

### Exempel 1: Ta bort en samling för arbetsytor
```
PS C:\>Remove-AzPowerBIWorkspaceCollection -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

Med det här kommandot tas arbetsytans samling med namnet WCN11 bort i den angivna resursgruppen.
