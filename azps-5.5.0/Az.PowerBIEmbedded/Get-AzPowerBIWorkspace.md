# Get-AzPowerBIWorkspace

## SYNOPSIS
Hämtar arbetsytorna i en Power BI-arbetsyta.

## SYNTAX

```
Get-AzPowerBIWorkspace [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzPowerBIWorkspace** är en samling arbetsytor i Power BI-arbetsytor.

## EXEMPEL

### Exempel 1: Hämta arbetsytor i en arbetsyta
```
PS C:\>Get-AzPowerBIWorkspace -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

Det här kommandot hämtar arbetsytorna som tillhör arbetsytans samling med namnet WCN11 i den angivna resursgruppen.
