# New-AzPowerBIWorkspaceCollection

## SYNOPSIS
Skapar en samling för Power BI-arbetsytor.

## SYNTAX

```
New-AzPowerBIWorkspaceCollection [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzPowerBIWorkspaceCollection** skapar en Power BI-arbetsyta för din Azure-prenumeration i den angivna resursgruppen och -platsen.

## EXEMPEL

### Exempel 1: Skapa en samling för arbetsytor
```
PS C:\>New-AzPowerBIWorkspaceCollection -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Location "Japan West"
```

Med det här kommandot skapas en arbetsytesamling med namnet WCN11 i den angivna resursgruppen på den angivna platsen.
