# Get-AzHpcCacheSku

## SYNOPSIS
Får alla SKU:er tillgängliga i prenumerationen.

## SYNTAX

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzHpcCacheSku** returnerar en lista med SKU:er som är tillgängliga i prenumerationen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
