# Remove-AzHpcCache

## SYNOPSIS
Tar bort en HPC-cache.

## SYNTAX

```
Remove-AzHpcCache -ResourceGroupName <String> -Name <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzHpcCache** tar bort en Azure HPC-cache.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Remove-AzHpcCache -ResourceGroupName testRG -CacheName testCache
```
