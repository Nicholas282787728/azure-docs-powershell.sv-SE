# New-AzCommunicationService

## SYNOPSIS
Skapa en ny CommunicationService eller uppdatera en befintlig CommunicationService.

## SYNTAX

```
New-AzCommunicationService -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DataLocation <String>] [-Location <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Skapa en ny CommunicationService eller uppdatera en befintlig CommunicationService.

## EXEMPEL

### Exempel 1: Skapa en ACS-resurs
```powershell
PS C:\> New-AzCommunicationService -ResourceGroupName ContosoResourceProvider1 -Name ContosoAcsResource1 -DataLocation UnitedStates -Location Global

Location Name           Type                                          AzureAsyncOperation
-------- ----           ----                                          -------------------
Global   ContosoAcsResource1 Microsoft.Communication/communicationServices
```

Skapar en ACS-resurs med de angivna parametrarna.
