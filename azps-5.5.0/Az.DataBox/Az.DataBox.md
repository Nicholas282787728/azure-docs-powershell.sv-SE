# Az.DataBox-modul
## Beskrivning
Modul för att skapa, avbryta, ta bort och hämta databox-jobb

## Az.DataBox-cmdlets
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
Hämtar autentiseringsuppgifterna för datarutan för ett visst jobb

### [Get-AzDataBoxLow](Get-AzDataBoxJob.md)
Hämtar information om Databox-jobb

### [New-AzDataBoxLow](New-AzDataBoxJob.md)
Skapar ett nytt databoxjobb med hjälp av de angivna parametrarna

### [Remove-AzDataBox Etta](Remove-AzDataBoxJob.md)
Tar bort databoxjobbet

### [Stop-AzDataBox Det här kan du göra](Stop-AzDataBoxJob.md)
Avbryter ett pågående databoxjobb om jobbet är i avbokningsbart läge.
