# New-AzMetricAlertRuleV2DimensionSelection

## SYNOPSIS
Skapar ett lokalt måtturvalsobjekt som kan användas för att konstruera ett måttaviseringsvillkor.

## SYNTAX

```
New-AzMetricAlertRuleV2DimensionSelection -DimensionName <String> -ValuesToInclude <String[]>
 [-ValuesToExclude <String[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzMetricAlertRuleV2DimensionSelection** skapar ett lokalt dimensionsmarkeringsobjekt som hjälper till att skapa metriska aviseringsvillkor med cmdleten [New-AzMetricAlertRuleV2Criteria.](https://docs.microsoft.com/en-us/powershell/module/az.monitor/new-azmetricalertrulev2criteria)

## EXEMPEL

### Exempel 1: Skapa ett måttmarkeringsobjekt

```powershell
PS C:\> New-AzMetricAlertRuleV2DimensionSelection -DimensionName LUN -ValuesToInclude 1,3,4

Dimension IncludeValues ExcludeValues
--------- ------------- -------------
LUN       {1, 3, 4}
```

Det här kommandot skapar ett dimensionsurvalsobjekt som {1,3,4} anger att värden måste väljas för Dimension "LUN"
