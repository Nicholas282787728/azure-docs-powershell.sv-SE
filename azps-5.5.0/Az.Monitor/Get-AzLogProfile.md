# Get-AzLogProfile

## SYNOPSIS
Hämtar en loggprofil.

## SYNTAX

```
Get-AzLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzLogProfile** får en loggprofil.
