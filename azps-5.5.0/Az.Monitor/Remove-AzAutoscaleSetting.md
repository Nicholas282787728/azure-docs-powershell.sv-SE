# Remove-AzAutoscaleSetting

## SYNOPSIS
Tar bort en inställning för automatisk skalning.

## SYNTAX

```
Remove-AzAutoscaleSetting -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzAutoscaleSetting** tar bort en inställning för automatisk skalning.
Du måste ange namnet på inställningen och namnet på den resursgrupp som den har tilldelats till.
Den här cmdleten implementerar mönstret ShouldProcess, dvs. den kan begära bekräftelse från användaren innan den faktiskt skapar, ändrar eller tar bort resursen.

## EXEMPEL

### Exempel 1

Tar bort en inställning för automatisk skalning. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzAutoscaleSetting -Name 'LogAlertRule1' -ResourceGroupName MyResourceGroup
```
