# New-AzMetricFilter

## SYNOPSIS
Skapar ett måttfilter som kan användas för frågemått.

## SYNTAX

```
New-AzMetricFilter [-Dimension] <String> [-Operator] <String> [-Value] <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzMetricFilter** skapar ett måttfilter som kan användas för frågemått.

## EXEMPEL

### Exempel 1: Skapa ett måttdimensionsfilter
```
PS C:\>New-AzMetricFilter -Dimension City -Operator eq -Value "Seattle","New York"
```

Det här kommandot skapar ett måttfilter för formatet "Ort eq 'Seattle' eller City eq 'New York'".
