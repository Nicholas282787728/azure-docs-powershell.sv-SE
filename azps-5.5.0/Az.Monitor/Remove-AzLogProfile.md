# Remove-AzLogProfile

## SYNOPSIS
Tar bort en loggprofil.

## SYNTAX

```
Remove-AzLogProfile -Name <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzLogProfile** tar bort en loggprofil.
Den här cmdleten implementerar mönstret ShouldProcess, dvs. den kan begära bekräftelse från användaren innan den faktiskt skapar, ändrar eller tar bort resursen.

## EXEMPEL

### Exempel 1

Tar bort en loggprofil. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzLogProfile -Name 'LogAlertRule1'
```
