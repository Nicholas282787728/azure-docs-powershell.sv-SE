# Update-AzDataCollectionRule

## SYNOPSIS
Uppdaterar en egenskap med regeltaggar för datainsamling.

## SYNTAX

### ByName (standard)
```
Update-AzDataCollectionRule 
      -ResourceGroupName <string> 
      -RuleName <string> 
      [-Tag <hashtable>] 
      [-DefaultProfile <IAzureContextContainer>] 
      [-WhatIf] 
      [-Confirm]
      [<CommonParameters>]
```

### ByResourceId
```
Update-AzDataCollectionRule 
      -RuleId <string> 
      [-Tag <hashtable>] 
      [-DefaultProfile <IAzureContextContainer>] 
      [-WhatIf] 
      [-Confirm]
      [<CommonParameters>]
```

### ByInputObject
```
Update-AzDataCollectionRule 
      -InputObject <PSDataCollectionRuleResource> 
      [-Tag <hashtable>] 
      [-DefaultProfile <IAzureContextContainer>]
      [-WhatIf]
      [-Confirm]
      [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Update-AzDataCollectionRule** uppdaterar egenskapen Taggar för en datainsamlingsregel.

DCR-regler (Data Collection Rules) definierar data som kommer till Azure Monitor och anger var dessa data ska skickas eller lagras. Här är den fullständiga [dcr-översiktsartikeln.](https://docs.microsoft.com/en-us/azure/azure-monitor/platform/data-collection-rule-overview)
