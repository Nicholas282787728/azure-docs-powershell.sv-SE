# Get-AzAutoscaleSetting

## SYNOPSIS
Hämtar inställningar för automatisk skalning.

## SYNTAX

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutoscaleSetting** hämtar alla inställningar för automatisk skalning som är kopplade till en resursgrupp eller en angiven inställning för automatisk skalning.
