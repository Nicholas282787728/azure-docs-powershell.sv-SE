# Get-AzRelayOperation

## SYNOPSIS
Lista med reläåtgärder som stöds

## SYNTAX

```
Get-AzRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRelayOperation** listar den relä som stöds.
