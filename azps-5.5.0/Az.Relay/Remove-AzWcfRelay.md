# Remove-AzWcfRelay

## SYNOPSIS
Tar bort WcfRelay från det angivna relänamnområdet.

## SYNTAX

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med **cmdleten Remove-AzWcfRelay** tas WcfRelay bort från det angivna relänamnområdet.

## EXEMPEL

### Exempel 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Tar bort WcfRelay `TestWCFRelay1` från `TestNameSpace-Relay1` namnområdet.
