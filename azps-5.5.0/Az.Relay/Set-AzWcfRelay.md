# Set-AzWcfRelay

## SYNOPSIS
Uppdaterar beskrivningen av en WcfRelay i det angivna relänamnområdet.

## SYNTAX

### WcfRelayInputObjectSet
```
Set-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-InputObject <PSWcfRelayAttributes>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### WcfRelayPropertiesSet
```
Set-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String> [-UserMetadata <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
CmdletSet-AzWcfRelay för relä uppdaterar beskrivningen för WcfRelay i det angivna relänamnområdet.
