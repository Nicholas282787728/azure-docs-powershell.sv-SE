# New-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Skapar en adresspool för en programgateway i backend.

## SYNTAX

```
New-AzApplicationGatewayBackendAddressPool -Name <String> [-BackendIPAddresses <String[]>]
 [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzApplicationGatewayBackendAddressPool** skapar en adresspool i backend för en Azure-programgateway.
En serveradress kan anges som en IP-adress, ett fullständigt kvalificerat domännamn (FQDN) eller ett IP-konfigurations-ID.

## EXEMPEL

### Exempel 1: Skapa en adresspool för serverslut med hjälp av FQDN för en serverserver
```
PS C:\>$Pool = New-AzApplicationGatewayBackendAddressPool -Name "Pool01" -BackendFqdns "contoso1.com", "contoso2.com"
```

Det här kommandot skapar en adresspool för serverslut med namnet Pool01 med hjälp av FQDN för backend-servrar och lagrar den i $Pool variabeln.

### Exempel 2: Skapa en serverslutsadresspool med IP-adressen för en serverserver
```
PS C:\>$Pool = New-AzApplicationGatewayBackendAddressPool -Name "Pool02" -BackendFqdns "10.10.10.10", "10.10.10.11"
```

Med det här kommandot skapas en adresspool för serverslut med namnet Pool02 med IP-adresserna för backend-servrarna och den lagras i $Pool variabeln.
