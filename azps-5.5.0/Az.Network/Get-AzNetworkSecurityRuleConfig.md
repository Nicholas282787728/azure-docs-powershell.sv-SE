# Get-AzNetworkSecurityRuleConfig

## SYNOPSIS
Skapa en konfiguration av en nätverkssäkerhetsregel för en nätverkssäkerhetsgrupp.

## SYNTAX

```
Get-AzNetworkSecurityRuleConfig [-Name <String>] -NetworkSecurityGroup <PSNetworkSecurityGroup> [-DefaultRules]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzNetworkSecurityRuleConfig** får en konfiguration av nätverkssäkerhetsregel för en Azure-nätverkssäkerhetsgrupp.

## EXEMPEL

### 1: Hämtar en nätverkssäkerhetsregelkonfiguration
```
Get-AzNetworkSecurityGroup -Name nsg1 -ResourceGroupName rg1 
    | Get-AzNetworkSecurityRuleConfig -Name AllowInternetOutBound -DefaultRules
```

Med det här kommandot hämtas standardregeln med namnet "AllowInternetOutBound" från Azure-nätverkssäkerhetsgruppen med namnet "nsg1" i resursgruppen "rg1"

### 2: Hämtar en nätverkssäkerhetsregel med bara namnet
```
Get-AzNetworkSecurityGroup -Name nsg1 -ResourceGroupName rg1 
    | Get-AzNetworkSecurityRuleConfig -Name "rdp-rule"
```

Det här kommandot hämtar en användardefinierad regel med namnet "rdp-regel" från Azure-nätverkssäkerhetsgruppen med namnet "nsg1" i resursgruppen "rg1"
