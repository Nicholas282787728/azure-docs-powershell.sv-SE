# New-AzO365PolicyProperty

## SYNOPSIS
Skapa ett principobjekt för trafikbrytning i Office 365.

## SYNTAX

```
New-AzO365PolicyProperty [-Allow] [-Optimize] [-Default] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Skapa en office 365-breakout-princip som ska användas New-AzVpnSite och Update-AzVpnSite cmdlets.
## EXEMPEL

### Exempel 1
```powershell
PS C:\> $policy = New-AzO365PolicyProperty -Allow -Optimize
```

Skapa en princip för trafikbrytning i Office 365 med tillåten utbrytning för att tillåta och optimera trafikkategori.
