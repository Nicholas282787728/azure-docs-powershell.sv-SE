# Get-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Hämtar en principdefinition för tjänsteslutpunkt.

## SYNTAX

```
Get-AzServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzServiceEndpointPolicyDefinition** får en principdefinition för tjänstslutpunkt.

## EXEMPEL

### Exempel 1
```
$policydef= Get-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ServiceEndpointPolicy $Policy
```

Det här kommandot hämtar tjänsteslutpunktsprincipdefinitionen med namnet ServiceEndpointPolicyDefinition1 i ServiceEndpointPolicy $Policy lagrar den i $policydef variabeln.
