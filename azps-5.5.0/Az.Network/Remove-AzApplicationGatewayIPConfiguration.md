# Remove-AzApplicationGatewayIPConfiguration

## SYNOPSIS
Tar bort en IP-konfiguration från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayIPConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewayIPConfiguration** tar bort en IP-konfiguration från en Azure-programgateway.

## EXEMPEL

### Exempel 1: Ta bort en IP-konfiguration från en Azure-programgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Subnet02"
```

Det första kommandot hämtar en programgateway och lagrar den i $AppGw variabeln.
Det andra kommandot tar bort IP-konfigurationen med namnet Undernät02 från programmets gateway som lagras $AppGw.
