# New-AzFirewallHubIpAddress

## SYNOPSIS
IP-adresser som ärassoicerade till brandväggen på virtuellt nav

## SYNTAX

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
IP-adresser som ärassoicerade till brandväggen på virtuellt nav. Detta kan vara offentliga och privata adresser

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

I det här exemplet skapas ett Hub Ip-adressobjekt med antalet 2 offentliga IP-adresser. HubIPAddress-objektet är separerat till brandväggen på det virtuella navet.
