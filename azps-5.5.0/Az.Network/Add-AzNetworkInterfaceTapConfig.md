# Add-AzNetworkInterfaceTapConfig

## SYNOPSIS
Skapar en TapConfiguration-resurs kopplad till ett NetworkInterface. Detta refererar till en befintlig VirtualNetworkTap-resurs.

## SYNTAX

### SetByResource (standard)
```
Add-AzNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTap <PSVirtualNetworkTap>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SetByResourceId
```
Add-AzNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTapId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Add-AzNetworkInterfaceTapConfig** skapar en TapConfiguration-resurs kopplad till ett NetworkInterface. Detta refererar till en befintlig VirtualNetworkTap-resurs.

## EXEMPEL

### Exempel 1: Lägga till TapConfiguration i ett givet NetworkInterface
```
PS C:\>Add-AzNetworkInterfaceTapConfig -NetworkInterface $sourceNic -VirtualNetworkTap $vVirtualNetworkTap -Name 'myTapConfig'
```

Lägg till TapConfiguration till en sourceNic. Trafiken från sourceNic VM speglas till mål-VM som refereras i vVirtualNetworkTap-resursen.
