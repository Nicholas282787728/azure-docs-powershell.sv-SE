# New-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Skapar en betrodd klient-CA-certifikatkedja för en programgateway.

## SYNTAX

```
New-AzApplicationGatewayTrustedClientCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med New-AzApplicationGatewayTrustedClientCertificate-cmdleten skapas en betrodd klient-CA-certifikatkedja för en programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $trustedClient = New-AzApplicationGatewayTrustedClientCertificate -Name "ClientCert" -CertificateFile "C:\clientCAChain.cer"
```
Kommandot skapar en ny sökväg för klientcertifikatutfärdaren med certifikatutfärdaren som indata för klientcertifikatet.
