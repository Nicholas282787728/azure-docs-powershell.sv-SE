# Get-AzRouteTable

## SYNOPSIS
Hämtar routetabeller.

## SYNTAX

### NoExpand (standard)
```
Get-AzRouteTable [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Utöka
```
Get-AzRouteTable -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRouteTable** hämtar Azure-routetabeller.
Du kan få en enda routetabell eller hämta alla routetabeller i en resursgrupp eller i din prenumeration.
