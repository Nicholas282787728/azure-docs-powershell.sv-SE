# Remove-AzPublicIpAddress

## SYNOPSIS
Tar bort en offentlig IP-adress.

## SYNTAX

```
Remove-AzPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzPublicIpAddress** tar bort en offentlig Azure-IP-adress.

## EXEMPEL

### 1: Ta bort en offentlig IP-adressresurs
```
Remove-AzPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Med det här kommandot tas den offentliga IP-adressresursen $publicIpName i resursgruppen $rgName.
