# Remove-AzApplicationGateway

## SYNOPSIS
Tar bort en programgateway.

## SYNTAX

```
Remove-AzApplicationGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGateway** tar bort en programgateway.

## EXEMPEL

### Exempel 1: Ta bort en angiven programgateway
```
PS C:\>Remove-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
```

Med det här kommandot tas programgatewayen med namnet ApplicationGateway01 bort i resursgruppen ResourceGroup01.
