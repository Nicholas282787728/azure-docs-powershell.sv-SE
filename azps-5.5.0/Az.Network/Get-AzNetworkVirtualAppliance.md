# Get-AzNetworkVirtualAppliance

## SYNOPSIS
Hämta eller lista virtuella nätverksutrustning.

## SYNTAX

### ResourceNameParameterSet (standard)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Med Get-AzNetworkVirtualAppliance kan du få eller lista resurser för virtuell resurs för nätverk.
