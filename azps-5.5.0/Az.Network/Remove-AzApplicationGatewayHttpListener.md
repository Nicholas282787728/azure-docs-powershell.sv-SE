# Remove-AzApplicationGatewayHttpListener

## SYNOPSIS
Tar bort en HTTP-lyssnare från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayHttpListener -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewayHttpListener** tar bort en HTTP-lyssnare från en Azure-programgateway.

## EXEMPEL

### Exempel 1: Ta bort EN HTTP-lyssnare för programgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayHttpListener -ApplicationGateway $AppGw -Name "Listener02"
```

Det första kommandot hämtar en programgateway och lagrar den i $AppGw variabeln.
Det andra kommandot tar bort HTTP-lyssnaren med namnet Listener02 från programmets gateway som lagras $AppGw.
