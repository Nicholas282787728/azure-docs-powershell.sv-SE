# New-AzFirewallPublicIpAddress

## SYNOPSIS
Det här är platshållaren för ip-adressen som kan användas för flera pip i Azure-brandväggen.

## SYNTAX

```
New-AzFirewallPublicIpAddress [-Address <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Det här är platshållaren för ip-adressen som kan användas för flera pip i Azure-brandväggen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $publicIp = New-AzFirewallPublicIpAddress -Address 20.2.3.4
```

$publicIp platshållare för IP-adressen 20.2.3.4
