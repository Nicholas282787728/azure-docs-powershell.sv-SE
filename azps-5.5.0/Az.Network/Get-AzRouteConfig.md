# Get-AzRouteConfig

## SYNOPSIS
Hämtar vägar från en routetabell.

## SYNTAX

```
Get-AzRouteConfig -RouteTable <PSRouteTable> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRouteConfig** hämtar vägar från en Azure-routetabell.
Du kan ange en route efter namn.

## EXEMPEL

### Exempel 1: Hämta en routetabell
```
PS C:\>Get-AzRouteTable -ResourceGroupName "ResourceGroup11" -Name "RouteTable01" | Get-AzRouteConfig -Name "Route07"
Name              : route07
Id                : 
Etag              : 
ProvisioningState : 
AddressPrefix     : 10.1.0.0/16
NextHopType       : VnetLocal
NextHopIpAddress  :
```

Med det här kommandot får du routetabellen RouteTable01 med hjälp av **cmdleten Get-AzRouteTable.**
Kommandot skickar tabellen till den aktuella cmdleten med hjälp av rörledningsoperatorn.
Den aktuella cmdleten får routen Route07 i tabellen RouteTable01.
