# New-AzFirewallPolicyDnsSetting

## SYNOPSIS
Skapar en ny DNS-inställning för Azure-brandväggsprincipen

## SYNTAX

```
New-AzFirewallPolicyDnsSetting [-EnableProxy] [-Server <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzFirewallPolicyDnsSetting** skapar ett DNS-inställningsobjekt för Azure-brandväggsprincipen

## EXEMPEL

### 1. Skapa en tom princip
```powershell
PS C:\> New-AzFirewallPolicyDnsSetting -EnableProxy
```

I det här exemplet skapas ett dns-inställningsobjekt med inställning som aktiverar dns-proxy.

### 2. Skapa en tom princip med ThreatIntel-läge
```powershell
PS C:\> $dnsServers = @("10.10.10.1", "20.20.20.2")
PS C:\> New-AzFirewallPolicyDnsSetting -EnableProxy -Server $dnsServers
```

I det här exemplet skapas ett dns Setting-objekt med inställning som aktiverar DNS-proxy och ställer in anpassade DNS-servrar.
