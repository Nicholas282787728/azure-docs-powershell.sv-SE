# Get-AzNetworkServiceTag

## SYNOPSIS
Hämtar listan med informationsresurser för tjänsttaggar.

## SYNTAX

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzNetworkServiceTag** hämtar listan med informationsresurser för tjänsttaggar.

Observera att azure-regioninformationen du anger används som referens för version (inte som ett filter baserat på plats). Till exempel visas listan med tjänsttaggar med prefixinformation i alla regioner men begränsad till molnet som prenumerationen tillhör (dvs. offentlig, amerikansk myndighet, Kina eller `-Location eastus2` Tyskland).
