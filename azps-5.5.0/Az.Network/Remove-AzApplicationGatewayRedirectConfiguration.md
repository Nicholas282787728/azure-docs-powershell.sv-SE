# Remove-AzApplicationGatewayRedirectConfiguration

## SYNOPSIS
Tar bort en omdirigeringskonfiguration från en befintlig Application Gateway.

## SYNTAX

```
Remove-AzApplicationGatewayRedirectConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGatewayRedirectConfiguration** tar bort en omdirigeringskonfiguration från en befintlig Application Gateway.

## EXEMPEL

### Exempel 1
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\>$AppGw = Remove-AzApplicationGatewayRedirectConfiguration -ApplicationGateway $AppGw -Name "Redirect01"
```

Det första kommandot får en programgateway och lagrar den i $AppGw variabeln.
Det andra kommandot tar bort omdirigeringskonfigurationen med namnet Redirect01 från programgatewayen som lagras $AppGw.
