# New-AzContainerNicConfig

## SYNOPSIS
Skapar ett nytt konfigurationsobjekt för behållarnätverksgränssnittet.

## SYNTAX

```
New-AzContainerNicConfig [-Name <String>] [-IpConfiguration <PSIPConfigurationProfile[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzContainerNicConfig** skapar ett nytt konfigurationsobjekt för behållarnätverksgränssnitt. Det här objektet avgör egenskaperna för behållarnätverksgränssnittet som skapas med referens till den överordnade nätverksprofilen.

## EXEMPEL

### Exempel 1
```powershell
$containerNicConfig = New-AzContainerNicConfig -Name cnicConfig1

$networkProfile = New-AzNetworkProfile -Name np1 -ResourceGroupName rg1 -Location westus -ContainerNetworkInterfaceConfiguration $containerNicConfig
```

Med det första kommandot skapas en tom konfiguration av behållarnätverksgränssnittet. Den andra skapar en ny nätverksprofil som passerar den tidigare skapade konfigureringen av behållarnätverksgränssnittet som ett argument till New-NetworkProfile cmdleten.

### Exempel 2

Skapar ett nytt konfigurationsobjekt för behållarnätverksgränssnittet. (autogenererat)
