# Get-AzAutoApprovedPrivateLinkService

## SYNOPSIS
Hämtar en matris med tjänst-ID för privat länk som kan länkas till en privat ändpunkt med automatiskt godkänt.

## SYNTAX

```
Get-AzAutoApprovedPrivateLinkService -Location <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Get-AzAutoApprovedPrivateLinkService** får en matris med privat länktjänst-ID som kan länkas till en privat punkt med automatiskt godkänt.

## EXEMPEL

### Exempel
```
Get-AzAutoApprovedPrivateLinkService -Location westus -ResourceGroupName TestResourceGroup
```

Det här exemplet returnerar en matris med tjänst-ID för privat länk som kan länkas till en privat ändpunkt med automatiskt godkänt.
