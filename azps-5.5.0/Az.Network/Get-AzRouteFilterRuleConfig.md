# Get-AzRouteFilterRuleConfig

## SYNOPSIS
Hämtar en routefilterregel i ett routefilter.

## SYNTAX

```
Get-AzRouteFilterRuleConfig [-Name <String>] -RouteFilter <PSRouteFilter>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRouteFilterRuleConfig** får en routefilterregel eller en lista med routefilterregler i ett routefilter.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "MyRouteFilter" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf
```

Det första kommandot får routefiltret med namnet MyRouteFilter och lagrar det sedan i variabeln $rf.
Det andra kommandot får routefilterregeln Rule01 kopplad till det routefiltret.
Det tredje kommandot får en lista med routefilterregler som är kopplade till det routefiltret.
