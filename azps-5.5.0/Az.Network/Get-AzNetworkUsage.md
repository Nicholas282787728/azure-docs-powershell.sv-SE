# Get-AzNetworkUsage

## SYNOPSIS
Visar nätverksanvändning för en prenumeration

## SYNTAX

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Med Get-AzNetworkUsage-cmdleten får du begränsningar och aktuell användning för nätverksresurser.
