# Remove-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Tar bort en IP-konfiguration på framsidan från en belastningsutjämnare.

## SYNTAX

```
Remove-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzLoadBalancerFrontendIpConfig** tar bort en IP-konfiguration på frontend-sidan från en Azure-belastningsutjämnare.

## EXEMPEL

### Exempel 1: Ta bort en IP-konfiguration på framsidan från en belastningsutjämnare
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerFrontendIpConfig -Name "frontendName" -LoadBalancer $loadbalancer
```

Det första kommandot hämtar belastningsutjämaren som är kopplad till frontend-IP-konfigurationen som du vill ta bort och lagrar den sedan i $loadbalancer variabeln.
Det andra kommandot tar bort den associerade frontend-IP-konfigurationen från belastningsutjämaren i $loadbalancer.
