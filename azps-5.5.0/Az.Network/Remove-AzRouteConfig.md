# Remove-AzRouteConfig

## SYNOPSIS
Tar bort en route från en routetabell.

## SYNTAX

```
Remove-AzRouteConfig -RouteTable <PSRouteTable> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRouteConfig** tar bort en route från en Azure-routetabell.

## EXEMPEL

### Exempel 1: Ta bort en färdväg
```
PS C:\>Get-AzRouteTable -ResourceGroupName "ResourceGroup11" -Name "RouteTable01" | Remove-AzRouteConfig -Name "Route02" | Set-AzRouteTable

ResourceGroupName : ResourceGroup11

                    k/routeTables/RouteTable01
Etag              : W/"47099b62-60ec-4bc1-b87b-fad56cb8bed1"
ProvisioningState : Succeeded
Tags              : 
Routes            : [
                      {
                        "Name": "Route07",
                        "Etag": "W/\"47099b62-60ec-4bc1-b87b-fad56cb8bed1\"",
                        "Id": "/subscriptions/xxxx-xxxx-xxxx-xxxx/resourceGroups/ResourceGroup11/providers/Micro
                    soft.Network/routeTables/RouteTable01/routes/Route07",
                        "AddressPrefix": "10.1.0.0/16",
                        "NextHopType": "VnetLocal",
                        "NextHopIpAddress": null, 
                        "ProvisioningState": "Succeeded"
                      }
                    ] 
Subnets           : []
```

Med det här kommandot får du routetabellen RouteTable01 med hjälp av **cmdleten Get-AzRouteTable.**
Kommandot skickar tabellen till den aktuella cmdleten med hjälp av rörledningsoperatorn.
Den aktuella cmdleten tar bort routen med namnet Route02 och skickar resultatet till cmdleten **Set-AzRouteTable,** som uppdaterar tabellen så att den återspeglar dina ändringar.
Tabellen innehåller inte längre sträckan Route02.
