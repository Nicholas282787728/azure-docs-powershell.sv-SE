# New-AzApplicationGatewayRewriteRuleCondition

## SYNOPSIS
Lägger till ett villkor i Omskrivningsregel för en programgateway.

## SYNTAX

```
New-AzApplicationGatewayRewriteRuleCondition -Variable <String> [-Pattern <String>] [-IgnoreCase] [-Negate]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten AzApplicationGatewayRewriteRuleCondition** skapar ett omskrivningsregelvillkor för en Azure-programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayRewriteRuleCondition -Variable "var_request_uri" -Pattern "http" -IgnoreCase
PS C:\> $condition

Variable   : var_request_uri
Pattern    : http
IgnoreCase : True
Negate     : False

PS C:\> $condition | Format-Table

Variable        Pattern IgnoreCase Negate
--------        ------- ---------- ------
var_request_uri http          True  False
```
Det här kommandot skapar ett villkor i en omskrivningsregel och lagrar resultatet i variabeln $condition.
