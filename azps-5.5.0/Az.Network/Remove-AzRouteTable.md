# Remove-AzRouteTable

## SYNOPSIS
Tar bort en routetabell.

## SYNTAX

```
Remove-AzRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRouteTable** tar bort en Azure-routetabell.

## EXEMPEL

### Exempel 1: Ta bort en routetabell
```
PS C:\>Remove-AzRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Med det här kommandot tas routetabellen med namnet RouteTable01 bort i resursgruppen ResourceGroup11.
I cmdleten uppmanas du att bekräfta detta innan tabellen tas bort.
