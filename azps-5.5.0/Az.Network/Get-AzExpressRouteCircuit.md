# Get-AzExpressRouteCircuit

## SYNOPSIS
Hämtar en Azure ExpressRoute-krets från Azure.

## SYNTAX

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzExpressRouteCircuit** används för att hämta ett ExpressRoute-kretsobjekt från din prenumeration. Det kretsobjekt som returneras kan användas som indata till andra cmdlets som fungerar på ExpressRoute-kretsar.
