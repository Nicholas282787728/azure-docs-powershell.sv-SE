# Remove-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
Tar bort en utgående regelkonfiguration från en belastningsutjämare.

## SYNTAX

```
Remove-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzLoadBalancerOutboundRuleConfig** tar bort en utgående regelkonfiguration från en Azure-belastningsutjämnare.

## EXEMPEL

### Exempel 1: Ta bort en utgående regel från en Azure-belastningsutjämare
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Remove-AzLoadBalancerOutboundRuleConfig -Name "RuleName" -LoadBalancer $slb
PS C:\>Set-AzLoadBalancer -LoadBalancer $slb
```

Det första kommandot hämtar belastningsutjämaren som är kopplad till den utgående regelkonfiguration du vill ta bort och lagrar den sedan i $slb variabeln.
Det andra kommandot tar bort den associerade utgående regelkonfigurationen från belastningsutjämaren.
Det tredje kommandot uppdaterar belastningsutjämaren.
