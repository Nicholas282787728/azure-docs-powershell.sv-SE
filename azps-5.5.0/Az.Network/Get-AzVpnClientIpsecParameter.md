# Get-AzVpnClientIpsecParameter

## SYNOPSIS
Hämtar vpn Ipsec-parametrarna som angetts för Virtuell nätverksgateway för Punkt till plats-anslutningar.

## SYNTAX

```
Get-AzVpnClientIpsecParameter [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den virtuella nätverksgatewayen är det objekt som representerar din gateway i Azure.
Cmdleten **Get-AzVpnClientIpsecParameter** returnerar objektet för vpn ipsec-parametrarna som angetts för gateway i Azure baserat på namn på gateway och resursgrupp.

## EXEMPEL

### Exempel 1: Hämtar vpn Ipsec-parametrarna som angetts i Virtual Network Gateway för Punkt till webbplatsanslutningar.
```powershell
PS C:\> $VpnClientIPsecParameters = Get-AzVpnClientIpsecParameter -Name myGateway -ResourceGroupName myRG
```

Returnerar objektet för vpn ipsec-parametrarna som angetts i den virtuella nätverksgatewayen med namnet "myGateway" i resursgruppen "myRG"
