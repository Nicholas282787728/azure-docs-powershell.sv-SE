# New-AzFirewallThreatIntelWhitelist

## SYNOPSIS
Skapa en ny lista över hotinformation för Azure-brandväggen

## SYNTAX

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzFirewallThreatIntelWhitelist** skapar ett hot som intel-vitlisteobjekt, som kan användas när du skapar eller anger en Azure-brandvägg.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

Det här exemplet skapar en hotlista för intel vitlista som innehåller en FQDN-vitlista med två poster och en IP-adress vitlista över två poster
