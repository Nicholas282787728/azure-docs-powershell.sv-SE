# Remove-AzLoadBalancerRuleConfig

## SYNOPSIS
Tar bort en regelkonfiguration för en belastningsutjämare.

## SYNTAX

```
Remove-AzLoadBalancerRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzLoadBalancerRuleConfig** tar bort en regelkonfiguration för en Azure-belastningsutjämnare.

## EXEMPEL

### Exempel 1: Ta bort en regelkonfiguration från en belastningsutjämare
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerRuleConfig -Name "MyLBruleName" -LoadBalancer $loadbalancer
```

Det första kommandot hämtar belastningsutjämaren med namnet MyLoadBalancer och lagrar den sedan $loadbalancer variabeln.
Med det andra kommandot tas regelkonfigurationen med namnet MyLBruleName bort från belastningsutjämaren i $loadbalancer.
