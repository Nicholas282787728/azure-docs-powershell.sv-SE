# New-AzVirtualHubRoute

## SYNOPSIS
Skapar ett Azure Virtual Hub-routeobjekt.

## SYNTAX

```
New-AzVirtualHubRoute -AddressPrefix <String[]> -NextHopIpAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Skapar ett Azure Virtual Hub-routeobjekt.

## EXEMPEL

### Exempel 1

```powershell
PS C:\> $route1 = New-AzVirtualHubRoute -AddressPrefix @("10.0.0.0/16", "11.0.0.0/16") -NextHopIpAddress "12.0.0.5"

AddressPrefixes            NextHopIpAddress
---------------            ----------------
{10.0.0.0/16, 11.0.0.0/16} 12.0.0.5
```

Ovanstående skapar ett virtuellt hubb-routeobjekt som kan inkluderas i tabellen för den virtuella hubb-routen.

Den virtuella hubb-vägen är ett minnesobjekt som kan användas för att skapa ett VirtualHubRouteTable-objekt.
