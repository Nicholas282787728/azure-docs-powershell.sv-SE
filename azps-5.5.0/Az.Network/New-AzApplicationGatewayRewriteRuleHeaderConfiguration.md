# New-AzApplicationGatewayRewriteRuleHeaderConfiguration

## SYNOPSIS
Skapar en konfiguration av en omskrivningsregelrubrik för en programgateway.

## SYNTAX

```
New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName <String> [-HeaderValue <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten AzApplicationGatewayRewriteRuleHeaderConfiguration** skapar en regelåtgärdsuppsättning för omskrivning för en Azure-programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $hc = New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName abc -HeaderValue def
```

Det här kommandot skapar en konfiguration av regelrubriken för omskrivning och lagrar resultatet i variabeln $hc.
