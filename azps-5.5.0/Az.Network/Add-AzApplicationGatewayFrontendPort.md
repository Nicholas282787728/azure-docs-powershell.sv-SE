# Add-AzApplicationGatewayFrontendPort

## SYNOPSIS
Lägger till en frontend-port i en programgateway.

## SYNTAX

```
Add-AzApplicationGatewayFrontendPort -ApplicationGateway <PSApplicationGateway> -Name <String> -Port <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Add-AzApplicationGatewayFrontendPort** lägger till en frontend-port i en programgateway.

## EXEMPEL

### Exempel 1: Lägga till en frontend-port i en programgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort01" -Port 80
```

Det första kommandot hämtar programgatewayen med namnet ApplicationGateway01 som tillhör resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln.
Det andra kommandot lägger till port 80 som en frontend-port för programgatewayen som lagras i $AppGw och namnger porten FrontEndPort01.
