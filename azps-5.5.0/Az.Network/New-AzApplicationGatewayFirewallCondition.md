# New-AzApplicationGatewayFirewallCondition

## SYNOPSIS
Skapar ett matchningsvillkor för anpassad regel

## SYNTAX

```
New-AzApplicationGatewayFirewallCondition -MatchVariable <PSApplicationGatewayFirewallMatchVariable[]>
 -Operator <String> [-NegationCondition <Boolean>] -MatchValue <String[]> [-Transform <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**New-AzApplicationGatewayFirewallCondition** skapar ett matchningsvillkor för den anpassade brandväggsregeln.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallCondition -MatchVariable $variable -Operator Contains -NegationCondition false -Transforms Lowercase, Trim -MatchValue abc, cde
```

Kommandot skapar ett nytt matchningsvillkor med den matchningsvariabel som definieras i $variable, operatorn är Innehåller och negationsvillkor är falskt, Transfroms inklusive gemener och trim, matchningsvärdet är abc och cde. Det nya matchningsvillkoret sparas i $condition.
