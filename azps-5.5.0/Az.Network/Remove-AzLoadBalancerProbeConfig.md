# Remove-AzLoadBalancerProbeConfig

## SYNOPSIS
Tar bort en avsökningskonfiguration från en belastningsutjämare.

## SYNTAX

```
Remove-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzLoadBalancerProbeConfig** tar bort en sonderkonfiguration från en belastningsutjämnare.

## EXEMPEL

### Exempel 1: Ta bort en sonde konfiguration från en belastningsutjämare
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $loadbalancer
```

Det första kommandot hämtar belastningsutjämaren med namnet MyLoadBalancer och lagrar den sedan $loadbalancer variabeln.
Det andra kommandot tar bort konfigurationen med namnet MyProbe från belastningsutjämaren i $loadbalancer.
