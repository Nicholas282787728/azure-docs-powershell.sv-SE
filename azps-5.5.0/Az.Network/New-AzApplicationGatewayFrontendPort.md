# New-AzApplicationGatewayFrontendPort

## SYNOPSIS
Skapar en frontend-port för en programgateway.

## SYNTAX

```
New-AzApplicationGatewayFrontendPort -Name <String> -Port <Int32> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzApplicationGatewayFrontendPort** skapar en frontend-port för en Azure-programgateway.

## EXEMPEL

### Exempel 1: Exempel1: Skapa en port på framsidan
```powershell
PS C:\>$FrontEndPort = New-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -Port 80
```

Med det här kommandot skapas en frontend-port med namnet FrontEndPort01 på port 80 och resultatet lagras i variabeln $FrontEndPort.
