# Test-AzDnsAvailability

## SYNOPSIS
Kontrollerar om ett domännamn i cloudapp.azure.com är tillgängligt för användning.

## SYNTAX

```
Test-AzDnsAvailability -DomainNameLabel <String> -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Kontrollerar om ett domännamn i cloudapp.azure.com är tillgängligt för användning.

## EXEMPEL

### Exempel 1: Kontrollera contoso.westus.cloudapp.azure.com är tillgängligt för användning.
```
Test-AzDnsAvailability -DomainNameLabel contoso -Location westus
```
