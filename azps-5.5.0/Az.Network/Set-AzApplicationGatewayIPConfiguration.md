# Set-AzApplicationGatewayIPConfiguration

## SYNOPSIS
Ändrar en IP-konfiguration för en programgateway.

## SYNTAX

### SetByResourceId
```
Set-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-SubnetId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResource
```
Set-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-Subnet <PSSubnet>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Set-AzApplicationGatewayIPConfiguration** ändrar en IP-konfiguration.
En IP-konfiguration innehåller undernätet där en programgateway distribueras.

## EXEMPEL

### Exempel 1: Uppdatera en IP-konfiguration för en programgateway
```
PS C:\>$VNet = Get-AzVirtualNetwork -Name "VNet01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Subnet = Get-AzVirtualNetworkSubnetConfig -Name "Subnet01" -VirtualNetwork $VNet 
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "AppgwSubnet01" -Subnet $Subnets
```

Det första kommandot får det virtuella nätverket med namnet VNet01 som tillhör resursgruppen ResourceGroup01 och lagrar det i $VNet variabeln.
Det andra kommandot hämtar undernätets konfiguration med namnet Subnet01 $VNet den och lagrar den $Subnet variabeln.
Det tredje kommandot hämtar en programgateway med namnet ApplicationGateway01 som tillhör resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln.
Med det här kommandot anger du IP-konfigurationen av programgatewayen som lagras i $AppGw till undernätets konfiguration som lagras i $Subnet.
