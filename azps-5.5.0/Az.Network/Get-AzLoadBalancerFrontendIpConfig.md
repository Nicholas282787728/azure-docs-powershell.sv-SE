# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Får en IP-konfiguration på frontend-sidan i en belastningsutjämnare.

## SYNTAX

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzLoadBalancerFrontendIpConfig** får en IP-konfiguration på frontend-sidan eller en lista med IP-konfigurationer på frontend-sidan i en belastningsutjämnare.

## EXEMPEL

### Exempel 1: Hämta en IP-konfiguration på framsidan i en belastningsutjämare
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

Det första kommandot får belastningsutjämaren MyLoadBalancer och lagrar den sedan i variabeln $slb.
Det andra kommandot hämtar IP-konfigurationen på frontend som är kopplad till den belastningsutjämaren.
