# Start-AzApplicationGateway

## SYNOPSIS
Startar en programgateway.

## SYNTAX

```
Start-AzApplicationGateway -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Start-AzApplicationGateway** startar en Azure-programgateway

## EXEMPEL

### Exempel1: Starta en programgateway
```
PS C:\>$AppGw = Start-AzApplicationGateway -ApplicationGateway $AppGw
```

Det här kommandot startar programmets gateway som lagras i den $AppGw variabeln.
