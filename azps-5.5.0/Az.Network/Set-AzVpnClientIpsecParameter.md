# Set-AzVpnClientIpsecParameter

## SYNOPSIS
Anger vpn ipsec-parametrar för befintlig virtuell nätverksgateway.

## SYNTAX

### ByFactoryName (standard)
```
Set-AzVpnClientIpsecParameter -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 -VpnClientIPsecParameter <PSVpnClientIPsecParameters> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Set-AzVpnClientIpsecParameter -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 -VpnClientIPsecParameter <PSVpnClientIPsecParameters> -InputObject <PSVirtualNetworkGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Set-AzVpnClientIpsecParameter -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 -VpnClientIPsecParameter <PSVpnClientIPsecParameters> -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Set-AzVpnClientIpsecParameter** cmdleten ställer in vpn ipsec-parametrar för befintlig virtuell nätverksgateway.
När virtuell nätverksgateway skapas anges standardprinciper för vpn ipsec för Gateway. Om punkt-till-webbplats-användaren vill använda vissa anpassade ipsec-policyer för att ansluta till VPN Gateway måste användaren först ange ipsec-principen på VPN Gateway. **Set-AzVpnClientIpsecParameter** är ett sätt att göra det.

## EXEMPEL

### Exempel 1: Anger en egen princip för vpn ipsec till befintlig virtuell nätverksgateway.
```powershell
PS C:\>$vpnclientipsecparams = New-AzVpnClientIpsecParameter -IpsecEncryption AES256 -IpsecIntegrity SHA256 -SALifeTime 86473 -SADataSize 429498 -IkeEncryption AES256 -IkeIntegrity SHA384 -DhGroup DHGroup2 -PfsGroup PFS2
PS C:\> $setvpnIpsecParams = Set-AzVpnClientIpsecParameter -VirtualNetworkGatewayName "ContosoLocalGateway" -ResourceGroupName "ContosoResourceGroup" -VpnClientIPsecParameter $vpnclientipsecparams
```

Det här exemplet anger en anpassad princip för vpn ipsec till befintlig virtuell nätverksgateway med namnet ContosoVirtualGateway från resursgruppen ContosoResourceGroup.
New-AzVpnClientIpsecParameter-cmdlet används för att skapa vpn ipsec-parametrarsobjektet för att använda de godkända parametrarna, som användaren kan ange för alla befintliga virtuella nätverksgateway i ResourceGroup.
Det skapade VpnClientIPsecParameters-objektet skickas till Set-AzVpnClientIpsecParameter-kommandot för att ange den angivna anpassade Vpn ipsec-principen på virtuell nätverksgateway som i exemplet ovan. Det här kommandot returnerar objektet för VpnClientIPsecParameters som visar ange parametrar.
