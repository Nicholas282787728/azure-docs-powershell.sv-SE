# Remove-AzFirewall

## SYNOPSIS
Ta bort en brandvägg.

## SYNTAX

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzFirewall** tar bort en Azure-brandvägg.

## EXEMPEL

### Exempel 1: Skapa och ta bort en brandvägg
```powershell
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

I det här exemplet skapas en brandvägg och sedan tas den bort. Om du vill dölja uppmaningen när du tar bort brandväggen använder du flaggan -Force.
