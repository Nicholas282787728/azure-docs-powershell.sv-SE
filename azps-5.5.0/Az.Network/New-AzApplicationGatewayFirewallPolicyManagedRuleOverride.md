# New-AzApplicationGatewayFirewallPolicyManagedRuleOverride

## SYNOPSIS
Skapar en managedRuleOverride-post för RuleGroupOverrideGroup-post.

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId <String> [-State <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Ny-AzApplicationGatewayFirewallPolicyManagedRuleOverride** skapar en ruleOverride-post.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> ruleOverrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId $ruleId -State Disabled
```

Skapar en ruleOverride-post med RuleId som $ruleId och State som inaktiverad.
