# Remove-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Tar bort en konfiguration av en inkommande NAT-regel från en belastningsutjämare.

## SYNTAX

```
Remove-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzLoadBalancerInboundNatRuleConfig** tar bort en konfiguration av regler för inkommande nätverksadressöversättning (NAT) från en Azure-belastningsutjämnare.

## EXEMPEL

### 1: Ta bort en inkommande NAT-regel från en Azure-belastningsutjämare
```
$loadbalancer = Get-AzLoadBalancer -Name mylb -ResourceGroupName myrg

 Remove-AzLoadBalancerInboundNatRuleConfig -Name "myinboundnatrule" -LoadBalancer $loadbalancer
```

Det första kommandot läser in ett redan befintligt belastningsutjämning med namnet "mylb" och lagrar det i variabeln $load balanseraren. Det andra kommandot tar bort den inkommande NAT-regeln som är kopplad till den här belastningsutjämaren.
