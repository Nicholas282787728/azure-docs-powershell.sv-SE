# Set-AzApplicationGatewayClientAuthConfiguration

## SYNOPSIS
Ändrar klientens autentiseringskonfiguration av ett SSL-profilobjekt.

## SYNTAX

```
Set-AzApplicationGatewayClientAuthConfiguration -SslProfile <PSApplicationGatewaySslProfile>
 [-VerifyClientCertIssuerDN] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzApplicationGatewayClientAuthConfiguration** ändrar klientauthkonfigurationen av ett ssl-profilobjekt.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "SslProfile01" -ApplicationGateway $AppGw
PS C:\> Set-AzApplicationGatewayClientAuthConfiguration -SslProfile $profile -VerifyClientCertIssuerDN
```

Det första kommandot hämtar programgatewayen med namnet ApplicationGateway01 i resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot hämtar SSL-profilen med namnet SslProfile01 för $AppGw lagrar inställningarna i variabeln $profile. Det senaste kommandot ändrar klientens autentiseringskonfiguration av ssl-profilobjektet som lagras i $profile.
