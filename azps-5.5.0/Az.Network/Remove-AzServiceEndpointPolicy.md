# Remove-AzServiceEndpointPolicy

## SYNOPSIS
Tar bort en tjänstslutpunktsprincip.

## SYNTAX

### RemoveByNameParameterSet (standard)
```
Remove-AzServiceEndpointPolicy -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzServiceEndpointPolicy -ResourceId <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzServiceEndpointPolicy -InputObject <PSServiceEndpointPolicy> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzServiceEndpointPolicy** tar bort en tjänstslutpunktsprincip.

## EXEMPEL

### Exempel 1: Tar bort en tjänstslutpunktsprincip med namn
```
Remove-AzServiceEndpointPolicy -Name "Policy1" -ResourceGroup "resourcegroup1"
```

Det här kommandot tar bort en tjänstslutpunktsprincip med namnprincip1 som tillhör resursgruppen med namnet "resourcegroup1"

### Exempel 2: Ta bort en tjänstslutpunktsprincip med hjälp av indataobjekt
```
Remove-AzServiceEndpointPolicy -InputObject $Policy1 -ResourceGroup "resourcegroup1"
```

Med det här kommandot tas en principprincip för tjänstslutpunktsobjekt1 bort som tillhör resursgruppen med namnet "resourcegroup1"
