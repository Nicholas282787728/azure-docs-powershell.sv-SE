# Remove-AzVirtualNetworkGateway

## SYNOPSIS
Tar bort en virtuell nätverksgateway

## SYNTAX

```
Remove-AzVirtualNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den virtuella nätverksgatewayen är det objekt som representerar din gateway i Azure.
Cmdleten **Get-AzVirtualNetworkGateway** returnerar gatewayens objekt i Azure baserat på namn och resursgruppnamn.

## EXEMPEL

### Exempel 1: Ta bort en virtuell nätverksgateway
```powershell
Remove-AzVirtualNetworkGateway -Name myGateway -ResourceGroupName myRG
```

Tar bort objektet för den virtuella nätverksgatewayen med namnet "myGateway" i resursgruppen "myRG"- Obs! Du måste först ta bort alla anslutningar till den virtuella nätverksgatewayen med **cmdleten Remove-AzVirtualNetworkGatewayConnection.**
