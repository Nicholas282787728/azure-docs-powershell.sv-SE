# Move-AzExpressRouteCircuit

## SYNOPSIS
Flyttar en ExpressRoute-krets från den klassiska distributionsmodellen till resurshanterarens distributionsmodell.

## SYNTAX

```
Move-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> -Location <String> -ServiceKey <String>
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Move-AzExpressRouteCircuit** flyttar en ExpressRoute-krets från den klassiska distributionsmodellen till resurshanterarens distributionsmodell. Efter flytten fungerar ExpressRoute-kretsen som vilken annan ExpressRoute-krets som helst som skapas i resurshanterarens distributionsmodell. Kretslänkar, virtuella nätverk och VPN-gatewayer flyttas inte genom den här åtgärden. Dessa resurser måste konfigureras om efter flytten.

## EXEMPEL

### Exempel 1: Flytta en ExpressRoute-krets till resurshanterarens distributionsmodell
```
Move-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $RG -Location $Location -ServiceKey $ServiceKey
```
