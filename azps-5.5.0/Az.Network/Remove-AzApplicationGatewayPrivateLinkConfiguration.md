# Remove-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Tar bort en privateLink-konfiguration från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewayPrivateLinkConfiguration** tar bort en privateLink-konfiguration från en Azure-programgateway.

## EXEMPEL

### Exempel 1: Ta bort en konfiguration av en programgateway för PrivateLink
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

Det första kommandot hämtar en programgateway och lagrar den i $AppGw variabeln.
Det andra kommandot tar bort privateLink-konfigurationen med namnet privateLinkConfig01 från programgatewayen som lagras i $AppGw.
