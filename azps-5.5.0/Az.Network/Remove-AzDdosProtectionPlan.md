# Remove-AzDdosProtectionPlan

## SYNOPSIS
Tar bort en DDoS-plan.

## SYNTAX

```
Remove-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzDdosProtectionPlan cmdleten tas en DDoS-skyddsplan bort.

## EXEMPEL

### Exempel 1: Ta bort en tom DDoS-skyddsplan
```
D:\> Remove-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

I det här fallet tar vi bort en DDoS-skyddsplan enligt givna.

### Exempel 2: Ta bort en DDoS-skyddsplan som är kopplad till ett virtuellt nätverk
```
D:\> $vnet = Get-AzVirtualNetwork -Name VnetName -ResourceGroupName ResourceGroupName
D:\> $vnet.DdosProtectionPlan = $null
```

DDoS-skyddsplaner kan inte tas bort om de är associerade med ett virtuellt nätverk. Det första steget är därför att ta bort associationer mellan båda objekten. Här får vi den mest uppdaterade versionen av det virtuella nätverket som är kopplat till planen och vi anger ett tomt värde för egenskapen **DdosProtectionPlan** och flaggan **EnableDdosProtection** (flaggan får inte vara sann utan en plan).
Sedan beständiga vi den nya statusen genom att pipa den lokala variabeln till **Set-AzVirtualNetwork.** I det här läget är planen inte längre kopplad till det virtuella nätverket.
Om det här är den sista som är kopplad till planen kan vi ta bort DDoS-skyddsplanen med kommandot Remove-AzDdosProtectionPlan.
