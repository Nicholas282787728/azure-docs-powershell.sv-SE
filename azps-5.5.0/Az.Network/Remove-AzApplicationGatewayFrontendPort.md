# Remove-AzApplicationGatewayFrontendPort

## SYNOPSIS
Tar bort en frontend-port från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayFrontendPort -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGatewayFrontendPort** tar bort en port på framsidan från en Azure-programgateway.

## EXEMPEL

### Exempel: Ta bort en frontend-port från en programgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort02"
```

Det första kommandot hämtar en programgateway med namnet ApplicationGateway01 som tillhör resursgruppen ResourceGroup01 och lagrar gatewayen i en $AppGw variabel.
Det andra kommandot tar bort porten FrontEndPort02 från programmets gateway.
