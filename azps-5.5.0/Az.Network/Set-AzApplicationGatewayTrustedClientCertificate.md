# Set-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Ändrar certifikatkedja för betrodd klient-CA för en programgateway.

## SYNTAX

```
Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzApplicationGatewayTrustedClientCertificate** ändrar certifikatkedja för betrodd klient-CA för en programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\clientCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Ovanstående exempelscenarier visar hur du uppdaterar ett befintligt certifikatkedjeobjekt för en betrodd klient-CERTIFIKATUTFÄRDARE. Det första kommandot får en programgateway och lagrar den i $gw variabeln. Det andra kommandot ändrar det befintliga objektet i certifikatkedjeobjektet för betrodd klient-CERTIFIKATUTFÄRDARE med en ny kedjefil för certifikatutfärdare. Det tredje kommandot uppdaterar programmets gateway i Azure.
