# Add-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Lägger till ett autentiseringscertifikat i en programgateway.

## SYNTAX

```
Add-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Add-AzApplicationGatewayAuthenticationCertificate** lägger till ett autentiseringscertifikat i en Azure-programgateway.

## EXEMPEL

### Exempel 1: Lägga till autentiseringscertifikat i en programgateway
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Add-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01" -CertificateFile "C:\cert.cer"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Det första kommandot får en programgateway med namnet appGwName och lagrar det i en $appgw variabel.
Det andra kommandot lägger till autentiseringscertifikatet cert01 i programmets gateway.
Det tredje kommandot uppdaterar programmets gateway.
