# Get-AzExpressRouteCrossConnectionRouteTable

## SYNOPSIS
Hämtar en routetabell från en ExpressRoute-korsanslutning.

## SYNTAX

### SpecifyByParameterValues
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName <String> -CrossConnectionName <String>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### SpecifyByReference
```
Get-AzExpressRouteCrossConnectionRouteTable -ExpressRouteCrossConnection <PSExpressRouteCrossConnection>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**Get-AzExpressRouteCrossConnectionRouteTable-cmdleten** hämtar en detaljerad routetabell av en ExpressRoute-krets. I tabellen Route visas alla vägar eller kan filtreras för att visa routes för en viss peeringtyp. Du kan använda routetabellen för att verifiera din peeringkonfiguration och anslutning.

## EXEMPEL

### Exempel 1: Visa routetabellen för den primära sökvägen
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName $RG -ExpressRouteCrossConnectionName $CircuitName -DevicePath 'Primary'
```
