# Get-AzApplicationGatewayAvailableSslOption

## SYNOPSIS
Hämtar alla tillgängliga SSL-alternativ för SSL-princip för Application Gateway.

## SYNTAX

```
Get-AzApplicationGatewayAvailableSslOption [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzApplicationGatewayAvailableSslOption** hämtar alla tillgängliga ssl-alternativ för ssl-principen

## EXEMPEL

### Exempel 1
```
PS C:\>$sslOptions = Get-AzApplicationGatewayAvailableSslOption
```

Med de här kommandona returneras alla tillgängliga SSL-alternativ för SSL-principen.
