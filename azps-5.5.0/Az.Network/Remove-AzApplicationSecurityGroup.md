# Remove-AzApplicationSecurityGroup

## SYNOPSIS
Tar bort en programsäkerhetsgrupp.

## SYNTAX

```
Remove-AzApplicationSecurityGroup -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationSecurityGroup** tar bort en programsäkerhetsgrupp.

## EXEMPEL

### Exempel 1
```
PS C:\>Remove-AzApplicationSecurityGroup -Name "MyApplicationSecurityGrouo" -ResourceGroupName "MyResourceGroup"
```

Med det här kommandot tas en programsäkerhetsgrupp med namnet MyApplicationSecurityGrouo bort i resursgruppen MyResourceGroup.
