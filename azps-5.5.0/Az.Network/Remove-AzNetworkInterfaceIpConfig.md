# Remove-AzNetworkInterfaceIpConfig

## SYNOPSIS
Tar bort en IP-konfiguration för nätverksgränssnittet från ett nätverksgränssnitt.

## SYNTAX

```
Remove-AzNetworkInterfaceIpConfig -Name <String> -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzNetworkInterfaceIpConfig** tar bort en IP-konfiguration för nätverksgränssnitt från ett Azure-nätverksgränssnitt.

## EXEMPEL

### Exempel 1: Ta bort en IP-konfiguration från ett nätverksgränssnitt
```powershell
$nic = Get-AzNetworkInterface -Name mynic -ResourceGroupName myrg

Remove-AzNetworkInterfaceIpConfig -Name IPConfig-1 -NetworkInterface $nic

Set-AzNetworkInterface -NetworkInterface $nic
```

Det första kommandot får ett nätverksgränssnitt som kallas mynic och lagrar det i variabeln $nic. Det andra kommandot tar bort IP-konfigurationen som kallas IPConfig-1 som är kopplad till det här nätverksgränssnittet. Det tredje kommandot anger ändringar gjorda i nätverksgränssnittet.
