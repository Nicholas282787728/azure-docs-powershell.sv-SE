# Get-AzLoadBalancerInboundNatPoolConfig

## SYNOPSIS
Hämtar en eller flera inkommande NAT-poolkonfigurationer från en belastningsutjämare.

## SYNTAX

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzLoadBalancerInboundNatPoolConfig** får en eller flera konfigurationer av inkommande NAT-pooler från en belastningsutjämare.

## EXEMPEL

### 1: Skaffa
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
