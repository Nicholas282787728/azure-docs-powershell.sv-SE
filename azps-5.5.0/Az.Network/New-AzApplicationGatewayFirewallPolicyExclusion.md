# New-AzApplicationGatewayFirewallPolicyExclusion

## SYNOPSIS
Skapar ett undantag i brandväggsprincipen

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicyExclusion -MatchVariable <String> -SelectorMatchOperator <String>
 -Selector <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzApplicationGatewayFirewallPolicyExclusion** är en ny undantagsregellista för brandväggsprincipen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $exclusionEntry = New-AzApplicationGatewayFirewallPolicyExclusion -MatchVariable "RequestHeaderNames" -SelectorMatchOperator "StartsWith" -Selector "xyz"
```

Det här kommandot skapar en ny exkluderingspost för variabeln RequestHeaderNames och operatorn StartsWith och Selector med namnet xyz. Undantagsposten sparas i en $exclusionEntry.
