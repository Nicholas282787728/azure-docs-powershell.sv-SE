# Set-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Uppdaterar en principdefinition för tjänstslutpunkt.

## SYNTAX

```
Set-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzServiceEndpointPolicyDefinition** skapar en principdefinition för tjänstslutpunkt.

## EXEMPEL

### Exempel 1: Uppdaterar en slutpunktsprincipdefinition för en tjänstslutpunkt i en tjänstslutpunktsprincip
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicyDefinition -Name "Policydef1" -ServiceEndpointPolicy $serviceEndpointPolicy
```

Det här kommandot uppdaterar en tjänstslutpunktsprincipdefinition med namnet Policydef1 i tjänstslutpunktsprincipen som definieras av $ServiceEndpointPolicy.
