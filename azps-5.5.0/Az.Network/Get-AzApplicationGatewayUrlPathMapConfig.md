# Get-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
Hämtar en matris med URL-sökvägsmappningar till en serverpool för serverbackend.

## SYNTAX

```
Get-AzApplicationGatewayUrlPathMapConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApplicationGatewayURLPathMapConfig** får en matris med URL-sökvägsmappningar till en serverpool för serverbackend.

## EXEMPEL

### Exempel 1: Hämta en konfiguration av url-sökvägskartan
```
PS C:\>Get-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway Gateway
```

Det här kommandot hämtar konfigurationer av URL-sökvägskartan från serverbackendservern som finns på programmets gateway med namnet Gateway.
