# Remove-AzLocalNetworkGateway

## SYNOPSIS
Tar bort en lokal nätverksgateway

## SYNTAX

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den lokala nätverksgatewayen är det objekt som representerar din lokala VPN-enhet.
Cmdleten **Remove-AzLocalNetworkGateway** tar bort objektet som representerar den lokala gatewayen baserat på namn och resursgruppnamn.

## EXEMPEL

### Exempel 1: Ta bort en lokal nätverksgateway
```powershell
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Tar bort objektet för den lokala nätverksgatewayen med namnet "myLocalGW" i resursgruppen "myRG": Du måste först ta bort alla anslutningar till den lokala nätverksgatewayen med **cmdleten Remove-AzVirtualNetworkGatewayConnection.**
