# New-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Skapar en konfiguration för automatisk skalning för Application Gateway.

## SYNTAX

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzApplicationGatewayAutoscaleConfiguration** skapar automatisk skalkonfiguration för en Azure-programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Det första kommandot skapar en konfiguration för automatisk skalning med minsta kapacitet 3.
Det andra kommandot skapar en programgateway med automatisk skalkonfiguration.
