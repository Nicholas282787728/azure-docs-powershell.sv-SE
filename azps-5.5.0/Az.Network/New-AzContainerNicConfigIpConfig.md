# New-AzContainerNicConfigIpConfig

## SYNOPSIS
Skapar ett konfigurationsobjekt för konfigurations-IP för behållare.

## SYNTAX

```
New-AzContainerNicConfigIpConfig -Name <String> -Subnet <PSSubnet> [-SubnetId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzContainerNicConfigIpConfig** skapar en ny konfigurations-IP-konfiguration för behållarnätverksgränssnitt. 

## EXEMPEL

### Exempel 1
```powershell
$subnet = New-AzVirtualNetworkSubnetConfig -Name subnet -AddressPrefix 10.0.1.0/24

$vnet = New-AzVirtualNetwork -Name vnet -ResourceGroupName rg1 -Location "West US" -AddressPrefix 10.0.0.0/16 -Subnet $subnet

$containerNicConfigIpConfig = New-AzContainerNicConfigIpConfig -Name ipconfigprofile1 -Subnet $vnet.Subnets[0]

$containerNicConfig = New-AzContainerNicConfig -Name cnic -IpConfiguration containerNicConfigIpConfig

$networkProfile = New-AzNetworkProfile -Name np1 -Location "West US" -ResourceGroupName rg1 -ContainerNetworkInterfaceConfiguration $containerNicConfig
```

De två första kommandona skapar och initierar ett vnet och ett undernät. Det tredje kommandot skapar en konfigurationsprofil för behållare för ip som refererar till det skapade undernätet. Det fjärde kommandot skapar en konfiguration av behållarnätverksgränssnittet som tillhandahåller ip-konfigurationsprofilen som skapades i det föregående kommandot. Slutligen skapas en nätverksprofil som initieras med konfigurationen av behållarnätverksgränssnittet som lagras i $containerNicConfig.
