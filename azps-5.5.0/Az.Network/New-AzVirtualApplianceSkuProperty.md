# New-AzVirtualApplianceSkuProperty

## SYNOPSIS
Definiera en virtuell nätverksenhets-SKU för resursen.

## SYNTAX

```
New-AzVirtualApplianceSkuProperty -VendorName <String> -BundledScaleUnit <String> -MarketPlaceVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Kommandot New-AzVirtualApplianceSkuProperties definierar en SKU för virtuell resurs för nätverk.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $var=New-AzVirtualApplianceSkuProperty -VendorName "barracudasdwanrelease" -BundledScaleUnit 1 -MarketPlaceVersion 'latest'
```

Skapa ett SKU-egenskapsobjekt för virtuellt</New-AzNetworkVirtualAppliance objekt.
