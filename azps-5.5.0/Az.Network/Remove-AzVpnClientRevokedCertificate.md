# Remove-AzVpnClientRevokedCertificate

## SYNOPSIS
Tar bort ett VPN-klientåterkallningscertifikat.

## SYNTAX

```
Remove-AzVpnClientRevokedCertificate -VpnClientRevokedCertificateName <String>
 -VirtualNetworkGatewayName <String> -ResourceGroupName <String> -Thumbprint <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzVpnClientRevokedCertificate** tar bort ett klientåterkallningscertifikat från en virtuell nätverksgateway.
Klientåterkallningscertifikat hindrar klientdatorer från att använda det angivna certifikatet för autentisering.
Om du tar bort en klientåterkallningscertifikatklientdator kan du använda det tidigare spärrade certifikatet för att upprätta en virtuell privat nätverksanslutning (VPN).

## EXEMPEL

### Exempel 1: Ta bort ett klientåterkallningscertifikat från en virtuell nätverksgateway
```powershell
PS C:\>Remove-AzVpnClientRevokedCertificate -VirtualNetworkGatewayName "ContosoVirtualNetwork" -ResourceGroupName"ContosoResourceGroup" -VpnClientRevokedCertificateName "ContosoRevokedClientCertificate" -Thumbprint "E3A38EBA60CAA1C162785A2E1C44A15AD450199C3"
```

Det här kommandot tar bort ett klientåterkallningscertifikat från en virtuell nätverksgateway med namnet ContosoVirtualNetwork.
Om du vill ta bort ett klientåterkallningscertifikat måste du ange både certifikatnamnet och certifikatets tumavtryck.
