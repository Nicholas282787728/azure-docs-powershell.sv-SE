# Remove-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Tar bort en omskrivningsregeluppsättning från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayRewriteRuleSet -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGatewayRewriteRuleSet** tar bort en regeluppsättning för omskrivning från en Azure-programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "RuleSet02"
```

Det första kommandot får en programgateway och lagrar den i $AppGw variabeln.
Det andra kommandot tar bort den omskrivningsregeluppsättning som heter RuleSet02 från programmets gateway som lagras i $AppGw.
