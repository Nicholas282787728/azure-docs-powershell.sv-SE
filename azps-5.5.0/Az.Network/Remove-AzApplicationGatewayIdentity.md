# Remove-AzApplicationGatewayIdentity

## SYNOPSIS
Tar bort en identitet från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewayIdentity** tar bort identitet från en programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $appgw = Remove-AzApplicationGatewayIdentity -ApplicationGateway $appgw
PS C:\> $updatedgateway = Set-AzApplicationGateway -ApplicationGateway $appgw
```

I det här exemplet tar vi bort identitet från en befintlig programgateway.
Obs! Om gatewayen refererar till en nyckelvaultshemlighet är det också viktigt att ta bort ssl-certifikatreferenserna i den här åtgärden.
