# Add-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Lägger till en kedja av betrodd klient-CERTIFIKATutfärdare i en programgateway.

## SYNTAX

```
Add-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Add-AzApplicationGatewayTrustedClientCertificate** lägger till en betrodd klient-CA-certifikatkedja i en Azure-programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClient = New-AzApplicationGatewayTrustedClientCertificate -Name "ClientCert" -CertificateFile "C:\clientCAChain.cer"
PS C:\> $AppGw = Add-AzApplicationGatewaySslProfile -Name $sslProfileName -ApplicationGateway $gw -TrustedClientCertificates $trustedClient
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Det första kommandot hämtar programmets gateway och lagrar den i en $gw variabel. Det andra kommandot skapar en ny betrodd klient-CA-certifikatkedja som tar sökvägen till klientcertifikatet för certifikatutfärdaren som indata. Det tredje kommandot skapar en SSL-profil med hjälp av ett betrott klientcertifikat. Det fjärde kommandot uppdaterar Application Gateway.
