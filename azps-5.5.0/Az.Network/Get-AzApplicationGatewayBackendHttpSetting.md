# Get-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
Hämtar HTTP-inställningarna i backend för en programgateway.

## SYNTAX

```
Get-AzApplicationGatewayBackendHttpSetting [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den Get-AzApplicationGatewayBackendHttpSetting cmdleten hämtar HTTP-inställningarna i backend-end för en programgateway.

## EXEMPEL

### Exempel 1: Hämta HTTP-inställningarna i backend efter namn
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Settings  = Get-AzApplicationGatewayBackendHttpSetting -Name "Settings01" -ApplicationGateway $AppGw
```

Det första kommandot hämtar programgatewayen med namnet ApplicationGateway01 i resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot hämtar HTTP-inställningarna med namnet Settings01 för $AppGw lagrar inställningarna i variabeln $Settings variabeln.

### Exempel 2: Hämta en samling HTTP-inställningar i backend
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SettingsList  = Get-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw
```

Det första kommandot hämtar programgatewayen med namnet ApplicationGateway01 i resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot hämtar samlingen http-inställningar för $AppGw och lagrar inställningarna i variabeln $SettingsList variabeln.
