# New-AzApplicationGatewayFirewallDisabledRuleGroupConfig

## SYNOPSIS
Skapar en ny inaktiverad regelgruppskonfiguration.

## SYNTAX

```
New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName <String> [-Rules <Int32[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzApplicationGatewayFirewallDisabledRuleGroupConfig** skapar en ny inaktiverad regelgruppskonfiguration.

## EXEMPEL

### Exempel 1
```
PS C:\> $disabledRuleGroup1 = New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName "REQUEST-942-APPLICATION-ATTACK-SQLI" -Rules 942130,942140
```

Kommandot skapar en ny inaktiverad regelgruppskonfiguration för regelgruppen med namnet "BEGÄR-942-APPLICATION-ATTACK-SQLI" med regel 942130 och regel 942140 inaktiverade. Den nya inaktiverade regelgruppskonfigurationen sparas i $disabledRuleGroup 1.
