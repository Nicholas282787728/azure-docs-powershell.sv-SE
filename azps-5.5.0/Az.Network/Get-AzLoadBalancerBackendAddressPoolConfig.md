# Get-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Hämtar en konfiguration av adresspoolen för en belastningsutjämning.

## SYNTAX

```
Get-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzLoadBalancerBackendAddressPoolConfig** får en enda backend-adresspool eller en lista med backend-adresspooler i en belastningsutjämnare.

## EXEMPEL

### Exempel 1: Hämta adresspoolen för backend
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" -LoadBalancer $loadbalancer
```

Det första kommandot hämtar en befintlig belastningsutjämnare med namnet MyLoadBalancer i resursgruppen MyResourceGroup och lagrar den sedan i $loadbalancer variabeln.
Det andra kommandot hämtar den associerade serverplatsadresspoolkonfigurationen med namnet BackendAddressPool02 för belastningsutjämning i $loadbalancer.
