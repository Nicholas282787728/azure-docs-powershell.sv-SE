# Get-AzBgpServiceCommunity

## SYNOPSIS
Visar en lista över alla tjänster/regioner, BGP-communities och tillhörande prefix.

## SYNTAX

```
Get-AzBgpServiceCommunity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den här cmdleten tillhandahåller en lista över alla tjänster/regioner, BGP-communities och tillhörande prefix.
