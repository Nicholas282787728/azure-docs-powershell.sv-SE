# New-AzApplicationGatewaySslCertificate

## SYNOPSIS
Skapar ett SSL-certifikat för en Azure-programgateway.

## SYNTAX

```
New-AzApplicationGatewaySslCertificate -Name <String> [-CertificateFile <String>] [-Password <SecureString>]
 [-KeyVaultSecretId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzApplicationGatewaySslCertificate** skapar ett SSL-certifikat för en Azure-programgateway.

## EXEMPEL

### Exempel 1: Skapa ett SSL-certifikat för en Azure Application Gateway.
```
PS C:\> $password = ConvertTo-SecureString $passwordPlainString -AsPlainText -Force
PS C:\> $cert = New-AzApplicationGatewaySslCertificate -Name "Cert01" -CertificateFile "D:\cert01.pfx" -Password $password
```

Det här kommandot skapar ett SSL-certifikat med namnet Cert01 för standardprogramgatewayen och lagrar resultatet i variabeln $Cert.

### Exempel 2: Skapa ett SSL-certifikat med KeyVault Secret (version mindre hemligtId) och lägg till i en programgateway.
```
PS C:\> $secret = Get-AzKeyVaultSecret -VaultName "keyvault01" -Name "sslCert01"
PS C:\> $secretId = $secret.Id.Replace($secret.Version, "") # https://<keyvaultname>.vault.azure.net/secrets/
PS C:\> $cert = New-AzApplicationGatewaySslCertificate -Name "Cert01" -KeyVaultSecretId $secretId
```

Hämta hemligheten och skapa ett SSL-certifikat med hjälp av `New-AzApplicationGatewaySslCertificate` .
Obs! Eftersom det finns ett hemligt Id med lägre versionsnummer här synkroniserar Application Gateway certifikatet med jämna mellanrum med KeyVault.

### Exempel 3: Skapa ett SSL-certifikat med KeyVault Secret och lägg till i en Application Gateway.
```
PS C:\> $secret = Get-AzKeyVaultSecret -VaultName "keyvault01" -Name "sslCert01"
PS C:\> $secretId = $secret.Id # https://<keyvaultname>.vault.azure.net/secrets/<hash>
PS C:\> $cert = New-AzApplicationGatewaySslCertificate -Name "Cert01" -KeyVaultSecretId $secretId
```

Hämta hemligheten och skapa ett SSL-certifikat med hjälp av `New-AzApplicationGatewaySslCertificate` .
Obs! Om det krävs att Application Gateway synkroniserar certifikatet med KeyVault anger du det hemliga ID:t (version mindre hemlig).
