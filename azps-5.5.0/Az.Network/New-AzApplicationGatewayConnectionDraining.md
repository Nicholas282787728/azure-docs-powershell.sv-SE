# New-AzApplicationGatewayConnectionDraining

## SYNOPSIS
Skapar en ny konfiguration för anslutningsav tömning för HTTP-inställningarna i backend.

## SYNTAX

```
New-AzApplicationGatewayConnectionDraining -Enabled <Boolean> -DrainTimeoutInSec <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzApplicationGatewayConnectionDraining** skapar en ny konfiguration för anslutningsavgång för HTTP-inställningar i serverslut.

## EXEMPEL

### Exempel 1
```
PS C:\> $connectionDraining = New-AzApplicationGatewayConnectionDraining -Enabled $True -DrainTimeoutInSec 42
```

Kommandot skapar en ny konfiguration för anslutningsfördring med Aktiverad inställd på True och DrainTimeoutInSec inställt på 42 sekunder och lagrar den $connectionDraining.
