# Set-AzVirtualNetwork

## SYNOPSIS
Uppdaterar ett virtuellt nätverk.

## SYNTAX

```
Set-AzVirtualNetwork -VirtualNetwork <PSVirtualNetwork> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzVirtualNetwork** uppdaterar ett virtuellt nätverk.

## EXEMPEL

### 1: Skapar ett virtuellt nätverk och tar bort ett av dess undernät
```powershell
New-AzResourceGroup -Name TestResourceGroup -Location centralus ## Create resource group 
$frontendSubnet = New-AzVirtualNetworkSubnetConfig -Name frontendSubnet -AddressPrefix "10.0.1.0/24" ## Create frontend subnet 
$backendSubnet = New-AzVirtualNetworkSubnetConfig -Name backendSubnet -AddressPrefix "10.0.2.0/24" ## Create backend subnet

$virtualNetwork = New-AzVirtualNetwork -Name MyVirtualNetwork -ResourceGroupName TestResourceGroup `
    -Location centralus -AddressPrefix "10.0.0.0/16" -Subnet $frontendSubnet,$backendSubnet ## Create virtual network

Remove-AzVirtualNetworkSubnetConfig -Name backendSubnet -VirtualNetwork $virtualNetwork ## Remove subnet from in memory representation of virtual network

$virtualNetwork | Set-AzVirtualNetwork ## Remove subnet from virtual network
```

I det här exemplet skapas ett virtuellt nätverk med namnet TestResourceGroup med två undernät: frontendSubnet och backendSubnet. Därefter tas backendSubnet-undernätet bort från det virtuella nätverkets representation av minnet. Den Set-AzVirtualNetwork cmdleten används sedan för att skriva den ändrade virtuella nätverkstillståndet på tjänstsidan. När Set-AzVirtualNetwork-cmdleten körs tas backendSubnet bort.
