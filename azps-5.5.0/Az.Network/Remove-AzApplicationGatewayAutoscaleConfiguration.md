# Remove-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Tar bort automatisk skalkonfiguration från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewayAutoscaleConfiguration** tar bort Automatisk skalkonfiguration från en befintlig Application Gateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Det första kommandot hämtar programmets gateway och lagrar den i en $gw variabel.
Det andra kommandot tar bort konfigurationen av automatisk skalning från programgatewayen.
Det tredje kommandot uppdaterar programmets gateway i Azure.
