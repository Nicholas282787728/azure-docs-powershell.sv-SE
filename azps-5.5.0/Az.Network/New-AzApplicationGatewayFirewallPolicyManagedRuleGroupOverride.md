# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## SYNOPSIS
Skapar RuleGroupOverride-post i ManagedRuleSets för brandväggsprincipen.

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** skapar en ruleGroupOverride-post i en managedRuleSet för en brandväggsprincip.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Skapar en RuleGroupOverride-post med gruppnamn som $ruleName och Regler som $rule 1, $rule 2. Tilldelar samma $overrideEntry
