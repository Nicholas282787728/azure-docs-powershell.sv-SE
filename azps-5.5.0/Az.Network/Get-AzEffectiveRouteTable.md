# Get-AzEffectiveRouteTable

## SYNOPSIS
Hämtar den effektiva routetabellen i ett nätverksgränssnitt.

## SYNTAX

```
Get-AzEffectiveRouteTable -NetworkInterfaceName <String> [-ResourceGroupName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzEffectiveRouteTable** returnerar den effektiva routetabellen som används i ett nätverksgränssnitt.

## EXEMPEL

### Exempel 1: Hämta den effektiva routetabellen i ett nätverksgränssnitt
```
PS C:\>Get-AzEffectiveRouteTable -NetworkInterfaceName "MyNetworkInterface" -ResourceGroupName "MyResourceGroup"
```

Med det här kommandot får du den effektiva routetabellen som är kopplad till nätverksgränssnittet MyNetworkInterface i resursgruppen MyResourceGroup.
