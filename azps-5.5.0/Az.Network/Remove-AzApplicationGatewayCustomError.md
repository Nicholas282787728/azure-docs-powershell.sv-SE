# Remove-AzApplicationGatewayCustomError

## SYNOPSIS
Tar bort ett anpassat fel från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayCustomError -StatusCode <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewayCustomError** tar bort ett anpassat fel från en programgateway.

## EXEMPEL

### Exempel 1: Tar bort anpassat fel från en programgateway
```
PS C:\> $updatedgateway = Remove-AzApplicationGatewayCustomError -ApplicationGateway $appgw -StatusCode HttpStatus502
```

Det här kommandot tar bort det anpassade felet för http-statuskod 502 från programmets gateway$appgw och returnerar den uppdaterade gatewayen.
