# Get-AzApplicationGatewaySslProfilePolicy

## SYNOPSIS
Hämtar SSL-principen för en SSL-profil för en programgateway.

## SYNTAX

```
Get-AzApplicationGatewaySslProfilePolicy -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzApplicationGatewaySslProfilePolicy hämtar SSL-principen** för en SSL-profil för programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SslProfile  = Get-AzApplicationGatewaySslProfile -Name "SslProfile01" -ApplicationGateway $AppGw
PS C:\> $sslpolicy = Get-AzApplicationGatewaySslProfilePolicy -SslProfile $SslProfile
```

Det första kommandot hämtar programgatewayen med namnet ApplicationGateway01 i resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot hämtar SSL-profilen med namnet SslProfile01 för $AppGw lagrar den $SslProfile variabeln. Det senaste kommandot hämtar SSL-principen från SSL-profilen $SslProfile lagrar den i variabeln $sslpolicy variabeln.
