# Remove-AzApplicationGatewayFrontendIPConfig

## SYNOPSIS
Tar bort en IP-konfiguration på framsidan från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayFrontendIPConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewayFrontendIPConfig** tar bort frontend IP från en Azure-programgateway.

## EXEMPEL

### Exempel 1: Ta bort en IP-konfiguration på framsidan
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendIPConfig -ApplicationGateway $AppGw -Name "FrontEndIP02"
```

Det första kommandot får en programgateway med namnet ApplicationGateway01 och lagrar den $AppGw variabeln.
Det andra kommandot tar bort frontend IP-konfigurationen med namnet FrontEndIP02 från programgatewayen som lagras i $AppGw.
