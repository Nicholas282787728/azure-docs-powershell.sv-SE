# Remove-AzApplicationGatewaySslProfile

## SYNOPSIS
Tar bort ssl-profilen från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslProfile -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApplicationGatewaySslProfile** tar bort ssl-profilen från en programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslProfile -ApplicationGateway $AppGw -Name "SslProfile01"
```

Det första kommandot hämtar en programgateway med namnet ApplicationGateway01 som tillhör resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot tar bort ssl-profilen med namnet SslProfile01 från programgatewayen som lagras i $AppGw.
