# New-AzApplicationGatewayFirewallPolicySetting

## SYNOPSIS
Skapar en principinställning för brandväggsprincipen

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicySetting [-Mode <String>] [-State <String>] [-DisableRequestBodyCheck]
 [-MaxRequestBodySizeInKb <Int32>] [-MaxFileUploadInMb <Int32>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**New-AzApplicationGatewayFirewallPolicySetting** skapar principinställningar för en brandväggsprincip.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallPolicySetting -State $enabledState -Mode $enabledMode -DisableRequestBodyCheck -MaxFileUploadInMb $fileUploadLimitInMb -MaxRequestBodySizeInKb $maxRequestBodySizeInKb
```

Kommandot skapar en principinställning med statusen $enabledState, läge som $enabledMode, RequestKommandoCheck som falskt, FileUploadLimitInMb som $fileUploadLimitInMb och MaxRequest DenSizeInKb som $$maxRequestBodySizeInKb.
De nya principinställningar lagras på $condition.
