# Add-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Lägger till en IP-konfiguration på framsidan till en belastningsutjämare.

## SYNTAX

### SetByResourceSubnet (standard)
```
Add-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-PrivateIpAddress <String>]
 [-PrivateIpAddressVersion <String>] [-Zone <String[]>] -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByResourceIdSubnet
```
Add-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-PrivateIpAddress <String>]
 [-PrivateIpAddressVersion <String>] [-Zone <String[]>] -SubnetId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

```

 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Add-AzLoadBalancerFrontendIpConfig** lägger till en FRONTEND IP-konfiguration i en Azure-belastningsutjämare.

## EXEMPEL

### Exempel 1 Lägga till en IP-konfiguration på framsidan med en dynamisk IP-adress
```
PS C:\> $Subnet = Get-AzVirtualNetwork -Name "MyVnet" -ResourceGroupName "MyRg" | Get-AzVirtualNetworkSubnetConfig -Name "MySubnet"
PS C:\> Get-AzLoadBalancer -Name "MyLB" -ResourceGroupName "NrpTest" | Add-AzLoadBalancerFrontendIpConfig -Name "FrontendName" -Subnet $Subnet | Set-AzLoadBalancer
```

Det första kommandot hämtar det virtuella Azure-nätverket med namnet MyVnet och skickar resultatet med hjälp av pipelinen till cmdleten **Get-AzVirtualNetworkSubnetConfig** för att hämta undernätet MittSubnet.
Kommandot lagrar sedan resultatet i variabeln med namnet $Subnet.
Det andra kommandot hämtar belastningsutjämaren Med namnet MyLB och skickar resultatet till cmdleten **Add-AzLoadBalancerFrontendIpConfig** som lägger till en frontend IP-konfiguration till belastningsutjämaren med en dynamisk privat IP-adress från undernätet som lagras i variabeln $MySubnet.

### Exempel 2 Lägga till en IP-konfiguration på framsidan med en statisk IP-adress
```
PS C:\> $Subnet = Get-AzVirtualNetwork -Name "MyVnet" -ResourceGroupName "RG001" | Get-AzVirtualNetworkSubnetConfig -Name "MySubnet"
PS C:\> Get-AzLoadBalancer -Name "MyLB" -ResourceGroupName "NrpTest" | Add-AzLoadBalancerFrontendIpConfig -Name "FrontendName" -Subnet $Subnet -PrivateIpAddress "10.0.1.6" | Set-AzLoadBalancer
```

Det första kommandot hämtar det virtuella Azure-nätverket med namnet MyVnet och skickar resultatet med hjälp av pipelinen till cmdleten **Get-AzVirtualNetworkSubnetConfig** för att hämta undernätet MittSubnet.
Kommandot lagrar sedan resultatet i variabeln med namnet $Subnet.
Det andra kommandot hämtar belastningsutjämaren Med namnet MyLB och skickar resultatet till cmdleten **Add-AzLoadBalancerFrontendIpConfig** som lägger till en frontend IP-konfiguration till belastningsutjämaren med en statisk privat IP-adress från undernätet som lagras i variabeln $Subnet.

### Exempel 3 Lägga till en frontend-IP-konfiguration med en offentlig IP-adress
```
PS C:\> $PublicIp = Get-AzPublicIpAddress -ResourceGroupName "myRG" -Name "MyPub"
PS C:\> Get-AzLoadBalancer -Name "MyLB" -ResourceGroupName "NrpTest" | Add-AzLoadBalancerFrontendIpConfig -Name "FrontendName" -PublicIpAddress $PublicIp | Set-AzLoadBalancer
```

Det första kommandot får den offentliga Azure-IP-adressen MyPub och lagrar resultatet i variabeln $PublicIp.
Det andra kommandot hämtar belastningsutjämaren MyLB och skickar resultatet till cmdleten **Add-AzLoadBalancerFrontendIpConfig** som lägger till en IP-konfiguration på frontend-sidan med offentlig IP-adress som lagras i variabeln $PublicIp.

### Exempel 4 Lägga till en IP-konfiguration på en frontend med ett offentligt IP-prefix
```
PS C:\> $PublicIpPrefix = Get-AzPublicIpPrefix -ResourceGroupName "myRG" -Name "MyPubPrefix"
PS C:\> Get-AzLoadBalancer -Name "MyLB" -ResourceGroupName "NrpTest" | Add-AzLoadBalancerFrontendIpConfig -Name "FrontendName" -PublicIpAddressPrefix $PublicIpPrefix | Set-AzLoadBalancer
```

Det första kommandot får det offentliga Azure-IP-prefixet MyPubPrefix och lagrar resultatet i variabeln med namnet $PublicIpPrefix.
Det andra kommandot hämtar belastningsutjämaren Med namnet MyLB och skickar resultatet till cmdleten **Add-AzLoadBalancerFrontendIpConfig** som lägger till en frontend IP-konfiguration till belastningsutjämaren med offentligt IP-prefix som lagras i variabeln $PublicIpPrefix.
