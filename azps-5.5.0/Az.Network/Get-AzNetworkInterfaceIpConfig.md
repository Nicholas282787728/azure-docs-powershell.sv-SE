# Get-AzNetworkInterfaceIpConfig

## SYNOPSIS
Hämtar en IP-konfiguration för nätverksgränssnittet.

## SYNTAX

```
Get-AzNetworkInterfaceIpConfig [-Name <String>] -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzNetworkInterfaceIPConfig** får en IP-konfiguration för nätverksgränssnitt från ett Azure-nätverksgränssnitt.

## EXEMPEL

### 1: Skaffa en IP-konfiguration av ett nätverksgränssnitt
```
$nic1 = Get-AzNetworkInterface -Name mynic -ResourceGroupName $myrg
Get-AzNetworkInterfaceIpConfig -Name ipconfig1 -NetworkInterface $nic1
```

Det första kommandot får ett befintligt nätverksgränssnitt som kallas mynic och lagrar det i variabeln $nic 1. Det andra kommandot får IP-konfigurationen ipconfig1 för det här nätverksgränssnittet.
