# New-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Skapar en frontend IP-konfiguration för en belastningsutjämning.

## SYNTAX

### SetByResourceSubnet (standard)
```
New-AzLoadBalancerFrontendIpConfig -Name <String> [-PrivateIpAddress <String>]
 [-PrivateIpAddressVersion <String>] [-Zone <String[]>] -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

```

```


## BESKRIVNING
**Cmdleten New-AzLoadBalancerFrontendIpConfig** skapar en frontend IP-konfiguration för en Azure-belastningsutjämare.

## EXEMPEL

### Exempel 1: Skapa en frontend-IP-konfiguration för en belastningsutjämare
```
PS C:\> $publicip = New-AzPublicIpAddress -ResourceGroupName "MyResourceGroup" -Name "MyPublicIP" -Location "West US" -AllocationMethod "Dynamic"
PS C:\> New-AzLoadBalancerFrontendIpConfig -Name "FrontendIpConfig01" -PublicIpAddress $publicip
```

Det första kommandot skapar en dynamisk offentlig IP-adress med namnet MyPublicIP i resursgruppen MyResourceGroup och lagrar den sedan $publicip variabeln.
Det andra kommandot skapar en frontend IP-konfiguration med namnet FrontendIpConfig01 med den offentliga IP-adressen i $publicip.

### Exempel 2: Skapa en FRONTEND IP-konfiguration för en belastningsutjämare med ip-prefix
```
PS C:\> $publicipprefix = New-AzPublicIpPrefix -ResourceGroupName "MyResourceGroup" -name "MyPublicIPPrefix" -location "West US" -Sku Standard -PrefixLength 28
PS C:\> $frontend = New-AzLoadBalancerFrontendIpConfig -Name "FrontendIpConfig01" -PublicIpAddressPrefix $publicipprefix
```

Det första kommandot skapar ett offentligt IP-prefix med namnet MyPublicIP med längd 28 i resursgruppen MyResourceGroup och lagrar det sedan i $publicipprefix variabeln.
Det andra kommandot skapar en frontend IP-konfiguration med namnet FrontendIpConfig01 med det offentliga IP-prefixet i $publicipprefix.
