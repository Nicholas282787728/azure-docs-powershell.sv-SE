# Get-AzApplicationGatewaySslCertificate

## SYNOPSIS
Hämtar ett SSL-certifikat för en programgateway.

## SYNTAX

```
Get-AzApplicationGatewaySslCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApplicationGatewaySslCertificate** får ett SSL-certifikat för en programgateway.

## EXEMPEL

### Exempel 1: Skaffa ett specifikt SSL-certifikat
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Cert = Get-AzApplicationGatewaySslCertificate -Name "Cert01" -ApplicationGateway $AppGW
```

Det första kommandot får Application Gateway med namnet ApplicationGateway01 och lagrar resultatet i variabeln $AppGW.
Det andra kommandot hämtar SSL-certifikatet cert01 från programgatewayen som lagras i variabeln $AppGW.
Kommandot lagrar certifikatet i variabeln med namnet $Cert.

### Exempel 2: Hämta en lista med SSL-certifikat
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Certs = Get-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW
```

Det första kommandot får Application Gateway med namnet ApplicationGateway01 och lagrar resultatet i variabeln $AppGW.
Det här andra kommandot hämtar en lista med SSL-certifikat från programgatewayen som lagras i variabeln $AppGW.
Med kommandot lagras sedan resultaten i variabeln $Certs.
