# Set-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Anger en konfiguration av en inkommande NAT-regel för en belastningsutjämare.

## SYNTAX

### SetByResource (standard)
```
Set-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfiguration <PSFrontendIPConfiguration>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByResourceId
```
Set-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfigurationId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzLoadBalancerInboundNatRuleConfig** anger en konfiguration av regler för inkommande nätverksadressöversättning (NAT) för en Azure-belastningsutjämnare.

## EXEMPEL

### Exempel 1: Ändra konfigurationen av den inkommande NAT-regeln för en belastningsutjämare
```powershell
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Add-AzLoadBalancerInboundNatRuleConfig -Name "NewNatRule" -FrontendIPConfiguration $slb.FrontendIpConfigurations[0] -Protocol "Tcp" -FrontendPort 3350 -BackendPort 3350 -EnableFloatingIP
PS C:\> $slb | Set-AzLoadBalancerInboundNatRuleConfig -Name "NewNatRule" -FrontendIPConfiguration $slb.FrontendIpConfigurations[0] -Protocol "Tcp" -FrontendPort 3350 -BackendPort 3350
```

Det första kommandot hämtar belastningsutjämaren med namnet MyLoadBalancer och lagrar den sedan $slb variabeln.
Det andra kommandot använder pipelineoperatorn för att överföra belastningsutjämningen i $slb till Add-AzLoadBalancerInboundNatRuleConfig, som lägger till en konfiguration av en inkommande NAT-regel till den.
Det tredje kommandot skickar belastningsutjämaren till **Set-AzLoadBalancerInboundNatRuleConfig,** som sparar och uppdaterar konfigurationen av den inkommande NAT-regeln.
Observera att regelkonfigurationen konfigurerades utan att aktivera flytande IP, som hade aktiverats av det föregående kommandot.
