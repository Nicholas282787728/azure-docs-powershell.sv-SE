# Remove-AzApplicationGatewayProbeConfig

## SYNOPSIS
Tar bort en hälsoprondens från en befintlig programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayProbeConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzApplicationGatewayProbeConfig-cmdleten tas en hedpronden bort från en befintlig programgateway.

## EXEMPEL

### Exempel 1: Ta bort en hälsoprondens från en befintlig programgateway
```
PS C:\>$Gateway = Remove-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe04"
```

Det här kommandot tar bort hälsoprondenten med namnet Sond04 från programgatewayen med namnet Gateway.
