# Remove-AzRouteFilterRuleConfig

## SYNOPSIS
Tar bort en routefilterregel från ett routefilter.

## SYNTAX

```
Remove-AzRouteFilterRuleConfig -Name <String> -RouteFilter <PSRouteFilter> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRouteFilterRuleConfig** tar bort en routefilterregel från ett routefilter.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
```

Det första kommandot får ett routefilter med namnet RouteFilter01 som tillhör resursgruppen ResourceGroup01 och lagrar det i $rf variabeln.
Med det andra kommandot tas routefilterregeln Rule01 bort från det routefilter som lagras i $rf.
