# Remove-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
Tar bort MAPPningar av URL-sökvägar till en serverpool för serverbackend.

## SYNTAX

```
Remove-AzApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGatewayUrlPathMapConfig** tar bort URL-sökvägsmappningar till en serverpool för server igen.

## EXEMPEL

### Exempel 1: Ta bort en mappning av URL-sökvägen från en programgateway
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway $appgw -Name "map01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Det första kommandot hämtar programmets gateway med namnet appGwName och lagrar resultatet i den $appgw variabeln.
Det andra kommandot tar bort URL-sökvägsmappningen med namnet map01 från programmets gateway.
Det tredje kommandot uppdaterar programmets gateway.
