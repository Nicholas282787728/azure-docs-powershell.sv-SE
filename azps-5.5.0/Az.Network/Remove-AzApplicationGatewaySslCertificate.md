# Remove-AzApplicationGatewaySslCertificate

## SYNOPSIS
Tar bort ett SSL-certifikat från en Azure-programgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGatewaySslCertificate** tar bort ett SSL (Secure Sockets Layer) (SSL)-certifikat från en Azure-programgateway.

## EXEMPEL

### Exempel 1: Ta bort ett SSL-certifikat från en programgateway
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Det första kommandot hämtar programgateway med namnet ApplicationGateway01 och lagrar resultatet i variabeln $AppGW.
Det andra kommandot tar bort SSL-certifikatet som heter Cert02 från programgatewayen som lagras i den $AppGW variabeln.
