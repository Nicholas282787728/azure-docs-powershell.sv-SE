# New-AzApplicationGatewayFirewallPolicyManagedRule

## SYNOPSIS
Create ManagedRules för brandväggsprincipen.

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicyManagedRule
 [-ManagedRuleSet <PSApplicationGatewayFirewallPolicyManagedRuleSet[]>]
 [-Exclusion <PSApplicationGatewayFirewallPolicyExclusion[]>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**New-AzApplicationGatewayFirewallPolicyManagedRule** skapar en hanterad regler för en brandväggsprincip.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallPolicyManagedRule -ManagedRuleSet $managedRuleSet -Exclusion $exclusion1,$exclusion2
```

Med kommandot skapas en lista över ManagedRuleSet med $managedRuleSet och en undantagslista med poster som $exclusion 1, $exclusion 2.
