# Get-AzEffectiveNetworkSecurityGroup

## SYNOPSIS
Får den effektiva nätverkssäkerhetsgruppen i ett nätverksgränssnitt.

## SYNTAX

```
Get-AzEffectiveNetworkSecurityGroup -NetworkInterfaceName <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzEffectiveNetworkSecurityGroup** returnerar den effektiva nätverkssäkerhetsgruppen som används i ett nätverksgränssnitt.

## EXEMPEL

### Exempel 1: Skaffa den effektiva nätverkssäkerhetsgruppen i ett nätverksgränssnitt
```
PS C:\>Get-AzEffectiveNetworkSecurityGroup -NetworkInterfaceName "MyNetworkInterface" -ResourceGroupName "myResourceGroup"
```

Med det här kommandot får du alla effektiva nätverkssäkerhetsregler som är kopplade till nätverksgränssnittet MyNetworkInterface i resursgruppen myResourceGroup.
