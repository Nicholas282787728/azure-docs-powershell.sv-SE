# Remove-AzApplicationGatewaySslProfilePolicy

## SYNOPSIS
Tar bort en SSL-princip från en SSL-profil för Azure-programgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslProfilePolicy -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzApplicationGatewaySslProfilePolicy-cmdleten tas en SSL-princip bort från en SSL-profil för Azure-programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "Profile01" -ApplicationGateway $AppGw
PS C:\> $profile = Remove-AzApplicationGatewaySslProfilePolicy -SslProfile $profile
```

Det första kommandot hämtar programgatewayen med namnet ApplicationGateway01 i resursgruppen ResourceGroup01 och lagrar den i $AppGw variabeln. Det andra kommandot hämtar SSL-profilen med namnet Profile01 för $AppGw lagrar det i den $profile variabeln. Det senaste kommandot tar bort SSL-principen för ssl-profilen som lagras i $profile.
