# Get-AzApplicationGatewaySslPolicy

## SYNOPSIS
Hämtar SSL-principen för en programgateway.

## SYNTAX

```
Get-AzApplicationGatewaySslPolicy -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApplicationGatewaySslPolicy hämtar SSL-principen** för en programgateway.

## EXEMPEL

### 1:
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $sslpolicy = Get-AzApplicationGatewaySslPolicy -ApplicationGateway $AppGW
```

Det första kommandot får Application Gateway med namnet ApplicationGateway01 och lagrar resultatet i variabeln $AppGW.
Det andra kommandot hämtar SSL-principen från Application Gateway som lagras i variabeln $AppGW.
