# Set-AzServiceEndpointPolicy

## SYNOPSIS
Uppdaterar en tjänstslutpunktsprincip.

## SYNTAX

```
Set-AzServiceEndpointPolicy -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzServiceEndpointPolicy** skapar en tjänsteslutpunktsprincip.

## EXEMPEL

### Exempel 1: Anger en tjänstslutpunktsprincip
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicy -Name "Policy1" -ServiceEndpointPolicy $serviceEndpointPolicy -ResourceGroup "resourcegroup1"
```

Det här kommandot uppdaterar en tjänstslutpunktsprincip med namnet Princip1 som definieras av $serviceEndpointPolicy tillhör resursgruppen "resourcegroup1".
