# Get-AzVirtualNetworkGatewayBGPPeerStatus

## SYNOPSIS
Visar BGP-peers för azure-nätverksgateway

## SYNTAX

```
Get-AzVirtualNetworkGatewayBGPPeerStatus -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 [-Peer <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Med det här kommandot räknas BGP-peers upp som en Azure-virtuell nätverksgateway som är konfigurerad att peer med. Status för varje peer ges också.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzVirtualNetworkGatewayBgpPeerStatus -ResourceGroupName resourceGroup -VirtualNetworkGatewayName gatewayName

Asn               : 65515
ConnectedDuration : 9.01:04:53.5768637
LocalAddress      : 10.1.0.254
MessagesReceived  : 14893
MessagesSent      : 14900
Neighbor          : 10.0.0.254
RoutesReceived    : 1
State             : Connected
```

Hämtar BGP-peers för Azure Virtual Network Gateway med namnet gatewayName i resource group resourceGroup.
I det här exemplet visas en ansluten BGP-peer med IP-adressen 10.0.0.254.
