# Set-AzVirtualNetworkTap

## SYNOPSIS
Uppdaterar ett virtuellt nätverkstryck.

## SYNTAX

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzVirtualNetworkTap** uppdaterar ett virtuellt nätverkstryck.

## EXEMPEL

### Exempel 1: Konfigurera ett virtuellt nätverkstryck
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

Kommandot uppdaterar Destination IpConfiguration och uppdaterar det virtuella nätverkstryckningen.
Om det finns tryckkonfigurationer som refererar till den kommer inte all källtrafik att börja speglas i den nya uppdateringen av mål-IP-konfigurationen.
