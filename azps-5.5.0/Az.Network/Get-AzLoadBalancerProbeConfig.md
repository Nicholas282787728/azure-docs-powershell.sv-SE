# Get-AzLoadBalancerProbeConfig

## SYNOPSIS
Hämtar en sonde konfiguration för en belastningsutjämare.

## SYNTAX

```
Get-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzLoadBalancerProbeConfig** får en eller flera sonde konfigurationer för en belastningsutjämning.

## EXEMPEL

### Exempel 1: Hämta sondens konfiguration av en belastningsutjämare
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $slb
```

Det första kommandot får belastningsutjämaren MyLoadBalancer och lagrar den sedan i variabeln $slb.
Det andra kommandot hämtar den associerade sondens konfiguration med namnet MyProbe från belastningsutjämaren i $slb.
