# Add-AzVirtualHubRouteTable

## SYNOPSIS
Skapar en tabellresurs med virtuell hubb-route som är underordnad VirtualHub.

## SYNTAX

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Tabellresursen för den virtuella hubb- routen innehåller listan över routes och en lista med anslutningar som den kan kopplas till och används till att dirigera trafik i ett virtuellt nav.
