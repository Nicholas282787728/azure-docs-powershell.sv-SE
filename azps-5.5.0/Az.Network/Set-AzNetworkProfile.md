# Set-AzNetworkProfile

## SYNOPSIS
Uppdaterar en nätverksprofil.

## SYNTAX

```
Set-AzNetworkProfile -NetworkProfile <PSNetworkProfile> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzPublicIpPrefix** uppdaterar en nätverksprofil.

## EXEMPEL

### Exempel 1
```powershell
$networkProfile = Get-AzNetworkProfile -Name np1 -ResourceGroupName rg1

$networkProfile.Tags = "TestTag"

$networkProfile.ContainerNetworkInterfaceConfigurations = New-AzNetworkProfileContainerNicConfig -Name cnicconfig1

$networkProfile | Set-AzNetworkProfile
```

Det första kommandot får en befintlig nätverksprofil. Det andra kommandot uppdaterar en tagg och det tredje lägger till en konfiguration av nätverksgränssnittet i nätverksprofilen. Det fjärde kommandot uppdaterar nätverksprofilen.
