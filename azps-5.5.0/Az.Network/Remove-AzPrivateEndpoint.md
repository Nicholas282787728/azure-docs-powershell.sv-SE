# Remove-AzPrivateEndpoint

## SYNOPSIS
Tar bort en privat slutpunkt.

## SYNTAX

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzPrivateEndpoint** tar bort en privat slutpunkt. 

## EXEMPEL

### Exempel 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

I det här exemplet tas en privat slutpunkt med namnet MyPrivateEndpoint1 bort.
