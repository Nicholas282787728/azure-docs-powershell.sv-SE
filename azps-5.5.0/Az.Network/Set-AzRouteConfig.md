# Set-AzRouteConfig

## SYNOPSIS
Uppdaterar en routekonfiguration för en routetabell.

## SYNTAX

```
Set-AzRouteConfig -RouteTable <PSRouteTable> [-Name <String>] [-AddressPrefix <String>] [-NextHopType <String>]
 [-NextHopIpAddress <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzRouteConfig** uppdaterar en routekonfiguration för en routetabell.

### -NextHopType
Anger hur denna route vidarebefordrar paket.
De godtagbara värdena för den här parametern är:
- Internet.
Standardgatewayen för Internet som tillhandahålls av Azure. 
- Inget.
Om du anger det här värdet vidarebefordrar inte vägen paket. 
- VirtualAppliance.
En virtuell enhet som du lägger till i ditt virtuella Azure-nätverk. 
- VirtualNetworkGateway.
En virtuell privat AzureServer-to-server-nätverksgateway. 
- VnetLocal.
Det lokala virtuella nätverket.
Om du har två undernät, 10.1.0.0/16 och 10.2.0.0/16 i samma virtuella nätverk, markerar du värdet VnetLocal för varje undernät som ska vidarebefordras till det andra undernätet.
