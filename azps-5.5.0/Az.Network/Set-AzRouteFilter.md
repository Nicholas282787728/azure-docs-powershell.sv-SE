# Set-AzRouteFilter

## SYNOPSIS
Uppdaterar ett routefilter.

## SYNTAX

```
Set-AzRouteFilter -RouteFilter <PSRouteFilter> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzApplicationGateway** uppdaterar ett routefilter

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzRouteFilter -RouteFilter $rf
```

Det här kommandot uppdaterar routefiltret med inställningar i $rf variabeln.
