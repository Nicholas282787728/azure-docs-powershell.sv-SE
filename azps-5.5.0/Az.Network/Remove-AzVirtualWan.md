# Remove-AzVirtualWan

## SYNOPSIS
Tar bort ett virtuellt Azure WAN.

## SYNTAX

### ByVirtualWanName (standard)
```
Remove-AzVirtualWan -ResourceGroupName <String> -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanObject
```
Remove-AzVirtualWan -InputObject <PSVirtualWan> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanResourceId
```
Remove-AzVirtualWan -ResourceId <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Tar bort ett virtuellt Azure WAN.

## EXEMPEL

### Exempel 1

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Passthru
```

I det här exemplet skapas ett virtuellt WAN i en resursgrupp och det tas sedan bort direkt. För att dölja uppmaningen när du tar bort det virtuella WAN:t använder du flaggan -Force.

### Exempel 2

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -InputObject $virtualWan -Passthru
```

I det här exemplet skapas ett virtuellt WAN i en resursgrupp och det tas sedan bort direkt. Det här borttagningen sker med hjälp av det virtuella wan-objektet som returneras av New-AzVirtualWan.
För att dölja uppmaningen när du tar bort det virtuella WAN:t använder du flaggan -Force.

### Exempel 3

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -ResourceId $virtualWan.Id -Passthru
```

I det här exemplet skapas ett virtuellt WAN i en resursgrupp och det tas sedan bort omedelbart. Det här borttagningen sker med hjälp av det virtuella wan-resurs-ID som returneras av New-AzVirtualWan.
För att dölja uppmaningen när du tar bort det virtuella WAN:t använder du flaggan -Force.
