# Remove-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Tar bort en adresspool för backend från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayBackendAddressPool -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGatewayBackendAddressPool** tar bort en adresspool på baksidan från en Azure-programgateway.

## EXEMPEL

### Exempel 1: Ta bort en adresspool i backend från en programgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "BackEndPool02"
```

Det första kommandot hämtar programgatewayen med namnet ApplicationGateway01 som tillhör resursgruppen ResourceGroup01 och sparar den i $AppGw variabeln.
Det andra kommandot tar bort adresspoolen för backend-adressen med namnet BackEndPool02 från programmets gateway.
