# Set-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Uppdaterar en frontend IP-konfiguration för en belastningsutjämare.

## SYNTAX

### SetByResourceSubnet (standard)
```
Set-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-PrivateIpAddress <String>]
 [-PrivateIpAddressVersion <String>] [-Zone <String[]>] -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzLoadBalancerFrontendIpConfig** uppdaterar en IP-konfiguration på frontend-sidan för en belastningsutjämning.

## EXEMPEL

### Exempel 1: Ändra frontend-IP-konfigurationen av en belastningsutjämare
```powershell
PS C:\> $Subnet = Get-AzVirtualNetwork -Name "MyVnet" -ResourceGroupName "MyResourceGroup" | Get-AzVirtualNetworkSubnetConfig -Name "Subnet"
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Add-AzLoadBalancerFrontendIpConfig -Name "NewFrontend" -Subnet $Subnet
PS C:\> $slb | Set-AzLoadBalancerFrontendIpConfig -Name "NewFrontend" -Subnet $Subnet
PS C:\> $slb | Set-AzLoadBalancer
```

Det första kommandot hämtar det virtuella undernätet med namnet Undernät och lagrar det sedan i den $Subnet variabeln.
Det andra kommandot hämtar den associerade belastningsutjämaren med namnet MyLoadBalancer och lagrar den sedan $slb variabeln.
Det tredje kommandot använder rörledningsoperatorn för att överföra belastningsutjämningen i $slb till Add-AzLoadBalancerFrontendIpConfig, som skapar en frontend IP-konfiguration med namnet NewFrontend för $slb.
Det fjärde kommandot skickar belastningsutjämaren i $slb till **Set-AzLoadBalancerFrontendIpConfig,** som sparar och uppdaterar IP-konfigurationen på framsidan.
