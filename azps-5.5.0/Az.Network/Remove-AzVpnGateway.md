# Remove-AzVpnGateway

## SYNOPSIS
Med Remove-AzVpnGateway-cmdleten tas en Azure VPN-gateway bort. Det här är en gateway som är specifik för Azure Virtual WAN:s programvarudefinierade anslutning.

## SYNTAX

### ByVpnGatewayName (standard)
```
Remove-AzVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayObject
```
Remove-AzVpnGateway -InputObject <PSVpnGateway> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayResourceId
```
Remove-AzVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzVpnGateway-cmdleten tas en Azure VPN-gateway bort. Det här är en gateway som är specifik för Azure Virtual WAN:s programvarudefinierade anslutning.

## EXEMPEL

### Exempel 1

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Remove-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -Passthru
```

I det här exemplet skapas en resursgrupp, Virtuell WAN, virtuellt nav, skalbar VPN-gateway i centrala USA och sedan tas den bort omedelbart. För att dölja uppmaningen när du tar bort den virtuella gatewayen använder du flaggan -Force.
Då tas VpnGateway och alla VpnConnections som är kopplade till den bort.

### Exempel 2

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Get-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" | Remove-AzVpnGateway-Passthru
```

I det här exemplet skapas en resursgrupp, Virtuell WAN, virtuellt nav, skalbar VPN-gateway i centrala USA och sedan tas den bort omedelbart. Den här borttagningen sker med powershell-rör, som använder VpnGateway-objektet som returneras av Get-AzVpnGateway kommando.
För att dölja uppmaningen när du tar bort den virtuella gatewayen använder du flaggan -Force.
Då tas VpnGateway och alla VpnConnections som är kopplade till den bort.
