# Get-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Hämtar regeluppsättningen för omskrivning av en programgateway.

## SYNTAX

```
Get-AzApplicationGatewayRewriteRuleSet [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämtar regeluppsättningen för omskrivning av en programgateway.

## EXEMPEL

### Exempel 1: Skapa en särskild regeluppsättning för omskrivning
```
PS C:\> $AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rule = Get-AzApplicationGatewayRewriteRuleSet -Name "RuleSet01" -ApplicationGateway $AppGW
```

Det första kommandot får Application Gateway med namnet ApplicationGateway01 och lagrar resultatet i variabeln $AppGW.
Det andra kommandot hämtar den omskrivningsregeluppsättning som heter RuleSet01 från programgatewayen som lagras i variabeln $AppGW.
