# Remove-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Tar bort det betrodda klient-CA-certifikatkedjeobjektet från en programgateway.

## SYNTAX

```
Remove-AzApplicationGatewayTrustedClientCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApplicationGatewayTrustedClientCertificate** tar bort det betrodda klient-CA-certifikatkedjeobjektet från en programgateway.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name "TrustedClientCertificate01"
```

Det första kommandot hämtar en programgateway och lagrar den i $gw variabeln. Det andra kommandot tar bort certifikatkedja för betrodd klient-CA med namnet "TrustedClientCertificate01" från programgatewayen som är lagrad $gw.
