# Get-AzStorageSyncService

## SYNOPSIS
Det här kommandot listar alla lagringssynkroniseringstjänster inom en viss omfattning för prenumeration/resursgrupp.

## SYNTAX

### ParentStringParameterSet (standard)
```
Get-AzStorageSyncService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### StringParameterSet
```
Get-AzStorageSyncService [-ResourceGroupName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Det här kommandot listar alla lagringssynkroniseringstjänster inom en viss omfattning för prenumeration/resursgrupp. Den kan användas för att även lista attributen för varje lagringssynkroniseringstjänst.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzStorageSyncService -ResourceGroupName "myResourceGroup"
```

Det här kommandot listar alla resurser för lagringssynkroniseringstjänsten i en viss resursgrupp. Den kan användas för att även lista attributen för varje lagringssynkroniseringstjänst. Specify -StorageSyncServiceName to return a specific one.
