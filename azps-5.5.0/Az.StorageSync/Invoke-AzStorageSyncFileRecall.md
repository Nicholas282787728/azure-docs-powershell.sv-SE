# Invoke-AzStorageSyncFileRecall

## SYNOPSIS
Det här kommandot återkallar alla nivåfiler tillbaka till den lokala hårddisken.

## SYNTAX

### StringParameterSet (standard)
```
Invoke-AzStorageSyncFileRecall [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name] <String> [-Pattern <String>] [-RecallPath <String>] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Invoke-AzStorageSyncFileRecall [-ResourceId] <String> [-Pattern <String>] [-RecallPath <String>] [-PassThru]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectParameterSet
```
Invoke-AzStorageSyncFileRecall [-InputObject] <PSServerEndpoint> [-Pattern <String>] [-RecallPath <String>]
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
När molnnivånivå är aktiverad på en serverslutpunkt (en viss plats på en registrerad server) kan det här kommandot användas för att återkalla alla nivåfiler till den lokala hårddisken. Vi rekommenderar starkt att du inaktiverar molnnivånivå på den här serverslutpunkten innan du startar återkallelsen. Om nivåindelad är fortfarande på kan återkallelsen leda till att andra filer blir nivåindelade och missar det önskade målet för allt innehåll som finns på den lokala hårddisken.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Invoke-AzStorageSyncFileRecall -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName" -ServerEndpointName "myServerEndpointName"
```

Med det här kommandot återkallas alla nivåfiler som finns under rotsökvägen för den angivna serverslutpunkten.
