# New-AzStorageSyncGroup

## SYNOPSIS
Med det här kommandot skapas en ny synkroniseringsgrupp inom en angiven lagringssynkroniseringstjänst.

## SYNTAX

### StringParameterSet (standard)
```
New-AzStorageSyncGroup [-ResourceGroupName] <String> [-StorageSyncServiceName] <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectParameterSet
```
New-AzStorageSyncGroup [-ParentObject] <PSStorageSyncService> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ParentStringParameterSet
```
New-AzStorageSyncGroup [-ParentResourceId] <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med det här kommandot skapas en ny synkroniseringsgrupp inom en angiven lagringssynkroniseringstjänst. En synkroniseringsgrupp används för att beskriva en topologi över platser, så kallade slutpunkter, som synkroniserar alla filer som lagras i någon av slutpunkterna. En synkroniseringsgrupp innehåller molnslutpunkter som refererar till Azure-filresurser, och den innehåller även serverslutpunkter som refererar till en specifik lokal sökväg på en registrerad server.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzStorageSyncGroup -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -Name "mySyncGroupName"
```

Med det här kommandot skapas en ny synkroniseringsgrupp inom en angiven lagringssynkroniseringstjänst.
