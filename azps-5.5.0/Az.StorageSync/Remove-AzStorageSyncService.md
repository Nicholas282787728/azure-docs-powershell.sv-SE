# Remove-AzStorageSyncService

## SYNOPSIS
Det här kommandot tar bort den angivna lagringssynkroniseringstjänsten.

## SYNTAX

### StringParameterSet (standard)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Det här kommandot tar bort den angivna lagringssynkroniseringstjänsten. En lagringssynkroniseringstjänst kan bara tas bort när alla de synkroniseringsgrupper och registrerade servrarna tas bort först.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Det här kommandot tar bort lagringssynkroniseringstjänsten.
