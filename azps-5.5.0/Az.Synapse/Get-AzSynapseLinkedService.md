# Get-AzSynapseLinkedService

## SYNOPSIS
Hämtar information om länkade tjänster på arbetsytan.

## SYNTAX

### GetByName (standard)
```
Get-AzSynapseLinkedService -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseLinkedService -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSynapseLinkedService** hämtar information om länkade tjänster i arbetsytan.
Om du anger namnet på en länkad tjänst hämtar den här cmdleten information om den länkade tjänsten.
Om du inte anger ett namn hämtar den här cmdleten information om alla länkade tjänster i arbetsytan.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace
```

Med det här kommandot får du information om alla länkade tjänster i arbetsytan ContosoWorkspace.

### Exempel 2
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
```

Det här kommandot hämtar information om den länkade tjänsten ContosoLinkedService i arbetsytan ContosoWorkspace.

### Exempel 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseLinkedService -Name ContosoLinkedService
```

Med det här kommandot får du information om den länkade tjänsten ContosoLinkedService i arbetsytan ContosoWorkspace via pipeline.
