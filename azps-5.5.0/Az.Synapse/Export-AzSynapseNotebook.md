# Export-AzSynapseNotebook

## SYNOPSIS
Exporterar notbooks.

## SYNTAX

### ExportByName (standard)
```
Export-AzSynapseNotebook -WorkspaceName <String> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByObject
```
Export-AzSynapseNotebook -WorkspaceObject <PSSynapseWorkspace> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByInputObject
```
Export-AzSynapseNotebook -InputObject <PSNotebookResource> -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Export-AzSynapseNotebook** exporterar en Azure Synapse-anteckningsbok till en anteckningsbokfil (.ipynb). Namnet på anteckningsboken blir namnet på den exporterade filen. Om du anger namnet på en anteckningsbok exporterar cmdleten anteckningsboken. Om du inte anger ett namn exporterar cmdleten alla anteckningsböcker i arbetsytan.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -OutputFolder "C:\Notebook"
```

Exporterar alla anteckningsböcker i arbetsytan ContosoWorkspace till mappen "C:\Notebook".

### Exempel 2
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Exporterar en anteckningsbok med namnet ContosoNotebook i arbetsytan ContosoWorkspace till mappen "C:\Notebook".

### Exempel 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Export-AzSynapseNotebook -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Exporterar en anteckningsbok som heter ContosoNotebook i arbetsytan ContosoWorkspace till mappen "C:\Notebook" via pipeline.

### Exempel 4
```powershell
PS C:\> $notebook = Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook
PS C:\> $notebook | Export-AzSynapseNotebook -OutputFolder "C:\Notebook"
```

Exporterar en anteckningsbok som heter ContosoNotebook i arbetsytan ContosoWorkspace till mappen "C:\Notebook" via pipeline.
