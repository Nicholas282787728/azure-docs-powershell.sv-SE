# Test-AzSynapseWorkspace

## SYNOPSIS
Söker efter en arbetsyta för Synapse-analys.

## SYNTAX

```
Test-AzSynapseWorkspace [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Test-AzSynapseWorkspace** kontrollerar att det finns en Synapse-analysarbetsyta.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Test-AzSynapseWorkspace -Name ContosoWorkspace
```

Det här kommandot kontrollerar att det finns en arbetsyta för Synapse-analys.
