# Set-AzSynapseDataFlow

## SYNOPSIS
Skapar eller uppdaterar ett dataflöde i arbetsytan.

## SYNTAX

### SetByName (standard)
```
Set-AzSynapseDataFlow -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataFlow -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzSynapseDataFlow** skapar ett dataflöde eller uppdaterar ett befintligt dataflöde i arbetsytan.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzSynapseDataFlow -WorkspaceName ContosoWorkspace -Name ContosoDataFlow -DefinitionFile "C:\\samples\\DataFlow.json"
```

Med det här kommandot skapas ett dataflöde med namnet ContosoDataFlow i arbetsytan ContosoWorkspace.
Kommandot baserar dataflödet på information i DataFlow.jsfilen.
