# Set-AzSynapseDataset

## SYNOPSIS
Skapar eller uppdaterar en datauppsättning i arbetsytan.

## SYNTAX

### SetByName (standard)
```
Set-AzSynapseDataset -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzSynapseDataset** skapar en datauppsättning eller uppdaterar en befintlig datauppsättning i arbetsytan.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset -DefinitionFile "C:\\samples\\Dataset.json"
```

Med det här kommandot skapas en datauppsättning med namnet ContosoDataset i arbetsytan ContosoWorkspace.
Kommandot baserar datauppsättningen på information i Dataset.jsbaserat på fil.
