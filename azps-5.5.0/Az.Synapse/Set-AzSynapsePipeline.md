# Set-AzSynapsePipeline

## SYNOPSIS
Skapar en pipeline i arbetsytan.

## SYNTAX

### SetByName (standard)
```
Set-AzSynapsePipeline -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapsePipeline -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzSynapsePipeline** skapar en rörledning i arbetsytan.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzSynapsePipeline -WorkspaceName ContosoWorkspace -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Det här kommandot skapar en pipeline med namnet ContosoPipeline i arbetsytan ContosoWorkspace.
Kommandot baserar pipelinen på information i pipeline.jsfilen.
Den här filen innehåller information om aktiviteter.

### Exempel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Set-AzSynapsePipeline -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Det här kommandot skapar en pipeline med namnet ContosoPipeline i arbetsytan ContosoWorkspace via pipeline.
Kommandot baserar pipelinen på information i pipeline.jsfilen.
Den här filen innehåller information om aktiviteter.
