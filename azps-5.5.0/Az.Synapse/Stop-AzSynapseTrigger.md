# Stop-AzSynapseTrigger

## SYNOPSIS
Stoppar en utlösare i en arbetsyta.

## SYNTAX

### StopByName (standard)
```
Stop-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByObject
```
Stop-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Stop-AzSynapseT azureer** stoppar en utlösare i en arbetsyta. Om utlösaren är i läget "Startad" stoppar cmdleten utlösaren och anropar inte längre pipelines. Om utlösaren redan har statusen Stoppad har den här cmdleten ingen effekt.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Stop-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Stoppar en utlösare som kallas ContosoT groove på arbetsytan ContosoWorkspace.

### Exempel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Stop-AzSynapseTrigger -Name ContosoTrigger
```

Stoppar en utlösare som kallas ContosoT groove på arbetsytan ContosoWorkspace via pipeline.

### Exempel 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Stop-AzSynapseTrigger
```

Stoppa en utlösare som kallas ContosoT groove på arbetsytan ContosoWorkspace via pipeline.
