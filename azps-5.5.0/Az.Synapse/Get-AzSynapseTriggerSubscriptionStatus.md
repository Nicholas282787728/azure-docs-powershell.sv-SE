# Get-AzSynapseTriggerSubscriptionStatus

## SYNOPSIS
Hämta status för prenumerationen för händelseutlösaren till de angivna externa tjänsthändelserna.

## SYNTAX

### GetByName (standard)
```
Get-AzSynapseTriggerSubscriptionStatus -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseTriggerSubscriptionStatus -WorkspaceObject <PSSynapseWorkspace> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByInputObject
```
Get-AzSynapseTriggerSubscriptionStatus -InputObject <PSTriggerResource>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSynapseTskripterSubscriptionStatus** får status för prenumerationen för händelseutlösaren till de angivna externa tjänsthändelserna. Utlösaren kan inte startas förrän den returnerade statusen är "Aktiverad".

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzSynapseTriggerSubscriptionStatus -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Det här kommandot får statusen för underscribtion för utlösaren ContosoTkommandot för de externa tjänsthändelserna.

### Exempel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseTriggerSubscriptionStatus -Name ContosoTrigger
```

Det här kommandot hämtar statusen för underscribtion för utlösaren ContosoTkommandot för de externa tjänsthändelserna via pipeline.

### Exempel 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Get-AzSynapseTriggerSubscriptionStatus
```

Det här kommandot hämtar statusen för underscribtion för utlösaren ContosoTkommandot för de externa tjänsthändelserna via pipeline.
