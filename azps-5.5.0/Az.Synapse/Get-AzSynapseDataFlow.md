# Get-AzSynapseDataFlow

## SYNOPSIS
Hämtar information om dataflöden på arbetsytan.

## SYNTAX

### GetByName (standard)
```
Get-AzSynapseDataFlow -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseDataFlow -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSynapseDataFlow** hämtar information om dataflöden på arbetsytan.
Om du anger namnet på ett dataflöde hämtar den här cmdleten information om dataflödet.
Om du inte anger ett namn hämtar den här cmdleten information om alla dataflöden i arbetsytan.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace
```

Det här kommandot hämtar information om alla dataflöden i arbetsytan ContosoWorkspace.

### Exempel 2
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace -Name ContosoDataFlow
```

Det här kommandot hämtar information om dataflödet ContosoDataFlow i arbetsytan ContosoWorkspace.
