# Get-AzSynapseTrigger

## SYNOPSIS
Hämtar information om utlösare i en arbetsyta.

## SYNTAX

### GetByName (standard)
```
Get-AzSynapseTrigger -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSynapseT azureer** hämtar information om utlösare i en arbetsyta. Om du anger namnet på en utlösare hämtar cmdleten information om den utlösaren. Om du inte anger ett namn hämtar cmdleten information om alla utlösare i arbetsytan.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace
```

Hämtar en lista över alla utlösare som har skapats i arbetsytan ContosoWorkspace.

### Exempel 2
```powershell
PS C:\> Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Får en enda utlösare som kallas ContosoTspireer i arbetsytan ContosoWorkspace.

### Exempel 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseTrigger -Name ContosoTrigger
```

Får en enda utlösare som kallas ContosoTvåren i arbetsytan ContosoWorkspace via pipeline.
