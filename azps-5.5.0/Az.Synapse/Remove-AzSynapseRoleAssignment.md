# Remove-AzSynapseRoleAssignment

## SYNOPSIS

## SYNTAX

```
Remove-AzSynapseRoleAssignment -WorkspaceName <String> -RoleDefinitionName <String> -SignInName <String>
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzSynapseRoleAssignment** tar bort en Azure Synapse Analytics-rolltilldelning permanent.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Remove-AzSynapseRoleAssignment -WorkspaceName ContosoWorkspace -RoleAssignmentId ContosoRoleAssignmentId
```

Det här kommandot tar bort en Azure Synapse Analytics-rolltilldelning med ett rolltilldelnings-ID.

### Exempel 2
```powershell
PS C:\> Remove-AzSynapseRoleAssignment -WorkspaceName ContosoWorkspace -RoleAssignmentName ContosoRole -SignInName ContosoName
```

Det här kommandot tar bort en Azure Synapse Analytics-rolltilldelning med ett rollnamn och ett huvudnamn för användaren.

### Exempel 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Remove-AzSynapseRoleAssignment -RoleAssignmentId ContosoRoleAssignmentId
```

Det här kommandot tar bort en Azure Synapse Analytics-rolltilldelning med ett rolltilldelnings-ID genom pipeline.
