# Start-AzVmss

## SYNOPSIS
Startar VMSS eller en uppsättning virtuella maskiner i VMSS.

## SYNTAX

```
Start-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Start-AzVmss-cmdleten** startar alla virtuella datorer i skalor för virtuella maskiner (VMSS) eller en uppsättning virtuella datorer.
Du kan använda *InstanceId-parametern* för att välja en uppsättning virtuella datorer.

## EXEMPEL

### Exempel 1: Starta en särskild uppsättning virtuella maskiner i VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"-InstanceId "0", "1"
```

Det här kommandot startar en specifik uppsättning virtuella maskiner som anges av instans-ID-strängmatrisen som tillhör VMSS som heter ContosoVMSS.

### Exempel 2: Starta alla virtuella datorer i VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"
```

Det här kommandot startar alla virtuella datorer som tillhör VMSS som heter ContosoVMSS.
