# Remove-AzVmssVMDataDisk

## SYNOPSIS
Tar bort en data disk från en virtuell datorskalauppsättning för VM

## SYNTAX

```
Remove-AzVmssVMDataDisk [-VirtualMachineScaleSetVM] <PSVirtualMachineScaleSetVM> [-Lun] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzVmssVMDataDisk** tar bort en data disk från en VM-skala med VM-skal

## EXEMPEL

### Exempel 1: Ta bort en data disk från en VM-skaluppsättning
```powershell
PS C:\> $VmssVM = Get-AzVmssVM -ResourceGroupName "myrg" -VMScaleSetName "myvmss" -InstanceId 0 
PS C:\> Remove-AzVmssVMDataDisk -VM $VirtualMachine -Lun 0
PS C:\> Update-AzVmssVM -VirtualMachineScaleSetVM $VmssVM
```

Det första kommandot hämtar befintliga VMSS som ges av resursgruppnamnet, vmss-namnet och instans-ID.
Med det andra kommandot tas datadisken 0 bort från skaluppsättningen VM som lagras i $VmssVM Det slutliga kommandot uppdaterar vmss VM med borttagen datadisk.
