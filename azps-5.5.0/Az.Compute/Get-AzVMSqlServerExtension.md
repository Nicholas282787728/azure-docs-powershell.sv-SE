# Get-AzVMSqlServerExtension

## SYNOPSIS
Hämtar inställningarna för ett SQL Server-tillägg på en virtuell dator.

## SYNTAX

```
Get-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVMSqlServerExtension** hämtar inställningarna för SQL Server-infrastrukturen som en tjänstagent (IaaS) på en virtuell dator.

## EXEMPEL

### Exempel 1: Hämta inställningarna för ett SQL Server-tillägg på en virtuell dator
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Det här kommandot hämtar inställningarna för SQL Server-tillägget på en virtuell dator med namnet ContosoVM07.

### Exempel 2: Hämta inställningarna med hjälp av pipelinen
```
PS C:\> Get-AzVM -ServiceName "Service08" -Name "ContosoVM22" | Get-AzVMSqlServerExtension
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Det här kommandot hämtar den virtuella datorn med namnet ContosoVM22 på tjänsten Service08 med hjälp Get-AzVM-cmdleten.
Kommandot skickar resultatet till den aktuella cmdleten med hjälp av rörledningsoperatorn.
Det aktuella kommandot hämtar inställningarna för SQL Server IaaS-agenten på den virtuella datorn.

### Exempel 3: Hämta inställningarna för en viss SQL Server-version
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07" -Version "1.0"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Det här kommandot hämtar inställningarna i version 1.0 av SQL Server-tillägget på en virtuell dator med namnet ContosoVM07.
