# Set-AzVMBginfoExtension

## SYNOPSIS
Lägger till BGInfo-tillägget på en virtuell dator.

## SYNTAX

```
Set-AzVMBginfoExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name <String>]
 [-TypeHandlerVersion <String>] [-Location <String>] [-DisableAutoUpgradeMinorVersion] [-ForceRerun <String>]
 [-NoWait] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzVMBGInfoExtension** lägger till BGInfo-tillägget på en virtuell dator.

## EXEMPEL

### Exempel 1: Lägga till BGInfo-tillägget för en virtuell dator
```
PS C:\> Set-AzVMBgInfoExtension -ResourceGroupName "ContosoRG" -VMName "ContosoVM" -Name "ExtensionName" -TypeHandlerVersion "2.1" -Location "West Europe"
```

Det här kommandot lägger till BGInfo-tillägget på den virtuella datorn ContosoVM.
Kommandot anger resursgruppen och platsen för den virtuella datorn.
Kommandot anger namn och version för tillägget.
