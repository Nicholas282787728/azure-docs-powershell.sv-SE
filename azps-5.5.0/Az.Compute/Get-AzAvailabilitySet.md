# Get-AzAvailabilitySet

## SYNOPSIS
Hämtar Azure-tillgänglighetsuppsättningar i en resursgrupp.

## SYNTAX

```
Get-AzAvailabilitySet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAvailabilitySet** får Azure-tillgänglighetsuppsättningar i en resursgrupp.
Du kan ange namnet på en viss tillgänglighetsuppsättning som ska fås.

## EXEMPEL
