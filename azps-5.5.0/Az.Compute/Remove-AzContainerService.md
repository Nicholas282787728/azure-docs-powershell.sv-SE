# Remove-AzContainerService

## SYNOPSIS
Tar bort en behållartjänst.

## SYNTAX

```
Remove-AzContainerService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzContainerService** tar bort en behållartjänst från ditt Azure-konto.

## EXEMPEL

### Exempel 1: Ta bort en behållartjänst
```
PS C:\> Remove-AzContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17"
```

Det här kommandot tar bort behållartjänsten med namnet CSResourceGroup17.
