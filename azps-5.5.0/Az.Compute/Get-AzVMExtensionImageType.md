# Get-AzVMExtensionImageType

## SYNOPSIS
Hämtar typen av Azure-tillägg.

## SYNTAX

```
Get-AzVMExtensionImageType -Location <String> -PublisherName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVMExtensionImageType** får typen av Azure-tillägg.

## EXEMPEL

### Exempel 1: Hämta en typ av tilläggsbild
```
PS C:\> Get-AzVMExtensionImageType -Location "Central US" -PublisherName "Fabrikam"
```

Det här kommandot hämtar filtilläggsbildtypen för den angivna utgivaren och platsen.
