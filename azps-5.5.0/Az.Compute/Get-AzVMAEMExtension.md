# Get-AzVMAEMExtension

## SYNOPSIS
Hämtar information om AEM-tillägget.

## SYNTAX

```
Get-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status]
 [[-OSType] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVMAEMExtension** hämtar information om tillägget Azure Enhanced Monitoring (AEM).

## EXEMPEL

### Exempel 1: Hämta AEM-tillägget
```
PS C:\> Get-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Det här kommandot hämtar information för AEM-tillägget för den virtuella datorn contoso-server.
