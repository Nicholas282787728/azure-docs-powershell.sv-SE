# Start-AzVM

## SYNOPSIS
Startar en virtuell Azure-dator.

## SYNTAX

### ResourceGroupNameParameterSetName (standard)
```
Start-AzVM [-Name] <String> [-NoWait] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [-NoWait] [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Start-AzVM** startar en virtuell Azure-dator.

## EXEMPEL

### Exempel 1: Starta en virtuell dator
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Det här kommandot startar den virtuella datorn med namnet VirtualMa machinee07 i ResourceGroup11.
