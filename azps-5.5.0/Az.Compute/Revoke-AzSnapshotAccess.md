# Revoke-AzSnapshotAccess

## SYNOPSIS
Återkallar en åtkomst till en ögonblicksbild.

## SYNTAX

```
Revoke-AzSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Revoke-AzSnapshotAccess** återkallar en åtkomst till en ögonblicksbild.

## EXEMPEL

### Exempel 1
```
PS C:\> Revoke-AzSnapshotAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

Återkalla åtkomsten till ögonblicksbilden med namnet Ögonblicksbild01 i resursgruppen med namnet Resursgrupp01
