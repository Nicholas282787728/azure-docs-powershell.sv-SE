# Remove-AzSnapshot

## SYNOPSIS
Tar bort en ögonblicksbild.

## SYNTAX

```
Remove-AzSnapshot [-ResourceGroupName] <String> [-SnapshotName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzSnapshot** tar bort en ögonblicksbild.

## EXEMPEL

### Exempel 1
```
PS C:\> Remove-AzSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -Force;
```

Med det här kommandot tas ögonblicksbilden med namnet Ögonblicksbild01 bort i resursgruppen Resursgrupp01.
