# Get-AzVMImagePublisher

## SYNOPSIS
Hämtar VMImage-utgivare.

## SYNTAX

```
Get-AzVMImagePublisher -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVMImagePublisher** hämtar VMImage-utgivare.

## EXEMPEL

### Exempel 1: Hämta VMImage-utgivare för en region
```
PS C:\> Get-AzVMImagePublisher -Location "Central US"
```

Det här kommandot hämtar utgivare av VMImage-instanser för regionen Central USA i din Azure-profil.
