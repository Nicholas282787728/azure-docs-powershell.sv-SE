# Test-AzVMAEMExtension

## SYNOPSIS
Kontrollerar konfigurationen av AEM-tillägget.

## SYNTAX

```
Test-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-OSType] <String>]
 [[-WaitTimeInMinutes] <Int32>] [-SkipStorageCheck] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Test-AzVMAEMExtension** kontrollerar konfigurationen av tillägget Azure Enhanced Monitoring (AEM).
Tillägget AEM samlar in prestandadata.
Den här cmdleten kontrollerar om prestandadata är tillgängliga.

## EXEMPEL

### Exempel 1: Kontrollera konfigurationen av AEM-tillägget
```
PS C:\> Test-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Det här kommandot kontrollerar konfigurationen av AEM-tillägget för den virtuella datorn contoso-server.
