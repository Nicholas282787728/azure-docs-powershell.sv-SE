# Update-AzAvailabilitySet

## SYNOPSIS
Uppdaterar en tillgänglighetsuppsättning.

## SYNTAX

```
Update-AzAvailabilitySet [-AvailabilitySet] <PSAvailabilitySet> [[-Sku] <String>]
 [-ProximityPlacementGroupId <String>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Update-AzAvailabilitySet** uppdaterar en tillgänglighetsuppsättning.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzAvailabilitySet -ResourceGroupName 'ResourceGroup01' -Name 'AvSet01' | Update-AzAvailabilitySet -Managed;
```

Med det här kommandot uppdateras tillgänglighetsuppsättningen med namnet AvSet01 i resursgruppen med namnet ResourceGroup01 till en hanterad tillgänglighetsuppsättning.
