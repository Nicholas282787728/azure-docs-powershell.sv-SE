# Get-AzVMUsage

## SYNOPSIS
Hämtar den virtuella datorns basräkningsanvändning för en plats.

## SYNTAX

```
Get-AzVMUsage [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVMUsage** får den virtuella datorns basräkningsanvändning för en plats.

## EXEMPEL

### Exempel 1: Hämta basräkningsanvändning för en plats
```
PS C:\> Get-AzVMUsage -Location "Central US"
```

Det här kommandot hämtar den virtuella datorns kärnräkningsanvändning för platsen i centrala USA.
