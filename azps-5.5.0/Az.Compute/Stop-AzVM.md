# Stop-AzVM

## SYNOPSIS
Stoppar en virtuell Azure-dator.

## SYNTAX

### ResourceGroupNameParameterSetName (standard)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-ResourceGroupName] <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Stop-AzVM-cmdleten** stoppar en virtuell Azure-dator.

## EXEMPEL

### Exempel 1: Stoppa en virtuell dator
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Det här kommandot stoppar den virtuella datorn med namnet VirtualMa machinee07 i ResourceGroup11.
