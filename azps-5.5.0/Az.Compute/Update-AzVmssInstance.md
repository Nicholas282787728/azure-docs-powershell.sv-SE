# Update-AzVmssInstance

## SYNOPSIS
Startar en manuell uppgradering av VMSS-instansen.

## SYNTAX

```
Update-AzVmssInstance [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-InstanceId] <String[]>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Update-AzVmssInstance-cmdleten startas en manuell uppgradering av den angivna VMSS-instansen (Virtual Machine Scale Set).
Det här används när uppgraderingsprincipen på VMSS-skaluppsättningen är inställd på manuell.

## EXEMPEL

### Exempel 1: Starta en uppgradering av VMSS-instansen
```
PS C:\> Update-AzVmssInstance -ResourceGroupName "Group011" -VMScaleSetName "VMScaleSet001" -InstanceId "0"
```

Det här kommandot startar en uppgradering av VMSS med namnet VMScaleSet001 som har instans-ID för 0.
