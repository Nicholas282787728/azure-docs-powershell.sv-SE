# Start-AzVmssRollingOSUpgrade

## SYNOPSIS
Startar en rullningsuppgradering för att flytta alla instanser av skalning för virtuella maskiner till den senaste tillgängliga versionen av plattformsbild-OS.

## SYNTAX

```
Start-AzVmssRollingOSUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Startar en rullningsuppgradering för att flytta alla instanser av skalning för virtuella maskiner till den senaste tillgängliga versionen av plattformsbild-OS.
Instanser som redan kör den senaste tillgängliga operativsystemet påverkas inte.

## EXEMPEL

### Exempel 1
```
PS C:\> Start-AzVmssRollingOSUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Det här kommandot startar en rullande uppgradering av alla VM-skalinstanser av skaluppsättningen "VMSS001" i resursgruppen "Group001".
