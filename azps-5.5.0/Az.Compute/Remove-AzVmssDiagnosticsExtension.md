# Remove-AzVmssDiagnosticsExtension

## SYNOPSIS
Tar bort ett diagnostiktillägg från VMSS.

## SYNTAX

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzVmssDiagnosticsExtension** tar bort ett diagnostiktillägg från VMSS (Virtual Machine Scale Set).

## EXEMPEL

### Exempel 1: Ta bort ett diagnostiktillägg från VMSS
```
PS C:\> Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

Det här kommandot tar bort diagnostiktillägget från VMSS.
