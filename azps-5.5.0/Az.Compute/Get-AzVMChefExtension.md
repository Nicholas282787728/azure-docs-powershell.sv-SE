# Get-AzVMChefExtension

## SYNOPSIS
Hämtar information om ett cheftillägg.

## SYNTAX

### Linux
```
Get-AzVMChefExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status] [-Linux]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Windows
```
Get-AzVMChefExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status] [-Windows]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVMChefExtension** hämtar information om ett Chef-tillägg som installeras på en virtuell dator.

## EXEMPEL

### Exempel 1: Få information om Chef-tillägget för en virtuell Windows-dator
```
PS C:\> Get-AzVMChefExtension -ResourceGroupName "ResourceGroup001" -VMName "WindowsVM001" -Windows
```

Det här kommandot får chef-tillägget från en virtuell Windows-dator med namnet WindowsVM001 som tillhör resursgruppen ResourceGroup001.

### Exempel 2: Få information om Chef-tillägget för en virtuell Linux-dator
```
PS C:\> Get-AzVMChefExtension -ResourceGroupName "ResourceGroup002" -VMName "LinuxVM001" -Linux
```

Det här kommandot får chef-tillägget från en virtuell Linux-dator med namnet LinuxVM001 som tillhör resursgruppen ResourceGroup002.
