# Disable-AzVMDiskEncryption

## SYNOPSIS
Inaktiverar kryptering på en virtuell IaaS-dator.

## SYNTAX

```
Disable-AzVMDiskEncryption [-ResourceGroupName] <String> [-VMName] <String> [[-VolumeType] <String>]
 [[-Name] <String>] [[-TypeHandlerVersion] <String>] [-Force] [-DisableAutoUpgradeMinorVersion]
 [-ExtensionType <String>] [-ExtensionPublisherName <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Disable-AzVMDiskEncryption** inaktiverar kryptering på en virtuell infrastruktur som en tjänstdator (IaaS).
Den här cmdleten stöds endast på virtuella Windows-datorer och inte på virtuella Linux-datorer.
Den här cmdleten installerar ett tillägg på den virtuella datorn för att inaktivera kryptering.
Om *parametern* Namn inte anges skapas ett tillägg med standardnamnet "AzureDiskEncryption for Windows VMs".
Varning! Den här cmdleten startar om den virtuella datorn.

## EXEMPEL

### Exempel 1: Inaktivera kryptering för alla volymer på en virtuell Windows-dator
```
PS C:\> Disable-AzVMDiskEncryption -ResourceGroupName "Group001" -VMName "VM002"
```

Det här kommandot inaktiverar kryptering för volymer av typen all för den virtuella datorn med namnet VM002 som tillhör resursgruppen med namnet Group001.
Eftersom *VolumeType-parametern* inte anges anger cmdleten värdet till Alla.

### Exempel 2: Inaktivera kryptering för datavolymer på en virtuell Windows-dator
```
PS C:\> $ResourceGroup = "Group002"
PS C:\> $VMName = "VM004"
PS C:\> $VolumeType = "Data"
PS C:\> Disable-AzVMDiskEncryption -ResourceGroupName $ResourceGroup -VMName $VMName -VolumeType $VolumeType
```

Det här kommandot inaktiverar kryptering för volymer av typdata för den virtuella datorn med namnet VM004 som tillhör resursgruppen med namnet Group002.
