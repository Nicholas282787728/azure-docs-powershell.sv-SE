# Stop-AzVmssRollingUpgrade

## SYNOPSIS
Avbryter den aktuella virtuella datorskalan för rullningsuppgradering.

## SYNTAX

```
Stop-AzVmssRollingUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Avbryter den aktuella virtuella datorskalan för rullningsuppgradering.

## EXEMPEL

### Exempel 1
```
PS C:\> Stop-AzVmssRollingUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Det här kommandot avbryter den rullningsbaserade uppgraderingen av VM-skaluppsättningen "VMSS001" i resursgruppen "Group001".
