# Remove-AzVMExtension

## SYNOPSIS
Tar bort ett tillägg från en virtuell dator.

## SYNTAX

```
Remove-AzVMExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzVMExtension** tar bort ett tillägg från virtual machine-tillägg för en virtuell dator.

## EXEMPEL

### Exempel 1: Ta bort ett tillägg från en virtuell dator
```
PS C:\> Remove-AzVMExtension -ResourceGroupName "ResourceGroup11" -Name "ContosoTest" -VMName "VirtualMachine22"
```

Det här kommandot tar bort tillägget ContosoTest från den virtuella datorn Med namnet VirtualMachine22 i ResourceGroup11.
