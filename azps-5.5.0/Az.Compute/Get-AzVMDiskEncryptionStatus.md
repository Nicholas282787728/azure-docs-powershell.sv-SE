# Get-AzVMDiskEncryptionStatus

## SYNOPSIS
Hämtar krypteringsstatusen för den virtuella datorn.

## SYNTAX

```
Get-AzVMDiskEncryptionStatus [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [-ExtensionType <String>] [-ExtensionPublisherName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVMDiskEncryptionStatus** får krypteringsstatusen för den virtuella datorn.
Här visas krypteringsstatus för operativsystemet och datavolymerna.
Förutom krypteringsstatusen visas också den hemliga URL:en för kryptering, nyckelkrypteringsnyckelns URL, resurs-ID:n för **KeyVaults där krypteringsnyckeln** och nyckelkrypteringsnyckeln för operativsystemsvolym finns.

## EXEMPEL

### Exempel 1: Hämta krypteringsstatusen för en virtuell dator
```
PS C:\> Get-AzVmDiskEncryptionStatus -ResourceGroupName "MyResourceGroup001" -VMName "VM001"
```

Det här kommandot får krypteringsstatusen för den virtuella datorn med namnet VM001.
