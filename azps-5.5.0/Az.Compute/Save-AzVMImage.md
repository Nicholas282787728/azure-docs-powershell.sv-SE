# Save-AzVMImage

## SYNOPSIS
Sparar en virtuell dator som ett VMImage.

## SYNTAX

### ResourceGroupNameParameterSetName (standard)
```
Save-AzVMImage [-Name] <String> [-DestinationContainerName] <String> [-VHDNamePrefix] <String> [-Overwrite]
 [[-Path] <String>] [-ResourceGroupName] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### IdParameterSetName
```
Save-AzVMImage [-DestinationContainerName] <String> [-VHDNamePrefix] <String> [-Overwrite] [[-Path] <String>]
 [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Spara-AzVMImage** sparar en virtuell dator som ett VMImage.
Innan du skapar en bild av en virtuell dator bör du använda den virtuella datorn och markera den som generaliserad med hjälp av Set-AzVM-cmdleten.
Utdata för den här cmdleten är en JavaScript Object Notation-mall (JSON).
Du kan distribuera virtuella datorer från din betagna bild.

## EXEMPEL

### Exempel 1: Avbilda en virtuell dator
```
PS C:\> Set-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07" -Generalized 
PS C:\> Save-AzVMImage -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -DestinationContainerName "VMContainer01" -VHDNamePrefix "VM07"
```

Det första kommandot markerar den virtuella datorn med namnet VirtualMa machinee07 som generaliserad.
Det andra kommandot avbildar en virtuell dator med namnet VirtualMa machinee07 som ett VMImage.
Egenskapen **Utdata** returnerar en JSON-mall.
