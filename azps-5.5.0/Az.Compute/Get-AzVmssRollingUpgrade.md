# Get-AzVmssRollingUpgrade

## SYNOPSIS
Visar statusen för den senaste uppgraderingsskalan för virtuell datorskala.

## SYNTAX

```
Get-AzVmssRollingUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Visar statusen för den senaste uppgraderingsskalan för virtuell datorskala.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzVmssRollingUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Det här kommandot visar statusen för den senaste rullande uppgraderingen av VMSSS med namnet VMSS001 som tillhör resursgruppen med namnet Group001.
