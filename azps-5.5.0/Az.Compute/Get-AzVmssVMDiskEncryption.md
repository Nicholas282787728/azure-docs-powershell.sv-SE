# Get-AzVmssVMDiskEncryption

## SYNOPSIS
Visar diskkrypteringsstatusen för virtuella maskiner i en VM-skaluppsättning.

## SYNTAX

```
Get-AzVmssVMDiskEncryption [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String>]
 [-ExtensionName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Visar diskkrypteringsstatusen för VM-skaluppsättningen.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzVmssVMDiskEncryption -ResourceGroupName "Group001" -VMScaleSetName "VMSS001" -InstanceId "1"
```

Visar diskkrypteringsstatusen för VM-instans 1 i skaluppsättningen för VMSS001 som tillhör resursgruppen med namnet Group001.
