# Get-AzRemoteDesktopFile

## SYNOPSIS
Hämtar en .rdp-fil.

## SYNTAX

### Ladda ned
```
Get-AzRemoteDesktopFile [-ResourceGroupName] <String> [-Name] <String> [-LocalPath] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Starta
```
Get-AzRemoteDesktopFile [-ResourceGroupName] <String> [-Name] <String> [[-LocalPath] <String>] [-Launch]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRemoteDesktopFile** får en .rdp-fil (Remote Desktop Protocol).

## EXEMPEL

### Exempel 1: Hämta en fjärrskrivbordsfil
```
PS C:\> Get-AzRemoteDesktopFile -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07" -LocalPath "D:\RemoteDesktopFile07.rdp"
```

Det här kommandot hämtar Fjärrskrivbord-filen för den virtuella datorn med namnet VirtualMa machinee07.
Kommandot lagrar resultatet i filen D:\RemoteDesktopFile07.rdp.
