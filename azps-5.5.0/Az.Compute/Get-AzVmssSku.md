# Get-AzVmssSku

## SYNOPSIS
Hämtar tillgängliga SKU:er för VMSS.

## SYNTAX

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Get-AzVmssSku-cmdleten** hämtar tillgängliga SKU:er för VMSS (Virtual Machine Scale Set).

## EXEMPEL

### Exempel 1: Hämta alla tillgängliga SKU:er från VMSS
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Det här kommandot hämtar alla tillgängliga SKU:er från VMSS med namnet ContosoVMSS som tillhör resursgruppen ContosoGroup.
