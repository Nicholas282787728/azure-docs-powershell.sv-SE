# Get-AzVMADDomainExtension

## SYNOPSIS
Hämtar information om ett AD-domäntillägg.

## SYNTAX

```
Get-AzVMADDomainExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzVKAMDomainExtension** hämtar information om det angivna Azure Active Directory-domäntillägget (AD).

## EXEMPEL

### Exempel 1

Hämtar information om ett AD-domäntillägg. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Get-AzVMADDomainExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```
