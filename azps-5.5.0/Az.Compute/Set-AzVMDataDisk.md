# Set-AzVMDataDisk

## SYNOPSIS
Ändrar egenskaperna för en virtuell datordatadisk.

## SYNTAX

### ChangeWithName
```
Set-AzVMDataDisk [-VM] <PSVirtualMachine> [-Name] <String> [[-Caching] <CachingTypes>]
 [[-DiskSizeInGB] <Int32>] [-StorageAccountType <String>] [-DiskEncryptionSetId <String>] [-WriteAccelerator]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ChangeWithLun
```
Set-AzVMDataDisk [-VM] <PSVirtualMachine> [-Lun] <Int32> [[-Caching] <CachingTypes>] [[-DiskSizeInGB] <Int32>]
 [-StorageAccountType <String>] [-DiskEncryptionSetId <String>] [-WriteAccelerator]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzVMDataDisk** ändrar egenskaperna för en virtuell datordatadisk.

## EXEMPEL

### Exempel 1: Ändra cachelagringsläge för en datadisk
```powershell
PS C:\> $VM = Get-AzVM -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07"
PS C:\> Set-AzVMDataDisk -VM $VM -Name "DataDisk01" -Caching ReadWrite | Update-AzVM
```

Det första kommandot får den virtuella datorn ContosoVM07 genom att använda **Get-AzVM.**
Kommandot lagrar det i den $VM variabeln.
Det andra kommandot ändrar cachelagringsläget för datadisken med namnet DataDisk01 på den virtuella datorn i $VM.
Kommandot skickar resultatet till cmdleten Update-AzVM, som implementerar ändringarna.
Om du ändrar betalningsläget startar den virtuella datorn om.

### Exempel 2

Ändrar egenskaperna för en virtuell datordatadisk. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Set-AzVMDataDisk -Caching None -Lun 1 -VM <PSVirtualMachine>
```
