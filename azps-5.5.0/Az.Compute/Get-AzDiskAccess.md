# Get-AzDiskAccess

## SYNOPSIS
Hämtar egenskaperna för diskåtkomst

## SYNTAX

### DefaultParameterSet (standard)
```
Get-AzDiskAccess [[-ResourceGroupName] <String>] [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIDParameterSet
```
Get-AzDiskAccess [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzDiskAccess** får egenskaperna för Diskåtkomst

## EXEMPEL

### Exempel 1: Använda standardparameteruppsättning 
```
PS C:\> Get-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -Name 'DiskAccess01'

ProvisioningState          : Succeeded
TimeCreated                : 8/13/2020 7:02:50 PM
Location                   : northcentralus

Name                       : DiskAccess01
Type                       : Microsoft.Compute/diskAccesses
Tags                       : {}
```

Det här kommandot hämtar egenskaperna för en Disk Access-resurs med namnet 'DiskAccess01' i resursgruppen Resursgrupp01.'
