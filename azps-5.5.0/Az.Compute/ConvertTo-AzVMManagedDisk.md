# ConvertTo-AzVMManagedDisk

## SYNOPSIS
Konverterar en virtuell dator med blob-baserade diskar till en virtuell dator med hanterade diskar.

## SYNTAX

```
ConvertTo-AzVMManagedDisk [-ResourceGroupName] <String> [-VMName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten ConvertTo-AzVMManagedDisk** konverterar en virtuell dator med blob-baserade diskar till en virtuell dator med hanterade diskar.
Den virtuella datorn måste stoppas innan du kan använda den här åtgärden.

## EXEMPEL

### Exempel 1
```
PS C:\> ConvertTo-AzVMManagedDisk -ResourceGroupName 'ResourceGroup01' -VMName 'VM01'
```

Det här kommandot konverterar blob-baserade diskar för den virtuella datorn med namnet VM01 i resursgruppen ResourceGroup01 till hanterade diskar.
