# Get-AzCognitiveServicesAccountNetworkRuleSet

## SYNOPSIS
Få egenskapen NetworkRule för ett kognitivt services-konto

## SYNTAX

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzCognitiveServicesAccountNetworkRuleSet** får egenskapen NetworkRule för ett Kognitiv Services-konto

## EXEMPEL

### Exempel 1: Få egenskapen NetworkRule för ett visst kognitivt services-konto
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

Det här kommandot får egenskapen NetworkRule för ett visst Kognitiva tjänster-konto
