# Get-AzCognitiveServicesAccount

## SYNOPSIS
Hämtar ett konto.

## SYNTAX

### ResourceGroupParameterSet
```
Get-AzCognitiveServicesAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzCognitiveServicesAccount** får de etablerade kognitiva tjänsterna-kontona i resursgruppen som anges av *parametern ResourceGroupName.*
Om du inte anger parametern *ResourceGroupName* får den här cmdleten alla Kognitiva tjänster-konton för den aktuella prenumerationen.
