# Get-AzCognitiveServicesAccountType

## SYNOPSIS
Hämtar de tillgängliga kontotyperna för kognitiva tjänster.

## SYNTAX

### GetAccountTypes (standard)
```
Get-AzCognitiveServicesAccountType [-Location <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetAccountTypeWithName
```
Get-AzCognitiveServicesAccountType -TypeName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzCognitiveServicesAccountType** får tillgängliga kontotyper för Kognitiva tjänster under den här prenumerationen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountType
```

Hämta listan över tillgängliga typer.

### Exempel 2
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -Location westus
```

Hämta listan över tillgängliga typer i westus.

### Exempel 3
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -TypeName Face

Face
```

Kontrollera om `Face` det är ett giltigt typnamn. Namnet returneras om det är ett giltigt namn.
