# Get-AzCognitiveServicesAccountKey

## SYNOPSIS
Hämtar API-nycklar för ett konto.

## SYNTAX

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzCognitiveServicesAccountKey får API-nycklar** för ett etablerat kognitivt services-konto.
Ett kognitivt services-konto har två API-nycklar: Key1 och Key2.
Nycklar möjliggör interaktion med slutpunkten för kognitiva tjänster-kontot.
Använd New-AzCognitiveServicesAccountKey för att generera en nyckel.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
