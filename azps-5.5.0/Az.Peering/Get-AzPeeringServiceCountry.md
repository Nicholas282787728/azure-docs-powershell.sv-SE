# Get-AzPeeringServiceCountry

## SYNOPSIS
Här listas de länder som är tillgängliga för peeringtjänsten.

## SYNTAX

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Här listas de länder som är tillgängliga för peeringtjänsten.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Lista över länder som är tillgängliga för peeringtjänst.
