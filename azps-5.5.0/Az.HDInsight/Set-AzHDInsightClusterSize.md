# Set-AzHDInsightClusterSize

## SYNOPSIS
Anger antalet arbetsnoder i ett angivet kluster.

## SYNTAX

```
Set-AzHDInsightClusterSize [-ClusterName] <String> [-TargetInstanceCount] <Int32> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzHDInsightClusterSize** anger antalet arbetsnoder i ett angivet Azure HDInsight-kluster.

## EXEMPEL

### Exempel 1: Ange storleken på ett angivet kluster
```
PS C:\>Set-AzHDInsightClusterSize -ClusterName "your-hadoop-001" -TargetInstanceCount 6
```

Det här kommandot anger storleken på klustret med namnet ditt-hadoop-001.
