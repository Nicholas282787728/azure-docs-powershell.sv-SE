# Disable-AzHDInsightMonitoring

## SYNOPSIS
Inaktiverar övervakning i ett HDInsight-kluster och relevanta loggar slutar flöda till den övervakningsarbetsyta som angetts under aktiverande.

## SYNTAX

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Disable-AzHDInsightMonitoring** inaktiverar övervakning i ett Azure HDInsight-kluster.

## EXEMPEL

### Exempel 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

Övervakning inaktiveras i HDInsight-klustret och relevanta loggar slutar att flöda till övervakningsarbetsytan.

### Exempel 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

Övervakning inaktiveras i HDInsight-klustret och relevanta loggar kommer inte att sluta flöda till övervakningsarbetsytan.
