# Remove-AzHDInsightCluster

## SYNOPSIS
Tar bort det angivna HDInsight-klustret från den aktuella prenumerationen.

## SYNTAX

```
Remove-AzHDInsightCluster [-ClusterName] <String> [-ResourceGroupName <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzHDInsightCluster** tar bort det angivna HDInsight-tjänstklustret från en prenumeration.
Den här åtgärden tar även bort data som lagrats i HDFS -systemet (Hadoop Distributed File System) i klustret.
Data som lagras i det associerade Azure Storage-kontot tas inte bort.
Data som lagras i externa metakällor tas inte bort.

## EXEMPEL

### Exempel 1: Ta bort ett Azure HDInsight-kluster
```
PS C:\>Remove-AzHDInsightCluster -ClusterName "your-hadoop-001"
```

Det här kommandot tar bort klustret med namnet ditt-hadoop-001.
