# Use-AzHDInsightCluster

## SYNOPSIS
Väljer ett kluster som ska användas med Invoke-RmAzureHDInsightHiveJob cmdlet.

## SYNTAX

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Use-AzHDInsightCluster** väljer Azure HDInsight-kluster för cmdleten Invoke-AzHDInsightHiveJob att skicka projekt.

## EXEMPEL

### Exempel 1: Välj ett kluster för inskickade urvalsfrågor
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Det här kommandot väljer ett kluster för en inskickade urvalsfråga.
