# Stop-AzHDInsightJob

## SYNOPSIS
Stoppar ett angivet körningsjobb i ett kluster.

## SYNTAX

```
Stop-AzHDInsightJob [-ClusterName] <String> [-JobId] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Stop-AzHDInsight En** cmdlet stoppar ett angivet körningsjobb på ett Azure HDInsight-kluster.

## EXEMPEL

### Exempel 1: Stoppa ett jobb på det angivna klustret
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential
PS C:\> Stop-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
            -JobId $jobId
```

Det här kommandot avbryter ett jobb i klustret med namnet your-hadoop-001.
