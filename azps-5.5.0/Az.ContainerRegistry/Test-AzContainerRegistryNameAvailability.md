# Test-AzContainerRegistryNameAvailability

## SYNOPSIS
Kontrollerar tillgängligheten för ett behållarregisternamn.

## SYNTAX

```
Test-AzContainerRegistryNameAvailability [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Med Test-AzContainerRegistryNameAvailability cmdleten kontrolleras om ett behållarregisternamn är giltigt och tillgängligt att använda.

## EXEMPEL

### Exempel 1: Kontrollerar tillgängligheten för ett behållarregisternamn
```powershell
PS C:\>Test-AzContainerRegistryNameAvailability -Name 'SomeRegistryName'

NameAvailable Reason Message
------------- ------ -------
         True
```

Det här kommandot kontrollerar tillgängligheten för behållarregisternamnet \` SomeRegistryName. \`
