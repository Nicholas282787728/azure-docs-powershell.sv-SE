# Remove-AzContainerRegistryTag

## SYNOPSIS
Ta bort taggen ACR-tagg.

## SYNTAX

```
Remove-AzContainerRegistryTag -RepositoryName <String> -Name <String> -RegistryName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Ta bort taggen ACR-tagg.
Om du vill använda den här cmdleten kör du `Update-AzContainerRegistryRepository -RegistryName XXX -Repository XXX -DeleteEnable $true`
först.

## EXEMPEL

### Exempel 1
```powershell
Remove-AzContainerRegistryTag -RegistryName registry -RepositoryName alpine -Name latest
True
```

Ta bort taggen alpine:tag under registret.
