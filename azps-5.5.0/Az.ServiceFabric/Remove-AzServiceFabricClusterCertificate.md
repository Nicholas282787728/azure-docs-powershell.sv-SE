# Remove-AzServiceFabricClusterCertificate

## SYNOPSIS
Ta bort ett klustercertifikat från att användas för klustersäkerhet.

## SYNTAX

```
Remove-AzServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Använd **Remove-AzServiceFabricClusterCertificate** för att ta bort ett klustercertifikat från klustret, så länge det finns ett annat giltigt certifikat som redan används i klustret.

## EXEMPEL

### Exempel 1
```
PS C:\> Remove-AzServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Det här kommandot tar bort certifikatet med thumbprint 5F3660C715EBBDA31DB1FFDCF508302348DE8E7A från att användas för klustersäkerhet.
