# Remove-AzServiceFabricNode

## SYNOPSIS
Ta bort noder från den specifika nodtypen från ett kluster.

## SYNTAX

```
Remove-AzServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Använd **Remove-AzServiceFabricNode för** att ta bort noder från en viss nodtyp från ett kluster. Borttagningen utförs endast om den uppfyller hälsostatistiken för kluster.

## EXEMPEL

### Exempel 1
```powershell
PS c:> Remove-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeType 'nt1' -NumberOfNodesToRemove 2
```

Det här kommandot tar bort 2 noder från NodeType 'nt1'.
