# Add-AzServiceFabricNode

## SYNOPSIS
Lägga till noder i den specifika nodtypen i klustret.

## SYNTAX

```
Add-AzServiceFabricNode -NumberOfNodesToAdd <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Använd **Lägg till AzServiceFabricNode för** att lägga till noder i den specifika nodtypen. Du behöver bara ange antalet noder som du vill lägga till i en nodtyp.

## EXEMPEL

### Exempel 1
```powershell
PS c:> Add-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Det här kommandot lägger till 2 noder i nodtypen 'n1'.
