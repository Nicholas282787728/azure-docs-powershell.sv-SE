## BESKRIVNING
Cmdleten **Invoke-AzStackEdgeDevice** anropar åtgärder för att söka, ladda ned och installera uppdateringarna på Stack Edge-enheten. En automatisk genomsökning körs på enheten varje dag, du kan utlösa genomsökningen explicit genom att köra den här cmdleten.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Invoke-AzStackEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -ScanForUpdate
```

### Exempel 2
```powershell
PS C:\> Invoke-AzStackEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -FetchUpdate
```

### Exempel 3
```powershell
PS C:\> Invoke-AzStackEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -InstallUpdate
```
