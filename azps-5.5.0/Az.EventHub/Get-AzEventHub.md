# Get-AzEventHub

## SYNOPSIS
Hämtar information om ett enskilt händelsenav eller får en lista över händelsehubben.

## SYNTAX

```
Get-AzEventHub [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>] [-MaxCount <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdlet Get-AzEventHub returnerar antingen informationen för ett händelsenav eller en lista över alla händelsehubben i det aktuella namnområdet.
Om händelsehubbens namn anges returneras information om ett enskilt händelsenav.
Om ett Namn för Händelsehubben inte anges returneras en lista över alla händelsehubben i det angivna namnområdet.

## EXEMPEL

### Exempel 1: angivet EventHub
```powershell
PS C:\> Get-AzEventHub -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName -EventHubName MyEventHubName
```

Returnerar information om Händelsehubben \` MyEventHubName. \`

### Exempel 2: Lista över EventHub i angivet namnområde
```powershell
PS C:\> Get-AzEventHub -ResourceGroup MyResourceGroupName -NamespaceName MyNamespaceName
```

Returnerar en lista över Händelsehubben i namnområdet \` MyNamespaceName. \`
