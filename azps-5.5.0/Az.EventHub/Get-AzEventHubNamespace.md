# Get-AzEventHubNamespace

## SYNOPSIS
Hämtar information om ett namnområde i Händelsehubben eller hämtar en lista över alla namnområden i Händelsehubben i den aktuella Azure-prenumerationen.

## SYNTAX

```
Get-AzEventHubNamespace [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Med Get-AzEventHubNamespace-cmdleten får du antingen information om ett angivet namnområde i Händelsehubben eller en lista över alla namnområden i Händelsehubben i den aktuella Azure-prenumerationen.
Om namnområdets namn anges returneras information om ett enskilt namnområde i Händelsehubben.
Om namnområdets namn inte anges returneras en lista med namnområden.

```

Sku                    : Name : Standard , Capacity : 1 , Tier : Standard
Tags                   :

Status                 : Active
```

Hämtar information om namnområdet \` MyNamespaceName \` i resursgruppen \` MyResourceGroupName. \`
