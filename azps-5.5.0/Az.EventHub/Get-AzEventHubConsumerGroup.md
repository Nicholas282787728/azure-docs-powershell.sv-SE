# Get-AzEventHubConsumerGroup

## SYNOPSIS
Hämtar information om en viss konsumentgrupp för händelsehubben eller får en lista över konsumentgrupper i ett händelsenav.

## SYNTAX

```
Get-AzEventHubConsumerGroup [-ResourceGroupName] <String> [-Namespace] <String> [-EventHub] <String>
 [[-Name] <String>] [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Med Get-AzEventHubConsumerGroup-cmdleten får du antingen information om en angiven konsumentgrupp för händelsehubben eller en lista över konsumentgrupper i ett visst händelsenav.
Om namnet på en konsumentgrupp anges returneras information om en enskild konsumentgrupp.
Om namnet på en konsumentgrupp inte anges returneras en lista över konsumentgrupper i det angivna händelsehubben.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzEventHubConsumerGroup -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName -EventHubName MyEventHubName -ConsumerGroupName MyConsumerGroupName
```

Hämtar konsumentgruppen \` MyConsumerGroupName i Händelsehubben MyEventHubName, som finns i \` \` \` namnområdet \` MyNamespaceName med \` resursgruppen \` MyResourceGroupName. \`

### Exempel 2
```
PS C:\> Get-AzEventHubConsumerGroup -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName -EventHubName MyEventHubName
```

Hämtar en lista över konsumentgrupper i Händelsehubben MyEventHubName, som finns i \` \` namnområdet \` MyNamespaceName med \` resursgruppen \` MyResourceGroupName. \`
