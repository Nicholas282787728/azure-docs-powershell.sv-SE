# Get-AzEventHubClustersAvailableRegion

## SYNOPSIS
Hämtar information om enstaka EventHub-kluster eller listan med kluster i den givna resursgruppen

## SYNTAX

```
Get-AzEventHubClustersAvailableRegion [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Listan Get-AzEventHubClustersAvailableRegion cmdleten med regioner där dedikerade är tillgängliga att skapa.
