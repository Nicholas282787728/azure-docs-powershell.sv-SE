# Get-AzEventHubCluster

## SYNOPSIS
Hämtar information om ett enstaka händelsenavkluster eller får en lista över händelsenavkluster.

## SYNTAX

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Med Get-AzEventHubCluster-cmdleten returneras antingen informationen om ett händelsenavskluster eller en lista över alla händelsehubbskluster i en viss resursgrupp.
Om klusternamnet anges returneras information om ett enstaka kluster.
Om ett klusternamn inte anges returneras en lista över alla kluster i den angivna resursgruppen.
