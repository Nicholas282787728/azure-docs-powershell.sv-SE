# Get-AzRedisCacheKey

## SYNOPSIS
Hämtar snabbtangenterna för en Redis-cache.

## SYNTAX

```
Get-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRedisCacheKey** får snabbtangenterna för en Azure Redis-cache.

## EXEMPEL

### Exempel 1: Hämta snabbtangenter för en Redis-cache
```
PS C:\>Get-AzRedisCacheKey -ResourceGroupName "MyResourceGroup" -Name "MyCacheKey"
PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Det här kommandot hämtar snabbtangenterna med namnet MyCacheKey.
