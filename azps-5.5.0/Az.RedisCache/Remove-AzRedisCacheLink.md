# Remove-AzRedisCacheLink

## SYNOPSIS
Ta bort en georeplikeringslänk mellan två Redis-cacher.

## SYNTAX

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Ta bort en georeplikeringslänk mellan två Redis-cacher.

## EXEMPEL

### Exempel 1: Ta bort en georeplikeringslänk
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Det här kommandot tar bort en georeplikeringslänkar där Redis-cache med namnet mycache1 är primär och Redis-cache med namnet mycache2 är sekundär.
