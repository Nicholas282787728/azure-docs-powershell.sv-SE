# Remove-AzRedisCacheDiagnostic

## SYNOPSIS
Inaktiverar diagnostik för Azure Redis-cache.

## SYNTAX

```
Remove-AzRedisCacheDiagnostic [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRedisCacheDiagnostic** inaktiverar diagnostik för Azure Redis-cache.

## EXEMPEL

### Exempel 1: Inaktivera diagnostik
```
PS C:\>Remove-AzRedisCacheDiagnostic -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -Force
```

Det här kommandot inaktiverar diagnostik på angiven Azure Redis-cache.
Detta inaktiverar diagnostik för alla Azure Redis-cacheminnen i samma region för prenumerationen.
