# Remove-AzRedisCachePatchSchedule

## SYNOPSIS
Tar bort korrigeringsschemat.

## SYNTAX

```
Remove-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRedisCachePatchSchedule** tar bort korrigeringsschemat från en cache i Azure Redis Cache.

## EXEMPEL

### Exempel 1: Ta bort korrigeringsschemat
```
PS C:\>Remove-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Det här kommandot tar bort korrigeringsschemat från cachen med namnet RedisCache06.
