# New-AzRedisCacheKey

## SYNOPSIS
Återskapar åtkomstnyckeln för en Redis-cache.

## SYNTAX

```
New-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> -KeyType <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den **nya cmdleten AzRedisCacheKey** återskapar åtkomstnyckeln för en Azure Redis-cache.

## EXEMPEL

### Exempel 1: Skapa en primärnyckel
```
PS C:\>New-AzRedisCacheKey -ResourceGroupName "ResourceGroup03" -Name "myCache" -KeyType "Primary" -Force

          PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=

          SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Det här kommandot återskapar primärnyckeln för en Redis-cache.

### Exempel 2: Skapa en sekundärnyckel
```
PS C:\>New-AzRedisCacheKey -ResourceGroupName "ResourceGroup03" -Name "myCache" -KeyType "Secondary" -Force
          PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=

          SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Det här kommandot återskapar sekundärnyckeln i en Redis-cache.
