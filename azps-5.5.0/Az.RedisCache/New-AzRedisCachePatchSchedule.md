# New-AzRedisCachePatchSchedule

## SYNOPSIS
Lägger till ett korrigeringsschema.

## SYNTAX

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzRedisCachePatchSchedule** lägger till ett korrigeringsschema i en cache i Azure Redis Cache.

## EXEMPEL

### Exempel 1: Skapa och lägga till ett korrigeringsschema i en cache
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Det här kommandot lägger till ett korrigeringsschema i cachen med namnet RedisCache06.
Parametern Entries använder ett kommando som använder **New-AzRedisCacheScheduleEntry** till att skapa ett schema.
