# Remove-AzRedisCache

## SYNOPSIS
Tar bort en Redis-cache.

## SYNTAX

```
Remove-AzRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRedisCache** tar bort en Azure Redis-cache.

## EXEMPEL

### Exempel 1: Ta bort en Redis-cache och returnera resultatet
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Det här kommandot tar bort en Redis-cache och visar om åtgärden lyckas.

### Exempel 2: Ta bort en Redis-cache och visa inte resultatet
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Med det här kommandot tas en Redis-cache bort.
Eftersom *passThru-parametern* inte har angetts visas inte åtgärdens resultat.
