# Get-AzBlueprintArtifact

## SYNOPSIS
Hämta artefakter från en skissdefinition.

## SYNTAX

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämta artefakter från en skissdefinition. Om en skissdefinitionsversion inte anges hämtas utkastversionen. Om det inte finns någon utkastversion returneras den senaste publicerade skissdefinitionen.
