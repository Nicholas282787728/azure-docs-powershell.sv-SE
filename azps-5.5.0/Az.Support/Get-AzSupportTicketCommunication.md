# Get-AzSupportTicketCommunication

## SYNOPSIS
Få meddelanden om supportmeddelanden.

## SYNTAX

### GetByNameParameterSet (standard)
```
Get-AzSupportTicketCommunication -SupportTicketName <String> [-Name <String>] [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount] [-Skip <UInt64>] [-First <UInt64>]
 [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSupportTicketCommunication [-Name <String>] -SupportTicketObject <PSSupportTicket> [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount] [-Skip <UInt64>] [-First <UInt64>]
 [<CommonParameters>]
```

## BESKRIVNING
Får meddelanden för ett support ärende. All kommunikation för en biljett hämtas om du inte anger några andra parametrar. Du kan också filtrera kommunikationen med CreatedDate eller CommunicationType med filterparametern. Här är några exempel på filtervärden som du kan ange.


| Scenario                                                        | Filter                                                     |
|-----------------------------------------------------------------|------------------------------------------------------------|
| Hämta webbkommunikation                                          | "CommunicationType eq 'Web'"                               |
| Hämta telefonkommunikation                                        | "CommunicationType eq 'Phone'"                             |
| Hämta information som skapades den 20 december 2019 eller senare | "CreatedDate ge 2019-12-20"                                |
| Hämta meddelanden som skapades efter 20 december 2019       | "CreatedDate gt 2019-12-20"                                |
| Webbkommunikation skapad efter 20 december 2019            | "CreatedDate gt 2019-12-20 and CommunicationType eq 'Web'" |


Den här cmdleten har stöd för parametrar för sidnumrering via first och skip.

Du kan också hämta ett enskilt supportmeddelande genom att ange kommunikationsnamnet. 

## EXEMPEL

### Exempel 1: Hämta all kommunikation för ett support ärende
```powershell
PS C:\> Get-AzSupportTicketCommunication -SupportTicketName "test1"

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage3 user@contoso.com     test message3  2/4/2020 9:38:14 PM
testmessage2 user@contoso.com     test message2  2/4/2020 9:36:36 PM
testmessage1 user@contoso.com     test message   2/4/2020 9:35:42 PM
```

### Exempel 2: Hämta en enskild kommunikation med namnet på ett support ärende
```powershell
PS C:\> Get-AzSupportTicketCommunication -SupportTicketName "test1" -Name "testmessage1"

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage1 user@contoso.com     test message   2/4/2020 9:38:14 PM
```

### Exempel 3: Hämta de första två meddelandena för ett support ärende
```powershell
PS C:\> Get-AzSupportTicketCommunication -SupportTicketName "test1" -First 2

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage3 user@contoso.com     test message3  2/4/2020 9:38:14 PM
testmessage2 user@contoso.com     test message2  2/4/2020 9:36:36 PM
```

### Exempel 4: Hämta nästa två meddelanden efter att du hoppat över de första två meddelandena för ett support ärende
```powershell
PS C:\> Get-AzSupportTicketCommunication -SupportTicketName "test1" -Skip 2 -First 2

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage4 user@contoso.com     test message4  2/4/2020 9:38:14 PM
testmessage5 user@contoso.com     test message5  2/4/2020 9:36:36 PM
```

### Exempel 5: Hämta all webbkommunikation för ett support ärende
```powershell
PS C:\> Get-AzSupportTicketCommunication -SupportTicketName "test1" -Filter "CommunicationType eq 'Web'"

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage3 user@contoso.com     test message3  2/4/2020 9:38:14 PM
testmessage2 user@contoso.com     test message2  2/4/2020 9:36:36 PM
```

### Exempel 6: Hämta all kommunikation som skapats den 20 december 2019 för ett support ärende
```powershell
PS C:\> Get-AzSupportTicketCommunication -SupportTicketName "test1" -Filter "CreatedDate ge 2019-12-20"

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage3 user@contoso.com     test message3  2/4/2020 9:38:14 PM
testmessage2 user@contoso.com     test message2  2/4/2020 9:36:36 PM
```

### Exempel 7: Hämta all webbkommunikation som skapats den 20 december 2019 eller senare för ett support ärende
```powershell
PS C:\> Get-AzSupportTicketCommunication -SupportTicketName "test1" -Filter "CommunicationType eq 'Web' and CreatedDate ge 2019-12-20"

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage3 user@contoso.com     test message3  2/4/2020 9:38:14 PM
testmessage2 user@contoso.com     test message2  2/4/2020 9:36:36 PM
```

### Exempel 8: Hämta all kommunikation för ett support ärende genom att pipa support ärendeobjekt
```powershell
PS C:\> Get-AzSupportTicket -Name "test1" | Get-AzSupportTicketCommunication

Name         Sender               Subject        CreatedDate
----         ------               -------        -----------
testmessage3 user@contoso.com     test message3  2/4/2020 9:38:14 PM
testmessage2 user@contoso.com     test message2  2/4/2020 9:36:36 PM
testmessage1 user@contoso.com     test message   2/4/2020 9:35:42 PM
```
