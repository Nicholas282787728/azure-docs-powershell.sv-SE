# Get-AzServiceBusOperation

## SYNOPSIS
Lista som stöds ServiceBus Operations

## SYNTAX

```
Get-AzServiceBusOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzServiceBusOperation** listar de ServiceBus-åtgärder som stöds.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzServiceBusOperation
```

Listar åtgärder som stöds av ServiceBus
