# Get-AzMarketplacePrivateStoreOffer

## SYNOPSIS
Få erbjudandet från en eller flera privata butiker.

## SYNTAX

```
Get-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> [-OfferId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Få en eller flera privata store-erbjudanden med offentliga + privata planer som har lagts till under klientorganisationens omfattning. Om prenumerations-ID visas får du ett eller flera privata butikers erbjudande med privata abonnemang endast under prenumerationsomfång
