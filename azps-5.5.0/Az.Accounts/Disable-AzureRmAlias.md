# Disable-AzureRmAlias

## SYNOPSIS
Inaktiverar AzureRm-prefixalias för Az-moduler.

## SYNTAX

```
Disable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Inaktiverar AzureRm-prefixalias för Az-moduler. Om -Module har angetts är det bara moduler som har inaktiverats. Annars är alla AzureRm-alias inaktiverade.

## EXEMPEL

### Exempel 1
```
PS C:\> Disable-AzureRmAlias
```

Inaktiverar alla AzureRm-prefix för den aktuella PowerShell-sessionen.

### Exempel 1
```
PS C:\> Disable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Inaktiverar AzureRm-alias för modulen Az.Accounts för både den aktuella processen och den aktuella användaren.
