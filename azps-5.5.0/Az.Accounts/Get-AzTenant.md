# Get-AzTenant

## SYNOPSIS
Får klientorganisationen som är auktoriserad för den aktuella användaren.

## SYNTAX

```
Get-AzTenant [[-TenantId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den Get-AzTenant cmdleten får klientorganisationens auktoriserade för den aktuella användaren.

## EXEMPEL

### Exempel 1: Skaffa alla klientorganisationar
```
PS C:\> Connect-AzAccount
PS C:\> Get-AzTenant

Id                                   Name        Category Domains
--                                   ----------- -------- -------
xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx Microsoft   Home     {test0.com, test1.com, test2.microsoft.com, test3.microsoft.com...}
yyyyyyyy-yyyy-yyyy-yyyy-yyyyyyyyyyyy Testhost    Home     testhost.onmicrosoft.com
```

Det här exemplet visar hur du hämtar alla auktoriserade innehavare för ett Azure-konto.

### Exempel 2: Skaffa en specifik klientorganisation
```
PS C:\> Connect-AzAccount
PS C:\> Get-AzTenant -TenantId xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

Id                                   Name        Category Domains
--                                   ----------- -------- -------
xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx Microsoft   Home     {test0.com, test1.com, test2.microsoft.com, test3.microsoft.com...}
```

Det här exemplet visar hur du får en specifik behörig klientorganisation för ett Azure-konto.
