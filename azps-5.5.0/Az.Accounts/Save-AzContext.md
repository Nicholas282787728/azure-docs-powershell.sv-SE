# Save-AzContext

## SYNOPSIS
Sparar den aktuella autentiseringsinformationen för användning i andra PowerShell-sessioner.

## SYNTAX

```
Save-AzContext [[-Profile] <AzureRmProfile>] [-Path] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Save-AzContext-cmdleten sparas den aktuella autentiseringsinformationen för användning i andra PowerShell-sessioner.

## EXEMPEL

### Exempel 1: Spara kontexten för den aktuella sessionen
```
PS C:\> Connect-AzAccount
PS C:\> Save-AzContext -Path C:\test.json
```

Det här exemplet sparar den aktuella sessionens Azure-kontext till den JSON-fil som tillhandahålls.

### Exempel 2: Spara ett visst sammanhang
```
PS C:\> Save-AzContext -Profile (Connect-AzAccount) -Path C:\test.json
```

Det här exemplet sparar Azure-kontexten som överförs till cmdleten till den JSON-fil som tillhandahålls.
