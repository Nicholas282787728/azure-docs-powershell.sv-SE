# Select-AzContext

## SYNOPSIS
Välj en prenumeration och ett konto att rikta i Azure PowerShell-cmdlets

## SYNTAX

### SelectByInputObject (standard)
```
Select-AzContext -InputObject <PSAzureContext> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SelectByName
```
Select-AzContext [-Scope <ContextModificationScope>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [-Name] <String> [<CommonParameters>]
```

## BESKRIVNING
Välj en prenumeration som du vill rikta (eller konto eller klientorganisation) i Azure PowerShell-cmdlets.  Efter den här cmdleten riktar framtida cmdlets mot det valda sammanhanget.

## EXEMPEL

### Exempel 1: Rikta ett namngivet sammanhang
```powershell
PS C:\> Select-AzContext "Work"

Name    Account             SubscriptionName    Environment         TenantId
----    -------             ----------------    -----------         --------
Work    test@outlook.com    Subscription1       AzureCloud          xxxxxxxx-x...
```

Rikta kommande Azure PowerShell-cmdlets för kontot, klientorganisationen och prenumerationen i sammanhanget "Arbete".

### Exempel 2

Välj en prenumeration och ett konto som du vill rikta i Azure PowerShell-cmdlets i den här PowerShell-sessionen. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Select-AzContext -Name TestEnvironment -Scope Process
```
