# Uninstall-AzureRm

## SYNOPSIS
Tar bort alla AzureRm-moduler från en dator.

## SYNTAX

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Tar bort alla AzureRm-moduler från en dator.

## EXEMPEL

### Exempel 1
```
PS C:\> Uninstall-AzureRm
```

När du kör det här kommandot tas alla AzureRm-moduler bort från datorn för versionen av PowerShell där cmdleten körs.
