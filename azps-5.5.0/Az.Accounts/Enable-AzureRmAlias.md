# Enable-AzureRmAlias

## SYNOPSIS
Aktiverar AzureRm-prefixalias för Az-moduler.

## SYNTAX

```
Enable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Aktiverar AzureRm-prefixalias för Az-moduler. Om -Module har angetts har endast moduler som listas alias aktiverade. Annars aktiveras alla AzureRm-alias.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Enable-AzureRmAlias
```

Aktiverar alla AzureRm-prefix för den aktuella PowerShell-sessionen.

### Exempel 2
```powershell
PS C:\> Enable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Aktiverar AzureRm-alias för modulen Az.Accounts för både den aktuella processen och den aktuella användaren.
