# Get-AzSubscription

## SYNOPSIS
Skaffa prenumerationer som det aktuella kontot kan komma åt.

## SYNTAX

### ListByIdInTenant (standard)
```
Get-AzSubscription [-SubscriptionId <String>] [-TenantId <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ListByNameInTenant
```
Get-AzSubscription [-SubscriptionName <String>] [-TenantId <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Med Get-AzSubscription-cmdleten får du prenumerations-ID, prenumerationsnamn och hemklientorganisation för prenumerationer som det aktuella kontot kan komma åt.

## EXEMPEL

### Exempel 1: Hämta alla prenumerationer i alla klientorganisationen
```
PS C:\>Get-AzSubscription

Name                               Id                      TenantId                        State
----                               --                      --------                        -----
Subscription1                      yyyy-yyyy-yyyy-yyyy     aaaa-aaaa-aaaa-aaaa             Enabled
Subscription2                      xxxx-xxxx-xxxx-xxxx     aaaa-aaaa-aaaa-aaaa             Enabled
Subscription3                      zzzz-zzzz-zzzz-zzzz     bbbb-bbbb-bbbb-bbbb             Enabled
```

Med det här kommandot får du alla prenumerationer i alla klientorganisationen som har behörighet för det aktuella kontot.

### Exempel 2: Hämta alla prenumerationer för en viss klientorganisation
```
PS C:\>Get-AzSubscription -TenantId "aaaa-aaaa-aaaa-aaaa"

Name                               Id                      TenantId                        State
----                               --                      --------                        -----
Subscription1                      yyyy-yyyy-yyyy-yyyy     aaaa-aaaa-aaaa-aaaa             Enabled
Subscription2                      xxxx-xxxx-xxxx-xxxx     aaaa-aaaa-aaaa-aaaa             Enabled
```

Lista alla prenumerationer i den angivna klientorganisationen som har godkänts för det aktuella kontot.

### Exempel 3: Hämta alla prenumerationer i den aktuella klientorganisationen
```
PS C:\>Get-AzSubscription

Name                               Id                      TenantId                        State
----                               --                      --------                        -----
Subscription1                      yyyy-yyyy-yyyy-yyyy     aaaa-aaaa-aaaa-aaaa             Enabled
Subscription2                      xxxx-xxxx-xxxx-xxxx     aaaa-aaaa-aaaa-aaaa             Enabled
```

Det här kommandot hämtar alla prenumerationer i den aktuella klientorganisationen som har godkänts för den aktuella användaren.

### Exempel 4: Ändra det aktuella sammanhanget för att använda en viss prenumeration
```
PS C:\>Get-AzSubscription -SubscriptionId "xxxx-xxxx-xxxx-xxxx" -TenantId "yyyy-yyyy-yyyy-yyyy" | Set-AzContext

Name                                     Account             SubscriptionName    Environment         TenantId
----                                     -------             ----------------    -----------         --------
Subscription1 (xxxx-xxxx-xxxx-xxxx)      azureuser@micros... Subscription1       AzureCloud          yyyy-yyyy-yyyy-yyyy
```

Det här kommandot får den angivna prenumerationen och anger sedan det aktuella sammanhanget till att använda det. Alla efterföljande cmdlets i den här sessionen använder den nya prenumerationen (Contoso Prenumeration 1) som standard.
