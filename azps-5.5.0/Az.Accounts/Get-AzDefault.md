# Get-AzDefault

## SYNOPSIS
Ange standardinställningarna för användaren i det aktuella sammanhanget.

## SYNTAX

```
Get-AzDefault [-ResourceGroup] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den Get-AzDefault cmdleten hämtar resursgruppen som användaren har angett som standard i det aktuella sammanhanget.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzDefault

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Med det här kommandot returneras de aktuella standardinställningarna om det finns standardinställningar eller inget returneras om inget standardvärde har angetts.

### Exempel 2
```
PS C:\> Get-AzDefault -ResourceGroup

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Med det här kommandot returneras den aktuella standardresursgruppen om det finns en standarduppsättning eller inget returneras om ingen standardinställning används.
