# Get-AzManagedApplicationDefinition

## SYNOPSIS
Får definitioner av hanterade program

## SYNTAX

### GetByNameAndResourceGroup (standard)
```
Get-AzManagedApplicationDefinition [-Name <String>] -ResourceGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzManagedApplicationDefinition -Id <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzManagedApplicationDefinition** får hanterade programdefinitioner

## EXEMPEL

### Exempel 1: Få alla definitioner av hanterade program under en resursgrupp
```
PS C:\>Get-AzManagedApplicationDefinition -ResourceGroupName "MyRG"
```

Det här kommandot får definitionerna för det hanterade programmet under resursgruppen "MyRG"

### Exempel 2: Hämta en hanterad programdefinition
```
PS C:\>Get-AzManagedApplicationDefinition -ResourceGroupName "MyRG" -Name "myManagedAppDef"
```

Det här kommandot får den hanterade programdefinitionen "myManagedAppDef" under resursgruppen "MyRG"
