## BESKRIVNING
Cmdleten **Set-AzPolicyDefinition** ändrar en principdefinition.

## EXEMPEL

### Exempel 1: Uppdatera beskrivningen av en principdefinition
```
PS C:\> $PolicyDefinition = Get-AzPolicyDefinition -Name 'VMPolicyDefinition'
PS C:\> Set-AzPolicyDefinition -Id $PolicyDefinition.ResourceId -Description 'Updated policy to not allow virtual machine creation'
```

Det första kommandot får en principdefinition som heter VMPolicyDefinition med hjälp av Get-AzPolicyDefinition-cmdleten.
Kommandot lagrar objektet i den $PolicyDefinition variabeln.
Det andra kommandot uppdaterar beskrivningen av principdefinitionen som identifieras av **egenskapen ResourceId** för $PolicyDefinition.

### Exempel 2: Uppdatera läge för en principdefinition
```
PS C:\> Set-AzPolicyDefinition -Name 'VMPolicyDefinition' -Mode 'All'
```

Det här kommandot uppdaterar principdefinitionen med namnet VMPolicyDefinition med hjälp av cmdleten Set-AzPolicyDefinition ange dess lägesegenskap till "Alla".

### Exempel 3: Uppdatera metadata för en principdefinition
```
PS C:\> Set-AzPolicyDefinition -Name 'VMPolicyDefinition' -Metadata '{"category":"Virtual Machine"}'
```
