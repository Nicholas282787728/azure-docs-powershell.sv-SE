# Remove-AzADAppCredential

## SYNOPSIS
Tar bort en autentiseringsuppgifter från ett program.

## SYNTAX

### ApplicationObjectIdWithKeyIdParameterSet (standard)
```
Remove-AzADAppCredential -ObjectId <String> [-KeyId <Guid>] [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ApplicationIdWithKeyIdParameterSet
```
Remove-AzADAppCredential -ApplicationId <Guid> [-KeyId <Guid>] [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ApplicationDisplayNameParameterSet
```
Remove-AzADAppCredential -DisplayName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ApplicationObjectWithKeyIdParameterSet
```
Remove-AzADAppCredential [-KeyId <Guid>] -ApplicationObject <PSADApplication> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzADAppCredential-cmdleten kan du ta bort en autentiseringsnyckel från ett program vid en kompromett eller som en del av förfallotiden för autentiseringsnyckelns återställning.
Programmet identifieras genom att ange objekt-ID eller AppId.
De autentiseringsuppgifter som ska tas bort identifieras med dess nyckel-ID.

## EXEMPEL

### Exempel 1: Ta bort en specifik autentisering från ett program

```powershell
PS C:\> Remove-AzADAppCredential -ObjectId 7663d3fb-6f86-4352-9e6d-cf9d50d5ee82 -KeyId 9044423a-60a3-45ac-9ab1-09534157ebb
```

Tar bort autentiseringsnyckeln med nyckel-ID '9044423a-60a3-45ac-9ab1-09534157ebb' från programmet med objekt-ID '7663d3fb-6f86-4352-9e6d-cf9d50d5ee82'.

### Exempel 2: Ta bort alla autentiseringsuppgifter från ett program

```powershell
PS C:\> Remove-AzADAppCredential -ApplicationId 4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58
```

Tar bort alla autentiseringsuppgifter från programmet med program-ID '4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58'.

### Exempel 3: Ta bort alla autentiseringsuppgifter med rörledning

```powershell
PS C:\> Get-AzADApplication -ObjectId 7663d3fb-6f86-4352-9e6d-cf9d50d5ee82 | Remove-AzADAppCredential
```

Hämtar programmet med objekt-ID '7663d3fb-6f86-4352-9e6d-cf9d50d5ee82' och rör som till Remove-AzADAppCredential-cmdleten och tar bort alla autentiseringsuppgifter från programmet.
