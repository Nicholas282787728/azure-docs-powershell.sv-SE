# Update-AzADUser

## SYNOPSIS
Uppdaterar en befintlig Active Directory-användare.

## SYNTAX

### UPNOrObjectIdParameterSet (standard)
```
Update-AzADUser -UPNOrObjectId <String> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UPNParameterSet
```
Update-AzADUser -UserPrincipalName <String> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectIdParameterSet
```
Update-AzADUser -ObjectId <String> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Update-AzADUser -InputObject <PSADUser> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Uppdaterar en befintlig Active Directory-användare (arbets-/skolkonto, som också är känt som organisations-id).
Mer information finns i: https://msdn.microsoft.com/en-us/library/azure/ad/graph/api/users-operations#UpdateUser

## EXEMPEL

### Exempel 1: Uppdatera visningsnamnet för en användare med hjälp av objekt-ID

```powershell
PS C:\> Update-AzADUser -ObjectId 155a5c10-93a9-4941-a0df-96d83ab5ab24 -DisplayName MyNewDisplayName
```

Uppdaterar användarens visningsnamn med objekt-ID '155a5c10-93a9-4941-a0df-96d83ab5ab24' till 'MyNewDisplayName'.

### Exempel 2: Uppdatera visningsnamnet för en användare med hjälp av huvudnamnet

```powershell
PS C:\> Update-AzADUser -UserPrincipalName foo@domain.com -DisplayName MyNewDisplayName
```

Uppdaterar visningsnamnet för användaren med huvudnamnet foo@domain.com ' till 'MyNewDisplayName'.

### Exempel 3: Uppdatera användarens visningsnamn med hjälp av rörledning

```powershell
PS C:\> Get-AzADUser -ObjectId 155a5c10-93a9-4941-a0df-96d83ab5ab24 | Update-AzADUser -DisplayName MyNewDisplayName
```

Hämtar användaren med objekt-ID '155a5c10-93a9-4941-a0df-96d83ab5ab24' och rör som till Update-AzADUser-cmdleten för att uppdatera användarens visningsnamn till "MyNewDisplayName".
