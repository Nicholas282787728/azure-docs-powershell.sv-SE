# Get-AzLocation

## SYNOPSIS
Hämtar alla platser och de resursleverantörer som stöds för varje plats.

## SYNTAX

```
Get-AzLocation [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzLocation** hämtar alla platser och de resursleverantörer som stöds för varje plats.

## EXEMPEL

### Exempel 1: Hämta alla platser och de resursleverantörer som stöds
```
PS C:\>Get-AzLocation
```

Det här kommandot hämtar alla platser och de resursleverantörer som stöds för varje plats.
