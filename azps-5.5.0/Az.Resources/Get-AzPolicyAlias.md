# Get-AzPolicyAlias

## SYNOPSIS
Get-AzPolicyAlias hämtar och matar ut Azure-leverantörsresurstyper som har alias definierade och matchar de givna parametervärdena. Om inga parametrar anges kommer alla leverantörsresurstyper som innehåller ett alias att matas ut.
Växeln -ListAvailable ändrar beteendet genom att visa alla matchande resurstyper, även de utan alias.

## SYNTAX

```
Get-AzPolicyAlias [-NamespaceMatch <String>] [-ResourceTypeMatch <String>] [-AliasMatch <String>]
 [-PathMatch <String>] [-ApiVersionMatch <String>] [-LocationMatch <String>] [-ListAvailable]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzPolicyAlias** får en lista med principalias.
Principalias används av Azure Policy för att referera till egenskaper för resurstyper.
Parametrar tillhandahålls som begränsar elementen i listan genom att matcha olika egenskaper för resurstypen eller dess alias.
Ett givet matchningsvärde matchar om målsträngen innehåller den med en okänslig jämförelse.
