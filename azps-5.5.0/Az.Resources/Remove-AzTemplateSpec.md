# Remove-AzTemplateSpec

## SYNOPSIS
Tar bort en mallspecifik

## SYNTAX

### RemoveByNameParameterSet (standard)
```
Remove-AzTemplateSpec [-Force] [-ResourceGroupName] <String> [-Name] <String> [[-Version] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByIdParameterSet
```
Remove-AzTemplateSpec [-Force] [[-Version] <String>] [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Tar bort en angiven mallspecifikt. Om **parametern** version –version tillhandahålls tas bara den angivna versionen bort. Om ingen specifik version tillhandahålls tas mallspecifikt och alla dess versioner bort. Om **-Force-flaggan** finns uppmanas du inte att bekräfta borttagningen.

## EXEMPEL

### Exempel 1: Ta bort en viss version med namn
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'MyTemplateSpec' -Version 'v1.0'
```

Tar bort version "v1.0" av mallspecifikt "MyTemplateSpec" i resursgruppen "myRG".

### Exempel 2: Ta bort en viss version med resurs-ID
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceId '/subscriptions/{subId}/resourceGroups/myRG/providers/Microsoft.Resources/templateSpecs/MyTemplateSpec' -Version 'v1.0'
```

Tar bort version "v1.0" av mallspecifikationen med namnet MyTemplateSpec inom resursgruppen "myRG" av \{ prenumerationsunder-ID. \}

### Exempel 3: Ta bort en mallspecifikt och alla versioner med namn
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'MyTemplateSpec'
```

Tar bort mallspecifikt "MyTemplateSpec" och alla dess versioner inom resursgruppen "myRG".

### Exempel 3: Ta bort en mallspecifikt och alla versioner med resurs-ID
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceId '/subscriptions/{subId}/resourceGroups/myRG/providers/Microsoft.Resources/templateSpecs/MyTemplateSpec' -ResourceGroupName 'myRG'
```

Tar bort mallspecifikationen med namnet MyTemplateSpec och alla dess versioner inom resursgruppen "myRG" av \{ prenumerationsunderid. \}
