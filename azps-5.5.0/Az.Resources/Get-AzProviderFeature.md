# Get-AzProviderFeature

## SYNOPSIS
Hämtar information om Azure-leverantörsfunktioner.

## SYNTAX

### ListAvailableParameterSet (standard)
```
Get-AzProviderFeature [-ProviderNamespace <String>] [-ListAvailable] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetFeature
```
Get-AzProviderFeature -ProviderNamespace <String> -FeatureName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzProviderFeature** får funktionsnamn, leverantörsnamn och registreringsstatus för Azure-providerfunktioner.

## EXEMPEL

### Exempel 1: Skaffa alla tillgängliga funktioner
```
PS C:\>Get-AzProviderFeature -ListAvailable
```

Med det här kommandot får du alla tillgängliga funktioner.

### Exempel 2: Få information om en specifik funktion
```
PS C:\>Get-AzProviderFeature -FeatureName "AllowPreReleaseRegions" -ProviderNamespace "Microsoft.Compute"
```

Det här kommandot hämtar information för funktionen AllowPreReleaseRegions för den angivna leverantören.
