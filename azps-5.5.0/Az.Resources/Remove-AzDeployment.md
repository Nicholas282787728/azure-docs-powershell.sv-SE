# Remove-AzDeployment

## SYNOPSIS
Tar bort en distribution och alla tillhörande åtgärder

## SYNTAX

### RemoveByDeploymentName (standard)
```
Remove-AzDeployment [-Name] <String> [-AsJob] [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByDeploymentId
```
Remove-AzDeployment -Id <String> [-AsJob] [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObject
```
Remove-AzDeployment -InputObject <PSDeployment> [-AsJob] [-PassThru] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzDeployment** tar bort en Azure-distribution med prenumerationens omfattning och alla associerade åtgärder.

## EXEMPEL

### Exempel 1: Ta bort en distribution med ett visst namn
```
PS C:\>Remove-AzDeployment -Name "RolesDeployment"
```

Det här kommandot tar bort distributionen "RolesDeployment" i den aktuella prenumerationsomfånget.

### Exempel 2: Hämta en distribution och ta bort den
```
PS C:\>Get-AzDeployment -Name "RolesDeployment" | Remove-AzDeployment
```

Med det här kommandot får distributionen "RolesDeployment" den aktuella prenumerationen och tar bort den.
