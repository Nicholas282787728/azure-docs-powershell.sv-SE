## BESKRIVNING
Ändrar en Templace-specifikation. Om mallspecifikt med det angivna namnet och/eller den specifika versionen inte redan finns, skapas den.

När du ändrar innehållet i en mallspecifikitetsversion av ARM-mallinnehållet kan innehållet antingen komma från en obearbetad JSON-sträng (med **updateVersionByNameFromJsonParameterSet-parameteruppsättning)** eller från en specifik JSON-fil (med parameteruppsättningen **UpdateVersionByNameFromJsonFileParameterSet).**

## EXEMPEL

### Exempel 1:
```powershell
PS C:\> $templateJson = @"
{
    "$schema": "http://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
    "contentVersion": "1.0.0.0",
    "parameters": {},
    "resources": []
}
"@
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v1.0' -Location 'West US' -TemplateJson $templateJson
```

Ändrar version "v1.0" av en mallspecifikt som heter "myTemplateSpec". Den angivna versionen måste $templateJson som versionens ARM mallinnehåll. Om rotmallsspecifiken och/eller versionen inte redan finns kommer de att skapas.


**Kommentarer:** 

* Mallen ARM i exemplet är en icke-operation eftersom den inte innehåller några faktiska resurser.
* Plats krävs endast när mallspecifikt inte redan finns

### Exempel 2:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v2.0' -Location 'West US' -TemplateFile 'myTemplateContent.json'
```

Ändrar version "v2.0" av en mallspecifikt som heter "myTemplateSpec". Den angivna versionen kommer att ha innehållet från den lokala filen "myTemplateContent.json" som versionens ARM mallinnehåll. Om rotmallsspecifiken och/eller versionen inte redan finns kommer de att skapas.

**Obs!** Plats krävs endast när mallspecifikt inte redan finns

### Exempel 3:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec'  -Location 'West US' -Description 'My updated Template Spec'
```

Ändrar beskrivningen av mallspecifikt "myTemplateSpec" i resursgruppen "myRG". Om mallspecifikt inte redan finns skapas det.

**Obs!** Plats krävs endast när mallspecifikt inte redan finns
