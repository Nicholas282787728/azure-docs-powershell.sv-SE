# Register-AzProviderFeature

## SYNOPSIS
Registrerar en Azure-leverantörsfunktion i ditt konto.

## SYNTAX

```
Register-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Register-AzProviderFeature** registrerar en Azure-providerfunktion i ditt konto.

## EXEMPEL

### Exempel 1: Registrera en funktion
```
PS C:\>Register-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Då läggs funktionen AllowApplicationSecurityGroups för Microsoft.Network till i ditt konto.
