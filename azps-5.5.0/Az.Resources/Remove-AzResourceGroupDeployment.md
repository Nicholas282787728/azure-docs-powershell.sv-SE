# Remove-AzResourceGroupDeployment

## SYNOPSIS
Tar bort en resursgruppsdistribution och alla associerade åtgärder.

## SYNTAX

### RemoveByResourceGroupName (standard)
```
Remove-AzResourceGroupDeployment [-ResourceGroupName] <String> [-Name] <String> [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupDeploymentId
```
Remove-AzResourceGroupDeployment -Id <String> [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING

Cmdleten **Remove-AzResourceGroupDeployment** tar bort en Azure-resursgruppdistribution och alla associerade åtgärder.

## EXEMPEL

### Exempel 1: Tar bort en resursgruppsdistribution med ResourceId

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceId /subscriptions/{subId}/resourceGroups/testGroup/providers/Microsoft.Resources/deployments/testDeployment1

True
```

Med det här kommandot tas en resursgruppsdistribution bort med distributionens fullständigt kvalificerade resurs-ID.
Om borttagningen lyckas returneras Sant.

### Exempel 2: Tar bort en resursgruppsdistribution med ResourceGroupName och ResourceName

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceGroupName testGroup -Name testDeployment1

True
```

Med det här kommandot tas en resursgruppsdistribution bort med det angivna ResourceGroupName och ResourceName.
Om borttagningen lyckas returneras sant.
