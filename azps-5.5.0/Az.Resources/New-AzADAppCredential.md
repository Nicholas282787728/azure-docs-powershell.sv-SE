# New-AzADAppCredential

## SYNOPSIS
Lägger till en autentiseringsuppgifter i ett befintligt program.

## SYNTAX

### ApplicationObjectIdWithPasswordParameterSet (Standard)
```
New-AzADAppCredential -ObjectId <String> -Password <SecureString> [-StartDate <DateTime>] [-EndDate <DateTime>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den New-AzADAppCredential cmdleten kan användas för att lägga till nya autentiseringsuppgifter eller för att samla autentiseringsuppgifter för ett program.
Programmet identifieras genom att ange antingen programmets objekt-ID eller program-ID.

## EXEMPEL

### Exempel 1 – Skapa ett nytt program med hjälp av ett lösenord

```
PS C:\> $SecureStringPassword = ConvertTo-SecureString -String "password" -AsPlainText -Force
PS C:\> New-AzADAppCredential -ObjectId 1f89cf81-0146-4f4e-beae-2007d0668416 -Password $SecureStringPassword
```

Ett nytt lösenord läggs till i det befintliga programmet med objekt-ID '1f89cf81-0146-4f4e-beae-2007d0668416'.

### Exempel 2 - Skapa ett nytt program med hjälp av ett certifikat

```
PS C:\> $cer = New-Object System.Security.Cryptography.X509Certificates.X509Certificate2
PS C:\> $cer.Import("C:\myapp.cer")
PS C:\> $binCert = $cer.GetRawCertData()
PS C:\> $credValue = [System.Convert]::ToBase64String($binCert)
PS C:\> New-AzADAppCredential -ApplicationId 4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58 -CertValue $credValue -StartDate $cer.NotBefore -EndDate $cer.NotAfter
```

Det medföljande base64-kodade offentliga X509-certifikatet ("myapp.cer") läggs till i det befintliga programmet med program-ID '4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58'.

### Exempel 3 – Skapa en ny program credential med rörledning

```
PS C:\> $SecureStringPassword = ConvertTo-SecureString -String "password" -AsPlainText -Force
PS C:\> Get-AzADApplication -ObjectId 1f89cf81-0146-4f4e-beae-2007d0668416 | New-AzADAppCredential -Password $SecureStringPassword
```

Hämtar programmet med objekt-ID '1f89cf81-0146-4f4e-beae-2007d0668416' och rör som till New-AzADAppCredential för att skapa ett nytt program med det angivna lösenordet.
