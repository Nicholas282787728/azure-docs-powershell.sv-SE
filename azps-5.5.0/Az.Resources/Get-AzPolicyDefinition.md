# Get-AzPolicyDefinition

## SYNOPSIS
Hämtar principdefinitioner.

## SYNTAX

### NameParameterSet (standard)
```
Get-AzPolicyDefinition [-Name <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ManagementGroupNameParameterSet
```
Get-AzPolicyDefinition [-Name <String>] -ManagementGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionIdParameterSet
```
Get-AzPolicyDefinition [-Name <String>] -SubscriptionId <Guid> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdParameterSet
```
Get-AzPolicyDefinition -Id <String> [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### BuiltinFilterParameterSet
```
Get-AzPolicyDefinition [-ManagementGroupName <String>] [-SubscriptionId <Guid>] [-Builtin]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### CustomFilterParameterSet
```
Get-AzPolicyDefinition [-ManagementGroupName <String>] [-SubscriptionId <Guid>] [-Custom]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzPolicyDefinition** får en samling principdefinitioner eller en viss principdefinition som identifieras med namn eller ID.

## EXEMPEL

### Exempel 1: Hämta alla principdefinitioner
```
PS C:\> Get-AzPolicyDefinition
```

Med det här kommandot får du alla principdefinitioner.

### Exempel 2: Hämta policydefinition från aktuell prenumeration med namn
```
PS C:\> Get-AzPolicyDefinition -Name 'VMPolicyDefinition'
```

Det här kommandot hämtar principdefinitionen med namnet VMPolicyDefinition från den aktuella standardprenumerationen.

### Exempel 3: Hämta principdefinitionen från hanteringsgruppen efter namn
```
PS C:\> Get-AzPolicyDefinition -Name 'VMPolicyDefinition' -ManagementGroupName 'Dept42'
```

Det här kommandot hämtar principdefinitionen med namnet VMPolicyDefinition från hanteringsgruppen Avd42.

### Exempel 4: Hämta alla inbyggda principdefinitioner från prenumerationen
```
PS C:\> Get-AzPolicyDefinition -SubscriptionId '3bf44b72-c631-427a-b8c8-53e2595398ca' -Builtin
```

Med det här kommandot får du alla inbyggda principdefinitioner från prenumerationen med ID 3bf44b72-c631-427a-b8c8-53e2595398ca.

### Exempel 5: Hämta principdefinitioner från en viss kategori
```
PS C:\> Get-AzPolicyDefinition | where-object {$_.Properties.metadata.category -eq "Virtual Machine"}
```

Det här kommandot får alla principdefinitioner i kategorin "Virtuell dator".
