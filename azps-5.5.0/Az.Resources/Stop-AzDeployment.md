# Stop-AzDeployment

## SYNOPSIS
Avbryta en distribution som körs

## SYNTAX

### StopByDeploymentName (standard)
```
Stop-AzDeployment [-Name] <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzDeployment -Id <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzDeployment -InputObject <PSDeployment> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Stop-AzDeployment** avbryter en distribution med en prenumerationsomfång som har startats men inte slutförts.
Om du vill stoppa en distribution måste distributionen ha en ofullständig etableringstillstånd, till exempel Etablering, och inte en slutförd status, till exempel Etablering eller Misslyckades.

Använd cmdleten New-AzDeployment för att skapa en distribution New-AzDeployment prenumeration.

Den här cmdleten stoppar endast en distribution som körs. Använd *parametern Name* för att stoppa en viss distribution.

## EXEMPEL

### Exempel 1
```
PS C:\>Stop-AzDeployment -Name "deployment01"
```

Det här kommandot avbryter en pågående distribution av "deployment01" i den aktuella prenumerationen.

### Exempel 2
```
PS C:\>Get-AzDeployment -Name "deployment01" | Stop-AzDeployment
```

Det här kommandot får distributionen "deployment01" i den aktuella prenumerationsomfattningen och avbryter den.
