# Remove-AzManagedApplicationDefinition

## SYNOPSIS
Tar bort en definition för hanterade program

## SYNTAX

### RemoveByNameAndResourceGroup (standard)
```
Remove-AzManagedApplicationDefinition -Name <String> -ResourceGroupName <String> [-Force]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### RemoveById
```
Remove-AzManagedApplicationDefinition -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzManagedApplicationDefinition** tar bort en hanterad programdefinition

## EXEMPEL

### Exempel 1: Ta bort den hanterade programdefinitionen efter resurs-ID
```
PS C:\>$ApplicationDefinition = Get-AzManagedApplicationDefinition -Name "myAppDef" -ResourceGroupName "myRG"
PS C:\>Remove-AzManagedApplicationDefinition -Id $ApplicationDefinition.ResourceId -Force
```

Det första kommandot får en hanterad programdefinition med namnet myAppDef med hjälp av cmdleten Get-AzManagedApplicationDefinition.
Kommandot lagrar det i den $ApplicationDefinition variabeln.
Det andra kommandot tar bort den hanterade programdefinitionen som identifieras av **egenskapen ResourceId** för $ApplicationDefinition.
