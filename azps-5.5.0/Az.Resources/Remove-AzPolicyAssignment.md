# Remove-AzPolicyAssignment

## SYNOPSIS
Tar bort en principtilldelning.

## SYNTAX

### NameParameterSet (standard)
```
Remove-AzPolicyAssignment -Name <String> [-Scope <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSet
```
Remove-AzPolicyAssignment -Id <String> [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzPolicyAssignment -InputObject <PsPolicyAssignment> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzPolicyAssignment** tar bort den angivna principtilldelningen.

## EXEMPEL

### Exempel 1: Ta bort principtilldelning efter namn och omfattning
```
PS C:\> $ResourceGroup = Get-AzResourceGroup -Name 'ResourceGroup11'
PS C:\> Remove-AzPolicyAssignment -Name 'PolicyAssignment07' -Scope $ResourceGroup.ResourceId -Confirm
```

Det första kommandot får en resursgrupp med namnet ResourceGroup11 med hjälp Get-AzResourceGroup cmdleten.
Kommandot lagrar objektet i den $ResourceGroup variabeln.
Det andra kommandot tar bort den principtilldelning med namnet PolicyAssignment07 som har tilldelats på en resursgruppnivå.
Egenskapen **ResourceId** för $ResourceGroup identifierar resursgruppen.

### Exempel 2: Ta bort principtilldelning med ID
```
PS C:\> $ResourceGroup = Get-AzResourceGroup -Name 'ResourceGroup11' 
PS C:\> $PolicyAssignment = Get-AzPolicyAssignment -Name 'PolicyAssignment07' -Scope $ResourceGroup.ResourceId
PS C:\> Remove-AzPolicyAssignment -Id $PolicyAssignment.ResourceId -Confirm
```

Det första kommandot får en resursgrupp med namnet ResourceGroup11 och lagrar sedan objektet i den $ResourceGroup variabeln.
Det andra kommandot hämtar principtilldelningen på en resursgruppnivå och lagrar den sedan $PolicyAssignment variabeln.
Egenskapen **ResourceId** för $ResourceGroup identifierar resursgruppen.
Det slutliga kommandot tar bort den principtilldelning som **Egenskapen ResourceId** $PolicyAssignment identifierar.
