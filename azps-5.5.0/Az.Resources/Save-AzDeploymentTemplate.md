# Save-AzDeploymentTemplate

## SYNOPSIS
Sparar en distributionsmall i en fil.

## SYNTAX

### SaveByDeploymentName (standard)
```
Save-AzDeploymentTemplate -DeploymentName <String> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SaveByDeploymentObject
```
Save-AzDeploymentTemplate -DeploymentObject <PSDeployment> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med **cmdleten Save-AzDeploymentTemplate**  sparas en distributionsmall som en JSON-fil.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Save-AzDeploymentTemplate -DeploymentName "TestDeployment"
```

Det här kommandot hämtar distributionsmallen från TestDeployment och sparar den som en JSON-fil i den aktuella katalogen.

### Exempel 2: Hämta en distribution och spara dess mall
```
PS C:\>Get-AzDeployment -Name "RolesDeployment" | Save-AzDeploymentTemplate
```

Med det här kommandot får distributionen "RolesDeployment" den aktuella prenumerationsomfånget och sparar mallen.
