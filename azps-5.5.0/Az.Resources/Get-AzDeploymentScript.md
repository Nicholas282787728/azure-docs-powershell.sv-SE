# Get-AzDeploymentScript

## SYNOPSIS
Hämtar eller visar distributionsskript.

## SYNTAX

### ListDeploymentScript (standard)
```
Get-AzDeploymentScript [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetDeploymentScriptByName
```
Get-AzDeploymentScript [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetDeploymentScriptByResourceId
```
Get-AzDeploymentScript [-Id] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzDeploymentScript** får ett enda distributionsskript eller en lista med distributionsskript.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzDeploymentScript
```

Visar distributionsskript i prenumerationen i användarens sammanhang.

### Exempel 2
```powershell
PS C:\> Get-AzDeploymentScript -ResourceGroupName DS-TestRg
```

Visar distributionsskript i DS-TestRg för resursgrupp.

### Exempel 3
```powershell
PS C:\> Get-AzDeploymentScript -Name MyDeploymentScript -ResourceGroupName DS-TestRg
```

Hämtar ett distributionsskript med namnet MyDeploymentScript i resursgruppen DS-TestRG.

### Exempel 4
```powershell
PS C:\> Get-AzDeploymentScript -Id "/subscriptions/{subId}/resourceGroups/{rgName}/providers/Microsoft.Resources/deploymentScripts/{deploymentScriptName}"
```

Hämtar ett distributionsskript med det angivna resurs-ID:t.
