# Az.Advisor Module
## Beskrivning
Kommandon för att interagera med Azure Advisor.

## Az.Advisor-cmdlets
### [Get-AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
Får en lista med rekommendationer om Azure-rådgivare.

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
Inaktivera en Rekommendation för Azure Advisor.

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
Aktiverar rekommendationerna för Azure Advisor.

### [Get-AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
Skaffa Azure Advisor-konfigurationer för den givna prenumerationen eller resursgruppen.

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
Uppdaterar eller skapar Azure Advisor-konfigurationen.
