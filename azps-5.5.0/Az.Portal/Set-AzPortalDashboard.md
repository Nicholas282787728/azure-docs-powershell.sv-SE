# Set-AzPortalDashboard

## SYNOPSIS
Skapar eller uppdaterar en instrumentpanel.

## SYNTAX

```
Set-AzPortalDashboard -Name <String> -ResourceGroupName <String> -DashboardPath <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Skapar eller uppdaterar en instrumentpanel.

## EXEMPEL

### Exempel 1: Uppdatera instrumentpanelsdefinitionen med hjälp av en instrumentpanelsmall
```powershell
PS C:\> Set-AzPortalDashboard -DashboardPath .\resources\dash1-update.json -ResourceGroupName my-rg -DashboardName dashbase03

Location Name       Type
-------- ----       ----
eastasia dashbase03 Microsoft.Portal/dashboards
```

Uppdatera en instrumentpanelsdefinition med hjälp av en bindestrecksmallfil.
