# New-AzRedisEnterpriseCacheDatabaseKey

## SYNOPSIS
Återskapar snabbnycklarna för RedisEnterprise-databasen.

## SYNTAX

```
New-AzRedisEnterpriseCacheDatabaseKey -ClusterName <String> -ResourceGroupName <String>
 -KeyType <AccessKeyType> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Återskapar snabbnycklarna för RedisEnterprise-databasen.

## EXEMPEL

### Exempel 1: Generera primäråtkomstnyckel
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Primary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
new-primary-key                              secondary-key

```

Det här kommandot återskapar den primära hemliga åtkomstnyckeln som används för att autentisera anslutningar till databasen av Redis Enterprise Cache med namnet MyCache.

### Exempel 2: Skapa sekundär åtkomstnyckel
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Secondary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
primary-key                                  new-secondary-key

```

Det här kommandot återskapar den sekundära hemliga åtkomstnyckeln som används för att autentisera anslutningar till databasen av Redis Enterprise Cache med namnet MyCache.
