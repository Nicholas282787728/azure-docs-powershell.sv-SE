# New-AzCustomProvider

## SYNOPSIS
Skapar eller uppdaterar den anpassade resursleverantören.

## SYNTAX

```
New-AzCustomProvider -Name <String> -ResourceGroupName <String> -Location <String> [-SubscriptionId <String>]
 [-Action <ICustomRpActionRouteDefinition[]>] [-ResourceType <ICustomRpResourceTypeRouteDefinition[]>]
 [-Tag <Hashtable>] [-Validation <ICustomRpValidations[]>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Skapar eller uppdaterar den anpassade resursleverantören.

## EXEMPEL

### Exempel 1: Skapa en anpassad leverantör
```powershell
PS C:\> New-AzCustomProvider -ResourceGroupName myRG -Name Namespace.Type -Location "West US 2" -ResourceType @{Name="CustomRoute1"; Endpoint="https://www.contoso.com/"}


Location  Name             Type
--------  ----             ----
West US 2 Namespace.Type   Microsoft.CustomProviders/resourceproviders
```

Skapa en anpassad resursleverantör

### Exempel 2: Skapa en anpassad leverantör med associationer
```powershell
PS C:\> New-AzCustomProvider -ResourceGroupName myRG -Name Namespace2.Type -Location "West US 2" -ResourceType @{Name="CustomRoute1"; Endpoint="https://www.contoso.com/"}, @{Name="Associations"; Endpoint="https://contoso.com/myService", RoutingType="Proxy,Cache,Extension"}

Location  Name             Type
--------  ----             ----
West US 2 Namespace2.Type   Microsoft.CustomProviders/resourceproviders
```

Skapa en anpassad leverantör med en rutt för anpassade leverantörsassociationer.

## ANTECKNINGAR

ALIAS

KOMPLEXA PARAMETEREGENSKAPER

Skapa de parametrar som beskrivs nedan genom att skapa en hash-tabell som innehåller rätt egenskaper. Om du vill ha information om hash-tabeller kör du Get-Help about_Hash_Tables.


ACTION <ICustomRpActionRouteDefinition[]>: En lista med åtgärder som har implementeras av den anpassade resursleverantören.
  - `Endpoint <String>`: Slutpunkts-URI för routedefinitionen som den anpassade resursleverantören proxybegäranden till. Det kan vara i form av en platt URI (t.ex. ' ') eller kan specificera att dirigeras via en sökväg https://testendpoint/ (t.ex. ' https://testendpoint/{requestPath} ')
  - `Name <String>`: Namnet på routedefinitionen. Det här blir namnet på ARM-tillägget (t.ex. '/subscriptions/{subscriptionId}/resourceGroups/{resourceGroupName}/providers/Microsoft.CustomProviders/resourceProviders/{resourceProviderName}/{name}')
  - `[RoutingType <ActionRouting?>]`: Routningstyper som stöds för åtgärdsförfrågningar.

RESOURCETYPE <ICustomRpResourceTypeRouteDefinition[]>: En lista över resurstyper som den anpassade resursleverantören implementerar.
  - `Endpoint <String>`: Slutpunkts-URI för routedefinitionen som den anpassade resursleverantören proxybegäranden till. Det kan vara i form av en platt URI (t.ex. ' ') eller kan specificera att dirigeras via en sökväg https://testendpoint/ (t.ex. ' https://testendpoint/{requestPath} ')
  - `Name <String>`: Namnet på routedefinitionen. Det här blir namnet på ARM-tillägget (t.ex. '/subscriptions/{subscriptionId}/resourceGroups/{resourceGroupName}/providers/Microsoft.CustomProviders/resourceProviders/{resourceProviderName}/{name}')
  - `[RoutingType <ResourceTypeRouting?>]`: Routningstyper som stöds för resursförfrågningar.

VALIDATION <ICustomRpValidations[]>: En lista med verifieringar som ska köras på begäran från den anpassade resursleverantören.
  - `Specification <String>`: En länk till verifieringsspecifikationen. Specifikationen måste vara värd för raw.githubusercontent.com.
  - `[ValidationType <ValidationType?>]`: Den typ av verifiering som ska köras mot en matchande begäran.
