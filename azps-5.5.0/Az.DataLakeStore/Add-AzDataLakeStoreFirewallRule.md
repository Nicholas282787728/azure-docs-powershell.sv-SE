# Add-AzDataLakeStoreFirewallRule

## SYNOPSIS
Lägger till en brandväggsregel för det angivna Data Lake Store-kontot.

## SYNTAX

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Add-AzDataLakeStoreFirewallRule** lägger till en brandväggsregel i det angivna Data Lake Store-kontot.

## EXEMPEL

### Exempel 1: Lägga till en ny brandväggsregel för ett Data Lake Store-konto
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Då skapas en ny brandväggsregel som kallas "MyRule" i kontot "ContosoADL" med ett IP-intervall på 127.0.0.1 – 127.0.0.2
