# Remove-AzDataLakeStoreItemAcl

## SYNOPSIS
Tar bort ACL för en fil eller mapp i Data Lake Store.

## SYNTAX

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzDataLakeStoreItemAcl** rensar åtkomstkontrollista (ACL) för en fil eller mapp i Data Lake Store.

## EXEMPEL

### Exempel 1: Ta bort ACL från en mapp
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Det här kommandot tar bort ACL för rotkatalogen för ContosoADL-kontot.
