# Test-AzDataLakeStoreItem

## SYNOPSIS
Testar att det finns en fil eller mapp i Data Lake Store.

## SYNTAX

```
Test-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-PathType] <PathType>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Test-AzDataLakeStoreItem** testar om det finns en fil eller mapp i Data Lake Store.

## EXEMPEL

### Exempel 1: Testa en fil
```powershell
PS C:\>Test-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

Det här kommandot testar om Test.csv finns i ContosoADL-kontot.

### Exempel 2

Testar att det finns en fil eller mapp i Data Lake Store. (autogenererat)

<!-- Aladdin Generated Example -->
```powershell
Test-AzDataLakeStoreItem -Account 'ContosoADL' -Path '/MyFiles/Test.csv' -PathType Any
```
