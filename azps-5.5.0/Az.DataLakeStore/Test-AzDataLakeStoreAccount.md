# Test-AzDataLakeStoreAccount

## SYNOPSIS
Testar om det finns ett Data Lake Store-konto.

## SYNTAX

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Test-AzDataLakeStoreAccount** testar om det finns ett Data Lake Store-konto.

## EXEMPEL

### Exempel 1: Testa ett konto
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Det här kommandot testar om kontot ContosoADL finns.
