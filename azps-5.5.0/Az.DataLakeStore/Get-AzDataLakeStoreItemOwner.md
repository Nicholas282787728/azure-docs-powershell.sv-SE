# Get-AzDataLakeStoreItemOwner

## SYNOPSIS
Hämtar ägaren till en fil eller mapp i Data Lake Store.

## SYNTAX

```
Get-AzDataLakeStoreItemOwner [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Type] <Owner>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzDataLakeStoreItemOwner** får ägaren av en fil eller mapp i Data Lake Store.

## EXEMPEL

### Exempel 1: Hämta ägaren till en katalog
```
PS C:\>Get-AzDataLakeStoreItemOwner -AccountName "ContosoADL" -Path / -Type User
```

Det här kommandot får användarens ägare för rotkatalogen för ContosoADL-kontot.
