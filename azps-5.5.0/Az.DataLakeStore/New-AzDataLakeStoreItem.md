# New-AzDataLakeStoreItem

## SYNOPSIS
Skapar en ny fil eller mapp i Data Lake Store.

## SYNTAX

```
New-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-Value] <Object>]
 [[-Encoding] <Encoding>] [-Folder] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzDataLakeStoreItem** skapar en ny fil eller mapp i Data Lake Store.

## EXEMPEL

### Exempel 1: Skapa en ny fil och en ny mapp
```
PS C:\>New-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/NewFile.txt"
PS C:\> New-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/NewFolder" -Folder
```

Med det första kommandot skapas NewFile.txt för det angivna kontot.
Med det andra kommandot skapas mappen NewFolder i rotmappen.
