# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Tar bort den angivna regeln för virtuellt nätverk i den angivna Data Lake Store.

## SYNTAX

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzDataLakeStoreVirtualNetworkRule** tar bort den angivna virtuella nätverksregeln i den angivna Data Lake Store.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Tar bort regeln för det virtuella nätverket "myVNET" från kontot "dls"
