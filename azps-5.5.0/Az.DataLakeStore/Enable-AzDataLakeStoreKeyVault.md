# Enable-AzDataLakeStoreKeyVault

## SYNOPSIS
Försöker aktivera ett användar hanterat nyckelvalv för kryptering av det angivna Data Lake Store-kontot.

## SYNTAX

```
Enable-AzDataLakeStoreKeyVault [-Account] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Enable-AzDataLakeStoreKeyVault** försöker aktivera ett användar hanterat nyckelvalv för kryptering av det angivna Data Lake Store-kontot.

## EXEMPEL

### Exempel 1: Aktivera nyckelvalvet för ContosoADLS-kontot
```
PS C:\>Enable-AzDataLakeStoreKeyVault -Name "ContosoADLS"
```

Det här kommandot försöker aktivera det användar hanterade nyckelvalvet för Data Lake Store-kontot ContosoADLS.
