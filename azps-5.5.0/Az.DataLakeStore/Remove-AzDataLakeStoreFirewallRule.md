# Remove-AzDataLakeStoreFirewallRule

## SYNOPSIS
Tar bort den angivna brandväggsregeln i den angivna Data Lake Store.

## SYNTAX

```
Remove-AzDataLakeStoreFirewallRule [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzDataLakeStoreFirewallRule** tar bort den angivna brandväggsregeln i den angivna Data Lake Store.

## EXEMPEL

### Exempel 1: Ta bort en brandväggsregel från ett konto
```
PS C:\> Remove-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule
```

Tar bort brandväggsregeln "MyFirewallRule" från kontot "ContosoADL"
