# Get-AzDtlVMsPerLabPolicy

## SYNOPSIS
Hämtar virtuella maskiner enligt labprincipen för ett labb i DevTest Labs.

## SYNTAX

```
Get-AzDtlVMsPerLabPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzDtlVMsPerLabPolicy** hämtar virtuella maskiner per labprincip för ett labb, vilket gör att du kan ange det totala antalet virtuella maskiner som tillåts i ett labb.
Cmdleten returnerar den aktiverade eller inaktiverade statusen för principen och det totala antalet virtuella datorer som tillåts i labbet som du har angett i principen.
