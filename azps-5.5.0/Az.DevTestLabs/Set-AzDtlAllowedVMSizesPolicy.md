# Set-AzDtlAllowedVMSizesPolicy

## SYNOPSIS
Anger principen för tillåtna storlekar på virtuella maskiner för ett labb i DevTest Labs.

## SYNTAX

### Aktivera (standard)
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Inaktivera
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzDtlAllowedVMSizesPolicy** anger principen för tillåtna storlekar på virtuella maskiner, som anger en lista över virtuella datorstorlekar som tillåts i ett labb.
Cmdleten använder den angivna resursgruppen och namnet på labbet för att ange principen.
