# Get-AzDtlAllowedVMSizesPolicy

## SYNOPSIS
Hämtar principen för tillåtna storlekar på virtuella maskiner för ett labb i DevTest Labs.

## SYNTAX

```
Get-AzDtlAllowedVMSizesPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzDtlAllowedVMSizesPolicy** får principen för virtuella datorstorlekar, så att du kan ange en lista över vilka virtuella datorstorlekar som tillåts i labbet.
Cmdleten returnerar den aktiverade eller inaktiverade statusen för principen och en lista över alla tillåtna storlekar för virtuella maskiner som du har angett i den angivna principen.
