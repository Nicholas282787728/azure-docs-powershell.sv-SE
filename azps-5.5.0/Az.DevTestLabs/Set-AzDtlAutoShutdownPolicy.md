# Set-AzDtlAutoShutdownPolicy

## SYNOPSIS
Anger principen för automatisk avstängning av ett labbutvecklingslabb.

## SYNTAX

### Aktivera (standard)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inaktivera
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzDtlAutoShutdownPolicy** ställer in principen för automatisk avstängning av ett lab, som automatiskt stänger av alla virtuella datorer i labbet vid en angiven tidpunkt.
Cmdleten använder den angivna resursgruppen och namnet på labbet för att ange principen.

## EXEMPEL

### Exempel 1

Anger principen för automatisk avstängning av ett labbutvecklingslabb. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoShutdownPolicy -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
