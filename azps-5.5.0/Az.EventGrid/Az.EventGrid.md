# Az.EventGrid-modul
## Beskrivning
I det här avsnittet visas hjälp för Cmdlets för Resurshanterare för Azure Event Grid PowerShell.

## Az.EventGrid-cmdlets
### [Get-AzEventGridDomain](Get-AzEventGridDomain.md)
Hämtar information om en domän i händelserutnätet eller får en lista över alla domäner i händelserutnätet i den aktuella Azure-prenumerationen.

### [Get-AzEventGridDomainKey](Get-AzEventGridDomainKey.md)
Hämtar de delade snabbtangenterna som används för att publicera händelser i en domän i ett händelserutnät.

### [Get-AzEventGridDomainTopic](Get-AzEventGridDomainTopic.md)
Hämtar information om ett domänämne i händelserutnätet eller får en lista över alla domänavsnitt om händelserutnätet under en specifik domän i händelserutnätet i den aktuella Azure-prenumerationen.

### [Get-AzEventGridSubscription](Get-AzEventGridSubscription.md)
Hämtar information om en händelseprenumeration eller får en lista över alla händelseprenumerationer i den aktuella Azure-prenumerationen.

### [Get-AzEventGridTopic](Get-AzEventGridTopic.md)
Hämtar information om ett ämne i händelserutnätet eller får en lista över alla avsnitt i händelserutnätet i den aktuella Azure-prenumerationen.

### [Get-AzEventGridTopicKey](Get-AzEventGridTopicKey.md)
Hämtar de delade snabbtangenterna som används för att publicera händelser i ett avsnitt i händelserutnätet.

### [Get-AzEventGridTopicType](Get-AzEventGridTopicType.md)
Hämtar information om de ämnestyper som stöds av Azure Event Grid.

### [New-AzEventGridDomain](New-AzEventGridDomain.md)
Skapar en ny Azure Event Grid-domän.

### [New-AzEventGridDomainKey](New-AzEventGridDomainKey.md)
Återskapar den delade åtkomstnyckeln för en Azure Event Grid-domän.

### [New-AzEventGridDomainTopic](New-AzEventGridDomainTopic.md)
Skapar ett nytt domänämne i Azure Event Grid.

### [New-AzEventGridSubscription](New-AzEventGridSubscription.md)
Skapar en ny Azure Event Grid-händelseprenumeration för ett ämne, En Azure-resurs, en Azure-prenumeration eller en resursgrupp.

### [New-AzEventGridTopic](New-AzEventGridTopic.md)
Skapar ett nytt Azure Event Grid-avsnitt.

### [New-AzEventGridTopicKey](New-AzEventGridTopicKey.md)
Återskapar den delade åtkomstnyckeln för ett Azure Event Grid-ämne.

### [Remove-AzEventGridDomain](Remove-AzEventGridDomain.md)
Tar bort en Azure Event Grid-domän.

### [Remove-AzEventGridDomainTopic](Remove-AzEventGridDomainTopic.md)
Tar bort ett domänämne i Azure Event Grid.

### [Remove-AzEventGridSubscription](Remove-AzEventGridSubscription.md)
Tar bort en Azure Event Grid-händelseprenumeration.

### [Remove-AzEventGridTopic](Remove-AzEventGridTopic.md)
Tar bort ett Azure Event Grid-avsnitt.

### [Set-AzEventGridTopic](Set-AzEventGridTopic.md)
Anger egenskaperna för ett avsnitt i händelserutnätet.

### [Update-AzEventGridSubscription](Update-AzEventGridSubscription.md)
Uppdatera egenskaperna för en händelseprenumeration på händelserutnätet.
