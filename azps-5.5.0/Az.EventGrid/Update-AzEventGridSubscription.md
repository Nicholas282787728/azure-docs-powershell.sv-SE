## BESKRIVNING
Uppdatera egenskaperna för en händelseprenumeration på händelserutnätet. Det här kan användas för att uppdatera filtret, målet eller etiketterna för en befintlig händelseprenumeration.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Update-AzEventGridSubscription -EventSubscriptionName ES1 -TopicName Topic1 -ResourceGroup MyResourceGroupName -Endpoint https://requestb.in/1kxxoui1
```

Uppdaterar slutpunkten för \` händelseprenumerationen ES1 \` för \` ämne1 \` i \` resursgruppen MyResourceGroupName \` till \`https://requestb.in/1kxxoui1\`

### Exempel 2
```powershell
PS C:\> Get-AzEventGridSubscription -EventSubscriptionName ES1 -TopicName Topic1 -ResourceGroup MyResourceGroupName | Update-AzEventGridSubscription -Endpoint https://requestb.in/1kxxoui1
```

Uppdaterar slutpunkten för \` händelseprenumerationen ES1 \` för \` ämne1 \` i \` resursgruppen MyResourceGroupName \` till \`https://requestb.in/1kxxoui1\`

### Exempel 3
```powershell
PS C:\> Update-AzEventGridSubscription -EventSubscriptionName ES1 -ResourceId "/subscriptions/55f3dcd4-cac7-43b4-990b-a139d62a1eb2/resourceGroups/TestRG/providers/Microsoft.EventHub/namespaces/ContosoNamespace" -Endpoint https://requestb.in/1kxxoui1 -SubjectEndsWith "jpg"
```

Uppdaterar egenskaperna för händelseprenumerationen ES1 för EventHub-namnområdet ContosoNamespace med den nya slutpunkten som ' och det nya \` \` \` https://requestb.in/1kxxoui1\ SubjectEndsWith-filtret som \` jpg\`

### Exempel 4
```powershell
PS C:\> $labels = "Finance", "HR"
PS C:\> Update-AzEventGridSubscription -EventSubscriptionName ES1 -ResourceGroup MyResourceGroupName -Label $labels
```

Uppdaterar egenskaperna för händelseprenumerationen \` ES1 för \` \` resursgruppen MyResourceGroupName \` med de nya etiketterna $labels.

## PARAMETERS

### -AdvancedFilter
Avancerat filter som anger en matris med flera hashtabellvärden som används för attributbaserad filtrering. Varje Hashtabellvärde har följande information om nyckelvärden: Åtgärd, nyckel och värde eller värden. Operatorn kan vara ett av följande värden: NumberIn, NumberNotIn, NumberLessQuals, NumberGreaterQuals, NumberLessQualOrEquals, NumberGreaterQuals, BoolEquals, StringIn, StringNotIn, StringBeginsWith, StringEndsWith eller StringContains. Nyckeln representerar den nyttolastegenskap där avancerade filtreringsprinciper tillämpas. Slutligen representerar värde eller värden det värde eller den uppsättning värden som ska matchas. Det kan vara ett enstaka värde av motsvarande typ eller en matris med värden. Som ett exempel på avancerade filterparametrar: $AdvancedFilters=@($AdvFilter 1, $AdvFilter 2) där $AdvFilter 1=@{operator="NumberIn"; key="Data.Key1"; Values=@(1,2)} och $AdvFilter 2=@{operator="StringBeginsWith"; key="Subject"; Values=@("SubjectPrefix1","SubjectPrefix2")}
