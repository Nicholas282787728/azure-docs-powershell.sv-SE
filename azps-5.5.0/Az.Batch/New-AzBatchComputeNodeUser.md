# New-AzBatchComputeNodeUser

## SYNOPSIS
Skapar ett användarkonto på en batchkalkylnod.

## SYNTAX

### Id
```
New-AzBatchComputeNodeUser [-PoolId] <String> [-ComputeNodeId] <String> -Name <String> -Password <SecureString>
 [-ExpiryTime <DateTime>] [-IsAdmin] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ParentObject
```
New-AzBatchComputeNodeUser [[-ComputeNode] <PSComputeNode>] -Name <String> -Password <SecureString>
 [-ExpiryTime <DateTime>] [-IsAdmin] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzBatchComputeNodeUser** skapar ett användarkonto på en Azure Batch-beräkningsnod.

## EXEMPEL

### Exempel 1: Skapa ett användarkonto med administrativa autentiseringsuppgifter
```
PS C:\>New-AzBatchComputeNodeUser -PoolId "MyPool01" -ComputeNodeId "ComputeNode01" -Name "TestUser" -Password "Password" -ExpiryTime ([DateTime]::Now.AddDays(7)) -IsAdmin -BatchContext $Context
```

Det här kommandot skapar ett användarkonto på beräkningsnoden som har ID ComputeNode01.
Noden finns i poolen som har ID MyPool01.
Användarnamnet är TestUser, lösenordet är Lösenord, kontot upphör att gälla om sju dagar och kontot har administrativa autentiseringsuppgifter.

### Exempel 2: Skapa ett användarkonto på en beräkningsnod med hjälp av pipelinen
```
PS C:\>Get-AzBatchComputeNode "MyPool01" -ComputeNodeId "ComputeNode01" -BatchContext $Context | New-AzBatchComputeNodeUser -Name "TestUser" -Password "Password" -BatchContext $Context
```

Med det här kommandot får du beräkningsnoden ComputeNode01 med cmdleten **Get-AzBatchComputeNode.**
Noden finns i poolen som har ID MyPool01.
Kommandot skickar den beräkningsnoden till den aktuella cmdleten med hjälp av rörledningsoperatorn.
Kommandot skapar ett användarkonto med användarnamnet TestUser och lösenordslösenordet.

Om du vill använda autentisering med delad nyckel i Get-AzBatchAccountKey använder du cmdleten för att få ett BatchAccountContext-objekt med dess snabbtangenter ifyllda. När du använder autentisering med delad nyckel används primärnyckeln som standard. Om du vill ändra nyckeln som ska användas anger du egenskapen BatchAccountContext.KeyInUse.
