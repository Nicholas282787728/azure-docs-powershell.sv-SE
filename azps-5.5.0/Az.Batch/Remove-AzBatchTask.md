# Remove-AzBatchTask

## SYNOPSIS
Tar bort en batchuppgift.

## SYNTAX

### Id
```
Remove-AzBatchTask [-JobId] <String> [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzBatchTask [-InputObject] <PSCloudTask> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzBatchTask** tar bort en Azure Batch-uppgift.
Den här cmdleten uppmanar dig att bekräfta, om du inte anger *force-parametern.*

## EXEMPEL

### Exempel 1: Ta bort en batchaktivitet med ID
```
PS C:\>Remove-AzBatchTask -JobId "Job-000001" -Id "Task23" -BatchContext $Context
```

Det här kommandot tar bort en aktivitet som har ID-aktiviteten23 under jobbet som har ID-jobbet-0000001.
Kommandot uppmanar dig att bekräfta.
Använd **cmdleten Get-AzBatchAccountKey** för att tilldela en kontext till den $Context variabeln.

### Exempel 2: Ta bort en batchaktivitet med hjälp av pipelinen utan bekräftelse
```
PS C:\>Get-AzBatchTask -JobId "Job-000001" -Id "Task26" -BatchContext $Context | Remove-AzBatchTask -Force -BatchContext $Context
```

Det här kommandot hämtar batchaktiviteten som har ID-aktiviteten26 i jobbet som har ID-jobbet-0000001 med hjälp av cmdleten **Get-AzBatchTask.**
Kommandot skickar aktiviteten till den aktuella cmdleten med hjälp av rörledningsoperatorn.
Kommandot tar bort uppgiften.
Det här kommandot anger *force-parametern.*
Därför uppmanas du inte att bekräfta med kommandot.
