# Get-AzBatchJob

## SYNOPSIS
Hämtar batchjobb för ett batchkonto eller jobbschema.

## SYNTAX

### ODataFilter (standard)
```
Get-AzBatchJob [-JobScheduleId <String>] [-Filter <String>] [-MaxCount <Int32>] [-Select <String>]
 [-Expand <String>] -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Id
```
Get-AzBatchJob [[-Id] <String>] [-Select <String>] [-Expand <String>] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ParentObject
```
Get-AzBatchJob [[-JobSchedule] <PSCloudJobSchedule>] [-Filter <String>] [-MaxCount <Int32>] [-Select <String>]
 [-Expand <String>] -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzBatch Enlsson** hämtar Azure-batchjobben för batchkontot som anges av *parametern BatchAccountContext.*
Du kan använda *Id-parametern* för att få ett enda jobb.
Du kan använda *filterparametern* för att få jobben som matchar ett OData-filter (Open Data Protocol).
Om du tillhandahåller ett ID för jobbschema eller **PSCloudCloudCloudSchedule-instans** returnerar den här cmdleten endast jobben för det jobbschemat.

Om du vill använda autentisering med delad nyckel i Get-AzBatchAccountKey använder du cmdleten för att få ett BatchAccountContext-objekt med dess snabbtangenter ifyllda.
