# Get-AzBatchComputeNode

## SYNOPSIS
Hämtar batchbearbetningsnoder från en pool.

## SYNTAX

### ODataFilter (standard)
```
Get-AzBatchComputeNode [-PoolId] <String> [-Filter <String>] [-MaxCount <Int32>] [-Select <String>]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Id
```
Get-AzBatchComputeNode [-PoolId] <String> [[-Id] <String>] [-Select <String>]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ParentObject
```
Get-AzBatchComputeNode [[-Pool] <PSCloudPool>] [-Filter <String>] [-MaxCount <Int32>] [-Select <String>]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzBatchComputeNode** hämtar Azure Batch-beräkningsnoder från en pool.
Ange antingen *pool-ID* eller *poolparameter.*
Ange *Id-parametern* för att få en enda beräkningsnod.
Ange *filterparametern* för att få de beräkningsnoder som matchar ett OData-filter (Open Data Protocol).
