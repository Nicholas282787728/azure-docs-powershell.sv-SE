# Remove-AzBatchAccount

## SYNOPSIS
Tar bort ett Batch-konto.

## SYNTAX

```
Remove-AzBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzBatchAccount** tar bort ett Azure Batch-konto.
Den här cmdleten uppmanar dig innan ett konto tas bort, såvida du inte anger *force-parametern.*

## EXEMPEL

### Exempel 1: Ta bort ett batchkonto
```
PS C:\>Remove-AzBatchAccount -AccountName "pfuller"
```

Med det här kommandot tas batchkontot pfuller bort.
Med det här kommandot uppmanas du att bekräfta detta innan kontot tas bort.
