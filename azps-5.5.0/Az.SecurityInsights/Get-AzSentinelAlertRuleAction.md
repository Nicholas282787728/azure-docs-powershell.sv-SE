# Get-AzSentinelAlertRuleAction

## SYNOPSIS
Få ett automatiserat svar (åtgärd aviseringsregel).

## SYNTAX

### AlertRuleId (standard)
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ActionId
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 -ActionId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSentinelAlertRuleAction** får en automatiserad svarsåtgärd (aviseringsregelåtgärd) från den angivna arbetsytan.
Om du anger *parametrarna ActionId* *och AlertRuleId* returneras ett enda **AlertRuleAction-objekt.**
Om du inte anger *ActionId-parametern* returneras en matris som innehåller alla åtgärder för den specifika aviseringsregeln på den angivna arbetsytan.
Du kan använda **åtgärdsobjektet** till att uppdatera åtgärden, till exempel ändra åtgärden **för** en aviseringsregel.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $AlertRuleActions = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId"
```

I det här exemplet visas alla **åtgärder** för den angivna aviseringsregeln på den angivna arbetsytan och lagras sedan i den $AlertRuleActions variabeln.

### Exempel 2
```powershell
PS C:\> $AlertRuleAction = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId" -ActionId "MyActionId"
```

I det här exemplet **får du en AlertRuleAction** för den angivna aviseringsregeln på den angivna arbetsytan och lagrar den i $AlertRuleAction variabeln.
