## BESKRIVNING
**Cmdleten Update-AzSentinelAlertRule** uppdaterar en analys (aviseringsregel) i den angivna arbetsytan.
Du kan använda ett -InputObject- eller -ResourceId eller -AlertId.  Du kan uppdatera 1 eller flera proprtery-parmaters.
Du kan använda *parametern Confirm* och $ConfirmPreference Windows PowerShell för att styra om cmdleten uppmanar dig att bekräfta.


## EXEMPEL

### Exempel 1
```powershell
PS C:\> Update-AzSentinelAlertRule -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId" -Disabled -DisplayName "Disabled-AlertRuleDisplayName"
```

Det här exemplet uppdaterar **en Aviseringsregel** som anger att *den är inaktiverad* och byter namn till *Disabled-AlertRuleDisplayName.*  Alla andra egenskaper förblir desamma.

### Exempel 2
```powershell
PS C:\> $AlertRule = Get-AzSentinelAlertRule -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId"
PS C:\> Update-AzSentinelAlertRule -InputObject $AlertRule -Disabled
```

Det här exemplet uppdaterar **en Avisering med en** InputObject-inställning som *inaktiverad.*  Alla andra egenskaper förblir desamma.
