# Remove-AzSentinelAlertRuleAction

## SYNOPSIS
Ta bort ett automatiserat svar från ett analys.

## SYNTAX

### ActionId (standard)
```
Remove-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 -ActionId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelAlertRuleAction -InputObject <PSSentinelActionResponse> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzSentinelAlertRuleAction** tar permanent bort ett automatiserat svar från aviseringsregeln i en angiven arbetsyta.
Du kan skicka ett **AlertRuleAction-objekt** med hjälp av rörledningsoperatorn, eller så kan du ange nödvändiga parametrar.
Du kan använda parametern Confirm och $ConfirmPreference Windows PowerShell för att styra om cmdleten uppmanar dig att bekräfta.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Remove-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId" -ActionId "MyActionId"
```

Med det här kommandot tas aviseringsregeln bort från arbetsytan.
