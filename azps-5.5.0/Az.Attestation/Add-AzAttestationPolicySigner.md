# Add-AzAttestationPolicySigner

## SYNOPSIS
Lägger till en betrodd principtecken för en klientorganisation i Azure-stationen.

## SYNTAX

### NameParameterSet
```
Add-AzAttestationPolicySigner [-Name] <String> [-ResourceGroupName] <String> -Signer <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Add-AzAttestationPolicySigner [-ResourceId] <String> -Signer <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Add-AzAttestationPolicySigner cmdleten läggs en betrodd principtecken till för en klientorganisation i Azure-bepansering.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $trustedSigner = Get-Content -Path .\trusted.signer.txt
PS C:\> Add-AzAttestationPolicySigner -Name pshtest -ResourceGroupName psh-test-rg -Signer $trustedSigner
```

Lägg till en betrodd undertecknare för atteestationsleverantören *pshtest.*
