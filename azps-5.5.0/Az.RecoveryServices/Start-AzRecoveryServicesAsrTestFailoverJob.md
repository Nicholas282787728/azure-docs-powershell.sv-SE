# Start-AzRecoveryServicesAsrTestFailoverJob

## SYNOPSIS
Startar en failover-teståtgärd.

## SYNTAX

```

 -Direction <String> [-CloudServiceCreationOption <String>] [-DataEncryptionPrimaryCertFile <String>]
 [-DataEncryptionSecondaryCertFile <String>] [-RecoveryPoint <ASRRecoveryPoint>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Start-AzRecoveryServicesAsrTestFailover Starts** test failover för ett skyddat objekt eller återställningsplan för Azure Site Recovery replication.
Du kan kontrollera om jobbet har lyckats med hjälp av Get-AzRecoveryServicesAsrJob-cmdleten.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $currentJob = Start-AzRecoveryServicesAsrTestFailoverJob -RecoveryPlan $RP -Direction PrimaryToRecovery -VMNetwork $TestRecoveryNetwork
```

Startar test redundansåtgärden för återställningsplanen med angivna parametrar och returnerar det ASR-jobb som använts för att spåra åtgärden.

### Exempel 2

Startar en failover-teståtgärd. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Start-AzRecoveryServicesAsrTestFailoverJob -AzureVMNetworkId <String> -Direction PrimaryToRecovery -RecoveryPlan $RP
```
