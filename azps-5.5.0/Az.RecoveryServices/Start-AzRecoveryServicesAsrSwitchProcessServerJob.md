# Start-AzRecoveryServicesAsrSwitchProcessServerJob

## SYNOPSIS
Växla replikering från en processserver till en annan för belastningsutjämning.

## SYNTAX

```
Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric <ASRFabric> -SourceProcessServer <ASRProcessServer>
 -TargetProcessServer <ASRProcessServer> [-ReplicationProtectedItem <ASRReplicationProtectedItem[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Start-AzRecoveryServicesAsrSwitchProcessServer Ett** växlat replikeringsdataförlopp för angivna virtuella datorer eller en angiven Process-server till den angivna målprocessservern. Används för belastningsutjämning eller växling av replikering mellan processservrar.

## EXEMPEL

### Exempel 1
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer
```

Jobb att spåra byte av processserver för alla replikeringsskyddade objekt från källa till målprocessserver.

### Exempel 2
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer -ReplicatedItem $rpList
```

Jobb att spåra växling av processserver för godkända replikeringsskyddade objekt från källa till målprocessserver.
