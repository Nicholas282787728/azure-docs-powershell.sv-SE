# Remove-AzRecoveryServicesAsrProtectionContainer

## SYNOPSIS
Tar bort den angivna skyddsbehållaren från dess fabric.

## SYNTAX

```
Remove-AzRecoveryServicesAsrProtectionContainer -InputObject <ASRProtectionContainer>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzRecoveryServicesAsrProtectionContainer cmdleten tas den angivna Azure Site Recovery Protection Container bort.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzRecoveryServicesAsrProtectionContainer -Name xxxxx  -Fabric $fabric
PS C:\> Remove-AzRecoveryServicesAsrProtectionContainer -InputObject $protectionContainer
```

Startar borttagningen av den angivna skyddbehållaren och returnerar asr-jobbet som användes för att spåra borttagningen.
