# Get-AzRecoveryServicesAsrNetworkMapping

## SYNOPSIS
Hämtar information om nätverksmappningar för webbplatsåterställning för det aktuella valvet.

## SYNTAX

### ByObject (standard)
```
Get-AzRecoveryServicesAsrNetworkMapping [-Name <String>] -Network <ASRNetwork>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFabricObject
```
Get-AzRecoveryServicesAsrNetworkMapping [-Name <String>] -PrimaryFabric <ASRFabric>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRecoveryServicesAsrNetworkMapping** hämtar information om nätverksmappningar för Azure-webbplatsåterställning för Recovery Services-valvet.

## EXEMPEL

### Exempel 1
```
PS C:\> $Networkmappings = Get-AzRecoveryServicesAsrNetworkMapping -Network $Network
```

Hämtar alla nätverksmappningar för det godkända nätverket.

### Exempel 2
```
PS C:\> $primaryFabric = Get-AzRecoveryServicesAsrFabric -Name xxxx
PS C:\> $Networkmappings = Get-AzRecoveryServicesAsrNetworkMapping -Name $networkMappingName -PrimaryFabric $primaryFabric
```

Hämtar nätverksmappning med angivet namn i angiven Azure-webbplatsåterställnings fabric.
