# New-AzRecoveryServicesVault

## SYNOPSIS
Skapar ett nytt Recovery Services-valv.

## SYNTAX

```
New-AzRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzRecoveryServicesVault** skapar ett nytt Recovery Services-valv.

## EXEMPEL

### Exempel 1
```
PS C:\> New-AzRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Skapa återställningstjänstvalv i resursgrupp och plats.
