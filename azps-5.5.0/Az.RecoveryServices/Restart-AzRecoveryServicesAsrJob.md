# Restart-AzRecoveryServicesAsrJob

## SYNOPSIS
Startar om ett Azure-webbplatsåterställningsjobb.

## SYNTAX

### ByObject (standard)
```
Restart-AzRecoveryServicesAsrJob -InputObject <ASRJob> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByName
```
Restart-AzRecoveryServicesAsrJob -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Restart-AzRecoveryServicesAsr Restart-AzRecoveryServicesAsr Engström** startar om ett Azure Site Recovery-jobb.

## EXEMPEL

### Exempel 1
```
PS C:\> $currentJob = Restart-AzRecoveryServicesAsrJob -Job $Job
```

Startar om det angivna ASR-jobbet och returnerar det uppdaterade ASR-jobbets objekt för ASR-jobbet.
