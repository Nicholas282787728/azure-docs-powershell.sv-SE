# Disable-AzRecoveryServicesBackupProtection

## SYNOPSIS
Inaktiverar skyddet för ett säkerhetskopierat objekt.

## SYNTAX

```
Disable-AzRecoveryServicesBackupProtection [-Item] <ItemBase> [-RemoveRecoveryPoints] [-Force]
 [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Disable-AzRecoveryServicesBackupProtection** inaktiverar skydd för ett säkerhetskopierat objekt i Azure.
Den här cmdleten stoppar den vanliga schemalagda säkerhetskopieringen av ett objekt.
Den här cmdleten kan också ta bort befintliga återställningspunkter för säkerhetskopian om den körs med parametern RemoveRecoveryPoints.
Ange valvkontexten med hjälp Set-AzRecoveryServicesVaultContext cmdleten innan du använder den aktuella cmdleten.

## EXEMPEL

### Exempel 1: Inaktivera säkerhetskopieringsskydd
```powershell
PS C:\> $Cont = Get-AzRecoveryServicesBackupContainer -ContainerType AzureVM -Status Registered 
PS C:\> $PI = Get-AzRecoveryServicesBackupItem -Container $Cont[0] -WorkloadType AzureVM 
PS C:\> Disable-AzRecoveryServicesBackupProtection -Item $PI[0]
```

Det första kommandot får en matris med behållare för säkerhetskopiering och lagrar den sedan $Cont matrisen.
Det andra kommandot får säkerhetskopian som motsvarar det första behållarobjektet och lagrar det sedan i den $PI variabeln.
Det senaste kommandot inaktiverar säkerhetskopieringsskydd för objektet i $PI \[ \] 0, men behåller data.

### Exempel 2

Inaktiverar skyddet för ett säkerhetskopierat objekt. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Disable-AzRecoveryServicesBackupProtection -Item $PI[0] -RemoveRecoveryPoints -VaultId $vault.ID
```
