# Get-AzRecoveryServicesAsrVaultContext

## SYNOPSIS
Hämtar information om ASR-valvinställningar för Recovery Services-valvet.

## SYNTAX

```
Get-AzRecoveryServicesAsrVaultContext [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRecoveryServicesAsrVaultContext** hämtar information om ASR-valvinställningar relaterade till Recovery Services-valvet.

## EXEMPEL

### Exempel 1
```
PS C:\> $VaultSettings = Get-AzRecoveryServicesAsrVaultContext
```

Hämtar ASR-valvinställningarna för det aktiva (i PowerShell-sessionen) Recovery Services-valvet.
