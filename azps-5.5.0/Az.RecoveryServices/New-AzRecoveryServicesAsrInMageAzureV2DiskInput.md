# New-AzRecoveryServicesAsrInMageAzureV2DiskInput

## SYNOPSIS
Skapar ett diskmappningsobjekt för virtuella vMWare-maskindiskar som ska replikeras.

## SYNTAX

```
New-AzRecoveryServicesAsrInMageAzureV2DiskInput -DiskId <String> -LogStorageAccountId <String>
 -DiskType <String> [-DiskEncryptionSetId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Skapar ett diskmappningsobjekt som mappar en virtuell vMWare-maskindisk till cachelagringskontot och mål hanterad disktyp (återställningsregion) som ska användas för att replikera disken.

## EXEMPEL

### Exempel 1
```powershell
PS C:> New-AzRecoveryServicesAsrInMageAzureV2DiskInput -DiskId $diskId -LogStorageAccountId $logStorageAccountId -DiskType $diskType
```

Skapa ett diskmappningsobjekt för virtuella vMWare-maskindiskar som ska replikeras. Används under aktivera skydd för vMWare-dator.
