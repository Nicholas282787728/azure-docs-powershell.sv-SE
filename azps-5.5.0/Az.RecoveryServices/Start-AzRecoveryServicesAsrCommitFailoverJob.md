# Start-AzRecoveryServicesAsrCommitFailoverJob

## SYNOPSIS
Startar failover-åtgärden för ett webbplatsåterställningsobjekt.

## SYNTAX

### ByRPIObject (standard)
```
Start-AzRecoveryServicesAsrCommitFailoverJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByRPObject
```
Start-AzRecoveryServicesAsrCommitFailoverJob -RecoveryPlan <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Start-AzRecoveryServicesAsrCommitFailoverKommando** startar redundansprocessen för ett Azure Site Recovery-objekt efter en redundansåtgärd.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> $currentJob = Start-AzRecoveryServicesAsrCommitFailoverJob -RecoveryPlan $RP
```

Startar commit failover för den angivna återställningsplanen och returnerar ASR-jobbet som använts för att spåra åtgärden.

### Exempel 2

Startar failover-åtgärden för ett webbplatsåterställningsobjekt. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Start-AzRecoveryServicesAsrCommitFailoverJob -ReplicationProtectedItem $ReplicationProtectedItem
```
