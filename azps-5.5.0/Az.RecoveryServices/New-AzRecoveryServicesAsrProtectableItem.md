# New-AzRecoveryServicesAsrProtectableItem

## SYNOPSIS
Lägg till(Upptäck) en fysisk server i listan med skyddbara objekt.

## SYNTAX

```
New-AzRecoveryServicesAsrProtectableItem -ProtectionContainer <ASRProtectionContainer> -FriendlyName <String>
 -IPAddress <String> -OSType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
**New-AzRecoveryServicesAsrProtectableItem** lägger till ett nytt skyddbart objekt i listan med upptäckta skyddbara objekt i en skyddsbehållare i en ASR-fabric (gäller endast för VMware-materialet).

## EXEMPEL

### Exempel 1

Lägg till(Upptäck) en fysisk server i listan med skyddbara objekt. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
New-AzRecoveryServicesAsrProtectableItem -FriendlyName XXXXXXXXXX -IPAddress $ipaddresss -OSType Windows -ProtectionContainer $pc
```
