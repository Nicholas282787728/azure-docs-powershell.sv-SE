# Remove-AzRecoveryServicesAsrFabric

## SYNOPSIS
Tar bort den angivna Azure Site Recovery Fabric från Recovery Services-valvet.

## SYNTAX

```
Remove-AzRecoveryServicesAsrFabric -InputObject <ASRFabric> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRecoveryServicesAsrFabric** tar bort den angivna Azure Site Recovery-strukturen från valv för återställningstjänster.

## EXEMPEL

### Exempel 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrFabric -Fabric $Fabric
```

Startar borttagningen av angivet material och returnerar det ASR-jobb som använts för att spåra åtgärden.
