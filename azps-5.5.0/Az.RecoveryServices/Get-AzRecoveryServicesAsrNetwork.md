# Get-AzRecoveryServicesAsrNetwork

## SYNOPSIS
Hämtar information om nätverken som hanteras av Webbplatsåterställning för det aktuella valvet.

## SYNTAX

### ByFabricObject (standard)
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByFriendlyName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -FriendlyName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRecoveryServicesAsrNetwork** hämtar information om Azure-nätverk för webbplatsåterställning för det aktuella Azure Site Recovery-valvet.

## EXEMPEL

### Exempel 1
```
PS C:\> $Networks = Get-AzRecoveryServicesAsrNetwork -Fabric $Fabric
```

Hämtar alla kända nätverk i det angivna materialet.
