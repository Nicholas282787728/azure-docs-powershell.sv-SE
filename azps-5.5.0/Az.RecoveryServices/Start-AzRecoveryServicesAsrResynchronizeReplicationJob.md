# Start-AzRecoveryServicesAsrResynchronizeReplicationJob

## SYNOPSIS
Startar replikeringssynkronisering.

## SYNTAX

### Standard (standard)
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Start-AzRecoveryServicesAsrResynchronizeReplicationAtions** cmdleten startar omsynkronisering av replikering för det angivna skyddade objektet om det skyddade är i ett obligatoriskt tillstånd för omsynkronisering.

## EXEMPEL

### Exempel 1
```
PS C:\> Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem $rpi
```

Startar jobbet för att synkronisera replikering på skickade replikeringsskyddade objekt.
