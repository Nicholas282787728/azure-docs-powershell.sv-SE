# Remove-AzRecoveryServicesAsrReplicationProtectedItem

## SYNOPSIS
Stoppar/inaktiverar replikering för ett skyddat objekt i Azure Site Recovery.

## SYNTAX

```
Remove-AzRecoveryServicesAsrReplicationProtectedItem -InputObject <ASRReplicationProtectedItem>
 [-WaitForCompletion] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzRecoveryServicesAsrReplicationProtectedItem** inaktiverar replikering av angivet skyddat Azure Site Recovery-objekt.
Den här åtgärden gör att replikering stoppas för det skyddade objektet.

## EXEMPEL

### Exempel 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrReplicationProtectedItem -ReplicationProtectedItem $ReplicationProtectedItem
```

Startar avaktiveringsreplikeringsåtgärden för det angivna replikeringsskyddade objektet och returnerar det ASR-jobb som använts för att spåra åtgärden.
