# New-AzRecoveryServicesAsrProtectionContainer

## SYNOPSIS
Skapar en Azure Site Recovery Protection-behållare inom den angivna strukturen.

## SYNTAX

```
New-AzRecoveryServicesAsrProtectionContainer -Name <String> -InputObject <ASRFabric>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med New-AzRecoveryServicesAsrProtectionContainer-cmdleten skapas en skyddsbehållare under angiven Azure Site Recovery Fabric.

## EXEMPEL

### Exempel 1
```
PS C:\> $job = New-AzRecoveryServicesAsrProtectionContainer -Name xyz -Fabric $fabric
PS C:\> Get-ASRJob -name $job.id
```

Startar skapandet av skyddsbehållaren med de angivna parametrarna och returnerar det ASR-jobb som använts för att spåra åtgärden.
