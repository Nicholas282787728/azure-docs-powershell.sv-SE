# Get-AzRecoveryServicesVaultProperty

## SYNOPSIS
Returnerar egenskaperna för ett Recovery Services-valv.

## SYNTAX

```
Get-AzRecoveryServicesVaultProperty [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzRecoveryServicesVaultProperty** returnerar egenskaperna för ett Återställningstjänstvalv.

## EXEMPEL

### Exempel 1: Hämta egenskaper för ett valv
```
PS C:\> $vault = Get-AzRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
PS C:\> $props = Get-AzRecoveryServicesVaultProperty -VaultId $vault.Id
PS C:\> $encryption.encryptionProperties
```

Det första kommandot får ett valv-objekt och lagrar det sedan i $vault variabeln.
Det andra kommandot hämtar egenskaper för valvet. Därefter kommer vi åt krypteringsegenskaperna för valvet.
