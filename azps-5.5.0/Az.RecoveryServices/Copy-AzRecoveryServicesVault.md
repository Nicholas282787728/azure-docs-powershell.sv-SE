# Copy-AzRecoveryServicesVault

## SYNOPSIS
Kopierar data från ett valv i en region till ett valv i en annan region.

## SYNTAX

```
Copy-AzRecoveryServicesVault [-Force] [-DefaultProfile <IAzureContextContainer>] [-SourceVault] <ARSVault>
 [-TargetVault] <ARSVault> [-RetryOnlyFailed] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Copy-AzRecoveryServicesVault** kopierar data från ett valv i en region till ett valv i en annan region. För närvarande har vi bara stöd för dataflyttning på valvnivå.

## EXEMPEL

### Exempel 1: Kopiera data från valv1 till valv2
```
PS C:\> $sourceVault = Get-AzRecoveryServicesVault -ResourceGroupName "rgName1" -Name "vault1"
PS C:\> $targetVault = Get-AzRecoveryServicesVault -ResourceGroupName "rgName2" -Name "vault2"
PS C:\> Copy-AzRecoveryServicesVault -SourceVault $sourceVault -TargetVault $targetVault
```

De två första cmdletarna hämtar Recovery Services Vault – valv1 respektive valv2.
Det andra kommandot utlöser en fullständig dataflyttning från valv1 till valv2. 

### Exempel 2: Kopiera data från valv1 till valv2 med endast misslyckade objekt
```
PS C:\> $sourceVault = Get-AzRecoveryServicesVault -ResourceGroupName "rgName1" -Name "vault1"
PS C:\> $targetVault = Get-AzRecoveryServicesVault -ResourceGroupName "rgName2" -Name "vault2"
PS C:\> Copy-AzRecoveryServicesVault -SourceVault $sourceVault -TargetVault $targetVault -RetryOnlyFailed
```

De två första cmdletarna hämtar Recovery Services Vault – valv1 respektive valv2.
Det andra kommandot utlöser en partiell dataflyttning från valv1 till valv2 med bara de objekt som misslyckades i tidigare flyttningsåtgärder.
