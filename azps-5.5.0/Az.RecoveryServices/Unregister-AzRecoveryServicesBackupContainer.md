# Unregister-AzRecoveryServicesBackupContainer

## SYNOPSIS
Avregistrerar en Windows Server eller en annan behållare från valvet.

## SYNTAX

```
Unregister-AzRecoveryServicesBackupContainer [-Container] <ContainerBase> [-PassThru] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Unregister-AzRecoveryServicesBackupContainer** avregistrerar en Windows Server- eller annan behållare för säkerhetskopiering från valvet.
Den här cmdleten tar bort referenser till en behållare från valvet.
Innan du kan avregistrera en behållare måste du ta bort alla skyddade data som är associerade med behållaren.
Ange valvkontexten med hjälp Set-AzRecoveryServicesVaultContext cmdleten innan du använder den aktuella cmdleten.

## EXEMPEL

### Exempel 1: Avregistrera en Windows Server från valvet
```powershell
PS C:\>$Cont = Get-AzRecoveryServicesBackupContainer -ContainerType "Windows" -BackupManagementType MARS -Name "server01.contoso.com"
PS C:\> Unregister-AzRecoveryServicesBackupContainer -Container $Cont
```

Det första kommandot får Windows-behållaren med namnet server01.contoso.com som är registrerad i valvet och lagrar den sedan i $Cont variabeln.
Det andra kommandot avregistrerar den angivna Windows Server från Azure Backup-valvet.

### Exempel 2

Avregistrerar en Windows Server eller en annan behållare från valvet. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
Unregister-AzRecoveryServicesBackupContainer -Container $Cont -VaultId $vault.ID
```
