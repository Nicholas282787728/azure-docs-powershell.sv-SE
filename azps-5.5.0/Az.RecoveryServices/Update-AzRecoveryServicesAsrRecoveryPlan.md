# Update-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Uppdaterar innehållet i en Återställningsplan för Azure-webbplatser.

## SYNTAX

### ByRPObject (standard)
```
Update-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByRPFile
```
Update-AzRecoveryServicesAsrRecoveryPlan -Path <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Update-AzRecoveryServicesAsrRecoveryPlan** uppdaterar innehållet i en återställningsplan med innehållet i den angivna ASR-återställningsplanobjektet eller ASR-återställningsplandefinition json-filen.

## EXEMPEL

### Exempel 1: Uppdatera en återställningsplan
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Starta åtgärden för att uppdatera en återställningsplan med hjälp av innehållet i det angivna asr-återställningsplanobjektet och returnera det ASR-jobb som användes för att spåra åtgärden.
