# Get-AzRecoveryServicesAsrAlertSetting

## SYNOPSIS
Hämtar de konfigurerade aviseringsinställningarna för Azure-webbplatsåterställning för valvet.

## SYNTAX

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRecoveryServicesAsrAlertSetting** hämtar de konfigurerade meddelandeinställningarna för Azure Site Recovery för valvet.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Inställning för avisering/avisering för Azure-webbplatsåterställning.
