# Az.WindowsIotServices-modul
## Beskrivning
Microsoft Azure PowerShell: WindowsIotServices-cmdlets

## Az.WindowsIotServices Cmdlets
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
Hämta icke-säkerhetsrelaterade metadata för en Windows IoT-enhetstjänst.

### [New-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Skapa eller uppdatera metadata för en Windows IoT-enhetstjänst.
Det vanliga mönstret för att ändra en egenskap är att hämta Windows IoT Device Service-metadata och säkerhetsmetadata och sedan kombinera dem med de ändrade värdena i en ny brödtext för att uppdatera Windows IoT-enhetstjänsten.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Ta bort en Windows IoT-enhetstjänst.

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Uppdaterar metadata för en Windows IoT-enhetstjänst.
Det vanliga mönstret för att ändra en egenskap är att hämta Windows IoT Device Service-metadata och säkerhetsmetadata och sedan kombinera dem med de ändrade värdena i en ny brödtext för att uppdatera Windows IoT-enhetstjänsten.
