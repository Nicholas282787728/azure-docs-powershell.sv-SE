# Get-AzAutomationSoftwareUpdateConfiguration

## SYNOPSIS
Hämtar en lista över konfigurationer för uppdatering av Azure-programvara.

## SYNTAX

### ByAll (standard)
```
Get-AzAutomationSoftwareUpdateConfiguration [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationSoftwareUpdateConfiguration -Name <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByVMId
```
Get-AzAutomationSoftwareUpdateConfiguration -AzureVMResourceId <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Data Get-AzAutomationSoftwareUpdateConfiguration en lista över konfigurationer för programvaruuppdateringar. Om du vill få en specifik konfiguration av programuppdateringen anger du namnparametern. Du kan också lista konfigurationer av programuppdateringar som riktar sig till specifik virtuell Azure-dator genom att ange Azure-resurs-ID för den virtuella datorn.

## EXEMPEL

### Exempel 1
Hämta en konfiguration av azure automationsprogramvara med namn.

```powershell
PS C:\> Get-AzAutomationSoftwareUpdateConfiguration -ResourceGroupName "mygroup" -AutomationAccountName "myaccount" -Name "MyWeeklySchedule"
```
