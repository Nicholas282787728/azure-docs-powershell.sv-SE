# Set-AzAutomationAccount

## SYNOPSIS
Ändrar ett automatiseringskonto.

## SYNTAX

```
Set-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Plan <String>] [-Tags <IDictionary>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzAutomationAccount** ändrar ett Azure Automation-konto.
Mer information om automatiseringskonton finns i New-AzAutomationAccount cmdlet.

## EXEMPEL

### Exempel 1: Ange taggarna för ett automatiseringskonto
```
PS C:\>$Tags = @{"tag01"="value01";"tag02"="value02"}
PS C:\> Set-AzAutomationAccount -Name "AutomationAccount01" -ResourceGroupName "ResourceGroup01" -Tags $Tags
```

Det första kommandot tilldelar två tangent-/värdepar till den $Tags variabeln.
Det andra kommandot anger taggar i $Tags automationskontot AutomationAccount01.

### Exempel 2: Ändra planen för ett automatiseringskonto
```
PS C:\>Set-AzAutomationAccount -Name "AutomationAccount01" -ResourceGroupName "ResourceGroup01" -Plan Basic
```

Med det här kommandot ändras planen till Basic för automatiseringskontot AutomationAccount01.
