# Get-AzAutomationScheduledRunbook

## SYNOPSIS
Hämtar automatiseringskörningsböcker och tillhörande scheman.

## SYNTAX

### ByAll (standard)
```
Get-AzAutomationScheduledRunbook [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByLowscheduleId
```
Get-AzAutomationScheduledRunbook -JobScheduleId <Guid> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByRunbookName
```
Get-AzAutomationScheduledRunbook -RunbookName <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByRunbookNameAndScheduleName
```
Get-AzAutomationScheduledRunbook -RunbookName <String> -ScheduleName <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByScheduleName
```
Get-AzAutomationScheduledRunbook -ScheduleName <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutomationScheduledRunbook** får en eller flera Azure Automation-runbooks och tillhörande scheman.
Som standard hämtar den här cmdleten alla schemalagda runbooks.
Ange namnet på en runbook eller ett schema, eller både och om du vill visa specifika runbook-scheman.

## EXEMPEL

### Exempel 1: Hämta alla schemalagda runbooks
```
PS C:\>Get-AzAutomationScheduledRunbook -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01"
```

Det här kommandot hämtar alla schemalagda runbooks i Azure Automation-kontot Contoso17.

### Exempel 2: Hämta alla scheman kopplade till en runbook
```
PS C:\>Get-AzAutomationScheduledRunbook -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -RunbookName "Runbk01"
```

Det här kommandot hämtar alla schemalagda runbooks för runbook Runbk01 på Azure Automation-kontot Contoso17.

### Exempel 3: Hämta alla runbooks som är kopplade till ett schema
```
PS C:\>Get-AzAutomationScheduledRunbook -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -ScheduleName "Schedule01"
```

Det här kommandot hämtar alla schemalagda runbooks för schemat Schedule01 i Azure Automation-kontot Contoso17.
