# Get-AzAutomationCertificate

## SYNOPSIS
Hämtar automatiseringscertifikat.

## SYNTAX

### ByAll (standard)
```
Get-AzAutomationCertificate [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByCertificateName
```
Get-AzAutomationCertificate [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutomationCertificate** får ett eller flera Azure Automation-certifikat.
Som standard hämtar den här cmdleten alla certifikat.
Ange namnet på ett certifikat för att få ett visst certifikat.

## EXEMPEL

### Exempel 1: Hämta alla certifikat
```
PS C:\>Get-AzAutomationCertificate -ResourceGroupName "ResourceGroup07" -AutomationAccountName "Contoso17"
```

Det här kommandot hämtar metadata för alla certifikat i automatiseringskontot Contoso17.

### Exempel 2: Skaffa ett certifikat
```
PS C:\>Get-AzAutomationCertificate -ResourceGroupName "ResourceGroup07" -AutomationAccountName "Contoso17" -Name "ContosoCertificate"
```

Det här kommandot får metadata för certifikatet ContosoCertificate.
