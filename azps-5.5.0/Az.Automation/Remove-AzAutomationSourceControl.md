# Remove-AzAutomationSourceControl

## SYNOPSIS
Tar bort en Azure Automation-källkontroll.

## SYNTAX

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzAutomationSourceControl-cmdleten tas en källkontroll bort från Azure Automation.

## EXEMPEL

### Exempel 1
Det här kommandot tar bort automationskällans kontroll med namnet VSTSNative i kontot devAccount.
Det här kommandot anger *force-parametern.* Därför uppmanas du inte att bekräfta.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
