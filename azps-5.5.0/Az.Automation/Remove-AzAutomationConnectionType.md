# Remove-AzAutomationConnectionType

## SYNOPSIS
Tar bort en automationsanslutningstyp.

## SYNTAX

```
Remove-AzAutomationConnectionType [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzAutomationConnectionType** tar bort en anslutningstyp från Azure Automation.
Alla anslutningar som är associerade med den anslutningstyp som du tar bort blir oanvändbara.
Ta bort dem, såvida du inte skapar en ny anslutningstyp som uppfyller följande villkor: 
- Typen har samma namn som den ursprungliga anslutningstypen. 
- Typen har samma fältdefinitioner som den ursprungliga anslutningstypen.
Det kan ha ytterligare fält.

## EXEMPEL

### Exempel 1: Ta bort en anslutningstyp
```
PS C:\>Remove-AzAutomationConnectionType -AutomationAccountName "Contoso17" -Name "ContosoConnectionType" -ResourceGroupName "ResourceGroup01"
```

Det här kommandot tar bort en anslutningstyp med namnet ContosoConnectionType i automationskontot Contoso17.
