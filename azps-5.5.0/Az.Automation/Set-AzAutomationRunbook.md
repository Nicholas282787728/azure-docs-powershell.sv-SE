# Set-AzAutomationRunbook

## SYNOPSIS
Ändrar en runbook.

## SYNTAX

```
Set-AzAutomationRunbook [-Name] <String> [-Description <String>] [-Tag <IDictionary>] [-LogProgress <Boolean>]
 [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzAutomationRunbook** ändrar konfigurationen av en Azure Automation-runbook i APS.

## EXEMPEL

### Exempel 1: Aktivera utförlig loggning för en runbook
```
PS C:\>Set-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook02" -LogVerbose $True -ResourceGroupName "ResourceGroup01"
```

Det här kommandot aktiverar utförlig loggning för jobben för den angivna runbooken i Azure Automation-kontot Contoso17.
