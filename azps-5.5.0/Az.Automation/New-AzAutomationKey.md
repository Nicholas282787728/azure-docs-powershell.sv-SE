# New-AzAutomationKey

## SYNOPSIS
Återskapar registreringsnycklar för ett automatiseringskonto.

## SYNTAX

```
New-AzAutomationKey [-KeyType] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzAutomationKey** återskapar registreringsnycklar för ett Azure Automation-konto.

## EXEMPEL

### Exempel 1: Generera en nyckel för ett automatiseringskonto
```
PS C:\>New-AzAutomationKey -KeyType Primary -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01"
```

Det här kommandot återskapar primärnyckeln för Azure Automation-kontot med namnet AutomationAccount01 i resursgruppen ResourceGroup01.
