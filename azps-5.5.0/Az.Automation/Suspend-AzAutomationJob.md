# Suspend-AzAutomationJob

## SYNOPSIS
Stänger av ett automatiseringsjobb.

## SYNTAX

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Suspend-AzAutomation Ett Azure Automation-jobb** avbryts.
Ange ett automatiseringsjobb som körs.
Om du vill återuppta ett pausat jobb använder Resume-AzAutomationJob-cmdleten.

## EXEMPEL

### Exempel 1: Avbryta ett jobb
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Det här kommandot stänger av jobbet som har det angivna ID:t.
