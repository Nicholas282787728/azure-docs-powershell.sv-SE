# Set-AzAutomationDscNode

## SYNOPSIS
Ändrar nodkonfigurationen som en DSC-nod är mappad till.

## SYNTAX

```
Set-AzAutomationDscNode -Id <Guid> -NodeConfigurationName <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzAutomationDscNode** ändrar nodkonfigurationen APS Desired State Configuration (DSC).
Azure Automation lagrar DSC-nodkonfiguration som ett konfigurationsdokument för hanterade objekt (MOF).

## EXEMPEL

### Exempel 1: Ändra nodkonfigurationsmappning
```
PS C:\>Set-AzAutomationDscNode -NodeConfigurationName "Contoso.NodeConfiguration01" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111c8a6067j8
```

Det här kommandot tilldelar nodkonfigurationen contoso.NodeConfiguration01 till noden som har den angivna GUID:en.
