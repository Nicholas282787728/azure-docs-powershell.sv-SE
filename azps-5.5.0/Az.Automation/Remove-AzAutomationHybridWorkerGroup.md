# Remove-AzAutomationHybridWorkerGroup

## SYNOPSIS
Tar bort hybridarbetaregruppen från Automation.

## SYNTAX

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzAutomationHybridWorkerGroup-cmdleten tas en hybridarbetaregrupp bort från Automation.

## EXEMPEL

### Exempel 1
Med det här kommandot tas en hybridarbetare bort med namnet.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
