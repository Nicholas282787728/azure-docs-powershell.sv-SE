## SYNOPSIS

## BESKRIVNING
Cmdleten **New-AzAutomationSchedule** skapar ett schema i Azure Automation.

## EXEMPEL

### Exempel 1: Skapa ett schema för ett enda tillfälle i lokal tid
```
PS C:\> $TimeZone = ([System.TimeZoneInfo]::Local).Id
PS C:\> New-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -StartTime "23:00" -OneTime -ResourceGroupName "ResourceGroup01" -TimeZone $TimeZone
```

Det första kommandot hämtar tidszons-ID från systemet och lagrar det i $TimeZone variabeln.
Det andra kommandot skapar ett schema som körs en gång på dagens datum kl. 23:00 i den angivna tidszonen.

### Exempel 2: Skapa ett återkommande schema
```
PS C:\> $StartTime = Get-Date "13:00:00"
PS C:\> $EndTime = $StartTime.AddYears(1)
PS C:\> New-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule02" -StartTime $StartTime -ExpiryTime $EndTime -DayInterval 1 -ResourceGroupName "ResourceGroup01"
```

Det första kommandot skapar ett datumobjekt med hjälp av cmdleten **Get-Date** och lagrar sedan objektet i $StartDate variabeln.
Ange en tid som är minst fem minuter i framtiden.
Det andra kommandot skapar ett datumobjekt med hjälp av cmdleten **Get-Date** och lagrar sedan objektet i $EndDate variabeln.
Kommandot anger en kommande tid.
Det slutliga kommandot skapar ett dagligt schema med namnet Schedule02 för att börja vid den tidpunkt som är lagrad i $StartDate och upphöra vid den tidpunkt som $EndDate.

### Exempel 3: Skapa ett återkommande veckoschema
```
PS C:\> $StartTime = (Get-Date "13:00:00").AddDays(1)
PS C:\> [System.DayOfWeek[]]$WeekDays = @([System.DayOfWeek]::Monday..[System.DayOfWeek]::Friday)
PS C:\> New-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule03" -StartTime $StartTime - WeekInterval 1 -DaysOfWeek $WeekDays -ResourceGroupName "ResourceGroup01"
```

Det första kommandot skapar ett datumobjekt med hjälp av cmdleten **Get-Date** och lagrar sedan objektet i $StartDate variabeln.
Det andra kommandot skapar en matris med veckodagar som innehåller måndag, tisdag, onsdag, torsdag och fredag.
Det slutliga kommandot skapar ett dagligt schema med namnet Schema03 som körs måndag till fredag varje vecka kl. 13:00. Schemat upphör aldrig att gälla.
