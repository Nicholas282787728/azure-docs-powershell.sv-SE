# Get-AzAutomationModule

## SYNOPSIS
Hämtar metadata för moduler från Automation.

## SYNTAX

### ByAll (standard)
```
Get-AzAutomationModule [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationModule [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutomationModule** hämtar metadata för moduler från Azure Automation.

## EXEMPEL

### Exempel 1: Hämta alla moduler
```
PS C:\>Get-AzAutomationModule -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01"
```

Med det här kommandot får du alla moduler i automatiseringskontot Contoso17.

### Exempel 2: Hämta en modul
```
PS C:\>Get-AzAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Med det här kommandot får du en modul som heter ContosoModule i automatiseringskontot Contoso17.
