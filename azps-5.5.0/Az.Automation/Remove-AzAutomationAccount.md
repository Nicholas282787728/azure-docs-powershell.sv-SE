# Remove-AzAutomationAccount

## SYNOPSIS
Tar bort ett automatiseringskonto.

## SYNTAX

```
Remove-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzAutomationAccount** tar bort ett Azure Automation-konto från en resursgrupp.
Mer information om automatiseringskonton finns i New-AzAutomationAccount cmdlet.

## EXEMPEL

### Exempel 1: Ta bort ett automatiseringskonto
```
PS C:\>Remove-AzAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

Med det här kommandot tas ett automatiseringskonto med namnet ContosoAutomationAccount bort utan att användaren behöver valideras.
