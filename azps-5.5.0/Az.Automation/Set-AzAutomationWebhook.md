# Set-AzAutomationWebhook

## SYNOPSIS
Ändrar en webhook för en automatiseringskörningskörningbok.

## SYNTAX

```
Set-AzAutomationWebhook [-Name] <String> [-IsEnabled] <Boolean> [[-Parameters] <IDictionary>] [-RunOn <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**Set-AzAutomationWebhook-cmdleten** ändrar en webhook för en Azure Automation-runbook.

## EXEMPEL

### Exempel 1: Inaktivera en webhook
```powershell
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -IsEnabled $False
```

Med det här kommandot inaktiveras webhook med namnet Webhook01 i automatiseringskontot AutomationAccount01.

### Exempel 2
```powershell
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -RunOn 'Windows'
```

Det här kommandot anger värdet Kör på för webhook och tvingar runbooken att köras på en Hybrid Worker-grupp med namnet Windows.

### Exempel 3
```powershell
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -RunOn $null
```

Det här kommandot uppdaterar värdet Kör på för webhook och tvingar runbooken att köras på en Azure Runbook-anställd.
