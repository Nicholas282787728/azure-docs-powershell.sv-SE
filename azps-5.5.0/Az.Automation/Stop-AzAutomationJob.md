# Stop-AzAutomationJob

## SYNOPSIS
Stoppar ett automatiseringsjobb.

## SYNTAX

```
Stop-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Stop-AzAutomation Enlsson** stoppar ett Azure Automation-jobb.
Ange ett automatiseringsjobb som körs.

## EXEMPEL

### Exempel 1: Avsluta ett jobb
```
PS C:\>Stop-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Det här kommandot avbryter jobbet som har det angivna ID:t.
