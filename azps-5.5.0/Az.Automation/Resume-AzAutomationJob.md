# Resume-AzAutomationJob

## SYNOPSIS
Återupptar ett automatiserat automatiseringsjobb.

## SYNTAX

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Resume-AzAutomation En** cmdlet återupptar Azure Automation-jobbet pausat.
Ange det inaktiverade jobbet.
Om du vill avbryta ett jobb använder Suspend-AzAutomationJob-cmdleten.

## EXEMPEL

### Exempel 1: Återuppta ett avbrutet jobb
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Det här kommandot återupptar jobbet som har det angivna ID:t.
