# Remove-AzAutomationModule

## SYNOPSIS
Tar bort en modul från Automation.

## SYNTAX

```
Remove-AzAutomationModule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzAutomationModule** tar bort en modul från ett automatiseringskonto i Azure Automation.

## EXEMPEL

### Exempel 1: Ta bort en modul
```
PS C:\>Remove-AzAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Med det här kommandot tas en modul med namnet ContosoModule bort från automatiseringskontot Contoso17.
