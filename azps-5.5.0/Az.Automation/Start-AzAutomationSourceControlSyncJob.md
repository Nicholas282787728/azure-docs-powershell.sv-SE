# Start-AzAutomationSourceControlSyncJob

## SYNOPSIS
Startar ett synkroniseringsjobb för Azure Automation-källkontroll.

## SYNTAX

```
Start-AzAutomationSourceControlSyncJob -SourceControlName <String> [-JobId <Guid>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Start-AzAutomationSourceControlSyncJob-cmdleten startas ett synkroniseringsjobb för källkontroll i Azure Automation för namnet på den givna källkontrollen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Start-AzAutomationSourceControlSyncJob -ResourceGroupName "rg1" `
                                                    -AutomationAccountName "devAccount" `
                                                    -Name "VSTSNative"

SourceControlSyncJobId               SyncType Status  StartTime EndTime
----------------------               -------- ------  --------- -------
b51aed78-bef6-40d4-a966-cd45fd5af576 FullSync Running
```
