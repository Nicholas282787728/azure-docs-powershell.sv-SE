# Export-AzAutomationRunbook

## SYNOPSIS
Exporterar en automatiseringskörningsbok.

## SYNTAX

```
Export-AzAutomationRunbook [-Name] <String> [-Slot <String>] [-OutputFolder <String>] [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Export-AzAutomationRunbook** exporterar en Azure Automation-runbook till en wps_2-skriptfil (.ps1), för wps_2- eller wps_2-arbetsflödeskörningsböcker eller till en grafisk runbook-fil (.graphrunbook) för grafiska körningsböcker.
Namnet på runbooken blir namnet på den exporterade filen.

## EXEMPEL

### Exempel 1: Exportera en runbook
```
PS C:\>Export-AzAutomationRunbook -ResourceGroupName "ResourceGroup01" -AutomationAccountName "ContosoAutomationAccount" -Name "Runbook03" -Slot "Published" -OutputFolder "C:\Users\PattiFuller\Desktop"
```

Det här kommandot exporterar den publicerade versionen av en automatiseringskörningsbok till en användares skrivbord.
