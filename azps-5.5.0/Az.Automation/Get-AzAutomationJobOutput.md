# Get-AzAutomationJobOutput

## SYNOPSIS
Hämtar utdata för ett automatiseringsjobb.

## SYNTAX

```
Get-AzAutomationJobOutput [-Id] <Guid> [-Stream <StreamType>] [-StartTime <DateTimeOffset>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutomation EttOutput** får resultatet av ett Azure Automation-jobb.

## EXEMPEL

### Exempel 1: Hämta utdata för ett automatiseringsjobb
```
PS C:\>Get-AzAutomationJobOutput -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01" -Stream "Any"
```

Det här kommandot hämtar alla utdata för jobbet som har det angivna ID:t.
