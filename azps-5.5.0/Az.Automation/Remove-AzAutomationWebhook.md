# Remove-AzAutomationWebhook

## SYNOPSIS
Tar bort en webhook från en automatiseringskörningsbok.

## SYNTAX

```
Remove-AzAutomationWebhook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzAutomationWebhook** tar bort en webhook från en Azure Automation-runbook.
Webhook tas bort.

## EXEMPEL

### Exempel 1: Ta bort en webhook
```
PS C:\>Remove-AzAutomationWebhook -Name "Webhook11" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -Force
```

Med det här kommandot tas en webhook med namnet Webhook11 bort i automatiseringskontot AutomationAccount01.
Kommandot anger *force-parametern.*
Därför uppmanas du inte att bekräfta.
