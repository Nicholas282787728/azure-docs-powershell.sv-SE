# Get-AzAutomationDscConfiguration

## SYNOPSIS
Hämtar DSC-konfigurationer från Automation.

## SYNTAX

### ByAll (standard)
```
Get-AzAutomationDscConfiguration [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByConfigurationName
```
Get-AzAutomationDscConfiguration [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutomationDscConfiguration** får DSC-konfigurationer (APS Desired State Configuration) från Azure Automation.

## EXEMPEL

### Exempel 1: Hämta alla DSC-konfigurationer
```
PS C:\>Get-AzAutomationDscConfiguration -ResourceGroupName "ResourceGroup03" -AutomationAccountName "Contoso17"
```

Det här kommandot hämtar metadata för alla DSC-konfigurationer i automatiseringskontot Contoso17.

### Exempel 2: Hämta en DSC-konfiguration efter namn
```
PS C:\>Get-AzAutomationDscConfiguration -ResourceGroupName "ResourceGroup03" -AutomationAccountName "Contoso17" -Name "ContosoConfiguration"
```

Det här kommandot hämtar metadata för en DSC-konfiguration med namnet MyConfiguration i automatiseringskontot Contoso17.
