# Get-AzAutomationRegistrationInfo

## SYNOPSIS
Hämtar registreringsinformation för att onboarda en DSC-nod eller hybridarbetare till Automation.

## SYNTAX

```
Get-AzAutomationRegistrationInfo [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutomationRegistrationInfo** får slutpunkten och nycklarna som krävs för att introducera en DSC-nod (Desired State Configuration) eller hybridarbetare i ett Azure Automation-konto.

## EXEMPEL

### Exempel 1: Hämta registreringsinformation
```
PS C:\>Get-AzAutomationRegistrationInfo -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01"
```

Med det här kommandot får du registreringsinformationen för automatiseringskontot AutomationAccount01 i resursgruppen ResourceGroup01.
