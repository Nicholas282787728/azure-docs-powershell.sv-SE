# Get-AzAutomationDscOnboardingMetaconfig

## SYNOPSIS
Skapar .mof-filer för metakonfiguration.

## SYNTAX

```
Get-AzAutomationDscOnboardingMetaconfig [-OutputFolder <String>] [-ComputerName <String[]>] [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAutomationDscOnboardingMetaconfig** skapar DSC-filer (APS Desired State Configuration) metakonfiguration Managed Object Format (MOF).
Den här cmdleten skapar en .mof-fil för varje datornamn som du anger.
Cmdleten skapar en mapp för .mof-filerna.
Du kan köra cmdleten Set-DscLocalConfigurationManager den här mappen för att introducera dessa datorer i ett Azure Automation-konto som DSC-noder.

## EXEMPEL

### Exempel 1: Onboard servers to Automation DSC
```
PS C:\>Get-AzAutomationDscOnboardingMetaconfig -ResourceGroupName "ResourceGroup03" -AutomationAccountName "Contoso17" -ComputerName "Server01", "Server02" -OutputFolder "C:\Users\PattiFuller\Desktop" 
PS C:\> Set-DscLocalConfigurationManager -Path "C:\Users\PattiFuller\Desktop\DscMetaConfigs" -ComputerName "Server01", "Server02"
```

Med det första kommandot skapas DSC-metakonfigurationsfiler för två servrar för automatiseringskontot Contoso17.
Kommandot sparar dessa filer på ett skrivbord.
Det andra kommandot använder cmdleten **Set-DscLocalConfigurationManager** för att använda metakonfigurationen på de angivna datorerna för att introducera dem som DSC-noder.
