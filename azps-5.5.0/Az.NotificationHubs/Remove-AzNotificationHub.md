# Remove-AzNotificationHub

## SYNOPSIS
Tar bort en befintlig meddelandehubben.

## SYNTAX

```
Remove-AzNotificationHub [-ResourceGroup] <String> [-Namespace] <String> [-NotificationHub] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzNotificationHub** tar bort ett befintligt meddelandenav.
Meddelandehubben används för att skicka push-meddelanden till flera klienter oavsett vilken plattform som används av dessa klienter.
Plattformarna omfattar, men är inte begränsade till: iOS, Android, Windows Phone 8 och Windows Store.
Meddelandehubben motsvarar på ungefär samma sätt enskilda appar: var och en av dina appar har vanligtvis ett eget meddelandenav.
Du kan ta bort ett befintligt meddelandenav med hjälp av **cmdleten Remove-AzNotificationHub.**
När ett nav har tagits bort kan du inte längre använda navet för att skicka push-meddelanden till användare.

## EXEMPEL

### Exempel 1: Ta bort ett meddelandenav
```
PS C:\>Remove-AzNotificationHub -Namespace "ContosoNamespace" -ResourceGroup "ContosoNotificationsGroup" -NotificationHub "ContosoInternalHub"
```

Det här kommandot tar bort meddelandehubben med namnet ContosoInternalHub.
Om du vill ta bort navet måste du ange namnområdet där navet finns samt resursgruppen som navet är tilldelat till.
