# Get-AzBlockchainConsortium

## SYNOPSIS
Listar de tillgängliga konsorterna för en prenumeration.

## SYNTAX

```
Get-AzBlockchainConsortium -Location <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Listar de tillgängliga konsorterna för en prenumeration.

## EXEMPEL

### Exempel 1: Skaffa blockchainkonsorter.
```powershell
PS C:\> Get-AzBlockchainConsortium -Location eastus

```

Det här kommandot listar konsorterna under en prenumeration för en viss plats.
