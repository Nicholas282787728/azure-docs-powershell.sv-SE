# Update-AzBlockchainTransactionNode

## SYNOPSIS
Uppdatera transaktionsnoden.

## SYNTAX

### UpdateExpanded (standard)
```
Update-AzBlockchainTransactionNode -BlockchainMemberName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-FirewallRule <IFirewallRule[]>] [-Password <SecureString>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzBlockchainTransactionNode -InputObject <IBlockchainIdentity> [-FirewallRule <IFirewallRule[]>]
 [-Password <SecureString>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Uppdatera transaktionsnoden.

## EXEMPEL

### Exempel 1: Uppdatera en transcationnod
```powershell
PS C:\> $tag = @{'key1'='update'}
PS C:\> Update-AzBlockchainTransactionNode -BlockchainMemberName dolauli002 -Name transacnode002 -ResourceGroupName testgroup -Tag $tag

Name           Type                                                    Location
----           ----                                                    --------
transacnode002 Microsoft.Blockchain/blockchainMembers/transactionNodes eastus
```

Det här kommandot uppdaterar en transaktionsnod.

### Exempel 2: Uppdatera en transcationnod
```powershell
PS C:\> $tag = @{'key2'='update'}
PS C:\> $tNode = Get-AzBlockchainMember -BlockchainMemberName dolauli002 -ResourceGroupName testgroup -Name transacnode002
PS C:\> Update-AzBlockchainTransactionNode -InputObject $tNode -Tag $tag

Name           Type                                                    Location
----           ----                                                    --------
transacnode002 Microsoft.Blockchain/blockchainMembers/transactionNodes eastus
```

Det här kommandot uppdaterar en transaktionsnod.

## ANTECKNINGAR

ALIAS

KOMPLEXA PARAMETEREGENSKAPER

Skapa de parametrar som beskrivs nedan genom att skapa en hash-tabell som innehåller rätt egenskaper. Om du vill ha information om hash-tabeller kör du Get-Help about_Hash_Tables.


FIREWALLRULE <IFirewallRule[]>: Hämtar eller anger brandväggsregler.
  - `[EndIPAddress <String>]`: Hämtar eller anger slut-IP-adressen för brandväggens regelintervall.
  - `[RuleName <String>]`: Hämtar eller anger namnet på brandväggsreglerna.
  - `[StartIPAddress <String>]`: Hämtar eller anger start-IP-adressen för brandväggens regelintervall.

INPUTOBJECT <IBlockchainIdentity> : Identity Parameter
  - `[BlockchainMemberName <String>]`: Blockchain-medlemsnamn.
  - `[Id <String>]`: Resursidentitetssökväg
  - `[Location <String>]`: Platsnamn.
  - `[OperationId <String>]`: Åtgärds-ID.
  - `[ResourceGroupName <String>]`: Namnet på resursgruppen som innehåller resursen. Du kan hämta det här värdet från Azure Resource Manager API eller portalen.
  - `[SubscriptionId <String>]`: Får det prenumerations-ID som unikt identifierar Microsoft Azure-prenumerationen. Prenumerations-ID är en del av URI:n för varje servicesamtal.
  - `[TransactionNodeName <String>]`: Namn på transaktionsnod.
