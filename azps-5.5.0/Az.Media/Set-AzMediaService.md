# Set-AzMediaService

## SYNOPSIS
Ändrar angivna egenskaper för en befintlig medietjänst.

## SYNTAX

```
Set-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Tag <Hashtable>]
 [-StorageAccounts <PSStorageAccount[]>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzMediaService** ändrar angivna egenskaper för en befintlig medietjänst.

## EXEMPEL

### Exempel 1: Ändra en befintlig medietjänst
```
PS C:\>$Tags = @{"tag1" = "value1"; "tag2" = "value2"}
Set-AzMediaService -ResourceGroupName "ResourceGroup123" -AccountName "MediaService001" -Tag $Tags -StorageAccounts $StorageAccounts
```

Det första kommandot skapar en serie taggar och lagrar de taggarna i variabeln $Tags.
Det här andra kommandot uppdaterar medietjänsten med namnet MediaService001 som tillhör resursgruppen ResourceGroup123 med de taggar som lagras i $Tags variabeln.
Kommandot använder också en matris med lagringskonfigurationsobjekt som lagras i $StorageAccounts variabeln.
