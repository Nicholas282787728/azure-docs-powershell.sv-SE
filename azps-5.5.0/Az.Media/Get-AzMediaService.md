# Get-AzMediaService

## SYNOPSIS
Hämtar information om en medietjänst.

## SYNTAX

### ResourceGroupParameterSet
```
Get-AzMediaService [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzMediaService** hämtar information om en medietjänst.

## EXEMPEL

### Exempel 1: Hämta alla medietjänster i en resursgrupp
```
PS C:\>Get-AzMediaService -ResourceGroupName "ResourceGroup001"
```

Det här kommandot hämtar egenskaper för alla medietjänster i resursgruppen med namnet ResourceGroup001.

### Exempel 2: Hämta egenskaper för medietjänsten
```
PS C:\>Get-AzMediaService -ResourceGroupName "ResourceGroup002" -AccountName "MediaService1"
```

Det här kommandot hämtar egenskaperna för medietjänsten med namnet MediaService1 som tillhör resursgruppen Med namnet ResourceGroup002.
