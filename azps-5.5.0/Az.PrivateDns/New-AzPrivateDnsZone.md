# New-AzPrivateDnsZone

## SYNOPSIS
Skapar en ny privat DNS-zon.

## SYNTAX

```
New-AzPrivateDnsZone -ResourceGroupName <String> -Name <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzPrivateDnsZone** skapar en ny privat DNS-zon (Domain Name System) i den angivna resursgruppen. Du måste ange ett unikt privat DNS-zonnamn för *parametern Name,* annars returnerar cmdleten ett fel. När zonen har skapats använder du cmdleten New-AzPrivateDnsRecordSet för att skapa postuppsättningar i zonen.
Du kan använda *parametern Confirm* och $ConfirmPreference Windows PowerShell för att styra om cmdleten uppmanar dig att bekräfta.

## EXEMPEL

### Exempel 1: Skapa en privat DNS-zon
```
PS C:\>$Zone = New-AzPrivateDnsZone -Name "myzone.com" -ResourceGroupName "MyResourceGroup"


This command creates a new private DNS zone named myzone.com in the specified resource group, and then
stores it in the $Zone variable. $Zone object looks something like this,

Name                          : myzone.com
ResourceId                    : "/subscriptions/xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/MyResourceGroup/PrivateZones/myzone.com"
ResourceGroupName             : MyResourceGroup
Location                      : 
Etag                          : xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
Tags                          : {}
NumberOfRecordSets            : 1
MaxNumberOfRecordSets         : 5000
```
