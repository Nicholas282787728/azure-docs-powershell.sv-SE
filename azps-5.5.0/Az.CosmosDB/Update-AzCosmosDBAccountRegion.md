# Update-AzCosmosDBAccountRegion

## SYNOPSIS
Uppdatera regioner för ett SDB-konto.

## SYNTAX

### ByNameParameterSet (standard)
```
Update-AzCosmosDBAccountRegion -ResourceGroupName <String> -Name <String> [-Location <String[]>]
 [-LocationObject <PSLocation[]>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Update-AzCosmosDBAccountRegion [-Location <String[]>] [-LocationObject <PSLocation[]>] -ResourceId <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObjectParameterSet
```
Update-AzCosmosDBAccountRegion [-Location <String[]>] [-LocationObject <PSLocation[]>]
 -InputObject <PSDatabaseAccountGetResults> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Uppdatera regioner för ett SDB-konto. Plats kan tillhandahållas antingen som ett objekt av typen PSLocation eller som strängar med platsnamn sorterat efter redundansprioritering.
LocationObject-parametern förväntar sig att listan över aktuella platser (failover prioritiies ingår) läggs till av de nya LocationObjects som motsvarar nya platser som ska läggas till.
Platsparametern förväntar sig listan över aktuell plats (ordnade efter redundansprioritet) och de nya platserna. Observera att vi bara stöder tillägg av regioner. Ange antingen Location eller LocationObject.
