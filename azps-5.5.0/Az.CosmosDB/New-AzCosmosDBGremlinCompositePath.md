# New-AzCosmosDBGremlinCompositePath

## SYNOPSIS
Skapar ett nytt objekt av typen PSCompositePath. Det kan skickas som ett parametervärde för Set-AzCosmosDBGremlinGraph.

## SYNTAX

```
New-AzCosmosDBGremlinCompositePath [-Path <String>] [-Order <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Objekt som motsvarar Gremlin API:s CompositePath.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzCosmosDBGremlinCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
