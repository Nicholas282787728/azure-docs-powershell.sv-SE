# New-AzCosmosDBCassandraClusterKey

## SYNOPSIS
Skapar en ny StudiosDB Ochandra Cluster Key.

## SYNTAX

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**New-AzCosmosDBCassandraClusterKey** skapar en ny AdamsDBAndra Cluster Key.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
