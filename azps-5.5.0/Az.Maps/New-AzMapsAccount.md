# New-AzMapsAccount

## SYNOPSIS
Skapar ett Azure Maps-konto.

## SYNTAX

```
New-AzMapsAccount [-ResourceGroupName] <String> [-Name] <String> [-SkuName] <String> [-Tag <Hashtable[]>]
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med New-AzMapsAccount-cmdleten skapas ett Azure Maps-konto med den angivna SKU:n.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzMapsAccount -ResourceGroupName MyResourceGroup -Name MyAccount -SkuName S0 -Tags @{Name="test";Value="true"}

ResourceGroupName AccountName Id
----------------- ----------- --
MyResourceGroup   MyAccount   /subscriptions/21a9967a-e8a9-4656-a70b-96ff1c4d05a0/resourceGroups/MyResourceGroup/providers/Microsoft.Maps/accounts/MyAccount
```

Skapar ett nytt Azure Maps-konto med namnet MyAccount i resursgruppen MyResourceGroup med SKU S0 och en tagg.
