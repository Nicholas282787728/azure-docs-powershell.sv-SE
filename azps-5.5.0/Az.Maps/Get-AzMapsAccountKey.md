# Get-AzMapsAccountKey

## SYNOPSIS
Hämtar API-nycklar för ett konto.
Dessa nycklar är den autentiseringsmekanism som används i efterföljande anrop till Azure Maps.

## SYNTAX

### NameParameterSet (standard)
```
Get-AzMapsAccountKey [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### InputObjectParameterSet
```
Get-AzMapsAccountKey [-InputObject <PSMapsAccount>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzMapsAccountKey [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den Get-AzMapsAccountKey cmdleten hämtar API-nycklarna för ett etablerat Azure Maps-konto.
Ett Azure Maps-konto har två API-nycklar: Primär och sekundär.
Nycklarna aktiverar interaktion med slutpunkten för Azure Maps-kontot.
Använd New-AzMapsAccountKey (New-AzMapsAccountKey.md)för att generera en nyckel.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzMapsAccountKey -ResourceGroupName MyResourceGroup -Name MyAccount

PrimaryKey                                  SecondaryKey
----------                                  ------------
******************************************* *******************************************
```

Returnerar nycklarna för kontot MyAccount i resursgruppen MyResourceGroup.

### Exempel 2
```powershell
PS C:\> Get-AzMapsAccountKey -ResourceId /subscriptions/21a9967a-e8a9-4656-a70b-96ff1c4d05a0/resourceGroups/MyResourceGroup/providers/Microsoft.Maps/accounts/MyAccount

PrimaryKey                                  SecondaryKey
----------                                  ------------
******************************************* *******************************************
```

Returnerar nycklarna för det angivna Azure Maps-kontot.
