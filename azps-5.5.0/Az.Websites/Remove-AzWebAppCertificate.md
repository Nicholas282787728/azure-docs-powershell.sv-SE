# Remove-AzWebAppCertificate

## SYNOPSIS
Skapar ett programtjänst hanterat certifikat för en Azure Web App. 

## SYNTAX

```
Remove-AzWebAppCertificate [-ResourceGroupName] <String> [-ThumbPrint] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzWebAppCertificate** skapar ett Azure App Service Managed Certificate

## EXEMPEL

### Exempel 1
```powershell
PS C:\>Remove-AzWebAppCertificate -ResourceGroupName Default-Web-WestUS -Thumbprint "E3A38EBA60CAA1C162785A2E1C44A15AD450199C3" 
```

Det här kommandot tar bort det hanterade certifikatet för programtjänsten för det givna webbprogrammet.
