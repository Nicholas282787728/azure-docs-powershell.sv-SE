# New-AzWebAppDatabaseBackupSetting

## SYNOPSIS

## SYNTAX

```
New-AzWebAppDatabaseBackupSetting [-Name] <String> [-DatabaseType] <String> [-ConnectionString] <String>
 [[-ConnectionStringName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzWebAppDatabaseBackupSetting** skapar en ny inställning för säkerhetskopiering av Azure Web App.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzWebAppDatabaseBackupSetting -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -ConnectionString "MyConnectionString" -DatabaseType "SqlAzure"
```

Skapar en inställning för säkerhetskopiering av databasen (anslutningssträng) av typen SqlAzure för den angivna appen ContosoWebApp som finns inom resursgruppen Default-Web-WestUS.

### Exempel 2

Med New-AzWebAppDatabaseBackupSetting-cmdleten skapas en ny inställning för säkerhetskopiering av Azure Web App. (autogenererat)

```powershell <!-- Aladdin Generated Example --> 
New-AzWebAppDatabaseBackupSetting -ConnectionString 'MyConnectionString' -ConnectionStringName <String> -DatabaseType 'SqlAzure' -Name 'ContosoWebApp'
```
