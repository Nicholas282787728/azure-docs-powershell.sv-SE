# Reset-AzWebAppPublishingProfile

## SYNOPSIS

## SYNTAX

### S1
```
Reset-AzWebAppPublishingProfile [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Reset-AzWebAppPublishingProfile [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Reset-AzWebAppPublishingProfile** återställer publiceringsprofilen för den angivna webbappen.

## EXEMPEL

### Exempel 1

I följande exempel återställs publiceringsprofilen för Web App IpRule som är kopplad till resursgruppen MyResourceGroup.

```powershell <!-- Aladdin Generated Example --> 
Reset-AzWebAppPublishingProfile -Name IpRule -ResourceGroupName MyResourceGroup
```
