# Add-AzWebAppTrafficRouting

## SYNOPSIS
Lägga till en routningsregel på plats.

## SYNTAX

```
Add-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RoutingRule <Hashtable>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Add-AzWebAppTrafficRouting** lägger till en routningsregel i en Azure Web App-slot.

## EXEMPEL

### Exempel 1 Lägga till en routningsregel för överföring av 15 % av produktionstrafik till Stg-plats
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=15;Name='Stg'}
```

Det här kommandot lägger till en routningsregel för överföring av 15 % av produktionstrafik till Stg-plats

### Exempel 2 Lägg till en routningsregel för att överföra produktionstrafik till Stg-intervall från 50 % till 90 % stegvis.
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=50;ChangeIntervalInMinutes=1;
MinReroutePercentage=50;MaxReroutePercentage=90;Name='Stg';ChangeStep=10}
```

Det här kommandot lägger till en routningsregel för att överföra produktionstrafik till Stg-fackintervall från 50 % till 90 % stegvis.
