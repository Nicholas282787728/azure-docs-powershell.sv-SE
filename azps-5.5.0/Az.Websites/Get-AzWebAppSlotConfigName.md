# Get-AzWebAppSlotConfigName

## SYNOPSIS
Hämta listan över Namn på Web App Slot-konfiguration

## SYNTAX

### S1
```
Get-AzWebAppSlotConfigName [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzWebAppSlotConfigName [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzWebAppSlotConfigName** hämtar listan med namn på appinställningar och anslutningssträngar som för närvarande markeras som platsinställningar

## EXEMPEL

### Exempel 1
```powershell
PS C:\>Get-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Det här kommandot hämtar appinställningar och anslutningssträngar som gäller webbappen ContosoSite som är kopplade till resursgruppen Default-Web-WestUS
