# Get-AzAppServicePlan

## SYNOPSIS
Hämtar en Azure App Service-plan i den angivna resursgruppen.

## SYNTAX

### S1
```
Get-AzAppServicePlan [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzAppServicePlan [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzAppServicePlan** får ett Azure App Service-abonnemang i den angivna resursgruppen.

## EXEMPEL

### Exempel 1: Hämta en apptjänstplan från en resursgrupp
```
PS C:\>Get-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Det här kommandot hämtar App Service-planen med namnet ContosoASP som tillhör resursgruppen Default-Web-WestUS.

### Exempel 2: Hämta alla App Service-abonnemang på en plats
```
PS C:\>Get-AzAppServicePlan -Location "West US"
```

Med det här kommandot får du alla App Service-abonnemang i regionen "Väst-USA".
