# Remove-AzWebAppSlot

## SYNOPSIS

## SYNTAX

### S1
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String> [-Slot] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzWebAppSlot** tar bort en Azure Web App Slot förutsatt att resursgruppen och Web App-namnet ingår.
Den här cmdleten tar som standard även bort alla platser och mått.

## EXEMPEL

### Exempel 1: Ta bort en Web App-plats
```
PS C:\>Remove-AzWebAppSlot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite" -Slot "Slot001"
```

Det här kommandot tar bort den plats som heter Slot001 som är kopplad till Web App ContosoSite som tillhör resursgruppen Default-Web-WestUS.
