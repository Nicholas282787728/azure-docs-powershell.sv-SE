# Remove-AzWebApp

## SYNOPSIS
Tar bort en Azure Web App.

## SYNTAX

### S1
```
Remove-AzWebApp [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzWebApp [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzWebApp** tar bort en Azure Web App förutsatt att resursgruppen och webbappens namn tas bort.
Den här cmdleten tar som standard även bort alla platser och mått.

## EXEMPEL

### Exempel 1: Ta bort en webbapp
```
PS C:\>Remove-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Det här kommandot tar bort Azure Web App med namnet ContosoSite som tillhör resursgruppen Default-Web-WestUS.
