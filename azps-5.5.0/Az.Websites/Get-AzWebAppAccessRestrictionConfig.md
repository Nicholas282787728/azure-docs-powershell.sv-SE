# Get-AzWebAppAccessRestrictionConfig

## SYNOPSIS
Hämtar Access Restiction-konfiguration för en Azure Web App.

## SYNTAX

```
Get-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String> [[-SlotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzWebAppAccessRestrictionConfig** får konfiguration av åtkomstbegränsningar om en Azure Web App.

## EXEMPEL

### Exempel 1: Hämta en konfiguration för åtkomstbegränsningar i Web App från en resursgrupp
```
PS C:\>Get-AzWebAppAccessRestrictionConfig -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Det här kommandot får åtkomstbegränsningskonfigurationen för en webbapp med namnet ContosoSite som tillhör resursgruppen Default-Web-WestUS.
