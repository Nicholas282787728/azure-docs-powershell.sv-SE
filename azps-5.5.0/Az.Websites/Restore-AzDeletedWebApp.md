# Restore-AzDeletedWebApp

## SYNOPSIS
Återställer en borttagna webbapp till en ny eller befintlig webbapp.

## SYNTAX

### FromDeletedResourceName (standard)
```
Restore-AzDeletedWebApp [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>] [-Location <String>]
 [-DeletedId <String>] [-TargetResourceGroupName <String>] [-TargetName <String>] [-TargetSlot <String>]
 [-TargetAppServicePlanName <String>] [-RestoreContentOnly] [-UseDisasterRecovery] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### FromDeletedApp
```
Restore-AzDeletedWebApp [-TargetResourceGroupName <String>] [-DeletedId <String>] [-TargetName <String>] 
 [-TargetSlot <String>] [-TargetAppServicePlanName <String>] [-RestoreContentOnly] [-UseDisasterRecovery] 
 [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-InputObject] <PSAzureDeletedWebApp> 
 [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Restore-AzDeletedWebApp** återställer en borttaget webbapp. Webbappen som anges av TargetResourceGroupName, TargetName och TargetSlot skrivs över med innehållet och inställningarna för den borttagna webbappen. Om målparametrar inte anges fylls de automatiskt i med den borttagna webbappens resursgrupp, namn och plats. Om målwebbappen inte finns skapas den automatiskt i den apptjänstplan som anges av TargetAppServicePlanName. Parametern RestoreContentOnly-växeln kan användas för att återställa bara de borttagna appfilerna utan appinställningarna.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Restore-AzDeletedWebApp -ResourceGroupName Default-Web-WestUS -Name ContosoApp -TargetAppServicePlanName ContosoPlan
```

Återställer en borttagna app med namnet ContosoApp som tillhör resursgruppen Default-Web-WestUS. En ny app med samma namn och resursgrupp skapas i apptjänstplanen ContosoPlan, och de borttagna appens filer och inställningar återställs till den.

### Exempel 2
```powershell
PS C:\> Restore-AzDeletedWebApp -ResourceGroupName Default-Web-WestUS -Name ContosoApp -Slot Staging -TargetResourceGroupName Default-Web-EastUS -TargetName ContosoRestore -RestoreContentOnly
```

Återställer mellanlagringsplatsen för en borttagna app med namnet ContosoApp som tillhör resursgruppen Default-Web-WestUS. Webbappen ContosoRestore som tillhör resursgruppen Default-Web-EastUS skrivs över. Inställningarna för borttagna webbappar återställs inte.

###Exempel 3
```powershell
PS C:\> Restore-AzDeletedWebApp -ResourceGroupName Default-Web-WestUS -Name ContosoApp -DeletedId /subscriptions/00000000-0000-0000-0000-000000000000/providers/Microsoft.Web/locations/location/deletedSites/1234 -TargetAppServicePlanName ContosoPlan
```

Om det finns två borttagna appar med samma namn (ContosoApp) får vi information om båda webbplatserna och återställer appen ContosoRestore med den app vi väljer genom att ringa Återställ med Id.

###Exempel 4
```powershell
PS C:\> $deletedSite = Get-AzDeletedWebApp -ResourceGroupName Default-Web-WestUS -Name ContosoApp
PS C:\> Restore-AzDeletedWebApp -TargetResourceGroupName Default-Web-EastUS -TargetName ContosoRestore -TargetAppServicePlanName ContosoPlan -InputObject $deletedSite[0]
```

Om det finns två borttagna appar med samma namn (ContosoApp) får vi information om båda webbplatserna och återställer appen ContosoRestore med valfri app genom att anropa Återställ med information om InputObject (Deletedsite)
