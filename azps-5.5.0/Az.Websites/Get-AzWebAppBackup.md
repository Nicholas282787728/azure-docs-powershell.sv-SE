# Get-AzWebAppBackup

## SYNOPSIS

## SYNTAX

### FromResourceName
```
Get-AzWebAppBackup [-BackupId] <String> [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### FromWebApp
```
Get-AzWebAppBackup [-BackupId] <String> [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzWebAppBackup** hämtar den angivna säkerhetskopian av en Azure Web App.

## EXEMPEL

### Exempel 1
```powershell
PS C:\>Get-AzWebAppBackup -ResourceGroupName "Default-Web-WestUS" -Name "WebAppStandard" -BackupId "12345"
```

Det här kommandot hämtar säkerhetskopian med ID "12345" från webbappen WebAppStandard som tillhör resursgruppen Default-Web-WestUS.
