# Remove-AzWebAppAccessRestrictionRule

## SYNOPSIS
Tar bort en regel för åtkomstbegränsning från en Azure Web App.

## SYNTAX

```
Remove-AzWebAppAccessRestrictionRule [-ResourceGroupName] <String> [-WebAppName] <String> [-Name <String>]
 [-Action <String>] [-SlotName <String>] [-TargetScmSite] [-IpAddress <String>] [-SubnetName <String>]
 [-VirtualNetworkName <String>] [-SubnetId <String>] [-ServiceTag <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzWebAppAccessRestrictionRule** tar bort en åtkomstbegränsningsregel från en Azure Web App

## EXEMPEL

### Exempel 1: Ta bort en begränsningsregel för Web App Access
```
PS C:\>Remove-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" -Name IpRule
```

Det här kommandot tar bort begränsningsregeln för IpRule-åtkomst från Azure Web App med namnet ContosoSite som tillhör resursgruppen Default-Web-WestUS.

### Exempel 2: Ta bort en begränsningsregel för tjänsttaggar i Web App Access
```
PS C:\>Remove-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" -ServiceTag AzureFrontDoor.Backend
```

Med det här kommandot tas åtkomstbegränsningsregeln bort med ServiceTag är lika med AzureFrontDoor.Backend från Azure Web App med namnet ContosoSite som tillhör resursgruppen Default-Web-WestUS.
