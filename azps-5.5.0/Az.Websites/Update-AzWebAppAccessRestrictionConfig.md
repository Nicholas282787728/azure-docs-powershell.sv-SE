# Update-AzWebAppAccessRestrictionConfig

## SYNOPSIS
Uppdaterar arvet för huvudwebbplatsens Access Restiction-konfiguration till SCM-webbplatsen för en Azure Web App.

## SYNTAX

### InputValuesParameterSet (standard)
```
Update-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String>
 [-ScmSiteUseMainSiteRestrictionConfig] [-SlotName <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Update-AzWebAppAccessRestrictionConfig [-PassThru] -InputObject <PSAccessRestrictionConfig>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Update-AzWebAppAccessRestrictionConfig** uppdaterar konfiguration för åtkomstbegränsning för en Azure Web App.

## EXEMPEL

### Exempel 1: Uppdatera en SCM-webbplats för Web App så att åtkomstbegränsningar från huvudwebbplatsen används

I följande exempel uppdateras en webbapp med namnet IpRule som tillhör resursgruppen MyResourceGroup för att använda åtkomstbegränsningskonfiguration för huvudwebbplatsen på SCM-webbplatsen.

```powershell <!-- Aladdin Generated Example --> 
Update-AzWebAppAccessRestrictionConfig -Name IpRule -ResourceGroupName MyResourceGroup -ScmSiteUseMainSiteRestrictionConfig
```
