# Suspend-AzAnalysisServicesServer

## SYNOPSIS
Stänger av en instans av Analysis Services-servern

## SYNTAX

```
Suspend-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Suspend-AzAnalysisServicesServer-cmdleten avbryts en instans av Analysis Services-servern

## EXEMPEL

### Exempel 1
```
PS C:\> Suspend-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Det här kommandot stänger av en aktiv server med namnet testserver i resourcegroup-testgruppen
