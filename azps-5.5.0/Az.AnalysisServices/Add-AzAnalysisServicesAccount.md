# Add-AzAnalysisServicesAccount

## SYNOPSIS
Lägger till ett autentiserat konto som du kan använda för Azure Analysis Services-server cmdlet-begäranden.

## SYNTAX

### UserParameterSetName (standard)
```
Add-AzAnalysisServicesAccount [[-RolloutEnvironment] <String>] [[-Credential] <PSCredential>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ServicePrincipalWithPasswordParameterSetName
```
Add-AzAnalysisServicesAccount [-RolloutEnvironment] <String> [-Credential] <PSCredential> [-ServicePrincipal]
 -TenantId <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ServicePrincipalWithCertificateParameterSetName
```
Add-AzAnalysisServicesAccount [-RolloutEnvironment] <String> [-ServicePrincipal] -TenantId <String>
 -ApplicationId <String> -CertificateThumbprint <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den Add-AzAnalysisServicesAccount cmdleten används för att logga in på en instans av Azure Analysis Services-servern

## EXEMPEL

### Exempel 1
```
PS C:\>Add-AzAnalysisServicesAccount
RolloutEnvironment: westcentralus.asazure.windows.net
Credential: $UserCredential
```

Det här exemplet lägger till det konto som anges $UserCredential variabeln i westcentralus.asazure.windows.net Analysis Services-miljön.

### Exempel 2
```
PS C:\>$ApplicationCredential = Get-Credential
PS C:\>Add-AzAnalysisServicesAccount -RolloutEnvironment 'westcentralus.asazure.windows.net' -ServicePrincipal -Credential $ApplicationCredential -TenantId "xxxx-xxxx-xxxx-xxxx"
```

Det första kommandot hämtar programmets huvudautentiseringsuppgifter och lagrar dem sedan i $ApplicationCredential variabeln.
Det andra kommandot lägger till programmets huvudkonto som anges av variabeln $ApplicationCredential och TenantId i westcentralus.asazure.windows.net Analysis Services-miljön.

### Exempel 3
```
PS C:\>Add-AzAnalysisServicesAccount -RolloutEnvironment 'westcentralus.asazure.windows.net' -ServicePrincipal -ApplicationId "yyyy-yyyy-yyyy-yyyy" -CertificateThumbprint 'zzzzzzzzzzzzzzzz' -TenantId "xxxx-xxxx-xxxx-xxxx"
```

Det här exemplet lägger till programmets huvudkonto som anges av ApplicationId, TenantId och CertificateThumbprint i westcentralus.asazure.windows.net Analysis Services-miljön.
