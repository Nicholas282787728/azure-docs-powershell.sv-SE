# Test-AzAnalysisServicesServer

## SYNOPSIS
Testar om det finns en instans av Analysis Services-servern

## SYNTAX

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Med Test-AzAnalysisServicesServer-cmdleten testas det att det finns en instans av Analysis Services-servern

## EXEMPEL

### Exempel 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Det här kommandot testar om det finns en server med namnet testserver i resursgruppens testgrupp
