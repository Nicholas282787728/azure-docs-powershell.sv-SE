# Remove-AzDataMigrationTask

## SYNOPSIS
Tar bort en Azure-databasmigreringstjänst från Azure.

## SYNTAX

### ComponentNameParameterSet (standard)
```
Remove-AzDataMigrationTask -ResourceGroupName <String> -ServiceName <String> -ProjectName <String>
 -Name <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Remove-AzDataMigrationTask [-InputObject] <PSProjectTask> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzDataMigrationTask [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzDataMigrationTask-cmdleten tas en Azure Database Migration Service-uppgift bort från Azure.

## EXEMPEL

### Exempel 1
```
PS C:\> Remove-AzDataMigrationTask -TaskName TestTask -ProjectName myTestProject -ServiceName MyTestService
 -ResourceGroupName MyResourceGroup
```

I föregående exempel tas en Azure-databasmigreringstjänstaktivitet med namnet TestTask bort från Azure baserat på uppgiftsnamnsparameter.

### Exempel 2
```
PS C:\> Remove-AzDataMigrationTask -InputObject $TestTask
```

I föregående exempel tas en migreringstjänst för Azure-databastjänst bort baserat på PSProjectTask-objekt som har överförs.
