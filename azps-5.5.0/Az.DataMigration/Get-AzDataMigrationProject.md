# Get-AzDataMigrationProject

## SYNOPSIS
Hämtar egenskaperna för ett Azure-databasmigreringsprojekt.

## SYNTAX

### ComponentNameParameterSet (standard)
```
Get-AzDataMigrationProject -ResourceGroupName <String> -ServiceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Get-AzDataMigrationProject [-InputObject] <PSDataMigrationService> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzDataMigrationProject [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdlet Get-AzDataMigrationProject hämtar egenskaperna för ett Azure Database Migration-projekt.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzDataMigrationProject -ServiceName testService -Name testProject -ResourceGroup testResourceGroup
```

I exemplet ovan hämtas Azure-databasmigreringsprojektet TestProject i resursgruppen testResourceGroup och under tjänsten testService

### Exempel 2
```
PS C:\> Get-AzDataMigrationProject -InputObject $myService
```

I exemplet ovan hämtas Azure-databasmigreringsprojektet baserat på PSProject-objektindataparameter som har överförs.
