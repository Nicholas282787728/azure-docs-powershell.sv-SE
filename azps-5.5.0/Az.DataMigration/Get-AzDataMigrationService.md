# Get-AzDataMigrationService

## SYNOPSIS
Hämtar egenskaperna som är kopplade till en instans av Azure-databasmigreringstjänsten. 

## SYNTAX

### ResourceGroupSet (standard)
```
Get-AzDataMigrationService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzDataMigrationService [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ServiceNameGroupSet
```
Get-AzDataMigrationService [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdlet Get-AzDataMigrationService hämtar egenskaperna som är associerade med en instans av Azure Database Migration Service baserat på Tjänstnamn och Azure Resource Group-namn som indataparametrar. 

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzDataMigrationService -ResourceGroupName testResourceGroup -Name testService
```

I exemplet ovan hämtas egenskaperna för Azure Database Migration Service-instansen testService. 

### Exempel 2
```
PS C:\> Get-AzDataMigrationService -ResourceGroupName testResourceGroup
```

I exemplet ovan hämtas Azure Database Migration Services i resursgruppen testResourceGroup.
