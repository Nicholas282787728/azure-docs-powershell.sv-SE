# Start-AzDataMigrationService

## SYNOPSIS
Startar en instans av Azure-databasmigreringstjänsten i ett stoppat tillstånd. 

## SYNTAX

### ComponentNameParameterSet (standard)
```
Start-AzDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Start-AzDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Start-AzDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Start-AzDataMigrationService-cmdleten startas en instans av Azure-databasmigreringstjänsten i ett stoppat tillstånd. 

## EXEMPEL

### Exempel 1
```
PS C:\> Start-AzDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

Exemplet ovan startar en Azure Database Migration Service-instans med namnet Test Service i ett stoppat tillstånd baserat på att tjänstnamn skickas som indata

### Exempel 2
```
PS C:\> Start-AzDataMigrationService -InputObject $TestService
```

Exemplet ovan startar en Azure Database Migration Service-instans baserat på PSDataMigrationService som överförs som indataparameter
