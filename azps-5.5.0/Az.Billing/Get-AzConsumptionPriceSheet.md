# Get-AzConsumptionPriceSheet

## SYNOPSIS
Hämta prislistor för prenumerationen.

## SYNTAX

```
Get-AzConsumptionPriceSheet [-BillingPeriodName <String>] [-ExpandMeterDetail] [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzConsumptionPriceSheet** får prislistor för prenumerationen.
