# Disable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Inaktivera borttagning av bevarandeprincip för Azure Storage Blob-tjänsten.

## SYNTAX

```
Disable-AzStorageDeleteRetentionPolicy [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Disable-AzStorageDeleteRetentionPolicy** inaktiverar borttagningsprincipen för Azure Storage Blob-tjänsten.

## EXEMPEL

### Exempel 1: Inaktivera borttagningsprincip för Blob-tjänsten
```
C:\PS>Disable-AzStorageDeleteRetentionPolicy
```

Med det här kommandot inaktiveras borttagningsprincipen för Blob-tjänsten.
