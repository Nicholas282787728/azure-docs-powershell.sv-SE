# Remove-AzStorageTableStoredAccessPolicy

## SYNOPSIS
Tar bort en lagrad åtkomstprincip från en Azure-lagringstabell.

## SYNTAX

```
Remove-AzStorageTableStoredAccessPolicy [-Table] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzStorageTableStoredAccessPolicy** tar bort en lagrad åtkomstprincip från en Azure-lagringstabell.

## EXEMPEL

### Exempel 1: Ta bort en lagrad åtkomstprincip från en lagringstabell
```
PS C:\>Remove-AzStorageTableStoredAccessPolicy -Table "MyTable" -Policy "Policy05"
```

Med det här kommandot tas principen med namnet Policy05 bort från lagringstabellen Mintabell.
