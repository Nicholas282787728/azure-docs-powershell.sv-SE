# New-AzStorageAccountKey

## SYNOPSIS
Återskapar en lagringsnyckel för ett Azure Storage-konto.

## SYNTAX

```
New-AzStorageAccountKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzStorageAccountKey** återskapar en lagringsnyckel för ett Azure Storage-konto.

## EXEMPEL

### Exempel 1: Återskapa en lagringsnyckel
```
PS C:\>New-AzStorageAccountKey -ResourceGroupName "MyResourceGroup" -Name "mystorageaccount" -KeyName "key1"
```

Det här kommandot återskapar en lagringsnyckel för det angivna lagringskontot.
