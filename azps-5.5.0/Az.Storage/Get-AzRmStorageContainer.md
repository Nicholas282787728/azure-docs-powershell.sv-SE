# Get-AzRmStorageContainer

## SYNOPSIS
Hämtar eller visar blob-behållare för lagring

## SYNTAX

### AccountName (standard)
```
Get-AzRmStorageContainer [-ResourceGroupName] <String> [-StorageAccountName] <String> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountObject
```
Get-AzRmStorageContainer -StorageAccount <PSStorageAccount> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzRmStorageContainer** hämtar eller visar blob-behållare för lagring

## EXEMPEL

### Exempel 1: Hämta en blob-behållare för lagring med namn på lagringskonto och behållarnamn
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer"
```

Det här kommandot får en blob-behållare för lagring med namn på lagringskonto och behållarnamn.

### Exempel 2: Lista alla blob-behållare för lagring i ett lagringskonto
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
```

Det här kommandot listar alla blob-lagringsbehållare i ett lagringskonto med namnet lagringskonto.

### Exempel 3: Hämta en blob-behållare för lagring med lagringskontoobjekt och behållarnamn.
```
PS C:\>$accountObject = Get-AzStorageAccount -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
PS C:\>Get-AzRmStorageContainer -StorageAccount $accountObject -ContainerName "myContainer"
```

Det här kommandot hämtar en blob-behållare för lagring med lagringskontoobjekt och behållarnamn.
