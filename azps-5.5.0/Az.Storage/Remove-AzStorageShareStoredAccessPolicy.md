# Remove-AzStorageShareStoredAccessPolicy

## SYNOPSIS
Tar bort en princip för lagrad åtkomst från en lagringsresurs.

## SYNTAX

```
Remove-AzStorageShareStoredAccessPolicy [-ShareName] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzStorageShareStoredAccessPolicy** tar bort en lagrad åtkomstprincip från en Azure Storage-delning.

## EXEMPEL

### Exempel 1: Ta bort en lagrad åtkomstprincip från en Azure Storage-delning
```
PS C:\>Remove-AzStorageShareStoredAccessPolicy -ShareName "ContosoShare" -Policy "GeneralPolicy"
```

Det här kommandot tar bort en lagrad åtkomstprincip med namnet GeneralPolicy från ContosoShare.
