# Set-AzStorageQueueStoredAccessPolicy

## SYNOPSIS
Anger en lagrad åtkomstprincip för en Azure-lagringskö.

## SYNTAX

```
Set-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-NoStartTime] [-NoExpiryTime] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzStorageQueueStoredAccessPolicy** anger en lagrad åtkomstprincip för en Azure-lagringskö.

## EXEMPEL

### Exempel 1: Ange en lagrad åtkomstprincip i kön med fullständig behörighet
```
PS C:\> Set-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy07" -Permission arup
```

Det här kommandot anger en åtkomstprincip med namnet Policy07 för lagringskö med namnet MyQueue.
