# New-AzStorageQueueSASToken

## SYNOPSIS
Genererar en signaturtoken för delad åtkomst för en Azure-lagringskö.

## SYNTAX

### SasPolicy
```
New-AzStorageQueueSASToken [-Name] <String> -Policy <String> [-Protocol <SharedAccessProtocol>]
 [-IPAddressOrRange <String>] [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-FullUri]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SasPermission
```
New-AzStorageQueueSASToken [-Name] <String> [-Permission <String>] [-Protocol <SharedAccessProtocol>]
 [-IPAddressOrRange <String>] [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-FullUri]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzStorageQueueSASToken** genererar signaturtoken för delad åtkomst för en Azure-lagringskö.

## EXEMPEL

### Exempel 1: Generera en SAS token för kö med fullständig behörighet
```
PS C:\>New-AzStorageQueueSASToken -Name "Test" -Permission raup
```

Det här exemplet genererar en SAS-token för kö med fullständig behörighet.
