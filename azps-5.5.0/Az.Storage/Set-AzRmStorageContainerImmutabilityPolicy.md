# Set-AzRmStorageContainerImmutabilityPolicy

## SYNOPSIS
Skapar eller uppdaterar ImmutabilityPolicy för en blob-behållare för lagring

## SYNTAX

### AccountName (standard)
```
Set-AzRmStorageContainerImmutabilityPolicy [-ResourceGroupName] <String> [-StorageAccountName] <String>
 -ContainerName <String> [-ImmutabilityPeriod <Int32>] [-AllowProtectedAppendWrite <Boolean>] [-Etag <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ExtendAccountName
```
Set-AzRmStorageContainerImmutabilityPolicy [-ResourceGroupName] <String> [-StorageAccountName] <String>
 -ContainerName <String> -ImmutabilityPeriod <Int32> -Etag <String> [-ExtendPolicy]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzRmStorageContainerImmutabilityPolicy** skapar eller uppdaterar ImmutabilityPolicy för en blob-behållare för lagring

## EXEMPEL

### Exempel 1: Skapa eller uppdatera ImmutabilityPolicy för en blob-behållare för lagring med namn på lagringskonto och behållarnamn
```
PS C:\>Set-AzRmStorageContainerImmutabilityPolicy -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer" -ImmutabilityPeriod 10
```

Med det här kommandot skapas eller uppdateras ImmutabilityPolicy för en blob-behållare för lagring med lagringskontonamn och behållarnamn.

### Exempel 2: Utöka ImmutabilityPolicy för en blob-behållare för lagring med lagringskontoobjekt
```
PS C:\>$accountObject = Get-AzStorageAccount -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
PS C:\>$policy = Get-AzRmStorageContainerImmutabilityPolicy -StorageAccount $accountObject -ContainerName "myContainer"
PS C:\>Set-AzRmStorageContainerImmutabilityPolicy -StorageAccount $accountObject -ContainerName "myContainer" -ImmutabilityPeriod 20 -Etag $policy.Etag -ExtendPolicy
```

Med det här kommandot utökas ImmutabilityPolicy för en blob-behållare för lagring med lagringskontoobjekt. Utvidga ImmutabilityPolicy kan endast köras när ImmutabilityPolicy är låst.

### Exempel 3: Uppdatera ImmutabilityPolicy för en blob-behållare för lagring
```
PS C:\>$containerObject = Get-AzStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -Name "myContainer"
PS C:\>$policy = Set-AzRmStorageContainerImmutabilityPolicy -Container $containerObject -ImmutabilityPeriod 12
PS C:\>$policy = Set-AzRmStorageContainerImmutabilityPolicy -Container $containerObject -ImmutabilityPeriod 9 -Etag $policy.Etag
PS C:\>$policy = Set-AzRmStorageContainerImmutabilityPolicy -Container $containerObject -AllowProtectedAppendWrite $true
```

Det här kommandot uppdaterar ImmutabilityPolicy för en blob-behållare för lagring med lagringsbehållareobjekt 3 gånger: Först till ImmutabilityPeriod 12 dagar utan etag, sedan till ImmutabilityPeriod 9 dagar med etag, slutligen aktiverat AllowProtectedAppendWrite.

### Exempel 4: Utöka ImmutabilityPolicy för en blob-behållare för lagring med ImmutabilityPolicy-objekt
```
PS C:\>Get-AzRmStorageContainerImmutabilityPolicy -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer" | Set-AzRmStorageContainerImmutabilityPolicy -ImmutabilityPeriod 15 -ExtendPolicy
```

Med det här kommandot utökas ImmutabilityPolicy för en blob-behållare för lagring med ImmutabilityPolicy-objekt. Utvidga ImmutabilityPolicy kan endast köras när ImmutabilityPolicy är låst.
