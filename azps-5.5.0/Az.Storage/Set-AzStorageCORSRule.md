# Set-AzStorageCORSRule

## SYNOPSIS
Ställer in CORS-reglerna för en typ av lagringstjänst.

## SYNTAX

```
Set-AzStorageCORSRule [-ServiceType] <StorageServiceType> -CorsRules <PSCorsRule[]> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzStorageCORSRule** anger reglerna för resursdelning mellan ursprung (CORS) för en typ av Azure Storage-tjänst.
Typerna av lagringstjänster för den här cmdleten är Blob, Table, Queue och File.
Den här cmdleten skriver över de befintliga reglerna.
Om du vill se de aktuella reglerna använder Get-AzStorageCORSRule-cmdleten.

## EXEMPEL

### Exempel 1: Tilldela CORS-regler till blob-tjänsten
```
PS C:\>$CorsRules = (@{
    AllowedHeaders=@("x-ms-blob-content-type","x-ms-blob-content-disposition");
    AllowedOrigins=@("*");
    MaxAgeInSeconds=30;
    AllowedMethods=@("Get","Connect")},
    @{
    AllowedOrigins=@("http://www.fabrikam.com","http://www.contoso.com"); 
    ExposedHeaders=@("x-ms-meta-data*","x-ms-meta-customheader"); 
    AllowedHeaders=@("x-ms-meta-target*","x-ms-meta-customheader");
    MaxAgeInSeconds=30;
    AllowedMethods=@("Put")})
PS C:\> Set-AzStorageCORSRule -ServiceType Blob -CorsRules $CorsRules
```

Med det första kommandot tilldelas en matris med regler till $CorsRules variabeln.
Det här kommandot använder standard utökas över flera rader i det här kodblocket.
Det andra kommandot tilldelar reglerna i $CorsRules till Blob-tjänsttypen.

### Exempel 2: Ändra egenskaper för en CORS-regel för blob-tjänst
```
PS C:\>$CorsRules = Get-AzStorageCORSRule -ServiceType Blob
PS C:\> $CorsRules[0].AllowedHeaders = @("x-ms-blob-content-type", "x-ms-blob-content-disposition")
PS C:\> $CorsRules[0].AllowedMethods = @("Get", "Connect", "Merge")
PS C:\> Set-AzStorageCORSRule -ServiceType Blob -CorsRules $CorsRules
```

Med det första kommandot får du aktuella KORS-regler för blob-typen med cmdleten **Get-AzStorageCORSRule.**
Kommandot lagrar reglerna i den $CorsRules matrisvariabeln.
Det andra och tredje kommandot ändrar den första regeln i $CorsRules.
Det slutliga kommandot tilldelar reglerna i $CorsRules till blob-tjänsttypen.
De reviderade reglerna skriver över de aktuella CORS-reglerna.

### -ConcurrentTaskCount
Anger maximalt antal samtidiga nätverkssamtal.
Du kan använda den här parametern för att begränsa samtidigheten för att begränsa den lokala CPU-belastningen och bandbreddsanvändningen genom att ange maximalt antal samtidiga nätverkssamtal.
Det angivna värdet är ett absolut antal och multipliceras inte med basantalet.
Den här parametern kan minska problem med nätverksanslutningen i miljöer med låg bandbredd, till exempel 100 kilobit per sekund.
Standardvärdet är 10.
