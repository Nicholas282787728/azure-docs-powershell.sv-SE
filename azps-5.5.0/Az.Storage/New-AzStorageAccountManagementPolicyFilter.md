# New-AzStorageAccountManagementPolicyFilter

## SYNOPSIS
Skapar ett ManagementPolicy-regelfilterobjekt som kan användas i New-AzStorageAccountManagementPolicyRule.

## SYNTAX

```
New-AzStorageAccountManagementPolicyFilter [-PrefixMatch <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzStorageAccountManagementPolicyFilter** skapar ett ManagementPolicy-regelfilterobjekt som kan användas i New-AzStorageAccountManagementPolicyRule.

## EXEMPEL

### Exempel 1: Skapar ett ManagementPolicy-regelfilterobjekt, lägger sedan till det i en hanteringsprincipregel och anger ett lagringskonto
```
PS C:\>$filter = New-AzStorageAccountManagementPolicyFilter -PrefixMatch blobprefix1,blobprefix2
PS C:\>$filter 

PrefixMatch                BlobTypes  
-----------                ---------  
{blobprefix1, blobprefix2} {blockBlob}

PS C:\>$action = Add-AzStorageAccountManagementPolicyAction -BaseBlobAction Delete -daysAfterModificationGreaterThan 100
PS C:\>$rule = New-AzStorageAccountManagementPolicyRule -Name Test -Action $action -Filter $filter
PS C:\>$policy = Set-AzStorageAccountManagementPolicy -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount" -Rule $rule
```

Med det här kommandot skapas ett ManagementPolicy-regelfilterobjekt. Lägg sedan till den i en principregel för hantering och ange ett lagringskonto.
