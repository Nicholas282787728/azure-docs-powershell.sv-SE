# Get-AzStorageQueueStoredAccessPolicy

## SYNOPSIS
Hämtar principen eller principerna för lagrad åtkomst för en Azure-lagringskö.

## SYNTAX

```
Get-AzStorageQueueStoredAccessPolicy [-Queue] <String> [[-Policy] <String>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzStorageQueueStoredAccessPolicy** visar principen eller principerna för lagrad åtkomst för en Azure-lagringskö.

## EXEMPEL

### Exempel 1: Hämta en lagrad åtkomstprincip i kön
```
PS C:\>Get-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy12"
```

Det här kommandot hämtar åtkomstprincipen med namnet Policy12 i lagringskön Med namnet MyQueue.

### Exempel 2: Hämta alla lagrade åtkomstprinciper i kön
```
PS C:\>Get-AzStorageQueueStoredAccessPolicy -Queue "MyQueue"
```

Det här kommandot hämtar alla lagrade åtkomstprinciper i kön MyQueue.
