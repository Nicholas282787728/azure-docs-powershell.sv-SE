# Set-AzStorageServiceMetricsProperty

## SYNOPSIS
Ändrar mätvärdesegenskaper för Azure Storage-tjänsten.

## SYNTAX

```
Set-AzStorageServiceMetricsProperty [-ServiceType] <StorageServiceType> [-MetricsType] <ServiceMetricsType>
 [-Version <Double>] [-RetentionDays <Int32>] [-MetricsLevel <MetricsLevel>] [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzStorageServiceMetricsProperty** ändrar mätvärden för Azure-lagringstjänsten.

## EXEMPEL

### Exempel 1: Ändra mätvärdesegenskaper för Blob-tjänsten
```
C:\PS>Set-AzStorageServiceMetricsProperty -ServiceType Blob -MetricsType Hour -MetricsLevel Service -PassThru -RetentionDays 10 -Version 1.0
```

Det här kommandot ändrar måtten för version 1.0 för blob-lagring till en tjänstnivå.
Azure Storage-tjänstmått behåller poster i 10 dagar.
Eftersom det här kommandot anger *parametern PassThru* visar kommandot de ändrade måttegenskaperna.
