# Remove-AzStorageQueue

## SYNOPSIS
Tar bort en lagringskö.

## SYNTAX

```
Remove-AzStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzStorageQueue** tar bort en lagringskö.

## EXEMPEL

### Exempel 1: Ta bort en lagringskö efter namn
```
PS C:\>Remove-AzStorageQueue "ContosoQueue01"
```

Det här kommandot tar bort en kö med namnet ContosoQueue01.

### Exempel 2: Ta bort flera lagringsköer
```
PS C:\>Get-AzStorageQueue "Contoso*" | Remove-AzStorageQueue
```

Det här kommandot tar bort alla köer med namn som börjar på Contoso.
