# Revoke-AzStorageAccountUserDelegationKeys

## SYNOPSIS
Återkalla alla användardelegeringsnycklar för ett lagringskonto.

## SYNTAX

### AccountName (standard)
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountObject
```
Revoke-AzStorageAccountUserDelegationKeys -InputObject <PSStorageAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountResourceId
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Revoke-AzStorageAccountUserDelegationKeys** återkallar alla användardelegeringsnycklar för ett lagringskonto, så alla SAS-token för Identity SAS för lagringskontot återkallas också.

## EXEMPEL

### Exempel 1: Återkalla alla nycklar för användardelegering för ett lagringskonto
```powershell
PS C:\>Revoke-AzStorageAccountUserDelegationKeys -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount"
```

Det här exemplet återkallar alla användardelegeringsnycklar för ett lagringskonto, så all SAS-identitetstoken som genereras från nycklar för användardelegering återkallas också.
