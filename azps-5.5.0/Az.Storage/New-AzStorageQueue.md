# New-AzStorageQueue

## SYNOPSIS
Skapar en lagringskö.

## SYNTAX

```
New-AzStorageQueue [-Name] <String> [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzStorageQueue** skapar en lagringskö i Azure.

## EXEMPEL

### Exempel 1: Skapa en Azure-lagringskö
```
PS C:\>New-AzStorageQueue -Name "queueabc"
```

I det här exemplet skapas en lagringskö med namnet queueabc.

### Exempel 2: Skapa flera Azure Storage-köer
```
PS C:\>"queue1 queue2 queue3".split() | New-AzStorageQueue
```

I det här exemplet skapas flera lagringsköer.
Den använder delningsmetoden för .NET-strängklassen och skickar sedan namnen på pipelinen.
