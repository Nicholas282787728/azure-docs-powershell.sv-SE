# Stop-AzSqlDatabaseActivity

## SYNOPSIS
Avbryter asynkron uppdateringsåtgärden i databasen.

## SYNTAX

```
Stop-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Stop-AzSqlDatabaseActivity** avbryter asynkrona uppdateringar i databasen.

## EXEMPEL

### Exempel 1: Avbryta den asynkrona uppdateringsåtgärden i databasen
