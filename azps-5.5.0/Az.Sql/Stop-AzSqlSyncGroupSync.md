# Stop-AzSqlSyncGroupSync

## SYNOPSIS
Stoppar synkronisering av synkroniseringsgrupp.

## SYNTAX

```
Stop-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Stop-AzSqlSyncGroupSync** avbryter en synkroniseringsgruppssynkronisering.

## EXEMPEL

### Exempel 1
```
PS C:\> Stop-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Det här kommandot stoppar synkroniseringen, som pågår för synkroniseringsgruppen MySG.
