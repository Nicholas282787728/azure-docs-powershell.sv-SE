# Invoke-AzSqlDatabaseFailover

## SYNOPSIS
Redundanser en databas.

## SYNTAX

```
Invoke-AzSqlDatabaseFailover [-DatabaseName] <String> [-ReadableSecondary] [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den Invoke-AzSqlDatabaseFailover cmdleten redundans för en Azure SQL-databas. Om databasen ligger i en pool med pooler med pooler, kommer det här kommandot att växla över den givna databasen utan att påverka de andra databaserna i samma pool med pooler.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Invoke-AzSqlDatabaseFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot kommer att redundans för den primära kopian av databasen med namnet "Database01" på servern med namnet "Server01"

### Exempel 2
```powershell
PS C:\> Invoke-AzSqlDatabaseFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -ReadableSecondary
```

Det här kommandot kommer att redundans för den läsbara sekundära kopian av databasen med namnet "Database01" på servern med namnet "Server01"
