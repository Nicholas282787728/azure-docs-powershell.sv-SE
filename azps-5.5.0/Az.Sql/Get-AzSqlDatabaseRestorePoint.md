# Get-AzSqlDatabaseRestorePoint

## SYNOPSIS
Hämtar de distinkta återställningspunkterna som ett SQL Data Warehouse kan återställas från.

## SYNTAX

```
Get-AzSqlDatabaseRestorePoint [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlDatabaseRestorePoint** hämtar de distinkta återställningspunkterna som ett Azure SQL Data Warehouse kan återställas från.
Återställningsfönstret är kontinuerligt för en Azure SQL-databas.
Det innebär att alla tidpunkter i databasens lagringstid för säkerhetskopior kan användas som återställningspunkt.
Den här cmdleten stöds också av tjänsten Stretcha databas för SQL Server i Azure.

## EXEMPEL

### Exempel 1: Hämta alla återställningspunkter
```
PS C:\>Get-AzSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : CONTINUOUS
RestorePointCreationDate : 
EarliestRestoreDate      : 8/12/2015 12:00:00 AM
RestorePointLabel        : RestorePoint01
```

Det här kommandot returnerar alla tillgängliga återställningspunkter för Azure SQL-databasen med namnet Database01.
