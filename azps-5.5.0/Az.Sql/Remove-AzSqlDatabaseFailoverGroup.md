# Remove-AzSqlDatabaseFailoverGroup

## SYNOPSIS
Tar bort en redundansgrupp för Azure SQL-databas.

## SYNTAX

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Med det här kommandot tas redundansgruppen bort med det angivna namnet och alla databaser och replikeringsrelationer förblir intakta. Slutpunkten för lyssnare avregistreras från DNS.
Redundansgruppens primära server ska användas för att köra kommandot.

## EXEMPEL

### Exempel 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Ta bort en redundansgrupp.
