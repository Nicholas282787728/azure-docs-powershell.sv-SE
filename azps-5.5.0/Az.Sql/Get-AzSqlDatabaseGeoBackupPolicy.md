# Get-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Hämtar en princip för geosäkerhetskopiering av databas.

## SYNTAX

```
Get-AzSqlDatabaseGeoBackupPolicy [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlDatabaseGeoBackupPolicy** får principen för geosäkerhetskopiering registrerad i den här databasen.
Det här är en Azure-säkerhetskopieringsresurs som används för att definiera en lagringsprincip för säkerhetskopiering.

## EXEMPEL

### Exempel 1

Hämtar en princip för geosäkerhetskopiering av databas. (autogenererat)

```powershell
<!-- Aladdin Generated Example --> 
Get-AzSqlDatabaseGeoBackupPolicy -DatabaseName db1 -ResourceGroupName MyResourceGroup -ServerName s1
```
