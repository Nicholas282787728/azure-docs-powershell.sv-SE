# Get-AzSqlServerActiveDirectoryAdministrator

## SYNOPSIS
Hämtar information om en Azure AD-administratör för SQL Server.

## SYNTAX

```
Get-AzSqlServerActiveDirectoryAdministrator [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlServerActiveDirectoryAdministrator** hämtar information om en Azure Active Directory-administratör (Azure AD) för en AzureSQL Server i den aktuella prenumerationen.

## EXEMPEL

### Exempel 1: Hämtar information om en administratör för en server
```
PS C:\>Get-AzSqlServerActiveDirectoryAdministrator -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
ResourceGroupName ServerName DisplayName ObjectId IsAzureADOnlyAuthentication
----------------- ---------- ----------- -------- -----------
ResourceGroup01   Server01   DBAs        40b79501-b343-44ed-9ce7-da4c8cc7353b true
```

Det här kommandot hämtar information om en Azure AD-administratör för en server med namnet Server01 som är associerad med en resursgrupp med namnet ResourceGroup01.
