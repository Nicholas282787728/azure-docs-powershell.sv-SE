# Invoke-AzSqlElasticPoolFailover

## SYNOPSIS
Failovers an pool.

## SYNTAX

```
Invoke-AzSqlElasticPoolFailover [-ElasticPoolName] <String> [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
CmdletInvoke-AzSqlElasticPoolFailover cmdleten redundanser en pool med pooler. Redundans uppstår för alla databaser i poolen med pooler, när du har kört den här cmdleten.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Invoke-AzSqlElasticPoolFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Det här kommandot redundans för poolen med namnet "ElasticitPool01" på servern "Server01".  Det innebär att redundans uppstår för alla databaser i poolen med namnet "ElasticitPool01".
