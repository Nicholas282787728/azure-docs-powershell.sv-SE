# Get-AzSqlDatabaseExpanded

## SYNOPSIS
Hämtar en databas och dess utökade egenskapsvärden.

## SYNTAX

```
Get-AzSqlDatabaseExpanded [-ServerName] <String> [[-DatabaseName] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzSqlDatabaseExpanded** hämtar en databas och dess utökade egenskapsvärden.
Den här cmdleten stöds också av tjänsten Stretcha databas för SQL Server i Azure.

## EXEMPEL

### Exempel 1: Hämta databasobjekt med information för tjänstnivårådgivare
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot returnerar databasen som har utökade egenskaper som innehåller information för tjänstenivårådgivaren.

### Exempel 2: Lista databasobjekt med filtrering
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database*"
```

Det här kommandot returnerar det utökade databasobjektet för alla databaser i Server01 som börjar med "Databas".
