# Get-AzSqlServer

## SYNOPSIS
Returnerar information om SQL Database-servrar.

## SYNTAX

```
Get-AzSqlServer [[-ResourceGroupName] <String>] [[-ServerName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlServer** returnerar information om en eller flera Azure SQL Database-servrar.
Ange namnet på en server för att visa information för endast den servern.

## EXEMPEL

### Exempel 1: Hämta alla instanser av SQL Server som tilldelats till en resursgrupp
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
Location                 : Central US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server01.database.windows.net

ResourceGroupName        : resourcegroup01
ServerName               : server02
Location                 : West US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server02.database.windows.net
```

Det här kommandot hämtar information om alla Azure SQL Database-servrar som tilldelats resursgruppen ResourceGroup01.

### Exempel 2: Få information om en Azure SQL Database-server
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
Location                 : Central US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server01.database.windows.net
```

Det här kommandot hämtar information om Azure SQL Database-servern med namnet Server01.

Med det här kommandot får du information om alla Azure SQL Database-servrar i den aktuella prenumerationen.
