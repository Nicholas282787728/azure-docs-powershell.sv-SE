# Get-AzSqlServerTransparentDataEncryptionProtector

## SYNOPSIS
Får den transparenta TDE-krypteringen (Transparent Data Encryption)

## SYNTAX

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den Get-AzSqlServerTransparentDataEncryptionProtector cmdleten hämtar information om TDE-the-

## EXEMPEL

### Exempel 1: Hämta TDE-kryptering (Transparent Data Encryption)
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Det här kommandot får TDE-namnet för servern ContosoServer i resursgruppen ContosoResourceGroup.
ResourceGroupName ServerName Type ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer ServiceManaged ServiceManaged
