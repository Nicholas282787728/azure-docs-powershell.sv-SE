# Get-AzSqlSyncGroup

## SYNOPSIS
Returnerar information om Azure SQL Database Sync Groups.

## SYNTAX

```
Get-AzSqlSyncGroup [[-Name] <String>] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlSyncGroup** returnerar information om en eller flera Azure SQL Database Sync Groups.
Ange namnet på en synkroniseringsgrupp om du bara vill visa information för den synkroniseringsgruppen.
