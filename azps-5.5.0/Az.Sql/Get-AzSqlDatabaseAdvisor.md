# Get-AzSqlDatabaseAdvisor

## SYNOPSIS
Får en eller flera rådgivare för en Azure SQL-databas.

## SYNTAX

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlDatabaseAdvisor** får en eller flera Azure SQL Database Advisors för en Azure SQL-databas.
