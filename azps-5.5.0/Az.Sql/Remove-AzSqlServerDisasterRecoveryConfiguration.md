# Remove-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Tar bort en konfiguration för återställning av SQL-databasserversystem.

## SYNTAX

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzSqlServerDisasterRecoveryConfiguration** tar bort en konfiguration för återställning av SQL-databasserversystem.
