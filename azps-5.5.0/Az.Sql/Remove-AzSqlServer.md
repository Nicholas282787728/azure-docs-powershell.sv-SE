# Remove-AzSqlServer

## SYNOPSIS
Tar bort en Azure SQL Database-server.

## SYNTAX

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzSqlServer** tar bort en Azure SQL Database-server.
Borttagningen är asynkron och kan ta lite tid, så kontrollera att åtgärden är slutförd innan du utför några ytterligare åtgärder som är beroende av att servern tas bort helt.
Du måste till exempel skapa en ny server som använder samma namn.

## EXEMPEL

### Exempel 1: Ta bort en server
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Det här kommandot tar bort Azure SQL Database-servern med namnet Server01.
