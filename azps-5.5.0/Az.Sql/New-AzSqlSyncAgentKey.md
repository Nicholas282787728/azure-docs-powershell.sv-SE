# New-AzSqlSyncAgentKey

## SYNOPSIS
Skapar en Azure SQL-synkroniseringsagentnyckel.

## SYNTAX

```
New-AzSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzSqlSyncAgentKey** skapar en Azure SQL Sync Agent-nyckel.

## EXEMPEL

### Exempel 1: Skapa en synkroniseringsagentnyckel för en Azure SQL-synkroniseringsagent.
```
PS C:\> New-AzSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Det här kommandot skapar en synkroniseringsagentnyckel för en Azure SQL-synkroniseringsagent.
