# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## SYNOPSIS
Hämtar advanced datasäkerhetsprincip för en hanterad instans.

## SYNTAX

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlInstanceAdvancedDataSecurityPolicy** hämtar Advanced Data Security-principen för en hanterad instans.

## EXEMPEL

### Exempel 1: Hämtar hanterad instans avancerad datasäkerhet
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Exempel 2: Hämtar hanterad instans avancerad datasäkerhet från hanterad instansresurs
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
