# Get-AzSqlElasticPoolRecommendedAction

## SYNOPSIS
Får en eller flera rekommenderade åtgärder för en rådgivare för Azure SQL-spoolpool.

## SYNTAX

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlElasticPoolRecomaction** får en eller flera rekommenderade åtgärder för en Azure SQL-bubbelpoolrådgivare.
