# Set-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Anger en princip för geosäkerhetskopiering i databasen.

## SYNTAX

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzSqlDatabaseGeoBackupPolicy** ställer in principen för geosäkerhetssäkerhet som är registrerad i en databas.
Det här är en Azure-säkerhetskopieringsresurs som används för att definiera en lagringsprincip för säkerhetskopiering.
