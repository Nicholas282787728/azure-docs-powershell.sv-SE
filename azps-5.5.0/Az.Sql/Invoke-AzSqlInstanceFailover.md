# Invoke-AzSqlInstanceFailover

## SYNOPSIS
Redundanser en Azure SQL Managed-instans.

## SYNTAX

```
Invoke-AzSqlInstanceFailover [-Name] <String> [-AsJob] [-PassThru] [-Force] [-ReadableSecondary]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Invoke-AzSqlInstanceFailover** redundanser en Azure SQL Managed Instance.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01"
```

Det här kommandot redundans för den primära kopian av instansen med namnet "ManagedInstance01".

### Exempel 2
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01" -ReadableSecondary
```

Det här kommandot redundans för den läsbara sekundära kopian av den hanterade instansen "ManagedInstance01".
