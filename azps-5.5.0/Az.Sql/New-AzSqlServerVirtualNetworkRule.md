# New-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Skapar en Azure SQL Server Virtual Network-regel. 

## SYNTAX

```
New-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Skapar en Azure SQL Server Virtual Network-regel. Regler för virtuellt nätverk används för att ansluta Azure SQL Server till ett specifikt virtuellt nätverk för att begränsa åtkomsten på Azure SQL Server till att endast vara tillgänglig i det virtuella nätverket. 

## EXEMPEL

### Exempel 1
```
PS C:\> $virtualNetworkRule = New-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Skapar en azure SQL Server-regel för virtuellt nätverk
