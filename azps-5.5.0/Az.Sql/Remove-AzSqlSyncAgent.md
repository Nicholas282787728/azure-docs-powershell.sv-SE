# Remove-AzSqlSyncAgent

## SYNOPSIS
Tar bort en Azure SQL-synkroniseringsagent.

## SYNTAX

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzSqlSyncAgent** tar bort en Azure SQL-synkroniseringsagent.

## EXEMPEL

### Exempel 1: Ta bort en synkroniseringsagent
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Det här kommandot tar bort Azure SQL-synkroniseringsagenten med namnet syncAgent01.
