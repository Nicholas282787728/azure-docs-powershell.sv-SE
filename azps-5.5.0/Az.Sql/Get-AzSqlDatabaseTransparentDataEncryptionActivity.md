# Get-AzSqlDatabaseTransparentDataEncryptionActivity

## SYNOPSIS
Hämtar förloppet för en TDE-genomsökning av en databas.

## SYNTAX

```
Get-AzSqlDatabaseTransparentDataEncryptionActivity [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlDatabaseTransparentDataEncryptionActivity** gör att en TDE-genomsökning (Transparent Data Encryption) görs av en Azure SQL-databas.
Om inget krypteringsintervall körs returnerar denna cmdlet en tom lista.
Mer information finns i Transparent Data Encryption med Azure SQL Database https://msdn.microsoft.com/library/dn948096 https://msdn.microsoft.com/library/dn948096) (i Microsoft Developer Network Library.
Den här cmdleten stöds också av tjänsten Stretcha databas för SQL Server i Azure.

## EXEMPEL

### Exempel 1: Hämta TDE-aktivitet för en databas
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryptionActivity -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName : resourcegroup01
ServerName        : server01
DatabaseName      : database01
Status            : Encrypting
PercentComplete   : 3.662109
```

Det här kommandot hämtar TDE-aktiviteten för databasen med namnet database01 på servern server01.
