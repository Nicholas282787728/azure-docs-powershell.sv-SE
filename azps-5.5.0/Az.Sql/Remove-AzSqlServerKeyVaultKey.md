# Remove-AzSqlServerKeyVaultKey

## SYNOPSIS
Tar bort en nyckel för ett nyckelvalv från en SQL-server.

## SYNTAX

```
Remove-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med Remove-AzSqlServerKeyVaultKey cmdleten tas nyckeln för nyckelvalv bort från den angivna SQL-servern.
Observera att SQL-serverns behörigheter till nyckelns valv inte ändras.
Om du vill ändra behörigheter använder du Set-AzKeyVaultAccessPolicy.
Observera att denna cmdlet inte gör några ändringar av nyckelvalv.
Om du vill ta bort en nyckel från nyckelvalv använder du Remove-AzKeyVaultKey.

## EXEMPEL

### Exempel 1: Ta bort en nyckel för ett nyckelvalv
```
PS C:\> Remove-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Det här kommandot tar bort nyckeln för nyckelvalv med Id https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 ' från den angivna servern.
