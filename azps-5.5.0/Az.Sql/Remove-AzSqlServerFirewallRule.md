# Remove-AzSqlServerFirewallRule

## SYNOPSIS
Tar bort en brandväggsregel från en SQL Database-server.

## SYNTAX

```
Remove-AzSqlServerFirewallRule [-FirewallRuleName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzSqlServerFirewallRule** tar bort en brandväggsregel från den angivna Azure SQL Database-servern.

## EXEMPEL

### Exempel 1: Ta bort en regel
```
PS C:\>Remove-AzSqlServerFirewallRule -FirewallRuleName "Rule01" -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Det här kommandot tar bort brandväggsregeln Rule01 på servern Server01.
