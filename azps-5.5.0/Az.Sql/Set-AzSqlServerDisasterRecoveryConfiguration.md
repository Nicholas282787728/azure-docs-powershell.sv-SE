# Set-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Ändrar en konfiguration för databasserveråterställning.

## SYNTAX

```
Set-AzSqlServerDisasterRecoveryConfiguration -VirtualEndpointName <String> [-Failover] [-AllowDataLoss]
 [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Set-AzSqlServerDisasterRecoveryConfiguration** ändrar en konfiguration för serveråterställning av SQL-databas.
