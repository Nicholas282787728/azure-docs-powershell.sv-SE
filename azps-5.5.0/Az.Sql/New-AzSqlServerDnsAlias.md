# New-AzSqlServerDnsAlias

## SYNOPSIS
Med det här kommandot skapas ett nytt Azure SQL Server DNS-alias.

## SYNTAX

```
New-AzSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Skapar nya Azure SQL Server DNS-alias som pekar på den angivna servern.

## EXEMPEL

### Exempel 1
```
PS C:\> $serverDNSAlias = New-AzSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Med det här kommandot skapas Azure SQL Server DNS-alias med namnaliaset Som pekar på servernamn
