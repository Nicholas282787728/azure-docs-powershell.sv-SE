# Set-AzSqlServerRecommendedActionState

## SYNOPSIS
Uppdaterar statusen för en rekommenderad åtgärd i Azure SQL Server.

## SYNTAX

```
Set-AzSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzSqlServerRecomactionState** uppdaterar statusen för en rekommenderad åtgärd i Azure SQL Server.
Den här cmdleten tillämpar, återställer eller ignorerar den rekommenderade åtgärden baserat på det nya läget.
