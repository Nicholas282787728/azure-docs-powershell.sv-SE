# Get-AzSqlElasticPool

## SYNOPSIS
Hämtar pooler med pooler och deras egenskapsvärden i en Azure SQL-databas.

## SYNTAX

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlElasticPool** får pooler med olika egenskapsvärden.
Ange namnet på en befintlig pool med pooler för att visa egenskapsvärdena för endast den poolen.
