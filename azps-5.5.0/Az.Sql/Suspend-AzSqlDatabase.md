# Suspend-AzSqlDatabase

## SYNOPSIS
Stänger av en SQL Data Warehouse-databas.

## SYNTAX

```
Suspend-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Suspend-AzSqlDatabase** stänger av en Azure SQL Data Warehouse-databas.

## EXEMPEL

### Exempel 1: Stänger av en Azure SQL Data Warehouse-databas
```
PS C:\>Suspend-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Det här kommandot stänger av en aktiv Azure SQL Data Warehouse-databas.
