# Set-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Ändrar konfigurationen av en regel för virtuell Azure SQL Server-nätverk.

## SYNTAX

```
Set-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Det här kommandot ändrar konfigurationen av en regel för virtuell Azure SQL Server-nätverk.
Om du vill styra uppsättningen av regler för virtuella nätverk på servern använder du i stället Add-AzSqlServerVirtualNetworkRule och Remove-AzSqlServerVirtualNetworkRule.

## EXEMPEL

### Exempel 1
```
PS C:\> $virtualNetworkRule = Set-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Ändrar en befintlig regel för virtuellt nätverk med det nya undernätets nya undernäts-ID som innehåller information om det nya virtuella nätverket
