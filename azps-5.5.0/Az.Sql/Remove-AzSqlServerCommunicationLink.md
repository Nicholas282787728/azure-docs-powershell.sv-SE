# Remove-AzSqlServerCommunicationLink

## SYNOPSIS
Tar bort en kommunikationslänk för transaktioner med databasdatabaser med databaser mellan två servrar.

## SYNTAX

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzSqlServerCommunicationLink** tar bort en kommunikationslänk mellan server och server för databastransaktioner med portaler med databaser som är av mindre storlek i Azure SQL Database.

## EXEMPEL

### Exempel 1: Ta bort en kommunikationslänk
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Det här kommandot tar bort en kommunikationslänk mellan server och server med namnet Link01 på ContosoServer17.
