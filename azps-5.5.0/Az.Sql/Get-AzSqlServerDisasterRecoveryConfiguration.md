# Get-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Hämtar en konfiguration för databasserversystemåterställning.

## SYNTAX

```
Get-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzSqlServerDisasterRecoveryConfiguration** får en konfiguration för återställning av SQL-databasserversystem.
