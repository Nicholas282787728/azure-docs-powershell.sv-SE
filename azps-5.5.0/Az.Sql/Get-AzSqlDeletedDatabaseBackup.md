# Get-AzSqlDeletedDatabaseBackup

## SYNOPSIS
Hämtar en borttagna databas som du kan återställa.

## SYNTAX

```
Get-AzSqlDeletedDatabaseBackup [-ServerName] <String> [[-DatabaseName] <String>] [[-DeletionDate] <DateTime>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzSqlDeletedDatabaseBackup** får en angiven säkerhetskopia av borttagna SQL-databaser som du kan återställa, eller alla borttagna säkerhetskopior som du kan återställa.
Den här cmdleten stöds också av tjänsten Stretcha databas för SQL Server i Azure.

## EXEMPEL

### Exempel 1: Hämta alla säkerhetskopior av borttagna databaser på en server
```
PS C:\>Get-AzSqlDeletedDatabaseBackup -ResourceGroupName "ContosoResourceGroup" -ServerName "ContosoServer"
```

Med det här kommandot säkerhetskopieras alla borttagna databaser på en server.

### Exempel 2: Hämta en angiven säkerhetskopia av borttagna databaser
```
PS C:\>Get-AzSqlDeletedDatabaseBackup -ResourceGroupName "ContosoResourceGroup" -ServerName "ContosoServer" -DatabaseName "ContosoDatabase"
```

Med det här kommandot får du säkerhetskopian av den borttagna databasen för ContosoDatabase.
