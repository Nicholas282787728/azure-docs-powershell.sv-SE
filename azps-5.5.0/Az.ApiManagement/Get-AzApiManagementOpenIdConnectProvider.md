# Get-AzApiManagementOpenIdConnectProvider

## SYNOPSIS
Hämtar OpenID Connect-leverantörer.

## SYNTAX

### GetAllOpenIdConnectProviders (standard)
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByOpenIdConnectProviderId
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-OpenIdConnectProviderId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByName
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApiManagementOpenIdConnectProvider** får OpenID Connect-leverantörer i Azure API Management.
ClientSecret inkluderas inte i resultatinformation. För att få klienthemlighet **använder du Get-AzApiManagementOpenIdConnectProviderClientSecret.**

## EXEMPEL

### Exempel 1: Hämta alla leverantörer
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext
```

Med det här kommandot får du alla OpenID Connect-leverantörer för den angivna kontexten.

### Exempel 2: Skaffa en leverantör med hjälp av ett ID
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01"
```

Det här kommandot hämtar providern som har ID OICProvider01.

### Exempel 3: Skaffa en leverantör med hjälp av ett namn
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -Name "Contoso OpenID Connect Provider"
```

Det här kommandot får leverantören Contoso OpenID Connect Provider.
