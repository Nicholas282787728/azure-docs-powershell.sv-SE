# Get-AzApiManagementTenantGitAccess

## SYNOPSIS
Hämtar git-åtkomstkonfigurationen för en klientorganisation.

## SYNTAX

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApiManagementTenantGitAccess** får git-åtkomstkonfigurationen för en klientorganisation.
Nycklar inkluderas inte i resultatinformationen. Om du vill få **klienthemlighet använder du Get-AzApiManagementTenantGitAccessSecret.**

## EXEMPEL

### Exempel 1: Hämta konfiguration för klientåtkomst
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```
