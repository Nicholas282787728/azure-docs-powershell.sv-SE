# Get-AzApiManagementIdentityProvider

## SYNOPSIS
Få information om identitetsleverantörens konfiguration.

## SYNTAX

### AllIdentityProviders (standard)
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdentityProviderByType
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Få information om identitetsleverantörens konfiguration.
ClientSecret inkluderas inte i resultatinformation. För att få klienthemlighet **kan du använda Get-AzApiManagementIdentityProviderClientSecret.**

## EXEMPEL

### Exempel 1: Hämta alla identitetsproviders

```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementIdentityProvider -Context $apimContext
```

Hämta all identitetsproviderkonfiguration för tjänsten.

### Exempel 2: Skaffa identitetsprovidern för AAD-typ
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\> Get-AzApiManagementIdentityProvider -Context $apimContext -Type Aad
```

Hämtar identitetsproviderkonfigurationen av Azure Active Directory.
