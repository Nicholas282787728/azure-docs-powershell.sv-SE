# Set-AzApiManagementNamedValue

## SYNOPSIS
Ändrar ett namngivet värde för API-hantering.

## SYNTAX

```
Set-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-Name <String>]
 [-Value <String>] [-Secret <Boolean>] [-Tag <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Set-AzApiManagementNamedValue** ändrar värdet för Azure API-hantering.

## EXEMPEL

### Exempel 1: Ändra taggarna för det namngivna värdet
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>$Tags = 'sdk', 'powershell'
PS C:\> Set-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -Tags $Tags -PassThru
```

Med det första kommandot tilldelas två värden $Tags variabeln.
Det andra kommandot ändrar det namngivna värdet som har ID-egenskapen11.
Kommandot tilldelar strängarna i $Tags som taggar på det namngivna värdet.

### Exempel 2: Ändra det namngivna värdet så att det får ett hemligt värde
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property12" -Secret $True -PassThru
```

Det här kommandot ändrar det namngivna värdet som ska krypteras.

### Exempel 3

Ändrar ett namngivet värde för API-hantering. (autogenererat)

```powershell
<!-- Aladdin Generated Example --> 
Set-AzApiManagementNamedValue -Context <PsApiManagementContext> -Name 'ContosoApi' -NamedValueId 'Property11' -Secret $true -Tag <String[]> -Value 'Property Value'
```
