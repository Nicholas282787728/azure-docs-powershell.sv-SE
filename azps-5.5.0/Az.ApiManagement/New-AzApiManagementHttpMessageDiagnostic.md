# New-AzApiManagementHttpMessageDiagnostic

## SYNOPSIS
Skapar en instans av **PsApiManagementHttpMessageDiagnostic** som är en http-meddelandediagnostikinställning för Diagnostik

## SYNTAX

```
New-AzApiManagementHttpMessageDiagnostic [-HeadersToLog <String[]>] [-BodyBytesToLog <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzApiManagementHttpMessageDiagnostic** skapar diagnostikinställningen Http Message.

## EXEMPEL

### Exempel 1: Skapa en grundläggande diagnostikinställning för HTTP-meddelanden
```powershell
PS C:\>  New-AzApiManagementHttpMessageDiagnostic -Headers 'Content-Type', 'UserAgent' -BodyBytes 100

Headers                   Body
-------                   ----
{Content-Type, UserAgent} Microsoft.Azure.Commands.ApiManagement.ServiceManagement.Models.PsApiManagementBodyDiagnosticSetting
```

Skapa en inställning för http-meddelandediagnostik `Content-Type` för logg och `User-Agent` sidhuvuden tillsammans med 100 byte `body`
