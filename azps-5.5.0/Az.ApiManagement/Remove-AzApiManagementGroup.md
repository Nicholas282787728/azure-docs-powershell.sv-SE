# Remove-AzApiManagementGroup

## SYNOPSIS
Tar bort en befintlig API-hanteringsgrupp.

## SYNTAX

```
Remove-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApiManagementGroup** tar bort en befintlig API-hanteringsgrupp.

## EXEMPEL

### Exempel 1: Ta bort en befintlig hanteringsgrupp
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementGroup -Context $apimContext -GroupId "Group0001" -Force
```

Med det här kommandot tas en befintlig hanteringsgrupp med namnet Group0001 bort och användaren uppmanas inte att bekräfta.
