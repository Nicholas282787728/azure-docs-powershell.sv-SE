# Publish-AzApiManagementTenantGitConfiguration

## SYNOPSIS
Publicerar ändringar från en Git-gren i konfigurationsdatabasen.

## SYNTAX

```
Publish-AzApiManagementTenantGitConfiguration -Context <PsApiManagementContext> -Branch <String> [-Force]
 [-ValidateOnly] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Publish-AzApiManagementTenantGitConfiguration** publicerar ändringarna från en Git-gren till konfigurationsdatabasen.
Du kan också verifiera ändringarna i en Git Branch utan publicering.

## EXEMPEL

### Exempel 1: Distribuera Git-ändringar
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Publish-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -PassThru
```

Det här kommandot publicerar ändringarna från den angivna grenen i konfigurationsdatabasen.

### Exempel 2: Validera Git-ändringar
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Publish-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -ValidateOnly -PassThru
```

Det här kommandot verifierar ändringarna i Git-grenen mot konfigurationsdatabasen.
Ändringar publiceras inte.
