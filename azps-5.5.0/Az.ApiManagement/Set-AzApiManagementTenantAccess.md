# Set-AzApiManagementTenantAccess

## SYNOPSIS
Aktiverar eller inaktiverar klientorganisationens åtkomst.

## SYNTAX

```
Set-AzApiManagementTenantAccess -Context <PsApiManagementContext> -Enabled <Boolean> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzApiManagementTenantAccess** aktiverar eller inaktiverar klientåtkomst.

## EXEMPEL

### Exempel 1: Aktivera klientorganisationsåtkomst
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementTenantAccess -Context $apimContext -Enabled $True
```

Det här kommandot ger klientorganisationsåtkomst i det angivna sammanhanget.
