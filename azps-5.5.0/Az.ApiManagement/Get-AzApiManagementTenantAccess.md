# Get-AzApiManagementTenantAccess

## SYNOPSIS
Hämtar åtkomstkonfigurationen för en klientorganisation.

## SYNTAX

```
Get-AzApiManagementTenantAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApiManagementTenantAccess** får konfigurationen klientåtkomst för en klientorganisation.
Nycklar inkluderas inte i resultatinformationen. För att få klienthemlighet **använder du Get-AzApiManagementTenantAccessSecret.**

## EXEMPEL

### Exempel 1: Hämta konfiguration för klientåtkomst
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccess -Context $apimContext
```

Det här kommandot hämtar konfigurationen för klientåtkomst för det angivna sammanhanget.
