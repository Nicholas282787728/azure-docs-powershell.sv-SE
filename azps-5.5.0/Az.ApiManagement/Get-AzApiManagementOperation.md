# Get-AzApiManagementOperation

## SYNOPSIS
Hämtar en lista eller en angiven API-åtgärd.

## SYNTAX

### GetAllapiOperations (standard)
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 -OperationId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Get-AzApiManagementOperation hämtar** en lista eller en angiven API-åtgärd.

## EXEMPEL

### Exempel 1: Hämta alla API-hanteringsåtgärder
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId
```

Det här kommandot hämtar alla API-hanteringsåtgärder.

### Exempel 2: Hämta en API-hanteringsåtgärd genom åtgärds-ID
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId -OperationId "Operation003"
```

Det här kommandot hämtar en API-hanteringsåtgärd genom åtgärds-ID:t med namnet Operation0003.
