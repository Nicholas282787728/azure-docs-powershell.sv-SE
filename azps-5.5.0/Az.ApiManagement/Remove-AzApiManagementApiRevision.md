# Remove-AzApiManagementApiRevision

## SYNOPSIS
En viss API-ändring har tagits bort

## SYNTAX

### ByApiId (standard)
```
Remove-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzApiManagementApiRevision -InputObject <PsApiManagementApi> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzApiManagementApiRevision** tar bort en viss API-revidering.

## EXEMPEL

### Exempel 1: Ta bort en API-revidering
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzAzureRmApiManagementApiRevision -Context $apimContext -ApiId "echo-api" -ApiRevision "2"
```

Det här kommandot tar bort `2` revideringen av `echo-api` API:t från API-hanteringstjänsten.

### Exempel 2

Det här kommandot tar bort 2 revideringen av API echo-api från API Management-tjänsten. (autogenererat)

```powershell
<!-- Aladdin Generated Example --> 
Remove-AzApiManagementApiRevision -ApiId '0001' -ApiRevision 6 -Context <PsApiManagementContext>
```
