# Get-AzApiManagementTenantAccessSecret

## SYNOPSIS
Hämtar en klientorganisations snabbkonfigurationsnycklar.

## SYNTAX

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämtar en klientorganisations snabbkonfigurationsnycklar.

## EXEMPEL

### Exempel 1: Hämta konfigurationsnycklar för klientåtkomst
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

Det här kommandot hämtar konfigurationsnycklarna för klientorganisationen för det angivna sammanhanget.
