# Remove-AzApiManagementAuthorizationServer

## SYNOPSIS
Tar bort en auktoriseringsserver.

## SYNTAX

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzApiManagementAuthorizationServer** tar bort en Azure API Management-auktoriseringsserver.

## EXEMPEL

### Exempel 1: Ta bort en auktoriseringsserver
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Med det här kommandot tas den angivna API Management Authorization Server bort.
Eftersom *force-parametern* har angetts krävs ingen bekräftelse.
