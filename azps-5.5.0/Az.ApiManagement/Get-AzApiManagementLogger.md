# Get-AzApiManagementLogger

## SYNOPSIS
Hämtar API Management Logger-objekt.

## SYNTAX

### GetAllLoggers (standard)
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByLoggerId
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApiManagementLogger** får en Azure API Management **Logger** eller alla loggers.

## EXEMPEL

### Exempel 1: Hämta alla loggare
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext
```

Med det här kommandot får du alla användare för den angivna kontexten.

### Exempel 2: Skaffa en särskild loggare
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123"
```

Det här kommandot tar bort en logger som har ID Logger123.
