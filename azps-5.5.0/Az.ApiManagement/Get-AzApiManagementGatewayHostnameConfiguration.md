# Get-AzApiManagementGatewayHostnameConfiguration

## SYNOPSIS
Hämtar all eller specifik värdnamnskonfiguration för den befintliga gatewayen.

## SYNTAX

### GetByGatewayId (standard)
```
Get-AzApiManagementGatewayHostnameConfiguration -Context <PsApiManagementContext> -GatewayId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByGatewayHostnameId
```
Get-AzApiManagementGatewayHostnameConfiguration -Context <PsApiManagementContext> -GatewayId <String>
 -GatewayHostnameConfigurationId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzApiManagementGatewayHostnameConfiguration** hämtar all eller specifik värdnamnskonfiguration för den befintliga gatewayen.

## EXEMPEL

### Exempel 1: Hämta alla värdnamnskonfigurationer för gatewayen
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGatewayHostnameConfiguration -Context $apimContext  -GatewayId "0123456789"
```

Det här kommandot hämtar alla värdnamnskonfigurationer för en "0123456789"-gateway.

### Exempel 2: Hämta en värdnamnskonfiguration för gatewayen
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGatewayHostnameConfiguration -Context $apimContext -GatewayId "0123456789" -GatewayHostnameConfigurationId "123"
```

Det här kommandot får konfigurationen "123" hostname för en "0123456789"-gateway.
