# Az.DevSpaces-modul
## Beskrivning
Kommandon för att interagera med Azure DevSpaces Controller.

## Az.DevSpaces-cmdlets
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Hämta eller lista Azure DevSpaces-kontroll.

### [New-AzDevSpacesController](New-AzDevSpacesController.md)
Skapa en ny Azure DevSpaces-kontroll.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
Ta bort en DevSpaces-handkontroll.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Uppdatera DevSpaces-handkontrollen för att lägga till taggar.
