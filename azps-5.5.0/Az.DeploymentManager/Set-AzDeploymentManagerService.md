# Set-AzDeploymentManagerService

## SYNOPSIS
Uppdaterar tjänsten.

## SYNTAX

```
Set-AzDeploymentManagerService [-InputObject] <PSServiceResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Set-AzDeploymentManagerService** uppdaterar en tjänst med det angivna tjänstobjektet.
Cmdleten returnerar det uppdaterade tjänstobjektet.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceObject
```

Det här kommandot uppdaterar en tjänst vars namn, tjänsttopologinamn och ResourceGroup matchar egenskaperna Namn, ServiceTopologyName och ResourceGroupName för $serviceObject respektive.
Tjänsten uppdaterades till egenskaperna som angetts i $serviceObject.
