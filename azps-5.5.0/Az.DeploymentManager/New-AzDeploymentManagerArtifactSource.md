# New-AzDeploymentManagerArtifactSource

## SYNOPSIS
Skapar en artefaktkälla.

## SYNTAX

```
New-AzDeploymentManagerArtifactSource -ResourceGroupName <String> -Name <String> -Location <String>
 -SasUri <String> [-Tag <Hashtable>] [-ArtifactRoot <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten New-AzDeploymentManagerArtifactSource** skapar en artefaktkälla.
Ange *namn,* *ResourceGroupName och* obligatoriska egenskaper.

Du kan ändra det returnerade objektet lokalt och sedan tillämpa ändringarna på artefaktkällan med hjälp av cmdleten Set-AzDeploymentManagerArtifactSource objekt.

Cmdleten returnerar ett ArtifactSource-objekt som har ett ResourceId som kan refereras till i cmdleten New-AzDeploymentManagerServiceTopology så att artefakter som krävs för en ServiceUnit-resurs, mall- och parametrarfilerna, kan refereras från den här platsen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzDeploymentManagerArtifactSource -ResourceGroupName ContosoResourceGroup -Name ContosoArtifactSource -Location "Central US" -SasUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts?sasParameters"
```

Skapar en artefaktkälla i ContosoResourceGroup med namnet ContosoArtifactSource med den centrala USA som resursplats. Egenskapen SasUri tillhandahåller SAS Uri för Azure Storage till lagringsbehållaren där artefakterna lagras.
