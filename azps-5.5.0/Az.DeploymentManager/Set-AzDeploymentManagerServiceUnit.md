# Set-AzDeploymentManagerServiceUnit

## SYNOPSIS
Uppdaterar tjänstenheten.

## SYNTAX

```
Set-AzDeploymentManagerServiceUnit [-InputObject] <PSServiceUnitResource>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Set-AzDeploymentManagerServiceUnit** uppdaterar en tjänstenhet med det angivna tjänstenhetsobjektet.
Cmdleten returnerar det uppdaterade tjänstenhetsobjektet.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzDeploymentManagerServiceUnit -InputObject $serviceUnitObject
```

Det här kommandot uppdaterar en tjänstenhet vars namn, tjänstnamn, tjänsttopologinamn och ResourceGroup matchar egenskaperna Namn, Tjänstnamn, ServiceTopologyName och ResourceGroupName för $serviceUnitObject respektive.
Kommandot returnerar det uppdaterade tjänstenhetsobjektet.
