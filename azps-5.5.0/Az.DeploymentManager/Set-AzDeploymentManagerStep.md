# Set-AzDeploymentManagerStep

## SYNOPSIS
Uppdaterar steget.

## SYNTAX

```
Set-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Set-AzDeploymentManagerStep** uppdaterar ett steg med det angivna stegobjektet.
Cmdleten returnerar det uppdaterade stegobjektet.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzDeploymentManagerStep -InputObject $stepObject
```

Det här kommandot uppdaterar ett steg vars namn och Resursgrupp matchar egenskaperna Namn och ResourceGroupName för $stepObject respektive.
Steget uppdateras till egenskaperna som anges i $stepObject.
