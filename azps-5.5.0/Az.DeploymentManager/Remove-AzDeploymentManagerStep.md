# Remove-AzDeploymentManagerStep

## SYNOPSIS
Tar bort steget.

## SYNTAX

### Interaktiv (standard)
```
Remove-AzDeploymentManagerStep [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerStep [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzDeploymentManagerStep** tar bort ett steg.
Ange steget efter dess namn och resursgruppens namn. Alternativt kan du ange Steg-objektet eller Resurs-ID:t.

## EXEMPEL

### Exempel 1: Ta bort ett steg
```powershell
PS C:\> Remove-AzDeploymentManagerStep -ResourceGroupName ContosoResourceGroup -Name ContosoService1WaitStep
```

Det här kommandot tar bort ett steg med namnet ContosoService1WaitStep i ContosoResourceGroup.

### Exempel 2: Ta bort ett steg med resursidentifieraren
```powershell
PS C:\> Remove-AzDeploymentManagerStep -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/steps/ContosoService1WaitStep"
```

Det här kommandot tar bort ett steg med namnet ContosoService1WaitStep i ContosoResourceGroup.

### Exempel 3: Ta bort ett steg med ett objekt som returneras av New-AzDeploymentManagerStep
### Exempel 1
```powershell
PS C:\> Remove-AzDeploymentManagerStep -InputObject $stepObject
```

 Det här kommandot tar bort ett steg vars namn och Resursgrupp matchar egenskaperna Namn och ResourceGroupName för $stepObject respektive.
