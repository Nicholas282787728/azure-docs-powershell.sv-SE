# Remove-AzDeploymentManagerServiceTopology

## SYNOPSIS
Tar bort tjänsttopologin. Alla tjänster som skapas i en tjänsttopologi måste tas bort innan du tar bort tjänsttopologin.

## SYNTAX

### Interaktiv (standard)
```
Remove-AzDeploymentManagerServiceTopology [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerServiceTopology [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerServiceTopology [-InputObject] <PSServiceTopologyResource> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzDeploymentManagerServiceTopology** tar bort en tjänsttopologi.

Ange tjänsttopologin efter dess namn och resursgruppnamnet. Alternativt kan du ange ServiceTopology-objektet eller ResourceId.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Remove-AzDeploymentManagerServiceTopology -ResourceGroupName ContosoResourceGroup -Name ContosoServiceTopology
```

Det här kommandot tar bort en tjänsttopologi med namnet ContosoServiceTopology i ContosoResourceGroup.

### Exempel 2: Ta bort en tjänsttopologi med resursidentifieraren.
```powershell
PS C:\> Remove-AzDeploymentManagerServiceTopology -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/serviceTopologies/ContosoServiceTopology"
```

Det här kommandot tar bort en tjänsttopologi med namnet ContosoServiceTopology i ContosoResourceGroup.

### Exempel 3: Ta bort en tjänsttopologi med tjänsttopologiobjektet.
```powershell
PS C:\> Remove-AzDeploymentManagerService -InputObject $serviceTopologyObject
```

Det här kommandot tar bort en tjänstetopologi vars namn och ResourceGroup matchar egenskaperna Namn och ResourceGroupName för $serviceTopologyObject respektive.
