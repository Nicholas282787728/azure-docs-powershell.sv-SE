# Get-AzOperationalInsightsCluster

## SYNOPSIS
Hämta eller lista kluster

## SYNTAX

```
Get-AzOperationalInsightsCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämta eller lista kluster, listkluster under resursgrupp när "-ClusterName" inte tillhandahållits, listkluster under prenumeration när "-ClusterName" och "ResourceGroupName" inte tillhandahållits.

## EXEMPEL

### Exempel 1
```powershell
Get-AzOperationalInsightsCluster -ResourceGroupName {rg-name} -ClusterName {cluster-name}

Name               : {cluster-name}
Type               : Microsoft.OperationalInsights/clusters
Tags               : {}
```

Hämta kluster
