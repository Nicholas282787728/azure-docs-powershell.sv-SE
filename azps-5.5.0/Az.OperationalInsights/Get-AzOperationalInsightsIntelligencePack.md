# Get-AzOperationalInsightsIntelligencePack

## SYNOPSIS
Hämtar de tillgängliga intelligence-paketen.

## SYNTAX

```
Get-AzOperationalInsightsIntelligencePack [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzOperationalInsightsIntelligencePack** får tillgängliga Intelligence Pack.

## EXEMPEL

### Exempel 1: Hämta informationspaket
```
PS C:\>Get-AzOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Det här kommandot hämtar de tillgängliga intelligence-paketen.
