# Get-AzSecurityAdaptiveApplicationControl

## SYNOPSIS
Hämtar en lista över vm/server-grupper för prenumerationen.

## SYNTAX

### SubscriptionScope (standard)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Adaptiva programkontroller beräknas automatiskt av Azure Säkerhetscenter. Använd den här cmdleten för att få en lista med resurser för adaptiva programkontroller i omfattningen för en prenumeration.
