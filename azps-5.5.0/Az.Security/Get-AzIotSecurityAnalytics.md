# Get-AzIotSecurityAnalytics

## SYNOPSIS
Hämta IoT-säkerhetsanalyser

## SYNTAX

```
Get-AzIotSecurityAnalytics -ResourceGroupName <String> -SolutionName <String> [-Default]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den Get-AzIotSecurityAnalytics cmdleten returnerar en uppsättning säkerhetsanalyser av en viss säkerhetslösning
