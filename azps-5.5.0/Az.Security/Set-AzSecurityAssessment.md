# Set-AzSecurityAssessment

## SYNOPSIS
Skapa eller uppdatera ett säkerhetsutvärderingsresultat för en resurs

## SYNTAX

### SubscriptionLevelResource (standard)
```
Set-AzSecurityAssessment -Name <String> [-StatusCode <String>] [-StatusCause <String>]
 [-StatusDescription <String>]
 [-AdditionalData <System.Collections.Generic.Dictionary`2[System.String,System.String]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdLevelResource
```
Set-AzSecurityAssessment -Name <String> -AssessedResourceId <String> -StatusCode <String>
 [-StatusCause <String>] [-StatusDescription <String>]
 [-AdditionalData <System.Collections.Generic.Dictionary`2[System.String,System.String]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Skapa eller uppdatera ett säkerhetsutvärderingsresultat för en resurs, kan användas för att ändra status för ett befintligt resultat eller lägga till ytterligare data.
kan endast användas för "CustomerManaged"-utvärderingstyper och endast efter att matchande utvärderingsmetadata har skapats.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzSecurityAssessment -Name 4FB6C0A0-1137-42C7-A1C7-4BD37C91DE8D -StatusCode "Unhealthy"
```

Markerar prenumerationsresultatet som "Ej fel" för utvärdering av typen "4FB6C0A0-1137-42C7-A1C7-4BD37C91DE8D" – mer information om utvärderingstypen hittar du under typen assessmentMetadata
