# Set-AzSecurityWorkspaceSetting

## SYNOPSIS
Uppdaterar arbetsytans inställningar för prenumerationen.

## SYNTAX

```
Set-AzSecurityWorkspaceSetting -Name <String> -Scope <String> -WorkspaceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Uppdaterar arbetsytans inställningar för prenumerationen.
Den konfigurerade arbetsytan kommer att innehålla säkerhetsdata som har samlats in av Azure Log Analytics-agenten som är installerad i virtuella maskiner i den här prenumerationen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Set-AzSecurityWorkspaceSetting -Name "default" -Scope "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869" -WorkspaceId  "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/resourcegroups/mainws/providers/microsoft.operationalinsights/workspaces/securityuserws"

Id                                                                                                         Name    WorkspaceId 
--                                                                                                         ----    ----
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/workspaceSettings/default default /...
```

Ställer in arbetsytan "securityuserws" till att innehålla alla säkerhetsdata som samlades in av Azure Log Analytics-agenten.
