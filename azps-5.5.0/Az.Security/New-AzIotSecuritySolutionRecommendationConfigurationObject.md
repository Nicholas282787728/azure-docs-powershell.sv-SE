# New-AzIotSecuritySolutionRecommendationConfigurationObject

## SYNOPSIS
Skapa ny rekommendationskonfiguration för en iot-säkerhetslösning

## SYNTAX

```
New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType <String> -Enabled <Boolean>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
AzIotSecuritySolutionRecommendationConfigurationObject skapar ett nytt rekommendationskonfigurationsobjekt för en iot-säkerhetslösning.
På så sätt konfigureras statusen för en rekommendation, oavsett om den är aktiverad eller inaktiverad.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType "IoT_ACRAuthentication" -Enabled $false

RecommendationType: "IoT_ACRAuthentication"
Name: "Service prinicpal not used with ACR repository"
Status: "Disabled"
```

Skapa ny rekommendationskonfiguration för rekommendationstyp "IoT_ACRAuthentication" med status inställd på inaktiverad
