# Get-AzSecurityLocation

## SYNOPSIS
Hämtar platsen där Azure Säkerhetscenter automatiskt sparar data för den specifika prenumerationen

## SYNTAX

### SubscriptionScope (standard)
```
Get-AzSecurityLocation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource
```
Get-AzSecurityLocation -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSecurityLocation -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Azure Säkerhetscenter bestämmer automatiskt var en del av dina data ska sparas.
Använd den här cmdleten för att identifiera den platsen.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzSecurityLocation
Id                                                                                                   Name
--                                                                                                   ----
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/locations/centralus centralus
```

Hämtar platsen där Azure Säkerhetscenter sparar beräknade säkerhetsdata.
