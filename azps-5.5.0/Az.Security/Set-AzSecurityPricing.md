# Set-AzSecurityPricing

## SYNOPSIS

Aktiverar eller inaktiverar Azure Defender-abonnemang för en prenumeration i Azure Säkerhetscenter.

## SYNTAX

### SubscriptionLevelResource (standard)

```powershell
Set-AzSecurityPricing -Name <String> -PricingTier <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObject

```powershell
Set-AzSecurityPricing -InputObject <PSSecurityPricing> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING

Aktivera eller inaktivera något av Azure Defender-abonnemangen för en prenumeration.

Mer information om Azure Defender och tillgängliga abonnemang finns i [Introduktion till Azure Defender.](https://docs.microsoft.com/azure/security-center/azure-defender)

## EXEMPEL

### Exempel 1

```powershell
PS C:\> Set-AzSecurityPricing -Name "virtualmachines" -PricingTier "Standard"
```

Aktiverar **Azure Defender för servrar** för prenumerationen.

"Standard" refererar till statusen "På" för en Azure Defender-plan som visas i Azure Säkerhetscenters pris- och inställningsområde på Azure-portalen.
