# Get-AzIotSecurityAnalyticsAggregatedAlert

## SYNOPSIS
Få aggregerad IoT-säkerhet

## SYNTAX

### SolutionScope (standard)
```
Get-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName <String> -SolutionName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SolutionLevelResource
```
Get-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName <String> -SolutionName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den Get-AzIotSecurityAnalyticsAggregatedAlert cmdleten returnerar en eller flera aggregerade aviseringar på enheter av iot-hubben.
Namnet på de aggregerade aviseringarna är en kombination av aviseringstyp och aviseringsdatum avgränsade med "/".
