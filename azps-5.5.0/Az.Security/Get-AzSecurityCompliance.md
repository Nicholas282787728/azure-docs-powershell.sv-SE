# Get-AzSecurityCompliance

## SYNOPSIS
Få den säkerhetsefterlevnad som gäller för en prenumeration över tid

## SYNTAX

### SubscriptionScope (standard)
```
Get-AzSecurityCompliance [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource
```
Get-AzSecurityCompliance -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSecurityCompliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämtar säkerhetsefterlevnad för en prenumeration baserat på den aktuella hälso- och icke-skyddade resurskvoten för den här prenumerationen.
Säkerhetsefterlevnaden beräknas varje dag och historiken sparas.
