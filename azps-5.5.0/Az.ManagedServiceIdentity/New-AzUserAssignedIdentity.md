# New-AzUserAssignedIdentity

## SYNOPSIS
Skapar en ny användar kopplad identitet eller uppdaterar en befintlig användar tilldelad identitet.

## SYNTAX

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzUserAssignedIdentity** skapar en ny användartilldelade identitet. När den används med en redan befintlig identitet uppdaterade den identiteten.
Om du vill lägga till Azure Resource Manager-taggar i identiteten använder du Set-AzResource cmdleten.
