# Add-AzIotHubEventHubConsumerGroup

## SYNOPSIS
Skapar en konsumentgrupp för eventhubben.

## SYNTAX

```
Add-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubConsumerGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Skapar en konsumentgrupp i Händelsehubben som är kopplad till angiven IotHub.

## EXEMPEL

### Exempel 1: Lägga till en konsumentgrupp i telemetrihändelsehubben
```
PS C:\> Add-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubConsumerGroupName "myconsumergroup"
```

Lägger till en ny konsumentgrupp med namnet "myconsumergroup" i händelsehubben för telemetrihändelser i iothub med namnet "myiothub"
