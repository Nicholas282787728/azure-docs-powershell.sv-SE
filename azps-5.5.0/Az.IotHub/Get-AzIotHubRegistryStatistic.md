# Get-AzIotHubRegistryStatistic

## SYNOPSIS
Hämtar registrystatistics för en IotHub.

## SYNTAX

```
Get-AzIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämtar registrystatistics för en IotHub.
Det här ger information om det totala antalet aktiverade och inaktiverade enheter i en IotHub.

## EXEMPEL

### Exempel 1 Hämta RegistryStatistics
```
PS C:\> Get-AzIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Hämtar RegistryStatistics för IotHub med namnet "myiothub"
