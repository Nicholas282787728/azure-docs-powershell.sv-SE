# Get-AzIotHubJob

## SYNOPSIS
Hämtar information om ett IotHub-jobb.

## SYNTAX

```
Get-AzIotHubJob [-ResourceGroupName] <String> [-Name] <String> [[-JobId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämtar information om ett IotHub-jobb.
Ett IotHub-jobb skapas när en import- eller exportåtgärd initieras med hjälp New-AzIotHubExportDevices eller New-AzIotHubImportDevices kommandon.
Du kan antingen lista alla jobb eller filtrera jobben med jobbidentifieraren.

## EXEMPEL

### Exempel 1 Lista alla jobb
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Hämtar alla jobb för IotHub med namnet "myiothub"

### Exempel 2 Skaffa ett visst jobb
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub" -JobId 3630fc31-4caa-43e8-a232-ea0577221cb2
```

Hämtar information om jobbet med identifieraren "3630fc31-4caa-43e8-a232-ea0577221cb2" för IotHub med namnet "myiothub"
