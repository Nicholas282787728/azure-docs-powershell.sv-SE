# Invoke-AzIotHubDeploymentMetricsQuery

## SYNOPSIS
Anropar en måttfråga för IoT Edge-distribution.

## SYNTAX

### ResourceSet (standard)
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Utvärdera ett anpassat målvärde eller systemmått som definierats i en IoT Edge-distribution.
Det finns fördefinierade systemmått som beräknas med Iot Hub och kan inte anpassas.
- "Riktad" visar IoT Edge-enheter som matchar villkoret för distributionsanpassning.
- "Används" visar de riktade IoT Edge-enheterna som inte har högre prioritet.
- "Reporting Success" visar IoT Edge-enheter som har rapporterat att modulerna har distribuerats.
- "Reporting Failure" visar IoT Edge-enheter som har rapporterat att en eller flera moduler inte har distribuerats. 
  Om du vill undersöka felet ytterligare ansluter du till de enheterna och visar loggfilerna.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Invoke-AzIotHubDeploymentMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "warningLimit"
```

Utvärdera det anpassade definierade "warningLimit"-måttet.

### Exempel 2
```powershell
PS C:\> Invoke-AzIotHubDeployMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "Reporting Success" -MetricType "system"
```

Utvärdera systemets mätvärde för rapporteringsresultat.
