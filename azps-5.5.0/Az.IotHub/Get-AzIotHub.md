# Get-AzIotHub

## SYNOPSIS
Hämtar information om IotHubs i en prenumeration.

## SYNTAX

### ListIotHubsByResourceGroup (standard)
```
Get-AzIotHub [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetIotHubByName
```
Get-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Hämtar information om IotHubs i en prenumeration.
Du kan visa alla IotHub-instanser i en prenumeration eller filtrera resultatet efter en resursgrupp eller ett visst IotHub-namn.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzIotHub
```

Hämtar alla IotHubs i prenumerationen.

### Exempel 2
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup"
```

Hämtar alla IotHubs i prenumerationen som tillhör resursgruppen med namnet "myresourcegroup".

### Exempel 3
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Hämtar information om IotHub med namnet "myiothub".
