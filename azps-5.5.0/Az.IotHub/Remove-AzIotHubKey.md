# Remove-AzIotHubKey

## SYNOPSIS
Tar bort en IotHub-nyckel.

## SYNTAX

### ResourceSet (standard)
```
Remove-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Remove-AzIotHubKey [-HubId] <String> [-KeyName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Tar bort en IotHub-nyckel.
Om det finns flera tangenter med samma namn tas den första nyckeln i listan bort.

## EXEMPEL

### Exempel 1 Ta bort en IotHub
```
PS C:\> Remove-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner1"
```

Tar bort nyckeln med namnet iothubowner1 från IotHub med namnet "myiothub"
