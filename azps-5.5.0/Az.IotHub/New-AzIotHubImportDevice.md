# New-AzIotHubImportDevice

## SYNOPSIS
Skapar ett nytt jobb för importenheter.

## SYNTAX

```
New-AzIotHubImportDevice [-ResourceGroupName] <String> [-Name] <String> [-InputBlobContainerUri] <String>
 [-OutputBlobContainerUri] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Skapar ett nytt importenheterjobb för IotHub.
Då importeras alla enheter till IotHub från den angivna behållaren. Se följande artikel om hur du skapar SAS URI.
https://docs.microsoft.com/azure/iot-hub/iot-hub-bulk-identity-mgmt#get-the-container-sas-uri .

## EXEMPEL

### Exempel 1
```

```

Skapar en ny importenhetsförfrågan för IotHub-"myiothub".
