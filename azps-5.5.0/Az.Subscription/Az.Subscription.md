# Az.Subscription Module
## Beskrivning
Avsnitten i det här avsnittet innehåller Azure PowerShell-cmdlets för Azure-prenumeration i Azure Resource Manager-ramverket (ARM). Cmdlet:arna finns i Microsoft.Azure.Commands.Subscription namespace.

## Az.Subscription Cmdlets
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Hämtar information om prenumerationsalias

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Skapar nytt alias och prenumeration

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Tar bort prenumerationsaliaset

### [Update-AzSubscription](Update-AzSubscription.md)
Uppdaterar en Azure-prenumeration
