# Update-AzMlCommitmentPlan

## SYNOPSIS
Uppdaterar egenskaper för en befintlig åtagandeplansresurs.

## SYNTAX

```
Update-AzMlCommitmentPlan -ResourceGroupName <String> -Name <String> -SkuName <String> -SkuTier <String>
 [-SkuCapacity <Int32>] [-Tag <Hashtable>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Uppdaterar en befintlig åtagandeplansresurs. Observera att de flesta egenskaperna för åtagandeplanen är oföränderliga och inte kan ändras. Egenskaper som kan ändras är SKU (så att du kan migrera åtagandeplanen från en SKU till en annan) och taggar.

## EXEMPEL

### Exempel 1: Uppdatera en åtagandeplan
```
Update-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName" -Tags @{'MyTagKey'='MyTagValue'}
```
