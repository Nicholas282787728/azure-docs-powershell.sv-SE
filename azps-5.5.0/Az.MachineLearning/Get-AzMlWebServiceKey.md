# Get-AzMlWebServiceKey

## SYNOPSIS
Hämtar webbtjänstens nycklar.

## SYNTAX

### GetByNameAndResourceGroup
```
Get-AzMlWebServiceKey -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByInstance
```
Get-AzMlWebServiceKey -MlWebService <WebService> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Hämtar snabbnycklarna för Azure Machine Learning-webbtjänstens körnings-API:er.

## EXEMPEL

### Exempel 1 – Hämta nycklar för en webbtjänst som anges av resursgrupp och namn
```
Get-AzMlWebServiceKey -ResourceGroupName "myresourcegroup" -Name "mywebservicename"
```

### Exempel 2 – Hämta nycklar för webbtjänstinstansen
```
Get-AzMlWebServiceKey -MlWebService $mlService
```

$mlService är ett objekt av typen Microsoft.Azure.Management.MachineLearning.WebServices.Models.WebService.
