# Set-AzCdnEndpoint

## SYNOPSIS
Uppdaterar en slutpunkt för CDN.

## SYNTAX

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Set-AzCdnEndpoint** uppdaterar en slutpunkt för Azure Content Delivery Network (CDN).
