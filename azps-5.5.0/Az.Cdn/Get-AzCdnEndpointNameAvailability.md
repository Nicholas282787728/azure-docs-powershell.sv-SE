# Get-AzCdnEndpointNameAvailability

## SYNOPSIS
Får tillgänglighetsstatus för CDN-slutpunkten.

## SYNTAX

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzCdnEndpointNameAvailability** får tillgänglighetsstatus för slutpunkten Azure Content Delivery Network (CDN).
