# New-AzCdnOriginGroup

## SYNOPSIS
Skapar en ny CDN-ursprungsgrupp

## SYNTAX

### ByFieldsParameterSet (standard)
```
New-AzCdnOriginGroup -EndpointName <String> -OriginGroupName <String>
 -OriginId <System.Collections.Generic.List`1[System.String]> [-ProbeIntervalInSeconds <Int32>]
 [-ProbePath <String>] [-ProbeProtocol <String>] [-ProbeRequestType <String>] -ProfileName <String>
 -ResourceGroupName <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByObjectParameterSet
```
New-AzCdnOriginGroup -CdnOriginGroup <PSOriginGroup> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den New-AzCdnOriginGroup skapar en ny ursprungsgrupp i den angivna slutpunkten. Om det här är den första ursprungsgruppen för slutpunkten måste egenskapen DefaultOriginGroup också anges.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> New-AzCdnOriginGroup -ResourceGroupName $resourceGroupName -ProfileName $profileName -EndpointName $endpointName -OriginGroupName $originGroupName -OriginId $originId
```
Denna cmdlet skapar en ny ursprungsgrupp i den angivna slutpunkten. Det använder de givna ursprungsnummer som ursprung.
