# Az.Cdn-modul
## Beskrivning
Avsnitten i det här avsnittet innehåller Azure PowerShell-cmdlets för Azure Content Delivery Network (CDN) i Azure Resource Manager-ramverket (ARM). Cmdletarna finns i namnområdet Microsoft.Azure.Commands.Cdn.

### [Disable-AzCdnCustomDomainHttps](Disable-AzCdnCustomDomainHttps.md)
