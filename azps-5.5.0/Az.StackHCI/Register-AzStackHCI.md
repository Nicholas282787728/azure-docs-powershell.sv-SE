# Register-AzStackHCI

## SYNOPSIS
Register-AzStackHCI skapar en Microsoft.AzureStackHCI-molnresurs som representerar det lokala klustret och registrerar det lokala klustret med Azure.

## SYNTAX

```
Register-AzStackHCI [-SubscriptionId] <String> [[-Region] <String>] [[-ResourceName] <String>]
 [[-TenantId] <String>] [[-ResourceGroupName] <String>] [[-ArmAccessToken] <String>]
 [[-GraphAccessToken] <String>] [[-AccountId] <String>] [[-EnvironmentName] <String>]
 [[-ComputerName] <String>] [[-CertificateThumbprint] <String>] [-RepairRegistration]
 [-UseDeviceAuthentication] [[-Credential] <PSCredential>] [<CommonParameters>]
```

## BESKRIVNING
Register-AzStackHCI skapar en Microsoft.AzureStackHCI-molnresurs som representerar det lokala klustret och registrerar det lokala klustret med Azure.
