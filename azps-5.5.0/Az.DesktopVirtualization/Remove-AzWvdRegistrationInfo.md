# Remove-AzWvdRegistrationInfo

## SYNOPSIS
Ta bort registreringsinformationen för den virtuella Windows-datorn.

## SYNTAX

```
Remove-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Ta bort registreringsinformationen för den virtuella Windows-datorn.

## EXEMPEL

### Exempel 1: Ta bort en registreringstoken för virtuell skrivbordsversionen av Windows
```powershell
PS C:\> Remove-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

Det här kommandot tar bort en Windows-registrering för virtuella skrivbord-token i en värdpool.
