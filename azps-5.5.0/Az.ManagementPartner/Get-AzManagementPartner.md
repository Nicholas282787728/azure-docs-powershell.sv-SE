# Get-AzManagementPartner

## SYNOPSIS
Hämtar Microsoft Partner Network(MPN)-ID för den aktuella autentiserade användaren eller tjänstens huvudnamn.

## SYNTAX

```
Get-AzManagementPartner [[-PartnerId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämtar Microsoft Partner Network(MPN)-ID för den aktuella autentiserade användaren eller tjänstens huvudnamn.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzManagementPartner
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Hämta aktuellt partner-ID för hanteringspartner

### Exempel 2
```powershell
PS C:\> Get-AzManagementPartner -PartnerId 4977985
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Hämta det aktuella id:t för hanteringspartner med ett partner-ID om det inte stämmer överens med det kommer det att misslyckas
