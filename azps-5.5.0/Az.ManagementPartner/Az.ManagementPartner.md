# Az.ManagementPartner-modul
## Beskrivning
Cmdlets för hanteringspartner för Azure Resource Manager.

## Az.ManagementPartner-cmdlets
### [Get-AzManagementPartner](Get-AzManagementPartner.md)
Hämtar Microsoft Partner Network(MPN)-ID för den aktuella autentiserade användaren eller tjänstens huvudnamn. 

### [New-AzManagementPartner](New-AzManagementPartner.md)
Associerar ett Microsoft Partner Network(MPN)-ID till den aktuella autentiserade användaren eller tjänstens huvudnamn.

### [Remove-AzManagementPartner](Remove-AzManagementPartner.md)
Ta bort Microsoft Partner Network(MPN)-ID för den aktuella autentiserade användaren eller tjänstens huvudnamn.

### [Update-AzManagementPartner](Update-AzManagementPartner.md)
Uppdaterar Microsoft Partner Network(MPN)-ID för den aktuella autentiserade användaren eller tjänstens huvudnamn.
