# Add-AzKustoClusterLanguageExtension

## SYNOPSIS
Lägg till en lista över språktillägg som kan köras i KQL-frågor.

## SYNTAX

### AddExpanded (standard)
```
Add-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-Value <ILanguageExtension[]>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### AddViaIdentityExpanded
```
Add-AzKustoClusterLanguageExtension -InputObject <IKustoIdentity> [-Value <ILanguageExtension[]>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Lägg till en lista över språktillägg som kan köras i KQL-frågor.

## EXEMPEL

### Exempel 1: Lägga till en lista över språktillägg
```powershell
PS C:\> Add-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster -Value (@{Name="R"}, @{Name="PYTHON"})
```

Kommandot ovan lägger till en lista över språktillägg som kan köras i KQL-frågor
