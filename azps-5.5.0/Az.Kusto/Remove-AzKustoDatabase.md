# Remove-AzKustoDatabase

## SYNOPSIS
Tar bort databasen med det angivna namnet.

## SYNTAX

### Ta bort (standard)
```
Remove-AzKustoDatabase -ClusterName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzKustoDatabase -InputObject <IKustoIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESKRIVNING
Tar bort databasen med det angivna namnet.

## EXEMPEL

### Exempel 1: Ta bort en befintlig Kusto-databas med namn
```powershell
PS C:\> Remove-AzKustoDatabase -ResourceGroupName testrg -ClusterName testnewkustocluster -Name mykustodatabase
```

Kommandot ovan tar bort Kusto-databasen med namnet "mykustodatabase" i klustret "testnewkustocluster" som finns i resursgruppen "testrg".
