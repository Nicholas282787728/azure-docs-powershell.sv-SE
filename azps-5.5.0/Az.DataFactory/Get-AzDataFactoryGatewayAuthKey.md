# Get-AzDataFactoryGatewayAuthKey

## SYNOPSIS
Hämtar gatewayauthnyckeln för en Azure Data Factory.

## SYNTAX

### ByFactoryName (standard)
```
Get-AzDataFactoryGatewayAuthKey [-DataFactoryName] <String> [-GatewayName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGatewayAuthKey [-InputObject] <PSDataFactory> [-GatewayName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzDataFactoryGatewayAuthKey** får autentiseringsnyckeln Gateway för en viss Azure Data Factory-gateway.
Du registrerar gatewayen i en molntjänst med hjälp av denna nyckel1 eller nyckel2 för denna autentiseringsnyckel.

## EXEMPEL

### Exempel 1: Hämtar autentiseringsnyckeln för en gateway
```
PS C:\> Get-AzDataFactoryGatewayAuthKey -ResourceGroup ADFResource -GatewayName 'MyGateway' -DataFactoryName MyADF
Key1 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@ZgBjjX6GfJcrzTQInEV9PoOqsDrqOmC
       gGHqUg1THLqA=
Key2 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@kFXxBdFCEBeL7LPB3hA3LqLd1uNFbyv
       YmWxtV4WD3JQ=
```

Det här kommandot får gatewayauthnyckeln för Data Factory-gatewayen med namnet MyGateway.
