# New-AzDataFactoryV2LinkedServiceEncryptedCredential

## SYNOPSIS
Kryptera autentiseringsuppgifter i länkad tjänst med angiven integrationskörning.

## SYNTAX

### ByFactoryName (standard)
```
New-AzDataFactoryV2LinkedServiceEncryptedCredential [-IntegrationRuntimeName] <String>
 [-DefinitionFile] <String> [-Force] [-ResourceGroupName] <String> [-DataFactoryName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryV2LinkedServiceEncryptedCredential [-IntegrationRuntimeName] <String>
 [-DefinitionFile] <String> [-Force] [-DataFactory] <PSDataFactory> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med New-AzDataFactoryV2LinkedServiceEncryptedCredential-cmdleten krypteras autentiseringsuppgifter i den länkade tjänsten med angiven integrationskörning.

Se till att följande krav uppfylls:
* **Alternativet fjärråtkomst** aktiveras på körningen av den självvärdiga integreringen.
* Powershell 7.0 eller senare används för att köra cmdleten.

## EXEMPEL

### Exempel 1: Kryptera autentiseringsuppgifter i en länkad tjänst

Det här kommandot krypterar autentiseringsuppgifter i filen C:\samples\WikiSample\TaxiDemo1.jspå med integrationskörningen test-selfhost-ir.

```powershell <!-- Aladdin Generated Example --> 
New-AzDataFactoryV2LinkedServiceEncryptedCredential -DataFactoryName WikiADF -DefinitionFile 'C:\samples\WikiSample\TaxiDemo1.json' -IntegrationRuntimeName 'test-selfhost-ir' -ResourceGroupName MyResourceGroup
```
