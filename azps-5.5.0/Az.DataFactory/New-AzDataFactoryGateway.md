# New-AzDataFactoryGateway

## SYNOPSIS
Skapar en gateway för en Azure Data Factory.

## SYNTAX

### ByFactoryName (standard)
```
New-AzDataFactoryGateway [-DataFactoryName] <String> [-Name] <String> [[-Description] <String>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryGateway [-DataFactory] <PSDataFactory> [-Name] <String> [[-Description] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzDataFactoryGateway** skapar en gateway i Azure Data Factory.

## EXEMPEL

### Exempel 1: Skapa en gateway
