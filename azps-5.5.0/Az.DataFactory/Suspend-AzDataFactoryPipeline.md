# Suspend-AzDataFactoryPipeline

## SYNOPSIS
Stänger av en pipeline i Azure Data Factory.

## SYNTAX

### ByFactoryName (standard)
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Suspend-AzDataFactoryPipeline** stänger av en rörledning i Azure Data Factory.
Du kan återuppta pipelinen med hjälp av Resume-AzDataFactoryPipeline cmdlet.

## EXEMPEL

### Exempel 1: Avbryta en pipeline
```
PS C:\>Suspend-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikiSample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to suspend pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Det här kommandot avbryter pipelinen med namnet DPWikiSample i data factory med namnet WikiADF.
Kommandot returnerar värdet $True.
