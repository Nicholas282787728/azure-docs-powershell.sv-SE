# Get-AzDataFactoryGateway

## SYNOPSIS
Hämtar information om logiska gateways i Azure Data Factory.

## SYNTAX

### ByFactoryName (standard)
```
Get-AzDataFactoryGateway [-DataFactoryName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGateway [-DataFactory] <PSDataFactory> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzDataFactoryGateway** hämtar information om logiska gateways i Azure Data Factory.
Om du anger namnet på en gateway hämtar den här cmdleten information om gatewayen.
Om du inte anger ett namn hämtar den här cmdleten information om alla gateways för en data factory.
Om du vill lägga till en lokal Microsoft SQL Server som en länkad tjänst i en data factory måste du installera en gateway på den lokala datorn.
