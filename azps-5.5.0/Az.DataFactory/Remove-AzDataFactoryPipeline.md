# Remove-AzDataFactoryPipeline

## SYNOPSIS
Tar bort en pipeline från Azure Data Factory.

## SYNTAX

### ByFactoryName (standard)
```
Remove-AzDataFactoryPipeline [-Force] [-Name] <String> [-DataFactoryName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryPipeline [-Force] [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzDataFactoryPipeline** tar bort en pipeline från Azure Data Factory.

## EXEMPEL

### Exempel 1: Ta bort en pipeline
```
PS C:\>Remove-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to remove pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Den här cmdleten tar bort pipelinen SOM heter DPWikisample från data factory med namnet WikiADF.
Kommandot returnerar värdet $True.
