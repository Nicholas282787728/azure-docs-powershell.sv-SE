# Start-AzDataFactoryV2Trigger

## SYNOPSIS
Startar en utlösare i en data factory.

## SYNTAX

### ByFactoryName (standard)
```
Start-AzDataFactoryV2Trigger [-Name] <String> [-ResourceGroupName] <String> [-DataFactoryName] <String>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Start-AzDataFactoryV2Trigger [-InputObject] <PSTrigger> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Start-AzDataFactoryV2Trigger [-ResourceId] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Start-AzDataFactoryV2Tfactorer** startar en utlösare i en data factory. Om utlösaren är i läget "Stoppad" startar cmdleten utlösaren och så småningom anropar den pipelines baserat på dess definition. Om utlösaren redan är i läget "Startad" har den här cmdleten ingen effekt. Om force-parametern anges uppmanas inte cmdleten innan utlösaren startas.

## EXEMPEL

### Exempel 1: Starta en utlösare
```
Start-AzDataFactoryV2Trigger -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -TriggerName "ScheduledTrigger"

Confirm
Are you sure you want to start trigger 'ScheduledTrigger' in data factory 'WikiADF'?
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
True
```

Startar en utlösare med namnet "ScheduledTlfer" i data factory "WikiADF".
