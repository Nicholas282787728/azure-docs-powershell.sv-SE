# New-AzDataFactory

## SYNOPSIS
Skapar en data factory.

## SYNTAX

```
New-AzDataFactory [-Name] <String> [-Location] <String> [[-Tag] <Hashtable>] [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzDataFactory** skapar en datakälla med angiven resursgrupps namn och plats.
Utför följande åtgärder i följande ordning: 
- Skapa en data factory. 
- Skapa länkade tjänster. 
- Skapa datauppsättningar. 
- Skapa en pipeline.

## EXEMPEL

### Exempel 1: Skapa en data factory
```
PS C:\>New-AzDataFactory -ResourceGroupName "ADF" -Name "WikiADF" -Location "WestUS"
DataFactoryName   : WikiADF
ResourceGroupName : ADF
Location          : WestUS
Tags              : {}
Properties        : Microsoft.WindowsAzure.Commands.Utilities.PSDataFactoryConfiguration
```

Med det här kommandot skapas en datakälla med namnet WikiADF i resursgruppen ADF på WestUS-platsen.
