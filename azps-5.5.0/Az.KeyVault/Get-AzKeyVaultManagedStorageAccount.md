# Get-AzKeyVaultManagedStorageAccount

## SYNOPSIS
Hämtar nyckelvalv som hanteras av Azure Storage Accounts.

## SYNTAX

### ByAccountName (standard)
```
Get-AzKeyVaultManagedStorageAccount [-VaultName] <String> [[-AccountName] <String>] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByInputObject
```
Get-AzKeyVaultManagedStorageAccount [-InputObject] <PSKeyVault> [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzKeyVaultManagedStorageAccount [-ResourceId] <String> [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Får ett nyckelvalv som hanteras av Azure Storage Account om namnet på kontot anges och kontonycklarna hanteras av det angivna valvet. Om kontonamnet inte anges visas alla konton vars nycklar hanteras av det angivna valvet.

### Exempel 1: Lista alla lagringskonton hanterade nyckelvalv
```powershell
PS C:\> Get-AzKeyVaultManagedStorageAccount -VaultName 'myvault'

Vault Name          : myvault
AccountName         : mystorageaccount

                      orage/storageAccounts/mystorageaccount
Enabled             : True
Created             : 4/25/2018 1:50:32 AM
Updated             : 4/25/2018 1:50:32 AM
Tags                :
```

Visar alla konton vars nycklar hanteras av valvet 'myvault'

### Exempel 2: Skaffa ett nyckelvalv hanterat lagringskonto
```powershell

```

Hämtar information om nyckelvalv hanterat lagringskonto för "mystorageaccount" om dess nycklar hanteras av valv 'myvault'
