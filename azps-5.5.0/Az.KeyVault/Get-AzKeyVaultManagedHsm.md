# Get-AzKeyVaultManagedHsm

## SYNOPSIS
Få hanterade HSMs.

## SYNTAX

```
Get-AzKeyVaultManagedHsm [[-Name] <String>] [[-ResourceGroupName] <String>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Get-AzKeyVaultManagedHsm** hämtar information om hanterade snabbmeddelanden i en prenumeration. Du kan visa alla hanterade HSMs-instanser i en prenumeration eller filtrera resultaten efter en resursgrupp eller en viss hanterad HSM.
Observera att även om det är valfritt att ange resursgruppen för den här cmdleten när du får en enda hanterad HSM bör du göra det för bättre prestanda.

## EXEMPEL

### Exempel 1: Få alla hanterade HSMs i din nuvarande prenumeration
```powershell
PS C:\> Get-AzKeyVaultManagedHsm

Name  Resource Group Name Location    SKU
----  ------------------- --------    ---
myhsm myrg1               eastus2euap StandardB1
```

Med det här kommandot får du alla hanterade virtuella maskiner i din nuvarande prenumeration.

### Exempel 2: Skaffa en särskild hanterad HSM
```powershell
PS C:\> Get-AzKeyVaultManagedHsm -Name 'myhsm'

Name  Resource Group Name Location    SKU
----  ------------------- --------    ---
myhsm myrg1               eastus2euap StandardB1
```

Det här kommandot får den hanterade HSM som heter myhsm i din nuvarande prenumeration.

### Exempel 3: Hanterade virtuella maskiner i en resursgrupp
```powershell
PS C:\> Get-AzKeyVaultManagedHsm -ResourceGroupName 'myrg1'

Name  Resource Group Name Location    SKU
----  ------------------- --------    ---
myhsm myrg1               eastus2euap StandardB1
```

Med det här kommandot får du alla hanterade HSM i resursgruppen myrg1.

### Exempel 4: Hanterade snabbmeddelanden (HSMs) med hjälp av filtrering
```powershell
PS C:\> Get-AzKeyVaultManagedHsm -Name 'myhsm*'

Name  Resource Group Name Location    SKU
----  ------------------- --------    ---
myhsm myrg1               eastus2euap StandardB1
```

Med det här kommandot får du alla hanterade HSM:er i prenumerationen som börjar med "myhsm".
