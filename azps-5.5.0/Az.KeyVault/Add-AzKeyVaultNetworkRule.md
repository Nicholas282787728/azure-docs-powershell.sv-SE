# Add-AzKeyVaultNetworkRule

## SYNOPSIS
Lägger till en regel som är avsedd att begränsa åtkomsten till ett nyckelvalv baserat på klientens internetadress.

## SYNTAX

### ByVaultName (standard)
```
Add-AzKeyVaultNetworkRule [-VaultName] <String> [[-ResourceGroupName] <String>] [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Add-AzKeyVaultNetworkRule [-InputObject] <PSKeyVault> [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Add-AzKeyVaultNetworkRule [-ResourceId] <String> [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Add-AzKeyVaultNetworkRule** beviljar eller begränsar åtkomsten till ett nyckelvalv till en uppsättning uppringare som anges av deras IP-adresser eller det virtuella nätverk de tillhör. Regeln kan begränsa åtkomsten för andra användare, program eller säkerhetsgrupper som har beviljats behörigheter via åtkomstprincipen.

Observera att alla IP-intervall inom `10.0.0.0-10.255.255.255` (privata IP-adresser) inte kan användas för att lägga till nätverksregler.
