# Remove-AzKeyVault

## SYNOPSIS
Tar bort ett nyckelvalv.

## SYNTAX

### ByAvailableVault (standard)
```
Remove-AzKeyVault [-VaultName] <String> [[-ResourceGroupName] <String>] [[-Location] <String>] [-Force]
 [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByDeletedVault
```
Remove-AzKeyVault [-VaultName] <String> [-Location] <String> [-InRemovedState] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectByAvailableVault
```
Remove-AzKeyVault [-InputObject] <PSKeyVault> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectByDeletedVault
```
Remove-AzKeyVault [-InputObject] <PSKeyVault> [-InRemovedState] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdByAvailableVault
```
Remove-AzKeyVault [-ResourceId] <String> [[-Location] <String>] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdByDeletedVault
```
Remove-AzKeyVault [-ResourceId] <String> [-Location] <String> [-InRemovedState] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzKeyVault** tar bort det angivna nyckelvalvet.
Den tar även bort alla nycklar och hemligheter som finns i den instansen.
Observera att även om det är valfritt att ange resursgruppen för den här cmdleten bör du göra det för bättre prestanda.

## EXEMPEL

### Exempel 1: Ta bort ett nyckelvalv
```powershell
PS C:\> Remove-AzKeyVault -VaultName "Contoso03Vault" -PassThru

True
```

Det här kommandot tar bort nyckelvalvet Contoso03Vault från din nuvarande prenumeration.

### Exempel 2: Ta bort ett nyckelvalv från en angiven resursgrupp
```powershell
PS C:\> Remove-AzKeyVault -Name "Contoso03Vault" -ResourceGroupName "Group14" -PassThru

True
```

Det här kommandot tar bort nyckelvalvet Contoso03Vault från den namngivna resursgruppen.
Om du inte anger resursgruppnamnet söker cmdleten efter det namngivna nyckelvalvet som ska tas bort i din nuvarande prenumeration.

### Exempel 3: Ta bort ett hanterat hsm
```powershell
PS C:\>  Remove-AzKeyVault -Name "testManagedHsm" -Hsm -PassThru

True
```

Det här kommandot tar bort den hanterade hsm-namngiven testManagedHsm från din nuvarande prenumeration.
