# Remove-AzKeyVaultManagedStorageSasDefinition

## SYNOPSIS
Tar bort SAS-definitioner för nyckelvalv som hanteras av Azure Storage.

## SYNTAX

### EfterDefinitionsnamn (standard)
```
Remove-AzKeyVaultManagedStorageSasDefinition [-VaultName] <String> [-AccountName] <String> [-Name] <String>
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKeyVaultManagedStorageSasDefinition [-InputObject] <PSKeyVaultManagedStorageSasDefinitionIdentityItem>
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Tar bort SAS-definitioner för nyckelvalv som hanteras av Azure Storage. Det här tar även bort hemligheten som används för att hämta SAS-token per denna SAS-definition.

## EXEMPEL

### Exempel 1: Ta bort ett nyckelvalv som hanteras av SAS-definitionen för Azure Storage.
```powershell
PS C:\> Remove-AzKeyVaultManagedStorageSasDefinition -VaultName 'myvault' -AccountName 'mystorageaccount' -Name 'mysasdef' -PassThru

Id          : https://myvault.vault.azure.net:443/storage/mystorageaccount/sas/mysasdef
Vault Name  : myvault
AccountName : mystorageaccount
Name        : mysasdef
Enabled     : True
Created     : 5/24/2018 9:11:08 PM
Updated     : 5/24/2018 9:11:08 PM
Tags        :
```

Tar bort en NYCKEL valv hanterad STORAGE SAS-definition 'mysasdef' som är kopplad till kontot 'mystorageaccount' i valvet 'myvault'.

### Exempel 2: Ta bort ett nyckelvalv som hanteras av SAS-definitionen för Azure Storage utan användarbekräftelse.
```powershell
PS C:\> Remove-AzKeyVaultManagedStorageSasDefinition -VaultName 'myvault' -AccountName 'mystorageaccount' -Name 'mysasdef' -PassThru -Force

Id          : https://myvault.vault.azure.net:443/storage/mystorageaccount/sas/mysasdef
Vault Name  : myvault
AccountName : mystorageaccount
Name        : mysasdef
Enabled     : True
Created     : 5/24/2018 9:11:08 PM
Updated     : 5/24/2018 9:11:08 PM
Tags        :
```

Tar bort en NYCKEL valv hanterad STORAGE SAS-definition 'mysasdef' som är kopplad till kontot 'mystorageaccount' i valvet 'myvault'.
