# Undo-AzKeyVaultManagedStorageAccountRemoval

## SYNOPSIS
Återställer ett tidigare borttagna KeyVault-hanterat lagringskonto.

## SYNTAX

### Standard (standard)
```
Undo-AzKeyVaultManagedStorageAccountRemoval [-VaultName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Undo-AzKeyVaultManagedStorageAccountRemoval [-InputObject] <PSDeletedKeyVaultManagedStorageAccountIdentityItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Kommandot **Undo-AzKeyVaultManagedStorageAccountRemoval** återställer ett tidigare borttagna hanterade lagringskonto, förutsatt att mjuk borttagning har aktiverats för det här valvet och att försöket att återställa uppstår under återställningsintervallet.

## EXEMPEL

### Exempel 1
```powershell
PS C:\> Get-AzKeyVaultManagedStorageAccount -VaultName myVault -Name myAccount -InRemovedState
PS C:\> Undo-AzKeyVaultManagedStorageAccountRemoval -VaultName myVault -Name myAccount
```

Den här sekvensen av kommandon bestämmer om det angivna lagringskontot finns i valvet i ett borttagna tillstånd. Det efterföljande kommandot återställer det borttagna lagringskontot och återställer det till aktivt läge.
