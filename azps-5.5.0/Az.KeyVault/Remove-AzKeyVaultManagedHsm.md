# Remove-AzKeyVaultManagedHsm

## SYNOPSIS
Tar bort en hanterad HSM.

## SYNTAX

### RemoveManagedHsmByName (standard)
```
Remove-AzKeyVaultManagedHsm [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByInputObject
```
Remove-AzKeyVaultManagedHsm [-InputObject] <PSManagedHsm> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByResourceId
```
Remove-AzKeyVaultManagedHsm [-ResourceId] <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzKeyVaultManagedHsm** tar bort den angivna hanterade HSM.
Den tar även bort alla nycklar som finns i den instansen.
Observera att även om det är valfritt att ange resursgruppen för den här cmdleten bör du göra det för bättre prestanda.

## EXEMPEL

### Exempel 1: Ta bort en hanterad HSM
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -Force

True
```

Det här kommandot tar bort den hanterade hsm som heter myhsm från din nuvarande prenumeration.

### Exempel 2: Ta bort en hanterad hsm från en angiven resursgrupp
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -ResourceGroupName "myrg1" -PassThru

True
```

Det här kommandot tar bort den hanterade hsm som heter myhsm från resursgruppen myrg1.
Om du inte anger resursgruppnamnet söker cmdleten efter den namngivna hanterade HSM som ska tas bort i din nuvarande prenumeration.
