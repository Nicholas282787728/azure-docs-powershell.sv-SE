# Get-AzKeyVaultCertificateContact

## SYNOPSIS
Hämtar kontakter som är registrerade för certifikataviseringar för ett nyckelvalv.

## SYNTAX

### VaultName (standard)
```
Get-AzKeyVaultCertificateContact [-VaultName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByInputObject
```
Get-AzKeyVaultCertificateContact [-InputObject] <PSKeyVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceId
```
Get-AzKeyVaultCertificateContact [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzKeyVaultCertificateContact** får kontakter som är registrerade för certifikataviseringar för ett nyckelvalv i Azure Key Vault.

## EXEMPEL

### Exempel 1: Hämta alla certifikatkontakter
```powershell
PS C:\> $Contacts = Get-AzKeyVaultCertificateContact -VaultName "Contoso"

Email                   VaultName
-----                   ---------
username@microsoft.com  Contoso
username1@microsoft.com Contoso
```

Det här kommandot hämtar alla kontakter för certifikatobjekten i Contosos nyckelvalv och lagrar dem sedan i den $Contacts variabeln.
