# Undo-AzKeyVaultRemoval

## SYNOPSIS
Återställer ett borttagna nyckelvalv till ett aktivt tillstånd.

## SYNTAX

### Standard (standard)
```
Undo-AzKeyVaultRemoval [-VaultName] <String> [-ResourceGroupName] <String> [-Location] <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Undo-AzKeyVaultRemoval [-InputObject] <PSDeletedKeyVault> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Undo-AzKeyVaultRemoval** återställer ett tidigare borttaget nyckelvalv. Det återställda valvet kommer att vara aktivt efter återställning
