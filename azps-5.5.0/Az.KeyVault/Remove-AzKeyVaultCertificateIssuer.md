# Remove-AzKeyVaultCertificateIssuer

## SYNOPSIS
Tar bort en certifikatutfärdare från ett nyckelvalv.

## SYNTAX

### Standard (standard)
```
Remove-AzKeyVaultCertificateIssuer [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzKeyVaultCertificateIssuer [-InputObject] <PSKeyVaultCertificateIssuerIdentityItem> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzKeyVaultCertificateIssuer** tar bort en certifikatutfärdare från ett nyckelvalv.

## EXEMPEL

### Exempel 1: Ta bort en certifikatutfärdare
```powershell
PS C:\> Remove-AzKeyVaultCertificateIssuer -VaultName "ContosoKV01" -Name "TestIssuer01" -Force

AccountId           :
ApiKey              :
OrganizationDetails :
Name                : TestIssuer01
IssuerProvider      : test
VaultName           : ContosoKV01
```

Det här kommandot tar bort certifikatutfärdaren TestIssuer01 från nyckelvalvet ContosoKV01.
