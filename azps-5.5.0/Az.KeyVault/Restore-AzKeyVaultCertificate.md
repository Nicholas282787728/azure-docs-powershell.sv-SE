# Restore-AzKeyVaultCertificate

## SYNOPSIS
Återställer ett certifikat i ett nyckelvalv från en säkerhetskopia.

## SYNTAX

### ByVaultName (standard)
```
Restore-AzKeyVaultCertificate [-VaultName] <String> [-InputFile] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Restore-AzKeyVaultCertificate [-InputObject] <PSKeyVault> [-InputFile] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Restore-AzKeyVaultCertificate [-ResourceId] <String> [-InputFile] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Restore-AzKeyVaultCertificate** skapar ett certifikat i det angivna nyckelvalvet från en säkerhetskopia.
Det här certifikatet är en kopia av det säkerhetskopierade certifikatet i indatafilen och har samma namn som det ursprungliga certifikatet.
Om nyckelvalvet redan innehåller ett certifikat med samma namn misslyckas den här cmdleten i stället för att skriva över det ursprungliga certifikatet.
Om säkerhetskopian innehåller flera versioner av ett certifikat återställs alla versioner.
Nyckelvalv som du återställer certifikatet till kan vara ett annat än nyckelvalvet som du säkerhetskopierade certifikatet från.
Men nyckelvalvet måste använda samma prenumeration och vara i en Azure-region i samma geografi (till exempel Nordamerika).
Se Microsoft Azure Trust Center https://azure.microsoft.com/support/trust-center/) (för mappning av Azure-regioner till geografiska områden).

## EXEMPEL

### Exempel 1: Återställa ett säkerhetskopierat certifikat
```powershell
PS C:\> Restore-AzKeyVaultCertificate -VaultName 'MyKeyVault' -InputFile "C:\Backup.blob"

                  CN=contoso.com

                [Issuer]
                  CN=contoso.com

                [Not After]
                  11/25/2018 2:57:41 AM

                [Thumbprint]
                  XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

SecretId      : https://mykeyvault.vault.azure.net:443/secrets/cert1/bd406f6d6b3a41a1a1c633494d8c3c3a
Thumbprint    : XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
RecoveryLevel : Purgeable
Enabled       : True
Expires       : 11/25/2018 10:57:41 AM
NotBefore     : 5/25/2018 10:47:41 AM
Created       : 5/25/2018 10:57:41 AM
Updated       : 5/25/2018 10:57:41 AM

```

Det här kommandot återställer ett certifikat, inklusive alla dess versioner, från säkerhetskopian med namnet Backup.blob till nyckelvalvet MyKeyVault.
