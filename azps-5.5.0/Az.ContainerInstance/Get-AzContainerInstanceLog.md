# Get-AzContainerInstanceLog

## SYNOPSIS
Hämta loggarna för en behållarinstans i en behållargrupp.

## SYNTAX

### GetContainerInstanceLogByNamesParamSet (standard)
```
Get-AzContainerInstanceLog [-ResourceGroupName] <String> -ContainerGroupName <String> [-Name <String>]
 [-Tail <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetContainerInstanceLogByPSContainerGroupParamSet
```
Get-AzContainerInstanceLog -InputContainerGroup <PSContainerGroup> [-Name <String>] [-Tail <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetContainerInstanceLogByResourceIdParamSet
```
Get-AzContainerInstanceLog -ResourceId <String> [-Name <String>] [-Tail <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzContainerInstanceLog** hämtar loggarna för en behållare i en behållargrupp.

## EXEMPEL

### Exempel 1: Hämta en behållarinstanss sidor
```
PS C:\> Get-AzContainerInstanceLog -ResourceGroupName demo -ContainerGroupName mycontainer -Name container1

Log line 1.
Log line 2.
Log line 3.
Log line 4.
```

Hämta loggen från `container1` en behållargrupp. `mycontainer` Som standard returneras upp till 4 MB-logginnehåll.

### Exempel 2: Hämta loggen för en behållarinstans som har samma namn som behållargruppen
```
PS C:\> Get-AzContainerInstanceLog -ResourceGroupName demo -ContainerGroupName mycontainer

Log line 1.
Log line 2.
Log line 3.
Log line 4.
```

Hämta loggen från `mycontainer` en behållargrupp. `mycontainer` Som standard returneras upp till 4 MB-logginnehåll.

### Exempel 3: Hämta de två radernas loggrader för en behållarinstans
```
PS C:\> Get-AzContainerInstanceLog -ResourceGroupName demo -ContainerGroupName mycontainer -Name container1 -Tail 2

Log line 3.
Log line 4.
```

Hämta de två radernas logg från i `container1` `mycontainer` behållargrupp.

### Exempel 4: Hämta en behållarinstanss sidor i en rörledning i behållargrupp
```
PS C:\> Get-AzContainerGroup -ResourceGroupName demo -Name mycontainer | Get-AzContainerInstanceLog

Log line 1.
Log line 2.
Log line 3.
Log line 4.
```

Hämta loggen från `mycontainer` ett rör i behållargruppen. `mycontainer` Som standard returneras upp till 4 MB-logginnehåll.
