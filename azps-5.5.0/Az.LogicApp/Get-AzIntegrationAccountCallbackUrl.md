# Get-AzIntegrationAccountCallbackUrl

## SYNOPSIS
Hämtar en URL för återringning av ett integrationskonto.

## SYNTAX

```
Get-AzIntegrationAccountCallbackUrl -ResourceGroupName <String> -Name <String> [-NotAfter <DateTime>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzIntegrationAccountCallbackUrl** får en URL för integrationskonto callback från en resursgrupp.
Denna cmdlet returnerar ett **CallbackUrl-objekt** som representerar URL:en för integrationskontots callback.
Ange namnet på integrationskontot och resursgruppens namn.
Den här modulen har stöd för dynamiska parametrar.
Om du vill använda en dynamisk parameter skriver du in den i kommandot.
Om du vill identifiera namnen på dynamiska parametrar skriver du ett bindestreck (-) efter cmdlet-namnet och trycker sedan på tabbtangenten flera gånger för att bläddra igenom tillgängliga parametrar.
Om du utelämnar en obligatorisk mallparameter uppmanas du att ange värdet i cmdleten.

## EXEMPEL

### Exempel 1: Hämta en URL för ett integrationskonto för återringning
```
PS C:\>Get-AzIntegrationAccountCallbackUrl -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -NotAfter "03/25/2016 18:23:22"
CallBackUrl : https://<baseurl>/integrationAccounts/8811f0155a364b5e9618ba28f7180601?api-version=2015-08-01-preview&se=2016-03
              -25T18%3A23%3A22.0000000Z&sp=%2F%2Fread&sv=1.0&sig=<value>
```

Det här kommandot får en URL för återringning av ett integrationskonto.
