# New-AzDataLakeAnalyticsCatalogCredential

## SYNOPSIS
Skapar en ny autentiseringsuppgifter för Azure Data Lake Analytics-katalogen.

## SYNTAX

### CreateByHostNameAndPort (standard)
```
New-AzDataLakeAnalyticsCatalogCredential [-Account] <String> [-DatabaseName] <String>
 [-CredentialName] <String> [-Credential] <PSCredential> [-Uri] <Uri>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### CreateByFullURI
```
New-AzDataLakeAnalyticsCatalogCredential [-Account] <String> [-DatabaseName] <String>
 [-CredentialName] <String> [-Credential] <PSCredential> [-DatabaseHost] <String> [-Port] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Med New-AzDataLakeAnalyticsCatalogCredential-cmdleten skapas en ny autentiseringsuppgifter som ska användas i en Azure Data Lake Analytics-katalog för anslutning till externa datakällor.

## EXEMPEL

### Exempel 1: Skapa en autentiseringsuppgifter för en katalog som anger värd och port
```
PS C:\> New-AzDataLakeAnalyticsCatalogCredential -AccountName "ContosoAdlAccount" `
                  -DatabaseName "databaseName" `
                  -CredentialName "exampleDbCred" `
                  -Credential (Get-Credential) `
                  -DatabaseHost "example.contoso.com" -Port 8080
```

Med det här kommandot skapas angivna autentiseringsuppgifter för det angivna kontot, databasen, värd och port med https-protokollet.

### Exempel 2: Skapa en autentiseringsuppgifter för en katalog som anger fullständig URI
```
PS C:\> New-AzDataLakeAnalyticsCatalogCredential -AccountName "ContosoAdlAccount" `
                  -DatabaseName "databaseName" `
                  -CredentialName "exampleDbCred" `
                  -Credential (Get-Credential) `
                  -Uri "http://httpExample.contoso.com:8080"
```

Med det här kommandot skapas de angivna autentiseringsuppgifterna för det angivna kontot, databasen och den externa URI:en för datakällan.
