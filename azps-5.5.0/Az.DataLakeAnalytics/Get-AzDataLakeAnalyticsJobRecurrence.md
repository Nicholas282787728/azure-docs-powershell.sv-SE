# Get-AzDataLakeAnalyticsJobRecurrence

## SYNOPSIS
Får ett återkommande återkommande eller återkommande Data Lake Analytics-jobb.

## SYNTAX

### GetAllInAccount (standard)
```
Get-AzDataLakeAnalyticsJobRecurrence [-Account] <String> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetBySpecificFlödetRecurrence
```
Get-AzDataLakeAnalyticsJobRecurrence [-Account] <String> [-RecurrenceId] <Guid>
 [-SubmittedAfter <DateTimeOffset>] [-SubmittedBefore <DateTimeOffset>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Get-AzDataLakeAnalytics Ett återkommande** Azure Data Lake Analytics-jobb anges eller en lista över återkommande uppgifter.

## EXEMPEL

### Exempel 1: Få en angiven återkommande händelse
```
PS C:\>Get-AzDataLakeAnalyticsJobRecurrence -Account "contosoadla" -RecurrenceId 83cb7ad2-3523-4b82-b909-d478b0d8aea3
```

Det här kommandot får angivet återkommande jobb med id '83cb7ad2-3523-4b82-b909-d478b0d8aea3' i kontot 'contosoadla'.

### Exempel 2: Visa en lista över alla återkommande förekomster i kontot
```
PS C:\>Get-AzDataLakeAnalyticsJobRecurrence -AccountName "contosoadla"
```

Det här kommandot får en lista över alla återkommande förekomster i kontot "contosoadla"
