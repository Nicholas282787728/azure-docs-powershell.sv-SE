# Test-AzDataLakeAnalyticsAccount

## SYNOPSIS
Söker efter ett Data Lake Analytics-konto.

## SYNTAX

```
Test-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Test-AzDataLakeAnalyticsAccount** kontrollerar att det finns ett Data Lake Analytics-konto.

## EXEMPEL

### Exempel 1: Testa om det finns ett konto
```
PS C:\>Test-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Det här kommandot testar om kontot ContosoAdlAccount finns.
