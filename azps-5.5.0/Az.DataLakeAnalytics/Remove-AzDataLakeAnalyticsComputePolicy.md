# Remove-AzDataLakeAnalyticsComputePolicy

## SYNOPSIS
Tar bort en angiven beräkningsprincip för Azure Data Lake Analytics

## SYNTAX

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Remove-AzDataLakeAnalyticsComputePolicy** tar bort en angiven beräkningsprincip för Azure Data Lake Analytics.

## EXEMPEL

### Exempel 1: Ta bort en beräkningsprincip
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Det här kommandot tar bort den angivna beräkningsprincipen med namnet "myPolicy" i kontot contosoadla.
