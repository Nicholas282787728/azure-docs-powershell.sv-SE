# Add-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Lägger till en brandväggsregel i ett Data Lake Analytics-konto.

## SYNTAX

```
Add-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Add-AzDataLakeAnalyticsFirewallRule** lägger till en brandväggsregel i ett Azure Data Lake Analytics-konto.

## EXEMPEL

### Exempel 1: Lägga till en brandväggsregel
```
PS C:\>Add-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

Det här kommandot lägger till brandväggsregeln med namnet "min brandväggsregel" från kontot "ContosoAdlAcct" med IP-intervallet: 127.0.0.1 – 127.0.0.10
