# Remove-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Tar bort en brandväggsregel från ett Data Lake Analytics-konto.

## SYNTAX

```
Remove-AzDataLakeAnalyticsFirewallRule [-Account] <String> [[-Name] <String>] [-PassThru]
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Remove-AzDataLakeAnalyticsFirewallRule** tar bort en brandväggsregel från ett Azure Data Lake Analytics-konto.

## EXEMPEL

### Exempel 1: Ta bort en brandväggsregel
```
PS C:\>Remove-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule"
```

Det här kommandot tar bort brandväggsregeln med namnet "min brandväggsregel" från kontot "ContosoAdlAcct"
