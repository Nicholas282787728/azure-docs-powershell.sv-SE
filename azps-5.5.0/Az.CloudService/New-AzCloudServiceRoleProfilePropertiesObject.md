# New-AzCloudServiceRoleProfilePropertiesObject

## SYNOPSIS
Skapa ett minnesobjekt för CloudServiceRoleProfileProperties

## SYNTAX

```
New-AzCloudServiceRoleProfilePropertiesObject [-Name <String>] [-SkuCapacity <Int64>] [-SkuName <String>]
 [-SkuTier <String>] [<CommonParameters>]
```

## BESKRIVNING
Skapa ett minnesobjekt för CloudServiceRoleProfileProperties

## EXEMPEL

### Exempel 1: Skapa egenskapsobjekt för rollprofil
```powershell
$role = New-AzCloudServiceRoleProfilePropertiesObject -Name 'WebRole' -SkuName 'Standard_D1_v2' -SkuTier 'Standard' -SkuCapacity 2
```

Med det här kommandot skapas rollprofilegenskaper som används för att skapa eller uppdatera en molntjänst.
Mer information finns i New-AzCloudService.
