# Get-AzReservationCatalog

## SYNOPSIS
Hämta katalogen med tillgängliga bokningar

## SYNTAX

```
Get-AzReservationCatalog [-SubscriptionId <Guid>] -ReservedResourceType <String> [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Hämta de regioner och SKU:er som är tillgängliga för köp av reserverad instans för den angivna Azure-prenumerationen.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzReservationCatalog -ReservedResourceType VirtualMachines -Location westus
```

Hämta VirtualMachines-katalogen i westus för standardprenumerationen

### Exempel 2
```
PS C:\> Get-AzReservationCatalog -SubscriptionId "1111aaaa-b1b2-c0c2-d0d2-00000fffff" -ReservedResourceType SuseLinux
```

Hämta SuseLinux-katalogen för den angivna prenumerationen
