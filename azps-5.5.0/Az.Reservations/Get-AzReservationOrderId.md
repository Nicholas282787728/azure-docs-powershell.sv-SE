# Get-AzReservationOrderId

## SYNOPSIS
Hämta en lista över `ReservationOrder` tillämpliga ID:n.

## SYNTAX

```
Get-AzReservationOrderId [-SubscriptionId <Guid>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Få ID:n för `ReservationOrder` tillämpliga s som kan tillämpas på den här prenumerationen.

## EXEMPEL

### Exempel 1
```
PS C:\> Get-AzReservationOrderId
```

Få `ReservationOrder` tillämpat för standardprenumeration

### Exempel 2
```
PS C:\> Get-AzReservationOrderId -SubscriptionId "1111aaaa-b1b2-c0c2-d0d2-00000fffff"
```

Få en `ReservationOrder` anteckning för angiven prenumeration
