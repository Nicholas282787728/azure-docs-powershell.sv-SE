# New-AzIoTDeviceProvisioningServiceCertificateVerificationCode

## SYNOPSIS
Generera en verifieringskod för ett Azure IoT Hub Device Provisioning Service-certifikat.

## SYNTAX

### ResourceSet (standard)
```
New-AzIoTDeviceProvisioningServiceCertificateVerificationCode [-ResourceGroupName] <String> [-Name] <String>
 [-CertificateName] <String> [-Etag] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### InputObjectSet
```
New-AzIoTDeviceProvisioningServiceCertificateVerificationCode [-InputObject] <PSCertificateResponse>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
New-AzIoTDeviceProvisioningServiceCertificateVerificationCode [-ResourceId] <String> [-Etag] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Den här verifieringskoden används för att slutföra kontrollsteget för ett certifikat. Använd den här verifieringskoden som CN för ett nytt certifikat signerat med den privata rotcertifikatnyckeln.
En detaljerad förklaring av CA-certifikat i Azure IoT Hub-enhetsetableringstjänsten finns i https://docs.microsoft.com/en-us/azure/iot-dps/how-to-verify-certificates .
