# Remove-AzStreamAnalyticsJob

## SYNOPSIS
Tar bort ett Stream Analytics-jobb.

## SYNTAX

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzStreamAnalytics En** cmdlet tas bort asynkront i ett visst Stream Analytics-jobb i Azure.

## EXEMPEL

### EXEMPEL 1: Ta bort ett jobb
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Det här kommandot tar bort jobbet Streaming Det här kommandot.
