# Test-AzStreamAnalyticsInput

## SYNOPSIS
Testar anslutningsstatus för en inmatning.

## SYNTAX

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Test-AzStreamAnalyticsInput** testar möjligheten för Stream Analytics att ansluta till en indata.

## EXEMPEL

### Exempel 1: Testa anslutningsstatusen för en indataström
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Detta testar anslutningsstatusen för input EntryStream i Streaming Streaming Ent.
