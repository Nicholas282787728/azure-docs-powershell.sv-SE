# Remove-AzStreamAnalyticsFunction

## SYNOPSIS
Tar bort en funktion från ett Stream Analytics-jobb.

## SYNTAX

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Remove-AzStreamAnalyticsFunction** tar bort en funktion asynkront från ett Azure Stream Analytics-jobb.

## EXEMPEL

### Exempel 1: Ta bort en streamanalysfunktion
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Det här kommandot tar bort funktionen ScoreTweet från jobbet Stream Det här 22-jobbet.
