# Get-AzStreamAnalyticsFunction

## SYNOPSIS
Hämtar funktioner i ett Stream Analytics-jobb.

## SYNTAX

```
Get-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzStreamAnalyticsFunction** får en lista över de funktioner som definieras i ett Azure Stream Analytics-jobb eller information om en viss funktion.

## EXEMPEL

### Exempel 1: Hämta alla funktioner för Stream Analytics
```
PS C:\>Get-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22"
```

Med det här kommandot får du de funktioner som definierats på jobbet StreamKommando22.

### Exempel 2: Hämta en specifik streamanalysfunktion
```
PS C:\>Get-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Med det här kommandot får du information om funktionen ScoreTweet definierad på jobbet StreamKommando22.
