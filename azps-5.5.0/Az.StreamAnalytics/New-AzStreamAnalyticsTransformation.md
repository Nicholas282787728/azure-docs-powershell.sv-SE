# New-AzStreamAnalyticsTransformation

## SYNOPSIS
Skapar eller uppdaterar en transformation inom ett jobb.

## SYNTAX

```
New-AzStreamAnalyticsTransformation [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **New-AzStreamAnalyticsTransformation** skapar en transformation inom ett Stream Analytics-jobb eller uppdaterar den befintliga transformationen.
Du kan ange namnet på transformationen i . JSON-fil eller på kommandoraden.
Om båda har angetts måste namnet på kommandoraden matcha namnet i filen.
Om du anger en transformation som redan finns och inte anger force-parametern frågar cmdleten om den befintliga transformationen ska ersättas.
Om du anger *force-parametern* och anger ett befintligt transformationsnamn ersätts transformationen utan bekräftelse.

## EXEMPEL

### Exempel 1: Skapa eller ersätta en transformation i ett jobb
```powershell
PS C:\>New-AzStreamAnalyticsTransformation -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Transformation.json" -JobName "StreamingJob" -Name "StreamingJobTransform"
```

Med det här kommandot skapas en transformering som kallas Streaming Streaming StreamingTransform i jobbet StreamingKommando.
Om en befintlig transformering redan har definierats med det namnet frågar cmdleten om den ska ersätta den.

### Exempel 2: Ersätta en omvandling i ett jobb
```powershell
PS C:\>New-AzStreamAnalyticsTransformation -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Transformation.json" -JobName "StreamingJob" -Name "StreamingJobTransform" -Force
```

Det här kommandot ersätter definitionen för Streaming Streaming StreamingTransform i jobbet StreamingKommando utan bekräftelse.
