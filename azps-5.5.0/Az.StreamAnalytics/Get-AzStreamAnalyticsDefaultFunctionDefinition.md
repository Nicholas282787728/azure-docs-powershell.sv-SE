# Get-AzStreamAnalyticsDefaultFunctionDefinition

## SYNOPSIS
Hämtar standarddefinitionen för en funktion i Stream Analytics.

## SYNTAX

```
Get-AzStreamAnalyticsDefaultFunctionDefinition [-JobName] <String> [-Name] <String> [-File] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Cmdleten **Get-AzStreamAnalyticsDefaultFunctionDefinition** får standarddefinitionen för en funktion i Azure Stream Analytics.
Du kan använda standarddefinitionen och cmdleten New-AzStreamAnalyticsFunction för att skapa en funktion.
Du kan ändra standarddefinitionen innan du skapar en funktion.

## EXEMPEL

### Exempel 1: Hämta standarddefinitionen för en Stream Analytics-funktion
```
PS C:\>Get-AzStreamAnalyticsDefaultFunctionDefinition -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\RetrieveDefaultDefinitionRequest.json" -Name "ScoreTweet"
```

Det här kommandot får standarddefinitionen för funktionen ScoreTweet med hjälp av parametrarna som anges i RetrieveDefaultDefinitionRequest.json-filen.
