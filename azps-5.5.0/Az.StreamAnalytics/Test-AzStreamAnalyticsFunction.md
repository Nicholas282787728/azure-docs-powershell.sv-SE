# Test-AzStreamAnalyticsFunction

## SYNOPSIS
Testar om Stream Analytics kan ansluta till en funktion.

## SYNTAX

```
Test-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
**Cmdleten Test-AzStreamAnalyticsFunction** testar om Azure Stream Analytics kan ansluta till en funktion.

## EXEMPEL

### Exempel 1: Testa en streamanalysfunktion
```
PS C:\>Test-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Det här kommandot testar anslutningsstatusen för funktionen ScoreTweet i jobbet StreamKommando22.
