# Get-AzureRmPolicySetDefinition

## Sammanfattning

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureRmPolicySetDefinition** hämtar en samling princip uppsättnings definitioner eller en specifik definition för princip uppsättning som identifieras med namn eller ID.

## BESKRIVS

### Exempel 1: Hämta alla definitioner för princip uppsättningar
```
PS C:\> Get-AzureRmPolicySetDefinition
```

Det här kommandot får alla definitioner för princip uppsättningar.

### Exempel 2: Hämta definitionen av princip uppsättning från aktuell prenumeration efter namn
```
PS C:\> Get-AzureRmPolicySetDefinition -Name 'VMPolicySetDefinition'
```

Det här kommandot hämtar princip uppsättnings definitionen med namnet VMPolicySetDefinition från den aktuella standard prenumerationen.

### Exempel 3: Hämta princip uppsättnings definition från abonnemang efter namn
```
PS C:\> Get-AzureRmPolicySetDefinition -Name 'VMPolicySetDefinition' -subscriptionId '3bf44b72-c631-427a-b8c8-53e2595398ca'
```

Det här kommandot hämtar princip definitionen med namnet VMPolicySetDefinition från prenumerationen med ID 3bf44b72-c631-427a-b8c8-53e2595398ca.

### Exempel 4: Hämta alla definitioner för anpassade principer uppsättning från en hanterings grupp
```
PS C:\> Get-AzureRmPolicySetDefinition -ManagementGroupName 'Dept42' -Custom
```

Det här kommandot får alla anpassade princip uppsättnings definitioner från hanterings gruppen med namnet Dept42.
