# Remove-AzureRmADGroupMember

## Sammanfattning

## PROBLEMBESKRIVNING
Tar bort en användare från en AD-grupp.

## BESKRIVS

### Exempel 1 – ta bort en användare från en grupp efter objekt-ID

```
PS C:\> Remove-AzureRmADGroup -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405 -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Tar bort användaren med objekt-ID ' D9076BBC-D62C-4105-9C78-A7F5BC4A3405 ' från gruppen med objekt-ID ' 85F89C90-780E-4AA6-9F4F-6F268D322EEE '.

### Exempel 2 – ta bort en användare från en grupp efter ledning

```
PS C:\> Get-AzureRmADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Remove-AzureRmADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405
```

Hämtar gruppen med objekt-ID ' 85F89C90-780E-4AA6-9F4F-6F268D322EEE ' och rör den till Remove-AzureRmADGroupMember cmdlet för att ta bort användaren till gruppen.
