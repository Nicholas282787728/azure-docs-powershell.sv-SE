# Get-AzureStorageShareStoredAccessPolicy

## Sammanfattning
Hämtar lagrade åtkomst principer för en lagrings resurs.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Get-AzureStorageShareStoredAccessPolicy [-ShareName] <String> [[-Policy] <String>] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Get-AzureStorageShareStoredAccessPolicy** hämtar åtkomst principer för en Azure Storage-resurs.
För att få en viss princip anger du den efter namn.

## BESKRIVS

### Exempel 1: Hämta en lagrad åtkomst princip i en resurs
```
PS C:\>Get-AzureStorageShareStoredAccessPolicy -ShareName "ContosoShare" -Policy "GeneralPolicy"
```

Det här kommandot får en lagrad åtkomst princip med namnet GeneralPolicy i ContosoShare.

### Exempel 2: Hämta alla lagrade åtkomst principer i dela
```
PS C:\>Get-AzureStorageShareStoredAccessPolicy -ShareName "ContosoShare"
```

Det här kommandot får alla lagrade åtkomst principer i ContosoShare.
