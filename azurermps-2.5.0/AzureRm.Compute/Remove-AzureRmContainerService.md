# Remove-AzureRmContainerService

## Sammanfattning
Tar bort en behållar tjänst.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmContainerService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmContainerService** tar bort en behållar tjänst från ditt Azure-konto.

## BESKRIVS

### Exempel 1: ta bort en behållar tjänst
```
PS C:\> Remove-AzureRmContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17"
```

Det här kommandot tar bort behållar tjänsten med namnet CSResourceGroup17.
