# Remove-AzureRmStorageAccount

## Sammanfattning
Tar bort ett lagrings konto från Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureRmStorageAccount [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureRmStorageAccount** tar bort ett lagrings konto från Azure.

## BESKRIVS

### Exempel 1: ta bort ett lagrings konto
```
PS C:\>Remove-AzureRmStorageAccount -ResourceGroupName "RG01" -AccountName "mystorageaccount"
```

Det här kommandot tar bort det angivna lagrings kontot.
