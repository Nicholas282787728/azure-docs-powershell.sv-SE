# Remove-AzureKeyVaultCertificateIssuer

## Sammanfattning
Tar bort en certifikat utfärdare från ett huvud valv.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## FRÅGESYNTAXEN

```
Remove-AzureKeyVaultCertificateIssuer [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Cmdleten **Remove-AzureKeyVaultCertificateIssuer** tar bort en certifikat utfärdare från ett nyckelord.

## BESKRIVS

### Exempel 1: ta bort en certifikat utfärdare
```
PS C:\>Remove-AzureKeyVaultCertificateIssuer -VaultName "ContosoKV01" -Name "TestIssuer01" -Force
```

Det här kommandot tar bort certifikat utfärdaren som heter TestIssuer01 från ContosoKV01-.
