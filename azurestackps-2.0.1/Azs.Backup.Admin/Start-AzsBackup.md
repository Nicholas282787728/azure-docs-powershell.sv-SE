# Start-AzsBackup

## Sammanfattning
Säkerhetskopiera en specifik plats.

## FRÅGESYNTAXEN

### Skapa (standard)
```
Start-AzsBackup [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentity
```
Start-AzsBackup -InputObject <IBackupAdminIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## PROBLEMBESKRIVNING
Säkerhetskopiera en specifik plats.

## BESKRIVS

### Exempel 1: starta azurestack säkerhets kopiering
```powershell
PS C:\>Start-AzsBackup

```

Starta en Azure-Stack-säkerhetskopia.
